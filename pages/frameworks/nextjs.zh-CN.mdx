# 部署 Next.js

在 Zeabur 上部署你的 Next.js 应用。

### 第一步: 创建 Next.js 应用（可选）

设备要求：
- 确保 node 版本是 14 或以上

打开终端，输入  
```bash 
npx create-next-app@latest 
# or
yarn create next-app  
# or
pnpm create next-app
```

创建完毕后，为其初始化一个 Git 仓库并部署到你的 Github 上。

### 第二步：部署 Next.js 应用

在你的项目中，点击 **Deploy service** 或者 **Add new service** 按钮，并选择 **Deploy your source code**.

![deploy](/framework/deploy.png)

搜索想要部署的 Next.js 应用仓库，并点击导入，你的 Next.js 应用就将自动开始部署。
