# Dev Plan (Developer Plan)

The Developer Plan is Zeabur's paid plan designed for individual developers. At **US$5/month**, it unlocks all deployment features and is ideal for personal production projects and side projects.

## Plan Details

| Item | Details |
| --- | --- |
| Monthly fee | US$5 / month |
| Included credits | US$5 / month (offsets resource costs) |
| Projects | Unlimited |
| Services | Unlimited |
| Custom domains | Supported |
| Persistent storage | Supported |
| Deployment regions | All available regions |
| Members | 1 (account owner only) |

### Rates

Dev Plan resource rates are the same as listed in the [Plans Overview](/pricing/pricing-plans):

- **vCPU** — Free
- **Memory** — US$0.00025 / GB-minute (~US$10.8 / GB-month)
- **Network egress** — US$0.1 / GB
- **Persistent storage** — US$0.2 / GB-month

The included US$5 monthly credits automatically offset resource usage. Any overage is charged to your credit card or prepaid balance.

## Free Plan vs. Dev Plan

| Feature | Free Plan | Dev Plan |
| --- | --- | --- |
| Monthly fee | Free | US$5 |
| Projects | 1 | Unlimited |
| Services | 2 per project | Unlimited |
| Custom domains | Not supported | Supported |
| Persistent storage | Not supported | Supported |
| Auto-sleep | Sleeps when idle | Always running |
| SLA | None | Standard |

## Use Cases

- **Personal side projects** — Deploy personal websites, blogs, API services, etc.
- **Development & testing** — Environments that need persistent storage (e.g., databases).
- **Small production services** — Low-traffic services that need to run 24/7.
- **Template development** — Create and maintain Zeabur templates with [Deploy Buttons](/deploy/deploy-button) to promote your projects.

## How to Subscribe

Go to [zeabur.com/pricing](https://zeabur.com/pricing) and select Dev Plan, or upgrade from the [Dashboard billing page](https://dash.zeabur.com/account/billing). Credit card and Alipay are supported — see [Payment Methods](/subscription/payment-methods) for details.
