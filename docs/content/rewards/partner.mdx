# Partner Program

The Zeabur Partner Program offers collaboration opportunities for instructors, event organizers, and sales consultants to earn rewards by promoting Zeabur.

## Two Partnership Models

### Education Advocacy

For **instructors, training institutions, and event organizers**. Promote Zeabur through teaching and technical events using Event Codes that give attendees free trial access, and earn rewards based on retained paying users.

- Receive a dedicated Event Code for your event
- One month later, earn US$5 per retained paying user
- Ideal for workshops, online courses, and hackathons

[Learn more →](/partner/education)

### Sales Partner

For **enterprise consultants, system integrators, and technology consulting firms**. Help enterprise clients adopt Zeabur and earn a 15% recurring commission for as long as the client continues to pay.

- Trigger: Single contract > US$1,000 or institutional bulk purchase
- Commission model: 15% recurring commission on client payments each month
- Ideal for enterprise onboarding and institutional procurement

[Learn more →](/partner/sales-partner)

## How It Differs from the Referral Program

Zeabur also offers a [Referral Program](/rewards/referral) that lets any user earn rewards through referral codes. Key differences:

| Aspect | Referral Program | Partner Program |
| --- | --- | --- |
| Who | Any user | Instructors / Organizers / Consultants |
| How | Automatic referral code flow | Manual review and contract |
| Rewards | Referral code commissions (20%–100% by tier) | Event Code retention rewards / 15% recurring commission |
| Threshold | None | Sales Partner: single deal > US$1,000 |

<Callout type="info">
A single transaction can only go through one path — referral code or partner program. They do not stack.
</Callout>
