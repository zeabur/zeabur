# はじめに

**Zeabur** は、ワンクリックでサービスをデプロイできるプラットフォームです。
使用するプログラミング言語やフレームワークに関係なく利用可能です。

[Discord](https://discord.gg/DrdGCvXEyY) サーバーに参加して、さらに詳しい情報を得たり、優先サポートを受けたりすることができます！

### Zeabur の利点

#### 便利で高速

従来のデプロイメント方法と比べ、Zeabur を使うと、デプロイメントに関連する多くの知識を学ぶことなく、わずか数分でデプロイメントを完了できます。これにより、製品開発に時間とエネルギーを集中させることができます。

- すぐに使える CI/CD
- 自動で行われるプロジェクトタイプの認識
- 迅速なドメイン生成
- 複数環境の隔離

#### 従量課金制

Zeabur は、従量課金制を採用しています。つまり、サービスに実際に使用されたリソースのみを支払う必要があり、サーバ全体に対して支払う必要はありません。

[詳細な情報](/en-US/billing/pricing) は料金に関するドキュメントで確認できます。

#### オールインワン統合

Zeabur は、強力なオールインワン統合機能を提供します。

フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージなど、様々なサービスを一つのプロジェクトでデプロイできます。別々のプラットフォームに個別にデプロイする必要はありません。
