# カスタムルートディレクトリ

フロントエンドとバックエンドの両方を持つプロジェクトなど、一部のGitリポジトリでは、プロジェクトのルートディレクトリを手動で指定する必要があります。

### ステップ1：サービスをデプロイする

[サービスをデプロイ](/en-US/get-started)して、アプリケーションをデプロイします

## ステップ2：プロジェクトのルートディレクトリを変更する

デプロイされたサービスをクリックし、設定タブに入り、「ルートディレクトリ」列までスクロールします

![setting](/deploy/root-directory/setting.png)

ここでプロジェクトのルートディレクトリを設定します。
