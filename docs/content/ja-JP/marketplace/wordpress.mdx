# WordPress をデプロイ

[WordPress](https://wordpress.org/) は、誰でも簡単に美しいウェブサイト、ブログ、アプリを構築できるオープンソースソフトウェアです。

この記事では、Marketplace を通じて Zeabur 上で希望するドメインを使用して自分の WordPress サービスをデプロイする方法を紹介します。

## テンプレートから WordPress をデプロイ

### ステップ1：WordPressテンプレートページを開く

WordPress は MySQL データベースを必要とするため、イメージから直接デプロイすることはできません。

MySQL を含む WordPress をワンクリックでデプロイできるテンプレートを用意しました。

[![Zeaburでデプロイ](https://zeabur.com/button.svg)](https://zeabur.com/templates/CV344X)

### ステップ2：テンプレートをデプロイ

![テンプレート](/marketplace/wordpress/template.png)

デプロイボタンをクリックし、サービスをデプロイする地域を選択します。

![地域を選択](/marketplace/wordpress/region.png)

選択後、自動的にプロジェクトページにリダイレクトされ、デプロイの進行状況が表示されます。

数秒待つと、WordPressサービスが正常にデプロイされ、稼働を開始します。

### ステップ3：ドメインをバインド

デプロイ後、WordPressサービスにアクセスするためにドメインをバインドする必要があります。

詳細については[ドメインをバインドするガイド](/deploy/domain-binding)をご覧ください。

### ステップ4：WordPressにログインして設定する

ドメインのバインドが完了したら、ドメイン名をクリックしてWordPressのインストール画面に入ります。まず、希望する言語を選択し、「続行」（または類似の内容）をクリックします：

![WordPressインストール、言語選択](/marketplace/wordpress/wordpress-inst-step1.png)

その後、インストール手順に従います：

![WordPressインストール、パスワードとサイト情報の設定](/marketplace/wordpress/wordpress-inst-step2.png)

この画面が表示されたら、インストールが完了です。

![WordPressインストール、インストール完了](/marketplace/wordpress/wordpress-inst-step3.png)
