# メンバーを招待する

プロジェクトには2人以上のメンバーがいる場合があります。プロジェクトオーナーはメンバーをプロジェクトに招待してサービスを管理することができます。

## メンバーを招待する方法

### ステップ1: プロジェクト設定を開く

まず、メンバーを招待したいプロジェクトを開きます。プロジェクトページの左上隅にある設定ボタンをクリックすると、プロジェクト設定ページに移動します。

![プロジェクト設定](/manage/invite/project-config.png)

### ステップ2: メンバーを招待する

サイドバーからチームパネルを選択すると、チームパネルにメンバーを招待するボタンが表示されます。

![プロジェクト設定](/manage/invite/project-settings.png)

招待したいメンバーのメールアドレスを入力し、招待ボタンをクリックします。メンバーはプロジェクトに参加するためのメール招待を受け取ります。

また、招待リンクをコピーして招待したいメンバーと共有することもできます。メンバーはリンクを開くだけでプロジェクトに参加できます。

![メンバーを招待する](/manage/invite/invite.png)

## よくある質問

- **Q: zeaburに登録していないメンバーを招待できますか？**
    - A: 現在、zeaburに登録しているメンバーのみを招待することができます。

- **Q: メンバーがプロジェクトに参加した後の権限はどうなりますか？**
    - A: メンバーはプロジェクトオーナーと同じ権限を持ちますが、プロジェクトを削除する権限はありません。

- **Q: プロジェクトからメンバーを削除できますか？**
    - A: はい、プロジェクト設定ページのチームパネルでメンバーをプロジェクトから削除することができます。
