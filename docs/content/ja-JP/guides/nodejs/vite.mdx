# Viteのデプロイ

ZeaburでViteアプリをデプロイします。

### Step1: Viteアプリの作成（オプション）

- Node.js v14.18またはそれ以上のバージョン。

ターミナルを開き、次のコマンドを実行してください
```bash
npx create vite@latest
# または
yarn create vite
# または
pnpm create vite
```

GitHubリポジトリを初期化し、GitHubにアップロードしてください。

### Step2: Viteアプリのデプロイ

**デプロイサービス**または**新しいサービスの追加**ボタンをクリックし、**ソースコードをデプロイ**を選択してください。

![deploy](/deploy/common/select-service.en-US.png)

Viteアプリのリポジトリを検索し、インポートをクリックし、Viteアプリがデプロイされます。
