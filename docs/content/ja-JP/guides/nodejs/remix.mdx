# Deploy Remix

<WorkingInProgress />

Deploy Remix App in Zeabur.

### Step1: Create Remix App (Optional)

- Node.js v14 or greater version.

Open your terminal and run
```bash
npx create-remix@latest
```

Init a GitHub repository for it and upload to your GitHub.

### Step2: Deploy Remix App

Click on **Deploy service** or **Add new service** button, then choose **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

Search for your Remix App repository, click on import, your Remix App will start deploying.
