# プリペイドカードの引換

Zeabur と提携している第三者パートナーから Zeabur プリペイドカードを購入または受け取ることがあります。これらのプリペイドカードは Zeabur サービスの残高に引き換えることができ、その残高を使って Zeabur の有料会員へのアップグレード、専用サーバーの購入、または共有クラスターの使用料の支払いに使用できます。

この記事では、購入または受け取った Zeabur プリペイドカードを残高に引き換える方法を説明します。

## 引換方法

1. Zeabur アカウントにログインします。Zeabur アカウントをお持ちでない場合は、まず[登録](https://zeabur.com/login)してください。
2. Zeabur コンソールの請求管理ページ [https://zeabur.com/account/billing](https://zeabur.com/account/billing) にアクセスします。
3. 請求管理ページで「アカウント残高」セクションを見つけ、右上の「ポイント引換」ボタンをクリックします。
4. ポップアップダイアログにプリペイドカードコードを入力し、「引換」ボタンをクリックします。
5. 引換が成功すると、画面右下に成功通知が表示されます。このとき、アカウント残高が相応の金額で増加します。

## 問題が発生した場合

プリペイドカードの引換で問題が発生した場合は、[Zeabur 公式 Discord チャンネル](https://zeabur.com/dc)でカスタマーサービスチームにお問い合わせください。できる限りのサポートをさせていただきます。
