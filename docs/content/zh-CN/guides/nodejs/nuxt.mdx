# 部署 Nuxt 项目

在 Zeabur 上部署你的 Nuxt 项目。

### 第一步: 建立 Nuxt 项目（可选）

打开终端，运行以下命令：

<Tabs items={['pnpm', 'npx']}>
  <Tabs.Tab>
    ```bash copy
    pnpm dlx nuxi init <project-name>
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    npx nuxi init <project-name>
    ```
  </Tabs.Tab>
</Tabs>

接下来，你需要安装依赖：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash copy
    pnpm install --shamefully-hoist
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    npm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    yarn
    ```
  </Tabs.Tab>
</Tabs>

<Callout type="info" emoji="ℹ️">
  这里需要注意由于 pnpm 的 `node_modules` 结构与其他依赖管理器不同，因此需要使用 `--shamefully-hoist` 来安装套件。
  并确保在部属前你有新增 `.npmrc` 文件在项目的根目录，并在文件中中加入 `shamefully-hoist=true`。
</Callout>

````.npmrc filename=".npmrc" copy
shamefully-hoist=true
````

### 第二步: 部署 Nuxt 应用

在你的项目中，点击 **Deploy service** 或者 **Add new service** 按钮，并选择 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜索想要部署的 Nuxt 应用仓库，并点击导入，你的 Nuxt 应用就将自动开始部署。
