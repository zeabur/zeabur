# 寻求帮助

有些时候你可能会在使用 Zeabur 的过程中遇到一些问题，你可以在 [我们的官方 Discord 服务器](https://discord.gg/DrdGCvXEyY) 中寻求帮助。

![server](/community/help/server.png)

## 如何得到帮助

在我们的 Discord 服务器中，我们有一个叫做 `问题求助` 的频道，你可以在这里发布你的问题。

![help channel](/community/help/help-channel.png)

### 发帖格式

遵循以下格式来描述你的问题可以帮助我们更快地定位和解决你的问题。

- 在帖子标题简单描述你的问题
- 在内容区块中描述你的问题细节
- 在帖子中提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在你的 Zeabur 控制台的 URL 中找到它们。
  ![url](/community/help/url.png)
