# 自定义根目录

对于某些仓库，例如前后端一体的应用，你可能需要手动为其指明项目根目录。

### 第一步：部署服务

参照 [部署服务](/zh-CN/get-started) 来部署你的应用

### 第二步：更改项目根目录

选择部署好的服务，选择其中的设置选项卡，下拉至 **Root Directory** 一栏

![setting](/deploy/root-directory/setting.png)

输入想要指定的根目录，并点击保存，保存成功后，点击 **Redeploy Service** 重新部署你的服务，服务将会根据指定的根目录重新构建运行。
