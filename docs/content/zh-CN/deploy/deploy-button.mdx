# 一键部署

使用我们的一键部署按钮，只需点击一下，就可以将服务部署到 Zeabur。按钮可以在任何 markdown 文件或 html 文件中使用。它可以将任何 [Zeabur 模板](https://zeabur.com/templates) 转换为自动在 Zeabur 部署的项目。

![button-demo](/deploy/button/button-demo.png)

## 按钮的作用

部署按钮是一个简单的 html 代码片段。您可以复制代码片段并粘贴到任何 markdown 文件或 html 文件中。每个按钮都将重定向用户到所需的模板。模板可以是单个服务或由模板作者决定的一组服务。

按钮也可以以作者的推荐代码结尾，每个使用推荐代码注册并升级到付费计划的新用户，作者将获得 5 美元的积分。

## 复制按钮

每个按钮都是与模板相关联的，只有模板作者才能生成按钮。

想要复制按钮，您需要在 [Zeabur Dashboard](https://dash.zeabur.com/account) 中的账户页面，选择 **模板** 标签，您可以看到您创建的模板列表。

![created-templates](/deploy/button/created-templates.png)

选择所需的模板，然后单击共享按钮。将弹出一个弹窗，其中包含按钮代码。

![popup](/deploy/button/popup.png)

如果您选择启用推荐代码，按钮代码将附加推荐代码，推荐代码是您在 Zeabur 的用户名。

## 使用场景

这是将您的项目分享给世界的最佳方式。每个人都可以轻松地一键部署您的项目。

比如，您可以将按钮粘贴到以下位置：

- GiHub 的 README 文件
- 个人网站
- 文档和教程
- 社交媒体简介

## 前置要求

你需要先创建一个模板才能获得部署按钮。您可以在 [Zeabur Dashboard](https://dash.zeabur.com/create-template) 中创建模板。

如下图所示，您需要填写模板名称、描述，并选择要包含在模板中的服务等等，就像填写表格一样来创建你的模板。

![create-template](/deploy/button/create-template.png)
