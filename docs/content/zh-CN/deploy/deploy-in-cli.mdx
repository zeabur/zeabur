# 使用 CLI 部署服务

在控制台中一键部署服务很简单，但是如果你想在终端中使用一条命令部署服务，我们也为你提供了这一方案。

本篇文章将会介绍如何使用 Zeabur CLI 在终端中部署服务。

## 登录账号

安装完成后，你需要登录账号才能部署服务。

```bash
npx zeabur auth login
```

点击 `Enter` 打开浏览器登录页面，然后点击 `Confirm` 按钮，你将会被重定向到终端并且看到登录成功的消息。

## 部署服务

现在你已经准备好部署服务了，只需要在你的服务目录中运行以下命令。

```bash
npx zeabur deploy
```

然后你需要按照指引选择或者创建一个项目，你的服务将会被部署到这个项目中。

选择好项目以后，你将会在终端中看到部署进度。等待几秒钟，你的服务将会被自动部署，你将会在终端中看到服务的 URL。
