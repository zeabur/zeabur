# Deploy Umami

[Umami](https://umami.is/) es una herramienta de análisis web simple y rápida para aquellos que se preocupan por la privacidad. Es la alternativa de código abierto a Google Analytics.

Este artículo te mostrará cómo desplegar Umami en Zeabur con un dominio personalizado.

## Desplegar Umami desde una plantilla

Dado que Umami requiere PostgreSQL como base de datos, no puedes desplegarlo directamente desde la imagen.

Hemos creado una plantilla para que puedas desplegar Umami con PostgreSQL en un solo clic.

[![Deploy on Zeabur](https://zeabur.com/button.svg)](https://zeabur.com/templates/01NQCC)

### Paso 1: Abrir la página de la plantilla de Umami

Primero, visita [la página de la plantilla](https://zeabur.com/templates/01NQCC) para Umami.

Puedes ver que la plantilla ya está configurada con el servicio de PostgreSQL, por lo que no necesitas hacer nada.

![Template](/marketplace/umami/template.png)

### Paso 2: Desplegar la plantilla

Haz clic en el botón Deploy, y luego selecciona una región para desplegar tus servicios.

![Select Region](/marketplace/umami/region.png)

Después de la selección, serás redirigido automáticamente a la página del proyecto donde podrás ver el progreso del despliegue.

![Deployment](/marketplace/umami/deployment.png)

Espera unos segundos más, y tu servicio de Umami se desplegará con éxito y comenzará a funcionar.

### Paso 3: Vincular el dominio

Necesitas vincular un dominio para acceder a tu servicio de Linkwarden después del despliegue.

Puedes seguir [la guía para vincular un dominio](/deploy/domain-binding) para más detalles.

### Paso 4: Iniciar sesión y configurar Umami

Después de vincular el dominio, puedes hacer clic en el dominio para entrar en la página de inicio de sesión de Umami, y usar la cuenta predeterminada `admin` y la contraseña `umami` para iniciar sesión.

![Umami login page](/marketplace/umami/screenshot.png)

Para obtener más información sobre Umami, por favor consulta la [documentación oficial de Umami](https://umami.is/docs).
