# 兌換預付卡

有些時候，你可能會在某些與 Zeabur 合作的第三方合作夥伴處購買或取得 Zeabur 預付卡。這些預付卡可以用來兌換 Zeabur 服務的餘額，你可以用這些兌換到的餘額升級 Zeabur 付費會員、購買專用伺服器或支付共享叢集的使用費。

本篇文章將教你如何將你購買或取得的 Zeabur 預付卡兌換至餘額。

## 如何兌換

1. 登入你的 Zeabur 帳號。如果你還沒有 Zeabur 帳號，請先[註冊](https://zeabur.com/login)一個。
2. 訪問 Zeabur 控制台的帳單管理頁面 [https://zeabur.com/account/billing](https://zeabur.com/account/billing)。
3. 你可以在帳單管理頁面找到「帳號餘額」區塊，點擊右上角的「兌換積分」按鈕。
4. 在彈出的對話框中輸入你的預付卡代碼，然後點擊「兌換」按鈕。
5. 如果兌換成功，你會在畫面右下角彈出的通知中看到兌換成功的提示。此時你的帳戶餘額將會增加相應的金額。

## 如果遇到問題

如果你在兌換預付卡時遇到問題，請到 [Zeabur 官方 Discord 頻道](https://zeabur.com/dc) 與我們的客服團隊聯繫，我們將竭誠為你提供幫助。
