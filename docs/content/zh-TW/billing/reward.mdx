# 貢獻獎勵計劃

為了鼓勵開發者們幫助 Zeabur 豐富 [模板](https://zeabur.com/templates) 及 [文件](https://zeabur.com/docs) 的內容，讓其他開發者可以更容易、更方便的使用 Zeabur，我們推出了「貢獻獎勵計劃」來提供免費使用額度給予貢獻者。

參考以下的貢獻方式，然後在 Discord 頻道中告訴我們您的貢獻，我們將會根據您的貢獻給予相應的免費使用額度。

請注意，本頁面提到的獎勵免費額度並不是固定的數值，我們會根據您的貢獻的內容、品質、難度等因素來決定獎勵的數值，建議你可以在開始貢獻前和我們討論你的貢獻內容，以確保你的貢獻可以獲得最大的獎勵。

## 貢獻模板

貢獻模板的方式主要有以下幾種：建立模板、補充模板以及翻譯模板。

### 建立模板

建立模板可以幫助其他開發者一鍵部署更多開源專案到 Zeabur 上，參考 [從 YAML 建立模板](../template/template-in-code) 來瞭解如何建立模板。

建立的每個模板可以獲得最少 **$10** 的免費使用額度，但建立的模板必須滿足以下**所有**條件：

1. 模板必須不重複於現有的模板，除非是新版本或不同場景、不同用途的模板。
2. 模板必須包含完整的資訊，包括模板名稱、描述、封面圖、介紹及使用說明。
3. 模板介紹必須至少有英文版本。
4. 模板必須可以一鍵部署，如果部署後有需要額外的操作才能使用，必須在模板介紹中說明。

除了以上的必要條件外，為了避免我們的貢獻獎勵計劃被濫用，我們還要求您建立的模板必須符合以下**至少一個**條件：

1. 模板對應的開源專案是來自一個超過 500 星的 GitHub 專案。
2. 模板對應的開源專案在其他 PaaS 平台已經有一鍵部署按鈕或模板，且使用量超過 30 次。

額外獎勵：如果您的模板被其他開發者使用超過 30 次（部署後必須持續運行），您將會獲得額外的 **$20** 免費使用額度。

### 補充模板

補充模板是指為已有的模板進行內容的補充，例如某個模板可能在部署後需要額外的設定步驟才能使用，但模板介紹中並沒有說明，您可以為這個模板補充這些設定步驟。

補充的每個模板可以獲得最少 **$5** 的免費使用額度，但補充的模板必須滿足以下**所有**條件：

1. 模板必須是部署次數超過 30 次的模板。
2. 補充的內容必須有具體的幫助，例如補充模板介紹、補充使用說明、補充部署步驟等。

另外一個情況是模板的設定本身就有問題，導致模板無法順利一鍵部署啟動，這種情況下您可以幫助我們修復這個模板，並且補充相應的說明，這樣的修復也可以獲得 **$5** 的免費使用額度。

### 翻譯模板

翻譯模板是指為已有的模板進行翻譯，例如某個模板的介紹只有英文版本，您可以為這個模板補充其他語言的介紹。

翻譯的每個模板可以獲得最少 **$5** 的免費使用額度，但翻譯的模板必須滿足以下**所有**條件：

1. 模板必須是部署次數超過 30 次的模板。
2. 翻譯的內容必須完整，例如模板介紹以及變數名稱及說明等欄位。
3. 必須翻譯英文、繁體中文、簡體中文、日文、西班牙文等模板頁面支援的語言。

## 如何開始

若你想要參與貢獻獎勵計畫，請您先加入 [Zeabur 的 Discord 頻道](https://zeabur.com/dc) 並告訴我們你有興趣參與貢獻獎勵計劃，告訴我們你想提交、補充或翻譯哪個模板或文件的哪個頁面。

如果你想用貢獻獎勵計畫賺取免費額度，但不知道應該幫哪個開源專案建立模板、修復或翻譯哪個模板，你也可以與我們取得聯繫，我們可以將其他用戶的需求告訴給你。

如果你在貢獻過程中遇到任何問題，也可以直接告訴我們，幫助我們讓參與貢獻的流程更加順暢。
