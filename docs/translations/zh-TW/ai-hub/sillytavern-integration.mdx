# 在 SillyTavern 中使用 Zeabur AI Hub

SillyTavern 是一款強大的 AI 角色扮演聊天介面，支援多種 AI 模型。本教學將說明如何在 SillyTavern 中使用 Zeabur AI Hub。

## 前置準備

在開始之前，請確保您具備以下條件：

1. 已安裝或部署 SillyTavern（您可以使用 [Zeabur 範本](https://zeabur.com/templates/DX09NM) 快速部署）
2. 已為 [Zeabur AI Hub](https://zeabur.com/ai-hub) 儲值額度 (Credits)
3. 已建立 API 金鑰 (API key)

## 設定步驟

### 步驟 1：設定 API 連線

1. 開啟 SillyTavern 介面
2. 點擊上方導覽列的 **API Connections** 圖示（外觀像插頭）
3. 在 **API** 下拉選單中，選擇 **Chat Completion (OpenAI)**
4. 在 **Chat Completion Source** 中，選擇 **Custom (OpenAI-compatible)**
5. 在 **Custom Endpoint (Base URL)** 中，輸入 Zeabur AI Hub 的端點 (Endpoint)：

```
https://hnd1.aihub.zeabur.ai/v1
```

根據您的地理位置，您也可以選擇其他區域的端點以獲得更好的延遲表現：
- **HND1 - 日本東京**: `https://hnd1.aihub.zeabur.ai/v1`
- **SFO1 - 美國舊金山**: `https://sfo1.aihub.zeabur.ai/v1`

6. 在 **Custom API key** 中，貼上您在 Zeabur AI Hub 建立的 API 金鑰
7. 點擊 **Connect** 按鈕
8. 如果設定正確，您會看到連線指示燈變為**綠色**並顯示 "Valid"
9. 在 **Model** 下拉選單中，選擇您想要使用的 AI 模型

![Configure API Key](https://cdn.zeabur.com/silly-tavern-tutorial/1.%20configure%20api%20key.png)

常用的模型包括：
- `gpt-4`: 最強大的模型，適用於複雜對話
- `gpt-4-turbo`: 效能與成本的平衡
- `gpt-3.5-turbo`: 快速且經濟
- `claude-3-5-sonnet`: 來自 Anthropic 的高品質模型
- `deepseek-chat`: DeepSeek V3 (V3.1/V3.2) 模型，極佳的性價比
- `kimi-k2`: Kimi k2 模型，表現優異
- `glm-4.6`: GLM 4.6 模型，表現優異

<Callout type="info">
  我們**強烈建議**使用 DeepSeek 模型 (V3.1/V3.2) 以獲得最佳體驗。如果您預算充足，我們也推薦使用 Kimi k2 與 GLM 4.6。
</Callout>


<Callout>
  不同模型的定價各不相同。我們建議造訪 [Zeabur Models](https://zeabur.com/models) 查看各模型的價格資訊。
</Callout>

如果連線失敗，請檢查：
- API 金鑰是否正確（無多餘空格）
- API URL 是否正確
- 您的 AI Hub 帳戶是否有足夠的額度

### 步驟 2：測試聊天

選擇或建立一個角色 (Character)，並在聊天框中輸入訊息進行測試。如果設定正確，您應該會看到 AI 成功回覆。
