# 訂閱

## 什麼是訂閱？

Zeabur 提供 3 種方案：**免費方案 (Free Plan)**、**開發者方案 (Developer Plan)** 和 **團隊方案 (Team Plan)**。
您可以在 [方案](/billing/plans) 頁面查看各個方案的詳細資訊。

如果您尚未訂閱任何方案，系統預設會將您視為 **免費方案** 使用者。

## 如何訂閱？

Zeabur 的訂閱服務是基於 [Stripe](https://stripe.com/)。

有兩種方式可以訂閱方案：**信用卡** 和 **預付額度 (Prepaid Credits)**。

### 使用信用卡訂閱

您可以使用信用卡訂閱方案，並以美金 (USD) 支付。

- 未訂閱時設定：前往 [帳單頁面](https://zeabur.com/account/billing)，您可以在 **Payment Method** (付款方式) 區塊設定您的信用卡。輸入信用卡資訊後，點擊 **Add** 按鈕儲存信用卡，接著只需在 [方案頁面](https://zeabur.com/pricing#upgrade) 點擊 **Subscribe** 按鈕即可更新您的訂閱。
- 訂閱時設定：前往控制台中的 [方案頁面](https://zeabur.com/pricing#upgrade)，選擇一個方案，然後輸入您的信用卡資訊，點擊 **Subscribe** 按鈕，您將被重新導向至 Stripe 的結帳頁面，填寫表單後即可完成訂閱並儲存您的信用卡。

### 使用預付額度訂閱

如果您沒有與 Stripe 服務相容的信用卡，您可以透過購買額度來訂閱方案。

前往控制台中的 [帳單頁面](https://zeabur.com/account/billing)，點擊 **Get More Credits** 按鈕購買額度，接著您可以前往控制台中的 [方案頁面](https://zeabur.com/pricing#upgrade)，使用您購買的額度來升級方案。

![Get More Credits](/billing/subscription/get-more-credits.png)

## 如何更改付款方式

如果您的卡片被拒絕或您想更改付款方式，您可以在 [帳單頁面](https://zeabur.com/account/billing) 進行操作。

向下捲動找到 **Unbind Payment Method** 以解除綁定目前的信用卡，接著您就可以設定新的信用卡。

## 如何降級至免費方案

如果您已訂閱付費方案，您可以隨時降級至 **免費方案**。

前往 [控制台中的帳單頁面](https://zeabur.com/account/billing)，向下捲動至危險區域 (danger zone) 進行取消訂閱。

![Unsubscribe Plan](/billing/subscription/unsubscribe.png)

<Callout>
    降級至免費方案後，您將不再產生任何費用。但是，您目前部署的容器化 (Containerized) 服務將會被系統自動暫停。
</Callout>

## 發票與收據

當您完成付款時，您會收到一封附有發票和收據的電子郵件。

您也可以在 [帳單頁面](https://zeabur.com/account/billing) 查看您的帳單歷史記錄並下載過去的發票。

如果您需要在發票上顯示自定義的帳單資訊（例如：公司名稱、統一編號或地址），您必須訂閱 **開發者方案** 或以上。如果您需要正式發票，則必須訂閱 **團隊方案**。
