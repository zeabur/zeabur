# 兌換儲值卡

有時，您可能會從與 Zeabur 合作的第三方合作夥伴處購買或收到 Zeabur 儲值卡。這些儲值卡可以兌換為 Zeabur 服務餘額，您可以使用這些餘額升級為 Zeabur 付費會員或購買專屬伺服器。

本文將教您如何將購買或收到的 Zeabur 儲值卡兌換至您的餘額。

## 如何兌換

1. 登入您的 Zeabur 帳號。如果您還沒有 Zeabur 帳號，請先[註冊](https://zeabur.com/login)。
2. 前往 Zeabur 控制台的帳單管理頁面：[https://zeabur.com/account/billing](https://zeabur.com/account/billing)。
3. 您可以在帳單管理頁面找到「帳戶餘額」區塊。點擊右上角的「兌換點數」按鈕。
4. 在彈出的對話框中輸入您的儲值卡代碼，然後點擊「兌換」按鈕。
5. 如果兌換成功，您將在螢幕右下角看到成功通知。您的帳戶餘額將增加相應的金額。

## 如果您遇到問題

如果您在兌換儲值卡時遇到任何問題，請在 [Zeabur 社群論壇](https://zeabur.com/forum) 聯繫我們的客服團隊，我們將竭誠為您提供協助。
