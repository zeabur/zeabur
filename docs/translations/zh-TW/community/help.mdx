# 取得協助

在使用 Zeabur 時，您有時可能會遇到一些問題。您可以透過以下管道從 Zeabur 團隊取得協助。

## 社群論壇

[Zeabur 社群論壇](https://zeabur.com/forum) 是取得協助的主要場所。您可以建立貼文描述您的問題，我們的團隊將根據您的方案支援優先級進行回覆。請參閱 [社群論壇](./forum) 頁面，瞭解如何建立貼文以及應包含哪些內容的詳細資訊。

![new post](/community/forum/new-post.png)

![server](/community/help/server.png)

## 升級您的方案

如果您需要更快的回應時間和專屬支援，請考慮升級您的方案。較高等級的方案在 [社群論壇](./forum) 上享有更短的保證回應時間。請參閱 [方案](../billing/plans) 頁面以獲取更多詳細資訊。

## 電子郵件

您也可以透過電子郵件與我們聯繫以尋求支援或諮詢：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為了獲得更快的回應速度，我們建議使用 [社群論壇](./forum)。但是，如果您偏好電子郵件溝通，請隨時給我們發送訊息，我們會盡快回覆您。
