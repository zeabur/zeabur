# 貢獻獎勵計畫

為了鼓勵開發者幫助 Zeabur 豐富[模板](https://zeabur.com/templates)與[文件](https://zeabur.com/docs)的內容，讓其他開發者能更輕鬆、方便地使用 Zeabur，我們推出了「貢獻獎勵計畫」，為貢獻者提供免費使用額度 (Usage Credits)。

請參考以下貢獻方式，並在社群論壇中告知我們您的貢獻。我們將根據您的貢獻內容發放相應的免費使用額度。

請注意，本頁面提到的獎勵額度並非固定金額。我們將根據您貢獻的內容、品質和難度來決定獎勵價值。我們建議您在開始之前先與我們討論您的貢獻內容，以確保您能獲得最大的獎勵。

## 貢獻模板

貢獻模板的方式主要包括建立模板、補充模板和翻譯模板。

### 建立模板

建立模板可以幫助其他開發者一鍵部署更多開源專案 (Open-source projects) 到 Zeabur。請參考 [從 YAML 建立模板](../template/template-format) 來學習如何建立模板。

每個建立的模板至少可以為您賺取 **$10** 的免費使用額度，但該模板必須滿足以下**所有**條件：

1. 模板不得與現有模板重複，除非它是新版本或針對不同的場景、目的。
2. 模板必須包含完整資訊，包括模板名稱、描述、封面圖片、介紹和使用說明。
3. 模板介紹至少必須有英文版本。
4. 模板必須能一鍵部署。如果部署後需要額外步驟，則需要在模板介紹中說明。

除了這些必要條件外，為了防止濫用我們的貢獻獎勵計畫，我們還要求您建立的模板至少滿足以下**其中一項**條件：

1. 相關開源專案在 GitHub 上擁有超過 500 顆星星 (Stars)。
2. 相關開源專案在其他 PaaS 平台上已有一鍵部署按鈕或模板，且已被使用超過 30 次。

額外獎勵：如果您的模板被其他開發者使用超過 30 次（部署後必須持續運行），您將獲得額外的 **$20** 免費使用額度。

### 補充模板

補充模板是指為現有模板提供額外內容。例如，如果某個模板在部署後需要額外的設定步驟才能使用，但這些步驟尚未被記錄，您可以補充這些資訊。

每個補充的模板至少可以為您賺取 **$5** 的免費使用額度，但補充的模板必須滿足以下**所有**條件：

1. 該模板必須已被部署超過 30 次。
2. 補充的內容必須提供實質幫助，例如補充模板介紹、使用說明或部署步驟。

另一種情況是模板本身的設定有問題，導致無法一鍵部署成功並啟動。在這種情況下，您可以幫助我們修復該模板並補充相關說明，這同樣可以賺取 **$5** 的免費使用額度。

### 翻譯模板

翻譯模板涉及為現有模板提供翻譯。例如，如果模板介紹只有英文，您可以提供其他語言的翻譯。

每個翻譯的模板至少可以為您賺取 **$5** 的免費使用額度，但翻譯的模板必須滿足以下**所有**條件：

1. 該模板必須已被部署超過 30 次。
2. 翻譯內容必須完整，包括模板介紹以及變數名稱、描述等欄位。
3. 您必須翻譯成模板頁面支援的語言，包括英文、繁體中文、簡體中文、日文和西班牙文。

## 如何開始

如果您想參與貢獻獎勵計畫，請先加入 [Zeabur 社群論壇](https://zeabur.com/forum) 並告知我們您有興趣參與。告訴我們您計畫提交、補充或翻譯哪個模板或文件頁面。

如果您想透過貢獻獎勵計畫賺取免費額度，但不知道該為哪個開源專案建立模板，或者不知道該修復或翻譯哪個模板，您也可以聯繫我們。我們可以告知您其他使用者的需求。

如果您在貢獻過程中遇到任何問題，也可以直接告訴我們，幫助我們讓貢獻流程更加順暢。
