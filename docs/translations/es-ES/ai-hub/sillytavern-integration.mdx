# Uso de Zeabur AI Hub en SillyTavern

SillyTavern es una potente interfaz de chat para juegos de rol (roleplay) con IA que admite múltiples modelos de IA. Este tutorial explica cómo usar Zeabur AI Hub en SillyTavern.

## Requisitos previos

Antes de comenzar, asegúrate de tener:

1. Haber instalado o desplegado SillyTavern (puedes usar la [plantilla de Zeabur](https://zeabur.com/templates/DX09NM) para un despliegue rápido)
2. Haber añadido créditos a [Zeabur AI Hub](https://zeabur.com/ai-hub)
3. Haber creado una clave API (API key)

## Pasos de configuración

### Paso 1: Configurar la conexión API

1. Abre la interfaz de SillyTavern
2. Haz clic en el icono de **API Connections** (Conexiones API) en la barra de navegación superior (parece un enchufe)
3. En el menú desplegable **API**, selecciona **Chat Completion (OpenAI)**
4. En **Chat Completion Source** (Fuente de finalización de chat), selecciona **Custom (OpenAI-compatible)**
5. En **Custom Endpoint (Base URL)** (Punto de conexión personalizado), introduce el endpoint de Zeabur AI Hub:

```
https://hnd1.aihub.zeabur.ai/v1
```

Según tu ubicación geográfica, también puedes elegir otros endpoints regionales para una mejor latencia:
- **HND1 - Tokio, Japón**: `https://hnd1.aihub.zeabur.ai/v1`
- **SFO1 - San Francisco, EE. UU.**: `https://sfo1.aihub.zeabur.ai/v1`

6. En **Custom API key**, pega la clave API que creaste en Zeabur AI Hub
7. Haz clic en el botón **Connect** (Conectar)
8. Si se ha configurado correctamente, verás que el indicador de conexión se vuelve **verde** y muestra "Valid" (Válido)
9. En el menú desplegable **Model** (Modelo), selecciona el modelo de IA que quieras usar

![Configure API Key](https://cdn.zeabur.com/silly-tavern-tutorial/1.%20configure%20api%20key.png)

Los modelos populares incluyen:
- `gpt-4`: El más potente, adecuado para conversaciones complejas
- `gpt-4-turbo`: Equilibrio entre rendimiento y coste
- `gpt-3.5-turbo`: Rápido y económico
- `claude-3-5-sonnet`: Modelo de alta calidad de Anthropic
- `deepseek-chat`: Modelos DeepSeek V3 (V3.1/V3.2), excelente relación rendimiento/precio
- `kimi-k2`: Modelo Kimi k2, excelente rendimiento
- `glm-4.6`: Modelo GLM 4.6, excelente rendimiento

<Callout type="info">
  **Recomendamos encarecidamente** usar los modelos DeepSeek (V3.1/V3.2) para obtener la mejor experiencia. También recomendamos Kimi k2 y GLM 4.6 si tienes presupuesto suficiente.
</Callout>


<Callout>
  Los diferentes modelos tienen precios distintos. Recomendamos visitar [Zeabur Models](https://zeabur.com/models) para consultar la información de precios de cada modelo.
</Callout>

Si la conexión falla, comprueba:
- La clave API es correcta (sin espacios adicionales)
- La URL de la API es correcta
- Tu cuenta de AI Hub tiene créditos suficientes

### Paso 2: Probar el chat

Selecciona o crea un Personaje (Character) e introduce un mensaje en el cuadro de chat para probarlo. Si se ha configurado correctamente, deberías ver una respuesta exitosa de la IA.
