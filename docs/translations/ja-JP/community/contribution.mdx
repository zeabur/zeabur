# 貢献報酬プログラム

Zeaburの[テンプレート](https://zeabur.com/templates)や[ドキュメント](https://zeabur.com/docs)のコンテンツを充実させ、他の開発者がZeaburをより簡単かつ便利に利用できるようにすることを奨励するため、貢献者に無料の利用クレジットを提供する「貢献報酬プログラム」を開始します。

以下の貢献方法を確認し、コミュニティフォーラムであなたの貢献についてお知らせください。貢献内容に基づいて、対応する無料の利用クレジットを付与します。

このページに記載されている報酬の無料クレジットは固定額ではないことにご注意ください。貢献の内容、品質、難易度などの要因に基づいて報酬額を決定します。最大限の報酬を獲得できるよう、開始前に貢献内容についてご相談いただくことをお勧めします。

## テンプレートへの貢献

テンプレートへの貢献方法には、主にテンプレートの作成、テンプレートの補足、およびテンプレートの翻訳が含まれます。

### テンプレートの作成

テンプレートを作成することで、他の開発者がより多くのオープンソースプロジェクトをZeaburにワンクリックでデプロイできるようになります。テンプレートの作成方法については、[YAMLからテンプレートを作成する](../template/template-format)を参照してください。

作成された各テンプレートにつき、少なくとも **$10** の無料利用クレジットを獲得できますが、テンプレートは以下の条件を**すべて**満たしている必要があります：

1. テンプレートは、新しいバージョンである場合や、異なるシナリオや目的を意図している場合を除き、既存のテンプレートと重複していないこと。
2. テンプレート名、説明、カバー画像、紹介文、使用方法を含む完全な情報が含まれていること。
3. テンプレートの紹介文に少なくとも英語版があること。
4. テンプレートがワンクリックでデプロイ可能であること。デプロイ後に追加の手順が必要な場合は、テンプレートの紹介文にそれらを記載する必要があります。

これらの必須条件に加えて、貢献報酬プログラムの悪用を防ぐため、作成されたテンプレートが以下の条件の少なくとも**1つ**を満たしていることも必要です：

1. 関連するオープンソースプロジェクトがGitHubで500以上のスターを獲得していること。
2. 関連するオープンソースプロジェクトが、他のPaaS（Platform as a Service）プラットフォームですでにワンクリックデプロイボタンやテンプレートを持っており、30回以上使用されていること。

ボーナス：作成したテンプレートが他の開発者によって30回以上使用された場合（デプロイ後も実行を継続している必要があります）、さらに **$20** の無料利用クレジットを受け取ることができます。

### テンプレートの補足

テンプレートの補足とは、既存のテンプレートに追加のコンテンツを提供することを指します。例えば、テンプレートの使用にデプロイ後の追加設定手順が必要であるにもかかわらず、それらの手順が記載されていない場合、この情報を補足できます。

補足された各テンプレートにつき、少なくとも **$5** の無料利用クレジットを獲得できますが、補足されたテンプレートは以下の条件を**すべて**満たしている必要があります：

1. テンプレートが30回以上デプロイされていること。
2. 補足された内容が、テンプレートの紹介、使用方法、またはデプロイ手順の補足など、具体的な助けとなるものであること。

もう一つのケースは、テンプレートの設定自体に問題があり、ワンクリックで正常にデプロイして起動できない場合です。この場合、テンプレートの修正を支援し、関連する説明を補足することで、同様に **$5** の無料利用クレジットを獲得できます。

### テンプレートの翻訳

テンプレートの翻訳には、既存のテンプレートの翻訳を提供することが含まれます。例えば、テンプレートの紹介が英語のみの場合、他の言語の翻訳を提供できます。

翻訳された各テンプレートにつき、少なくとも **$5** の無料利用クレジットを獲得できますが、翻訳されたテンプレートは以下の条件を**すべて**満たしている必要があります：

1. テンプレートが30回以上デプロイされていること。
2. 翻訳された内容が、テンプレートの紹介や、変数名、説明などのフィールドを含めて完全であること。
3. テンプレートページでサポートされている言語（英語、繁体字中国語、簡体字中国語、日本語、スペイン語）に翻訳すること。

## 開始方法

貢献報酬プログラムに参加したい場合は、まず [Zeaburコミュニティフォーラム](https://zeabur.com/forum) に参加し、貢献報酬プログラムへの参加に興味があることをお知らせください。提出、補足、または翻訳を予定しているテンプレートやドキュメントのページを教えてください。

貢献報酬プログラムを通じて無料クレジットを獲得したいが、どのオープンソースプロジェクトのテンプレートを作成すべきか、あるいはどのテンプレートを修正・翻訳すべきかわからない場合は、お問い合わせください。他のユーザーのニーズをお伝えすることができます。

貢献プロセス中に問題が発生した場合は、直接お知らせください。貢献プロセスをよりスムーズにするための助けとなります。
