# サポートを受ける

Zeabur を使用している際に、問題が発生することがあります。以下のチャネルを通じて Zeabur チームからサポートを受けることができます。

## コミュニティフォーラム

[Zeabur コミュニティフォーラム](https://zeabur.com/forum)は、サポートを受けるための主要な場所です。問題を説明する投稿を作成すると、プランのサポート優先度に基づいてチームが回答します。投稿の作成方法や記載すべき内容の詳細については、[コミュニティフォーラム](./forum)のページを参照してください。

![new post](/community/forum/new-post.png)

![server](/community/help/server.png)

## プランのアップグレード

より迅速な対応や専用のサポートが必要な場合は、プランのアップグレードをご検討ください。上位プランでは、[コミュニティフォーラム](./forum)での保証された応答時間（Response times）が短縮されます。詳細は[プラン](../billing/plans)のページを参照してください。

## メール

サポートやお問い合わせについては、メールで連絡することも可能です。

- **メール:** [contact@zeabur.com](mailto:contact@zeabur.com)

より迅速な対応を希望される場合は、[コミュニティフォーラム](./forum)の使用をお勧めします。ただし、メールでの連絡を希望される場合は、お気軽にお送りください。できるだけ早く返信いたします。
