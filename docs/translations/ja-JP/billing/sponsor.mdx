# スポンサー

## スポンサーシップとは？

Zeabur からのスポンサーシップでは、以下の特典を提供しています：

- すべてのコントリビューター（貢献者）に対して、Developer プランを 1 ヶ月間無料で提供します。

- プロジェクトの規模や個々の貢献度に応じて、コントリビューターにクレジットを付与します。

## スポンサーシップの基準

- プロジェクトが非営利（non-commercial）であり、完全にオープンソースであること。

- プロジェクトのウェブサイトのフッターおよび `README.md` ファイルに Zeabur のバナーを表示する必要があります。

リソースは、ダッシュボードの [紹介ページ（referral page）](https://zeabur.com/account/referral) で確認できます。

[![Deploy on Zeabur](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://zeabur.com/account/referral)

## スポンサーシップの申請方法

- 上記の基準を満たすすべてのプロジェクトがスポンサーシップの対象となります。

- 申請するには、以下の手順に従ってください：

    - プロジェクトが記載されているすべての基準を満たしていることを確認してください。

    - [コミュニティフォーラム](https://zeabur.com/forum) の `help` チャンネルに、プロジェクトのリンクと簡単な説明を含めて投稿を作成してください。
