# プリペイドカードの引き換え

Zeabur と提携しているサードパーティパートナーから、Zeabur プリペイドカード (Prepaid Card) を購入したり、受け取ったりすることがあります。これらのプリペイドカードは Zeabur サービスの残高 (Balance) に引き換えることができ、Zeabur の有料メンバーシップへのアップグレードや専用サーバーの購入に使用できます。

この記事では、購入または受け取った Zeabur プリペイドカードを残高に引き換える (Redeem) 方法について説明します。

## 引き換え方法

1. Zeabur アカウントにログインします。まだ Zeabur アカウントをお持ちでない場合は、まず[登録](https://zeabur.com/login)を行ってください。
2. Zeabur コンソールの請求管理ページ [https://zeabur.com/account/billing](https://zeabur.com/account/billing) にアクセスします。
3. 請求管理ページにある「Account Balance（アカウント残高）」セクションを確認します。右上にある「Redeem Points（ポイントを引き換える）」ボタンをクリックします。
4. ポップアップダイアログにプリペイドカードのコードを入力し、「Redeem（引き換える）」ボタンをクリックします。
5. 引き換えに成功すると、画面の右下に成功の通知が表示されます。アカウントの残高に対応する金額が加算されます。

## 問題が発生した場合

プリペイドカードの引き換え中に問題が発生した場合は、[Zeabur コミュニティフォーラム](https://zeabur.com/forum)のカスタマーサービスチームまでご連絡ください。解決に向けて最善を尽くします。
