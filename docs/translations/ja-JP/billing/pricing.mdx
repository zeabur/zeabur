# 料金

Zeabur は、プロジェクトに対して従量課金制（consumption-based pricing model）を採用しています。使用していないリソースに対して請求が発生することはありません。

その代わりに、Zeabur はサービスが実際に使用した各リソースに対して、分単位で課金します。

## 料金体系

<Callout type="warning">
  Zeabur は共有クラスター（shared clusters）を段階的に廃止しています。新規プロジェクトは共有クラスター上に作成できなくなり、既存の共有クラスタープロジェクトは2026年4月1日から新しいサービスの受け入れを停止します。既存のサービスは引き続き実行されます。詳細については、[変更履歴（changelog）](https://zeabur.com/changelogs/phasing-out-shared-cluster)をご覧ください。
</Callout>

Zeabur には、*共有クラスター（shared clusters）*と[*専用サーバー（dedicated servers）*](/dedicated-server)の2種類のサーバーがあります。以下の料金体系は**共有クラスター**にのみ適用され、専用サーバーについては使用料を請求しません。

同様に、Developer Plan の月額5米ドルの無料枠は共有クラスターの使用量にのみ適用され、専用サーバーのレンタル料金に充てることはできません。

**共有クラスター**での使用料は以下の通りです：

- vCPU - 無料 ($0.00 / vCPU-分)

- メモリ - **US$0.00025** / GB-分 (約 US$10.80 / GB-月)
  - サービスが実際に使用したメモリに対してのみ課金されます。サービスがクラッシュしたり停止したりしている間は、メモリ使用料は発生しません。

- ネットワーク送信（Network Egress） - **US$0.10** / GB (US$10.00 / 100GB)
  - 1GB は、CDN や圧縮なしで 1MiB のページにアクセスする約 1,000 人のユニークビジターに対応します。
  - [プライベートネットワーク](/networking/private)は、送信帯域幅としてカウントされません。

- 永続ボリューム（Persistent Volume） - **US$0.20** / GB-月

## 返金

返金に関する問題については、[コミュニティフォーラム](https://zeabur.com/forum)までお問い合わせください。

**Developer Plan** または **Team Plan** の場合、サブスクリプションをキャンセルした際に残高の返金をリクエストできます。例えば、Developer Plan を10日間使用した後にキャンセルした場合、残りの20日間分について返金を受けることができます。24時間以内に返金をリクエストした場合は、全額返金されます。

**専用サーバー（Dedicated Server）**のユーザーについては、2025年7月21日以降に作成または更新されたサーバーの返金はサポートしていません。次回の請求サイクルでの課金を避けるには、自動更新設定を無効にしてください。

> 2025年7月21日以降に作成または更新されたサーバーのみが対象となりますのでご注意ください。

![disable auto renewal](/billing/pricing/auto-renewal.png)

返金方法は、Zeabur クレジットとして受け取るか、元の支払い方法に戻すかを選択できます。Zeabur クレジットを選択した場合、通常、承認後すぐに処理されます。元の支払い方法への返金を選択した場合、処理に最大7日かかることがあり、返金額の約30%にあたる Stripe の決済手数料を負担していただく必要があります。

## よくある質問

### VPS とサブスクリプションプランの関係は何ですか？

VPS は、Zeabur を通じてクラウドプロバイダーから割引価格で購入するサーバーです。サブスクリプションプランは、統合ダッシュボード、監視、バックアップ、AI Agent、テクニカルサポートなど、VPS 上での追加の管理サービスを提供します。

### なぜ Zeabur を通じて VPS を購入すべきなのですか？

Zeabur は、主要なクラウドプロバイダーからの独占的な割引価格と、統合された管理プラットフォームを提供します。低価格、即時セットアップ、組み込みの監視、自動バックアップ、優先サポートをすべて一箇所で受けることができます。

### どのサブスクリプションプランが自分に適していますか？

Free Plan は、基本的なダッシュボードアクセスから始めるのに最適です。Developer Plan は、完全な監視、バックアップ、AI Agent を必要とする個人の開発者に理想的です。Team Plan は、カスタム要件を持つ複数のサーバーを管理するチーム向けに設計されています。

### クレジットカードなしで Zeabur を利用できますか？

はい。クレジットカードなしでプリペイドクレジットを購入できます。代替の支払い方法として Alipay も利用可能です。詳細は[サブスクリプションページ](/billing/subscription)をご覧ください。

### 領収書は発行されますか？

お支払い時に、請求書と領収書が添付されたメールが送信されます。また、[請求ページ](https://zeabur.com/account/billing)で請求履歴を確認することもできます。

請求書にカスタマイズされた請求情報（会社名、納税者番号、住所など）が必要な場合は、**Developer Plan** 以上を購読する必要があります。正式な請求書（インボイス）が必要な場合は、**Team Plan** を購読する必要があります。

### どのクラウドプロバイダーをサポートしていますか？

現在は AWS (Lightsail)、Hetzner、DigitalOcean、Tencent Cloud、Linode、Alibaba Cloud、Google Cloud (GCP) をサポートしています。今後も継続的にプロバイダーを追加していく予定です。
