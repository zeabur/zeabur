# AI Hub

AI Hub 是一个统一的 AI 服务平台，允许您通过单一接口访问多种 AI 模型（如 OpenAI、Claude 等），并根据实际使用情况进行透明的按量计费（Pay-as-you-go）。

## 核心特性

- **多模型支持**：访问 OpenAI 和 Anthropic Claude 等主流 AI 模型
- **统一 API Key 管理**：使用单个 API Key 即可访问所有支持的模型
- **透明计费**：按量计费，并提供完整的使用历史记录追踪
- **多区域端点 (Endpoints)**：选择最合适的 API 端点以获得最佳性能
- **易于集成**：兼容 OpenAI API 格式，可快速集成到现有应用中

## 快速上手

### 步骤 1：充值额度

1. 登录 [Zeabur 控制台 (Dashboard)](https://zeabur.com/projects)
2. 导航至 **AI Hub** 页面
3. 在“余额管理”部分，选择您想要充值的金额
4. 完成支付后，额度将立即添加到您的账户中

### 步骤 2：创建 API Key

1. 在 AI Hub 页面，点击“创建 API Key”
2. 为您的 API Key 命名（例如 "My Application"）
3. 复制生成的 API Key 并妥善保存

<Callout type="warning">
  API Key 在创建时仅显示一次。请立即将其复制并保存到安全的位置。如果丢失，您将需要创建一个新密钥。
</Callout>

### 步骤 3：使用 API

使用 Zeabur AI Hub API 与使用 OpenAI API 的方式相同。您只需要：

1. 将 API Base URL 设置为 Zeabur AI Hub 的端点
2. 使用您在 AI Hub 中创建的 API Key

```bash
curl https://hnd1.aihub.zeabur.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ZEABUR_API_KEY" \
  -d '{
    "model": "gpt-4",
    "messages": [
      {
        "role": "user",
        "content": "Hello!"
      }
    ]
  }'
```

## 计费说明

AI Hub 使用**预付额度系统**：

1. 首先向您的账户充值额度
2. 每次 API 调用将根据所使用的模型和 Token 数量扣除相应的额度
3. 在“使用历史”中查看详细的使用记录和费用

### 查看使用记录

在 AI Hub 页面上：
1. 进入“使用历史”选项卡
2. 查看每次 API 调用的详细信息：
   - 时间戳
   - 模型名称
   - Token 使用量
   - 消耗额度

## 安全最佳实践

- **保护您的 API Key**：不要将 API Key 提交到版本控制系统
- **使用环境变量**：将 API Key 存储在环境变量中
- **定期轮换**：定期撤销旧的 API Key 并创建新的密钥
- **监控使用情况**：定期检查使用历史，以发现异常的 API 调用

## 常见问题

### 如何查看我的剩余额度？

您可以在 AI Hub 页面的“余额管理”部分查看当前的剩余额度。

### 额度会过期吗？

目前，AI Hub 额度不会过期。

### 我可以申请退款吗？

充值到账户中的额度目前不可退款。请根据您的实际需求进行充值。

### 支持哪些编程语言？

Zeabur AI Hub 使用标准的 OpenAI API 格式，支持所有主流编程语言：
- Python (openai 软件包)
- JavaScript/TypeScript (openai 软件包)
- Java, Go, Ruby, PHP 等

您只需要将 API Base URL 和 API Key 设置为 Zeabur AI Hub 的对应值即可。
