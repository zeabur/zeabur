# 兑换预付卡

有时，您可能会从与 Zeabur 合作的第三方合作伙伴处购买或收到 Zeabur 预付卡。这些预付卡可以兑换为 Zeabur 服务余额，您可以将其用于升级到 Zeabur 付费会员或购买专用服务器。

本文将教您如何将购买或收到的 Zeabur 预付卡兑换至您的余额。

## 如何兑换

1. 登录您的 Zeabur 账号。如果您还没有 Zeabur 账号，请先[注册](https://zeabur.com/login)。
2. 访问 Zeabur 控制台的账单管理页面：[https://zeabur.com/account/billing](https://zeabur.com/account/billing)。
3. 您可以在账单管理页面找到“账户余额”部分。点击右上角的“兑换积分”按钮。
4. 在弹出对话框中输入您的预付卡代码，然后点击“兑换”按钮。
5. 如果兑换成功，您将在屏幕右下角看到成功通知。您的账户余额将增加相应的金额。

## 如果您遇到问题

如果您在兑换预付卡时遇到任何问题，请在 [Zeabur 社区论坛](https://zeabur.com/forum) 联系我们的客服团队，我们将尽力为您提供帮助。
