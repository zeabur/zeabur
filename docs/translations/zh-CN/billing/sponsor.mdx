# 赞助

## 什么是赞助？

Zeabur 的赞助提供以下权益：

- 为所有贡献者提供一个月免费的开发者计划 (Developer Plan)。

- 为贡献者提供代金券 (Credits)，具体金额取决于项目规模和个人贡献。

## 赞助标准

- 项目必须是非商业性的且完全开源。

- 必须在项目网站的页脚和 `README.md` 文件中展示 Zeabur 的横幅 (Banner)。

您可以在我们控制面板的 [推广页面](https://zeabur.com/account/referral) 中找到相关资源。

[![在 Zeabur 部署](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://zeabur.com/account/referral)

## 如何申请赞助

- 赞助面向任何符合上述标准的项目开放。

- 申请步骤如下：

    - 确保您的项目符合所有列出的标准。

    - 在我们的 [社区论坛](https://zeabur.com/forum) 的 `help` 频道中发布帖子，包括您的项目链接和简短描述。
