# 贡献奖励计划

为了鼓励开发者帮助 Zeabur 丰富[模板](https://zeabur.com/templates)和[文档](https://zeabur.com/docs)的内容，让其他开发者更轻松、更方便地使用 Zeabur，我们推出了“贡献奖励计划”，为贡献者提供免费的使用额度 (Usage Credits)。

请参考以下贡献方式，然后在社区论坛中告知我们您的贡献。我们将根据您的贡献情况发放相应的免费使用额度。

请注意，本页面提到的奖励额度并非固定金额。我们将根据您贡献的内容、质量和难度来决定奖励价值。我们建议您在开始之前先与我们讨论您的贡献计划，以确保您能获得最高奖励。

## 贡献模板

贡献模板的方式主要包括创建模板、补充模板和翻译模板。

### 创建模板

创建模板可以帮助其他开发者在 Zeabur 上一键部署更多开源项目。参考[从 YAML 创建模板](../template/template-format)了解如何创建模板。

每个创建的模板至少可以为您赚取 **$10** 的免费使用额度，但模板必须满足以下**所有**条件：

1. 模板不得与现有模板重复，除非它是新版本或针对不同的场景或用途。
2. 模板必须包含完整的信息，包括模板名称、描述、封面图、介绍和使用说明。
3. 模板介绍必须至少包含英文版本。
4. 模板必须支持一键部署。如果部署后需要额外步骤，则需要在模板介绍中记录。

除了这些必要条件外，为了防止滥用我们的贡献奖励计划，我们还要求您创建的模板至少满足以下条件之**一**：

1. 关联的开源项目在 GitHub 上拥有超过 500 个 Star。
2. 关联的开源项目在其他 PaaS (Platform as a Service，平台即服务) 平台上已有键部署按钮或模板，且已被使用超过 30 次。

额外奖励：如果您的模板被其他开发者使用超过 30 次（部署后必须持续运行），您将获得额外的 **$20** 免费使用额度。

### 补充模板

补充模板是指为现有模板提供额外内容。例如，如果某个模板在部署后需要额外的设置步骤才能使用，但这些步骤未被记录，您可以补充这些信息。

每个补充的模板至少可以为您赚取 **$5** 的免费使用额度，但补充的模板必须满足以下**所有**条件：

1. 模板必须已被部署超过 30 次。
2. 补充的内容必须提供实质性的帮助，例如补充模板介绍、使用说明或部署步骤。

另一种情况是模板本身的设置存在问题，导致无法通过一键部署成功启动。在这种情况下，您可以帮助我们修复该模板并补充相关说明，这同样可以为您赚取 **$5** 的免费使用额度。

### 翻译模板

翻译模板涉及为现有模板提供翻译。例如，如果模板介绍只有英文，您可以提供其他语言的翻译。

每个翻译的模板至少可以为您赚取 **$5** 的免费使用额度，但翻译的模板必须满足以下**所有**条件：

1. 模板必须已被部署超过 30 次。
2. 翻译的内容必须完整，包括模板介绍以及变量名称和描述等字段。
3. 您必须翻译成模板页面支持的语言，包括英语、繁体中文、简体中文、日语和西班牙语。

## 如何开始

如果您想参与贡献奖励计划，请先加入 [Zeabur 社区论坛](https://zeabur.com/forum) 并告知我们您有兴趣参与。告诉我们您计划提交、补充或翻译哪个模板或文档页面。

如果您想通过贡献奖励计划赚取免费额度，但不知道该为哪个开源项目创建模板，或者不知道该修复或翻译哪个模板，也可以联系我们。我们可以告知您其他用户的需求。

如果您在贡献过程中遇到任何问题，也可以直接告诉我们，帮助我们优化贡献流程。
