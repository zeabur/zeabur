# 获取帮助

在使用 Zeabur 的过程中，你可能会遇到一些问题。你可以通过以下渠道从 Zeabur 团队获取帮助。

## 社区论坛

[Zeabur 社区论坛](https://zeabur.com/forum) 是获取帮助的主要场所。你可以发布帖子描述你遇到的问题，我们的团队将根据你所选方案的支持优先级进行回复。有关如何发布帖子以及应包含哪些内容的详细信息，请参阅 [社区论坛](./forum) 页面。

![new post](/community/forum/new-post.png)

![server](/community/help/server.png)

## 升级你的方案

如果你需要更快的响应速度和专属支持，请考虑升级你的方案。更高级别的方案在 [社区论坛](./forum) 上享有更短的保证响应时间。更多详情请参阅 [方案](../billing/plans) 页面。

## 电子邮件

你也可以通过电子邮件联系我们以获取支持或进行咨询：

- **电子邮件：** [contact@zeabur.com](mailto:contact@zeabur.com)

为了获得更快的响应速度，我们建议使用 [社区论坛](./forum)。但是，如果你更倾向于使用电子邮件沟通，请随时给我们发送邮件，我们会尽快回复你。
