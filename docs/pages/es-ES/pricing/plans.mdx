# Planes

Zeabur ofrece cuatro planes de suscripción más una opción Enterprise personalizada:

- **[Free Plan](/pricing/free-plan)** — US$0/mes, para exploración personal.
- **[Dev Plan](/pricing/dev-plan)** — US$5/mes, prueba gratuita de 14 días. Para desarrolladores individuales.
- **[Pro Plan](/pricing/pro-plan)** — US$19/mes, prueba gratuita de 14 días. Recomendado para producción.
- **[Team Plan](/pricing/team-plan)** — US$79/mes para 3 asientos, +US$24/asiento. Para equipos.
- **Enterprise Plan** — [contáctanos](mailto:contact@zeabur.com) para un plan adaptado a tu caso de uso.

Para la comparación completa de funciones y la política de reembolsos, consulta [Planes de Precios](/pricing/pricing-plans). Para los precios más actualizados, visita [zeabur.com/pricing](https://zeabur.com/pricing).
