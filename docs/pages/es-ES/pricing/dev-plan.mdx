# Plan Dev (Plan Desarrollador)

El Plan Desarrollador es el plan de pago de Zeabur diseñado para desarrolladores individuales. A **US$5/mes**, desbloquea todas las funciones de despliegue y es ideal para proyectos personales en producción y proyectos secundarios.

## Detalles del plan

| Elemento | Detalles |
| --- | --- |
| Tarifa mensual | US$5 / mes |
| Credits incluidos | US$5 / mes (compensa costos de recursos) |
| Proyectos | Ilimitados |
| Servicios | Ilimitados |
| Dominios personalizados | Soportado |
| Almacenamiento persistente | Soportado |
| Regiones de despliegue | Todas las regiones disponibles |
| Miembros | 1 (solo el propietario de la cuenta) |

### Tarifas

Las tarifas de recursos del Plan Dev son las mismas que se indican en el [Resumen de Planes](/pricing/pricing-plans):

- **vCPU** — Gratis
- **Memoria** — US$0.00025 / GB-minuto (~US$10.8 / GB-mes)
- **Tráfico de red saliente** — US$0.1 / GB
- **Almacenamiento persistente** — US$0.2 / GB-mes

Los US$5 de Credits mensuales incluidos compensan automáticamente el uso de recursos. Cualquier exceso se cobra a su tarjeta de crédito o saldo prepagado.

## Plan Gratuito vs. Plan Dev

| Característica | Plan Gratuito | Plan Dev |
| --- | --- | --- |
| Tarifa mensual | Gratis | US$5 |
| Proyectos | 1 | Ilimitados |
| Servicios | 2 por proyecto | Ilimitados |
| Dominios personalizados | No soportado | Soportado |
| Almacenamiento persistente | No soportado | Soportado |
| Suspensión automática | Se suspende cuando está inactivo | Siempre en ejecución |
| SLA | Ninguno | Estándar |

## Casos de uso

- **Proyectos personales secundarios** — Despliegue sitios web personales, blogs, servicios API, etc.
- **Desarrollo y pruebas** — Entornos que necesitan almacenamiento persistente (ej., bases de datos).
- **Servicios pequeños en producción** — Servicios de bajo tráfico que necesitan ejecutarse 24/7.
- **Desarrollo de plantillas** — Cree y mantenga plantillas de Zeabur con [Deploy Buttons](/deploy/deploy-button) para promocionar sus proyectos.

## Cómo suscribirse

Vaya a [zeabur.com/pricing](https://zeabur.com/pricing) y seleccione Plan Dev, o actualice desde la [página de facturación del Dashboard](https://dash.zeabur.com/account/billing). Se aceptan tarjeta de crédito y Alipay — consulte [Métodos de Pago](/subscription/payment-methods) para más detalles.
