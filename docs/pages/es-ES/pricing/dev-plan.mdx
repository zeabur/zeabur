# Dev Plan (Developer Plan)

El Dev Plan es el nivel de pago básico de Zeabur, diseñado para desarrolladores individuales. A **US$5/mes** con una **prueba gratuita de 14 días**, desbloquea Zeabur AI, copias de seguridad automatizadas de bases de datos, compra de dominios y el servicio Zeabur Email — ideal para proyectos personales en producción y proyectos secundarios.

Para la lista completa de funciones y las cuotas más recientes, consulta la [tabla comparativa](/pricing).

## Casos de Uso

- **Proyectos personales secundarios** — Despliega sitios web personales, blogs, servicios API, etc.
- **Desarrollo y pruebas** — Entornos que necesitan almacenamiento persistente (p. ej., bases de datos) y copias de seguridad automatizadas.
- **Servicios pequeños en producción** — Servicios de bajo tráfico que necesitan ejecutarse 24/7.
- **Desarrollo de plantillas** — Crea y mantén plantillas de Zeabur con [Deploy Buttons](/deploy/deploy-button) para promocionar tus proyectos.

## Cuándo Actualizar a Pro

Si necesitas **reenvío de logs**, **mayores cuotas de Email/Dominio/Clave de API/Webhook**, **iconos de servicio personalizados** o **soporte prioritario**, el [Pro Plan](/pricing/pro-plan) (US$19/mes, prueba gratuita de 14 días) es el siguiente paso. Consulta la página del [Pro Plan](/pricing/pro-plan) para más detalles.

## Cómo Suscribirse

Ve a [zeabur.com/pricing](https://zeabur.com/pricing) y selecciona Dev Plan, o actualiza desde la [página de facturación del Dashboard](https://dash.zeabur.com/account/billing). Se aceptan tarjetas de crédito y Alipay — consulta [Métodos de Pago](/subscription/payment-methods) para más detalles.
