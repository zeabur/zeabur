# Plan Dev (Plan Desarrollador)

El Plan Dev es el plan de pago de Zeabur diseñado para desarrolladores individuales que quieren AI Agent, backups de base de datos y gestión de dominios. A **US$5/mes**, desbloquea las funciones de productividad esenciales para desarrolladores individuales.

<Callout type="info">
Elegible para la **prueba gratuita de 14 días** — las mismas funciones que la versión de pago, sin cargo hasta que termine la prueba. Consulta [Prueba gratuita de 14 días](/billing/subscription#prueba-gratuita-de-14-días).
</Callout>

## Detalles del plan

| Elemento | Detalles |
| --- | --- |
| Tarifa mensual | US$5 / mes |
| Servidores externos vinculados | Hasta 3 |
| Especificación CI de build | 2C4G |
| Retención de logs | 7 días |
| Límite de subida de archivos | Medio |
| Gestión de archivos de contenedor | Subida de archivo único |
| Zeabur AI (AI Agent) | Incluido |
| Compra de dominio | Soportado |
| Sistema de correo | Incluido |
| Exportación de backup de base de datos | Soportado |
| Miembros | 1 (solo el propietario de la cuenta) |
| Regiones de despliegue | Todas las regiones disponibles |

## Qué se añade (vs. Free)

Todo lo del plan Free, más:

- Hasta **3 servidores externos vinculados** (Free: 1)
- Retención de logs **7 días** (Free: 48 horas)
- **Zeabur AI** — AI Agent para gestionar tus servicios
- **Compra de dominio** directamente a través de Zeabur
- **Sistema de correo** — envía correos transaccionales
- Mayores límites de subida de archivos

## Tarifas de recursos

El uso del clúster compartido se factura por minuto. Consulta [Billing / Pricing](/billing/pricing) para la tabla completa.

- **vCPU** — Gratis
- **Memoria** — US$0.00025 / GB-minuto (~US$10.8 / GB-mes)
- **Tráfico de red saliente** — US$0.1 / GB
- **Almacenamiento persistente** — US$0.2 / GB-mes

## Casos de uso

- **Proyectos personales secundarios** — Despliega sitios web personales, blogs, servicios API.
- **Flujos asistidos por IA** — Deja que Zeabur AI gestione y solucione problemas en tus servicios.
- **Servicios pequeños en producción** — Servicios de bajo tráfico que necesitan ejecutarse 24/7.
- **Desarrollo de plantillas** — Crea y mantén plantillas de Zeabur con [Deploy Buttons](/deploy/deploy-button).

## Cómo suscribirse

Ve a [zeabur.com/pricing](https://zeabur.com/pricing) y selecciona Plan Dev, o actualiza desde la [página de facturación del Dashboard](https://dash.zeabur.com/account/billing). Se aceptan tarjeta de crédito y Alipay — consulta [Métodos de Pago](/subscription/payment-methods) para más detalles.
