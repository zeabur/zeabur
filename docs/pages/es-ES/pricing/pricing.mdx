# Precios

Zeabur ofrece cuatro niveles de suscripción más una opción Enterprise personalizada:

- **[Free Plan](/pricing/free-plan)** — US$0/mes
- **[Dev Plan](/pricing/dev-plan)** — US$5/mes (prueba gratuita de 14 días)
- **[Pro Plan](/pricing/pro-plan)** — US$19/mes (prueba gratuita de 14 días)
- **[Team Plan](/pricing/team-plan)** — US$79/mes para 3 asientos (+US$24/asiento)
- **Enterprise** — [contáctanos](https://zeabur.com/forum)

Para la comparación completa de funciones y la política de reembolsos, consulta [Planes de Precios](/pricing/pricing-plans). Para los precios más actualizados, visita [zeabur.com/pricing](https://zeabur.com/pricing).
