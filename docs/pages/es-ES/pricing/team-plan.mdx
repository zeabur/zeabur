# Plan Team

El Plan Team es el plan colaborativo de Zeabur diseñado para equipos. Soporta gestión de múltiples miembros y facturación unificada, ideal para equipos que necesitan que varias personas gestionen proyectos juntas.

## Detalles del plan

| Elemento | Detalles |
| --- | --- |
| Facturación | Basada en uso |
| Proyectos | Ilimitados |
| Servicios | Ilimitados |
| Dominios personalizados | Soportado |
| Almacenamiento persistente | Soportado |
| Regiones de despliegue | Todas las regiones disponibles |
| Miembros | Múltiples (por cantidad de puestos) |
| Facturación / Tax ID | Soportado |

### Modelo de puestos

El Plan Team soporta colaboración de múltiples miembros. La cantidad de miembros se basa en puestos — consulte [zeabur.com/pricing](https://zeabur.com/pricing) para tarifas específicas.

### Tarifas

Las tarifas de recursos del Plan Team son las mismas que las del Plan Dev. Consulte el [Resumen de Planes](/pricing/pricing-plans) para más detalles.

## Facturación y Tax ID

El Plan Team soporta emisión de facturas y configuración de Tax ID (número de identificación fiscal), lo que lo hace adecuado para empresas y organizaciones que necesitan recibos para contabilidad. Consulte [Facturación](/subscription/invoicing) para más detalles.

## Plan Dev vs. Plan Team

| Característica | Plan Dev | Plan Team |
| --- | --- | --- |
| Facturación | US$5/mes + uso | Basada en uso |
| Miembros | 1 | Múltiples (por puesto) |
| Permisos de miembros | N/A | Soportado |
| Facturación / Tax ID | No soportado | Soportado |
| Ideal para | Desarrolladores individuales | Equipos / Organizaciones |

<Callout type="info">
Para planes personalizados a mayor escala, soporte dedicado y SLAs, consulte el **Plan Enterprise** — [contáctenos](https://zeabur.com/dc) para más detalles.
</Callout>

## Casos de uso

- **Equipos de startups** — Múltiples desarrolladores gestionando entornos de despliegue conjuntamente.
- **Proyectos internos de empresa** — Equipos que necesitan facturación unificada y emisión de facturas.
- **Agencias** — Crear proyectos separados para diferentes clientes bajo una gestión de equipo unificada.
- **Comunidades de código abierto** — Múltiples mantenedores colaborando en despliegues de servicios.

## Cómo suscribirse

Vaya a [zeabur.com/pricing](https://zeabur.com/pricing) y seleccione Plan Team, o actualice desde la [página de facturación del Dashboard](https://dash.zeabur.com/account/billing). Se aceptan tarjeta de crédito y Alipay — consulte [Métodos de Pago](/subscription/payment-methods) para más detalles.
