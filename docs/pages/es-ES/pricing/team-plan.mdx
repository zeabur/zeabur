# Team Plan

El Team Plan es el nivel colaborativo de Zeabur diseñado para equipos. **US$79/mes para 3 asientos al nivel Pro**, con cada asiento adicional a **US$24/mes**. Soporta colaboración de múltiples miembros, gestión de roles y permisos, facturación e ID fiscal, y las mayores cuotas de recursos — ideal para equipos que necesitan que varias personas gestionen proyectos juntas.

## Modelo de Asientos

El Team Plan incluye **3 asientos al nivel Pro** en la cuota base de US$79/mes. Cada miembro adicional es **US$24/mes**. Cada asiento incluye el conjunto completo de funciones del nivel Pro para ese usuario. Consulta [zeabur.com/pricing](https://zeabur.com/pricing) para las tarifas más actualizadas.

## Facturación e ID Fiscal

El Team Plan soporta la emisión de facturas y la configuración de ID fiscal (統一編號), lo que lo hace adecuado para empresas y organizaciones que necesitan recibos para contabilidad. Consulta [Facturación](/subscription/invoicing) para más detalles.

## Casos de Uso

- **Equipos de startups** — Múltiples desarrolladores gestionando entornos de despliegue conjuntamente.
- **Proyectos internos de empresa** — Equipos que necesitan facturación unificada y emisión de facturas.
- **Agencias** — Crea proyectos separados para diferentes clientes bajo una gestión de equipo unificada.
- **Comunidades de código abierto** — Múltiples mantenedores colaborando en despliegues de servicios.

## Cómo Suscribirse

Ve a [zeabur.com/pricing](https://zeabur.com/pricing) y selecciona Team Plan, o actualiza desde la [página de facturación del Dashboard](https://dash.zeabur.com/account/billing). Se aceptan tarjeta de crédito y Alipay — consulta [Métodos de Pago](/subscription/payment-methods) para más detalles.
