# Free Plan

El Free Plan es el nivel gratuito de Zeabur, diseñado para la exploración personal y el aprendizaje. No se requiere tarjeta de crédito — simplemente regístrate y empieza a desplegar.

## Detalles del Plan

| Elemento | Free Plan |
| --- | --- |
| Cuota mensual | US$0 |
| Vinculaciones de servidor externo | 1 |
| Especificaciones del CI de compilación | 2C4G |
| Retención de logs | 48 horas |
| Gestión de archivos del contenedor | Subida de archivo único |
| Visor de base de datos | Compatible |
| Almacenamiento persistente | Compatible |
| Vinculación de dominio personalizado | Compatible |
| Compra de dominios (registro) | No compatible |
| Copia de seguridad de BD — exportación / importación | No compatible |
| Reenvío de logs | No compatible |
| Zeabur AI | No compatible |
| Zeabur Email — cuota diaria | 0 |
| Zeabur Email — cuota mensual | 0 |
| Dominios | 0 |
| Claves de API | 0 |
| Webhooks | 0 |
| Publicación en el foro | Limitada (básica) |
| Soporte prioritario | No incluido |
| SLA | No incluido |

## Limitaciones

- **Suspensión automática** — Los servicios en el Free Plan se suspenden automáticamente tras un período de inactividad. Se reactivan con la siguiente solicitud entrante, lo que puede causar unos segundos de latencia de arranque en frío.
- **Sin SLA** — El Free Plan no incluye un Acuerdo de Nivel de Servicio.
- **Sin Zeabur Email** — El envío de correos a través de [Zeabur Email](/email) no está disponible en el Free Plan.
- **Sin copia de seguridad automatizada de bases de datos/importación ni reenvío de logs** — estas funciones operativas avanzadas están disponibles a partir del Dev Plan.

<Callout type="info">
Los servicios en el Free Plan se suspenden automáticamente cuando están inactivos para conservar recursos. Si necesitas que los servicios se ejecuten de forma continua, considera actualizar al [Dev Plan](/pricing/dev-plan).
</Callout>

## Ruta de Actualización

Cuando tus necesidades superen el Free Plan, puedes actualizar a un plan de pago:

- **[Dev Plan](/pricing/dev-plan)** — US$5/mes (prueba gratuita de 14 días), ideal para desarrolladores individuales con proyectos en producción. Añade Zeabur AI, copia de seguridad de bases de datos, compra de dominios y Zeabur Email.
- **[Pro Plan](/pricing/pro-plan)** — US$19/mes (prueba gratuita de 14 días), recomendado para cargas de trabajo en producción. Añade reenvío de logs, cuotas avanzadas, iconos de servicio personalizados y soporte prioritario.
- **[Team Plan](/pricing/team-plan)** — US$79/mes para 3 asientos (+US$24/asiento), diseñado para la colaboración en equipo con gestión de roles y permisos.
- **Enterprise** — Planes personalizados con soporte dedicado y SLAs — [contáctanos](https://zeabur.com/forum) para más detalles.

Visita [zeabur.com/pricing](https://zeabur.com/pricing) para comparar todos los planes.
