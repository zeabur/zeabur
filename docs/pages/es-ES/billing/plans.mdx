# Planes

Zeabur ofrece diferentes planes de suscripción según las necesidades de cada usuario.

Las características y precios se pueden consultar en la [página de "Pricing" de Zeabur](https://zeabur.com/es-ES/pricing). Este documento explica principalmente el público objetivo de cada plan y los métodos de pago admitidos.

![Planes](/billing/plans/overview.png)

## Plan de Prueba Gratuita

Por defecto, todas las cuentas de Zeabur comienzan con el Plan de Prueba Gratuita.

Con este plan puedes probar cualquier servicio, pero los servicios solo pueden desplegarse en regiones gratuitas específicas. Después de actualizar, puedes migrar a otras regiones de pago.

Este plan es ideal solo para pruebas y evaluación.

## Plan Developer

El Plan Developer está pensado para todo tipo de desarrolladores y pequeños equipos. Además de generosas cuotas de recursos, prioridad más alta de soporte y la insignia exclusiva "Dev Plan" en el servidor de Discord, también permite "pago por uso" para crear todo tipo de aplicaciones interesantes en Zeabur sin preocuparte por restricciones.

El Plan Developer incluye una cuota gratuita de **US$5** cada mes:

- Si tu uso en un mes no supera los US$5, no se te cobrará nada adicional más allá del costo del plan.
- Si el uso supera los US$5, el exceso se facturará según la fórmula descrita en ["Pricing"](./pricing).

## Plan Team

El Plan Team está diseñado para proyectos medianos y grandes o con altos requerimientos de recursos. Además de ofrecer mayores cuotas de recursos, puedes contactar directamente con el equipo de Zeabur para discutir posibilidades personalizadas.

Admite pagos con tarjeta de crédito y Alipay. Si necesitas otros métodos de pago, puedes comunicarte con nosotros.

## Plan Enterprise

Si el Plan Team no cubre tus necesidades, puedes [contactarnos](mailto:contact@zeabur.com) para obtener un plan adaptado a tu escenario de uso.
