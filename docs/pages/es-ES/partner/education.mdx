# Promoción Educativa

El programa de Promoción Educativa es para **instructores, instituciones de formación y organizadores de eventos** que presentan Zeabur a sus estudiantes a través de la enseñanza y eventos técnicos, ganando recompensas basadas en el número de asistentes que se convierten en usuarios de pago.

## Cómo funciona

1. **Solicitar** — Contacte al equipo de Zeabur y describa su plan de curso o evento.
2. **Recibir un Event Code** — Zeabur crea un Event Code dedicado para su evento. Los estudiantes pueden usar este código para probar Zeabur de forma gratuita.
3. **Realizar su evento** — Guíe a los estudiantes a desplegar proyectos en Zeabur durante su curso o evento.
4. **Conteo de retención** — Un mes después del evento, Zeabur cuenta los usuarios que se registraron con su Event Code y siguen siendo clientes de pago.
5. **Reclamar recompensas** — Gane US$5 por cada usuario de pago retenido.

## Mecanismo del Event Code

Un Event Code es un código único que proporciona a los asistentes Credits de prueba durante un evento:

- Cada Event Code está vinculado a un evento específico.
- Los estudiantes ingresan el Event Code durante el registro en Zeabur para recibir Credits de prueba.
- Los Event Codes tienen un período de validez basado en el calendario del evento, generalmente cubriendo el evento y un período de gracia posterior.

<Callout type="info">
Los canjes de Event Code no activan el mecanismo de comisión del programa de referidos — porque los Event Codes proporcionan Credits de prueba, no pagos reales. Las comisiones de referidos solo se calculan cuando el pago real de un usuario es exitoso.
</Callout>

## Cálculo de recompensas

| Elemento | Detalles |
| --- | --- |
| Período de conteo | Un mes después de la distribución del Event Code |
| Criterio de retención | El estudiante se registró vía Event Code, actualizó a un plan de pago y sigue activo durante el período de conteo |
| Monto de recompensa | US$5 por usuario de pago retenido |
| Método de pago | Credits de Zeabur u otros métodos acordados |

### Ejemplo

Organiza un taller para 50 asistentes y distribuye un Event Code. Un mes después, 8 estudiantes siguen usando Zeabur y han actualizado a un plan de pago. Su recompensa es 8 × US$5 = **US$40**.

## Escenarios ideales

- **Cursos de programación** — Los estudiantes despliegan proyectos del curso en la nube.
- **Talleres técnicos** — Eventos prácticos donde los asistentes experimentan el proceso de despliegue en tiempo real.
- **Hackatones** — Los equipos usan Zeabur para desplegar rápidamente sus proyectos.
- **Cursos en línea** — Proporcione Event Codes para que los estudiantes remotos practiquen.

## Cómo solicitar

Contacte al equipo de Zeabur para solicitar una asociación de Promoción Educativa:

- **Discord**: Abra un ticket en el [servidor de Discord de Zeabur](https://zeabur.com/dc) describiendo su plan de evento.
- Proporcione el nombre del evento, asistencia esperada, fechas y formato. El equipo de Zeabur configurará un Event Code para usted.
