# Programa de Socios

El Programa de Socios de Zeabur ofrece oportunidades de colaboración para instructores, organizadores de eventos y consultores de ventas para ganar recompensas promoviendo Zeabur.

## Dos modelos de asociación

### Promoción Educativa

Para **instructores, instituciones de formación y organizadores de eventos**. Promueva Zeabur a través de la enseñanza y eventos técnicos utilizando Event Codes que dan a los asistentes acceso de prueba gratuito, y gane recompensas basadas en los usuarios que se convierten en clientes de pago.

- Reciba un Event Code dedicado para su evento
- Un mes después, gane US$5 por cada usuario de pago retenido
- Ideal para talleres, cursos en línea y hackatones

[Más información →](/partner/education)

### Socio de Ventas

Para **consultores empresariales, integradores de sistemas y firmas de consultoría tecnológica**. Ayude a clientes empresariales a adoptar Zeabur. Una vez que se firma un contrato y comienza el pago, gane una recompensa de devolución de efectivo.

- Requisito: Contrato individual > US$1,000 o compra institucional al por mayor
- Devolución: Total del contrato ÷ meses del contrato (es decir, el equivalente a un mes)
- Ideal para incorporación empresarial y adquisiciones institucionales

[Más información →](/partner/sales-partner)

## En qué se diferencia del Programa de Referidos

Zeabur también ofrece un [Programa de Referidos](/pricing/referral) que permite a cualquier usuario ganar recompensas a través de códigos de referido. Diferencias clave:

| Aspecto | Programa de Referidos | Programa de Socios |
| --- | --- | --- |
| Quién | Cualquier usuario | Instructores / Organizadores / Consultores |
| Cómo | Flujo automático de código de referido | Revisión manual y contrato |
| Recompensas | Comisiones por código de referido | Recompensas por retención de Event Code / Devolución por contrato |
| Umbral | Ninguno | Socio de Ventas: operación individual > US$1,000 |

<Callout type="info">
Una sola transacción solo puede seguir un camino — código de referido o programa de socios. No se acumulan.
</Callout>
