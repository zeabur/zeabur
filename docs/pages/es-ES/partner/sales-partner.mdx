# Socio de Ventas (Sales Partner)

El programa de Socio de Ventas (Sales Partner) está dirigido a **consultores empresariales, integradores de sistemas y firmas de consultoría tecnológica**. Cuando ayude a clientes empresariales, instituciones u organizaciones a adoptar Zeabur, obtendrá comisiones recurrentes basadas en las etapas en las que participe efectivamente.

## Cuándo aplica

El programa de Socio de Ventas aplica en los siguientes escenarios:

- **Migración empresarial a la nube** — Ayude a las empresas a migrar servicios existentes a Zeabur.
- **Adquisición institucional** — Compras al por mayor o por contrato realizadas por instituciones educativas, agencias gubernamentales o departamentos de TI empresariales, incluyendo Vouchers, facturación y casos que requieran contrato.
- **Consultoría técnica** — Como consultor técnico o integrador de sistemas, incorpore Zeabur a los planes de despliegue que diseña para sus clientes.
- **Alianzas de servicio a largo plazo** — Ofrezca servicios de despliegue, operaciones o cobro de pagos a clientes utilizando Zeabur como plataforma subyacente.

Si el monto está por debajo del umbral y el usuario paga individualmente a través del autoservicio, utilice el flujo automático del [Programa de Referidos](/rewards/referral) en su lugar. Los umbrales y condiciones específicos se acuerdan de forma individual en cada contrato.

<Callout type="info">
Una misma transacción solo puede ir por la vía del Programa de Referidos o la vía de Socio de Ventas — **no se acumulan**.
</Callout>

## Estructura de comisiones

Las comisiones del Socio de Ventas combinan dos capas, calculadas según su participación efectiva.

### Capa 1: Tasa escalonada por acuerdo

Cada acuerdo se calcula de forma independiente utilizando una tasa progresiva según el tamaño del acuerdo. Las tasas se **negocian individualmente** en función del tipo de socio, su posicionamiento en el mercado y la escala del compromiso — no se publican tasas estándar.

### Capa 2: Ponderación por etapa de contribución

Su participación en cada acuerdo se pondera según las etapas que efectivamente asuma. Las cuatro etapas suman el 100%. Las etapas que Zeabur gestiona internamente no se pagan.

| Etapa | Definición | Criterio de atribución |
| --- | --- | --- |
| Lead (Presentación) | Encontrar al cliente | Marca de tiempo del momento en que facilitó el primer contacto entre ambas partes (incluyendo la creación de un grupo de comunicación dedicado) |
| Close (Contrato) | Cerrar el acuerdo | Cotización firmada por ambas partes |
| Deliver (Servicio) | Entrega del servicio | Usted gestiona todas las consultas del cliente y el soporte operativo sin requerir al equipo de ingeniería de Zeabur |
| Collect (Cobro) | Cobro de pagos | Usted completa el cobro y la facturación sin que Zeabur tenga que reclamar |

### Patrones de participación comunes

| Escenario | Lead | Close | Deliver | Collect |
| --- | :---: | :---: | :---: | :---: |
| Servicio completo | ✓ | ✓ | ✓ | ✓ |
| Solo presentación | ✓ |   |   |   |
| Presentación + Servicio | ✓ |   | ✓ |   |
| Servicio + Cobro |   |   | ✓ | ✓ |
| Solo operación (sin Lead) |   | ✓ | ✓ | ✓ |
| Presentación + Servicio + Cobro | ✓ |   | ✓ | ✓ |

### División entre múltiples partes

Distintas etapas de un mismo acuerdo pueden pertenecer a diferentes socios. Cada acuerdo se calcula de forma aislada (sin fondo común) y cada parte liquida de manera independiente sin afectar a las demás.

## Reglas complementarias

- **Moneda de liquidación** — Las comisiones se liquidan en USD. Los costos de cambio de divisa corren por cuenta del Socio de Ventas.
- **Ciclo de liquidación** — Las comisiones se calculan mensualmente y se publican en el Partner Portal tras confirmación mutua.
- **Período de retención** — Se aplica un período de retención antes de que las comisiones puedan retirarse. La pérdida del cliente durante el período de retención resulta en una deducción automática; después del período de retención, los montos ya retirados no se recuperan. La duración específica se establece por contrato.
- **Compromiso mínimo de ventas** — Se aplica un compromiso periódico mínimo. El incumplimiento durante períodos consecutivos permite a Zeabur ajustar las tasas o rescindir el contrato. El monto específico se establece por contrato.
- **Atribución del Lead** — La atribución del Lead requiere la confirmación de Zeabur antes de quedar fijada.
- **Bloqueo de etapas** — La titularidad de cada etapa se fija al registrar el acuerdo; los cambios requieren consentimiento por escrito de ambas partes.
- **Continuidad de comisiones** — Las comisiones continúan mientras el cliente siga pagando; cesan cuando el cliente abandona.

<Callout type="warning">
El programa de Socio de Ventas y el Programa de Referidos no pueden aplicarse a la misma cuenta simultáneamente, para evitar conflictos de doble beneficio.
</Callout>

## Proceso de asociación

1. **Contactar a Zeabur** — Solicite en [partner.zeabur.com](https://partner.zeabur.com), o póngase en contacto a través del [foro de la comunidad de Zeabur](https://zeabur.com/forum), describiendo el perfil de su cliente y la escala estimada del compromiso.
2. **Negociar el contrato** — El equipo de Zeabur evalúa y negocia con usted las tasas específicas, la ponderación de etapas y los términos complementarios. Los contratos individuales son confidenciales.
3. **Firmar el contrato** — Una vez confirmados los términos, firme el contrato e inicie el seguimiento del acuerdo.
4. **Incorporación del cliente** — Ayude al cliente a evaluar, probar y poner en producción.
5. **Liquidación mensual** — Una vez que el pago del cliente sea exitoso, las comisiones mensuales se liquidan según el contrato.

## Anexos relacionados

Algunas asociaciones implican contratos o anexos separados, como:

- **Tarifas por servicio de cobro** — Si actúa como cobrador de pagos, las tarifas de servicio asociadas se rigen por un contrato aparte.
- **Subsidios educativos** — Las asociaciones que involucran escuelas o instituciones educativas se gestionan mediante un anexo/MOU aparte.

## Cómo solicitar

- Regístrese en [partner.zeabur.com](https://partner.zeabur.com) y envíe una solicitud con la información de su empresa, el perfil de su cliente y la escala estimada del compromiso.
- También puede manifestar su interés a través del [foro de la comunidad de Zeabur](https://zeabur.com/forum).
