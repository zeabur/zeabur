# Inicio Rápido

Esta guía le ayudará a habilitar InsForge en su proyecto de Zeabur y conectar su IDE de IA en minutos.

## Requisitos previos

Antes de comenzar, necesitará:

1. Una cuenta de Zeabur
2. Una tarjeta de crédito registrada en la configuración de facturación de Zeabur
3. Saldo suficiente en la cuenta
4. Un IDE de IA compatible con MCP (ej. Cursor, Claude Code, Windsurf)

## Paso 1: Habilitar InsForge

import { Callout, Steps } from 'nextra/components'

<Steps>

### Abra su proyecto

Visite [Zeabur Dashboard](https://zeabur.com) y navegue al proyecto donde desea habilitar InsForge.

### Navegue a la página de integración de InsForge

En la barra lateral del proyecto, busque y haga clic en "InsForge".

### Habilite InsForge

Haga clic en el botón **Habilitar InsForge**. Aparecerá un diálogo de confirmación — confirme para continuar.

<Callout type="info">
  Habilitar InsForge creará una instancia de backend dedicada para su proyecto. Esto puede tardar un momento.
</Callout>

### Anote sus credenciales de API

Una vez habilitado InsForge, verá:
- **API Endpoint** — La URL base de su instancia de InsForge
- **API Key** — Su clave de autenticación para acceso a la API

</Steps>

## Paso 2: Conecte su IDE de IA

Después de habilitar InsForge, haga clic en **Usar en IDE de IA** para abrir el modal de conexión. Consulte [Conectar IDE de IA](/es-ES/integrations/insforge/connect-ai-ide) para instrucciones detalladas.

## Paso 3: Comience a construir

Una vez que su IDE de IA esté conectado via MCP, envíe el siguiente prompt en el chat de su agente de IA:

```
I'm using InsForge as my backend platform, fetch InsForge instruction doc to learn more about InsForge
```

## Gestión de su instancia

### Abrir panel de InsForge

Haga clic en **Abrir panel de InsForge** en la página de integración para acceder a la interfaz completa de gestión.

### Pausar / Restaurar / Eliminar InsForge

- **Pausar** — Detenga temporalmente su instancia para ahorrar costos (los datos se conservan)
- **Restaurar** — Reanude una instancia pausada
- **Eliminar** — Elimine permanentemente la instancia y todos los datos asociados

<Callout type="error">
  Eliminar InsForge borrará permanentemente todos los datos. Esta acción no se puede deshacer.
</Callout>
