# Uso y Facturación

InsForge proporciona seguimiento detallado del uso para monitorear el consumo de recursos y los costos.

## Ver el uso

1. Navegue a la página de integración de InsForge en el panel de Zeabur
2. Desplácese hasta la sección **Resumen de uso**

### Pestaña de resumen

Muestra un resumen del uso para el período seleccionado:

- **Tamaño máximo de base de datos** — Pico de almacenamiento de base de datos
- **Almacenamiento máximo de archivos** — Pico de almacenamiento de archivos
- **Egress total** — Ancho de banda saliente total consumido
- **EC2 Compute** — Total de horas de cómputo utilizadas
- **AI Credits** — Créditos de IA LLM consumidos
- **AI Tokens** — Total de tokens de IA procesados
- **Llamadas MCP** — Total de llamadas del protocolo MCP
- **Solicitudes de email** — Total de solicitudes de API de email
- **Llamadas de funciones** — Total de ejecuciones de funciones serverless

### Pestaña de uso diario

Proporciona un desglose del uso de recursos por día en una tabla paginada.

### Filtros de período

- **1 día** — Últimas 24 horas
- **7 días** — Última semana
- **30 días** — Último mes
- **Máximo** — Todos los datos disponibles

## Facturación

<Callout type="warning">
  InsForge se factura por separado y no está incluido en los créditos gratuitos de Zeabur.
</Callout>

### Precios

| Concepto | Tarifa |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / hora |
| Database Size | $0.125 / GB / mes |
| Storage | $0.021 / GB / mes |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / crédito |

### Información importante

- **Sin límites artificiales** — Sin límites de tasa de API, filas ni usuarios máximos
- **Facturación unificada** — Todos los costos se consolidan en su factura de Zeabur
- **Pago por uso** — Solo paga por los recursos que su backend consume

### Problemas de facturación

Si su instancia de InsForge se pausa por problemas de facturación:

1. Vaya a **Facturación** en su panel de Zeabur
2. Agregue un método de pago o recargue su saldo
3. Regrese a la página de InsForge y haga clic en **Restaurar**

<Callout type="info">
  Los datos se conservan cuando una instancia se pausa por problemas de facturación. Restaure la instancia después de resolver los problemas para reanudar el servicio.
</Callout>
