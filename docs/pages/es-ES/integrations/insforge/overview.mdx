# InsForge

InsForge es una alternativa nativa de IA a Supabase integrada en Zeabur, que ayuda a los desarrolladores que necesitan un backend y base de datos simples y ligeros sin desplegar un servicio completo. Los agentes de IA pueden construir y gestionar aplicaciones full-stack de forma autónoma a través del protocolo MCP.

## Funcionalidades principales

- **Base de datos PostgreSQL** - APIs REST instantáneas sin configuración
- **Autenticación JWT** - Autenticación integrada con soporte de inicio de sesión social
- **Almacenamiento de archivos compatible con S3** - Almacene y sirva archivos con una API familiar
- **Funciones Edge sin servidor** - Despliegue lógica del lado del servidor sin gestionar infraestructura
- **Soporte del protocolo MCP** - Conecte su IDE de IA y permita que los agentes gestionen su backend de forma autónoma

## Cómo funciona

import { Callout } from 'nextra/components'

1. **Habilite InsForge** en el panel de su proyecto Zeabur
2. **Conecte su IDE de IA** (Cursor, Claude Code, Windsurf, Trae, Cline, Roo Code) a través del protocolo MCP
3. **Deje que su agente de IA construya** — el agente puede crear tablas, gestionar autenticación, manejar almacenamiento y más de forma autónoma

<Callout type="info">
  InsForge crea una instancia de backend dedicada para cada proyecto de Zeabur. Cada instancia tiene su propio API Endpoint y API Key.
</Callout>

## Precios

InsForge se factura por separado y **no está incluido en los créditos gratuitos**. Solo paga por los recursos subyacentes que su backend consume realmente:

| Concepto | Tarifa |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / hora |
| Database Size | $0.125 / GB / mes |
| Storage | $0.021 / GB / mes |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / crédito |

- **Sin límites artificiales** — Sin límites de tasa de API, límites de filas ni topes de usuarios
- **Facturación unificada** — Todos los costos se consolidan en su factura existente de Zeabur

<Callout type="warning">
  Antes de habilitar InsForge, asegúrese de tener una tarjeta de crédito registrada y saldo suficiente en la cuenta.
</Callout>
