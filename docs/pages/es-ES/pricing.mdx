# Precios

Zeabur ofrece cuatro niveles de suscripción más una opción Enterprise personalizada:

- **[Free Plan](/pricing/free-plan)** — US$0/mes
- **[Dev Plan](/pricing/dev-plan)** — US$5/mes (prueba gratuita de 14 días)
- **[Pro Plan](/pricing/pro-plan)** — US$19/mes (prueba gratuita de 14 días, recomendado para producción)
- **[Team Plan](/pricing/team-plan)** — US$79/mes para 3 plazas (+US$24/plaza/mes)
- **Enterprise** — Precios personalizados, [contáctenos](https://zeabur.com/forum)
