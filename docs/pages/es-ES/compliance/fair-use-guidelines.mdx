# Directrices de Uso Justo

Todos los planes en Zeabur están sujetos a las siguientes directrices de uso justo. Estas directrices están diseñadas para garantizar que todos los clientes reciban acceso justo e igualitario a los servicios de Zeabur.

## Ejemplos de uso justo

- Sitios estáticos como blogs, documentación y sitios web personales.
- Bases de datos como MySQL, PostgreSQL y MongoDB.
- APIs backend para aplicaciones móviles, aplicaciones web y dispositivos IoT.
- Almacenamiento de objetos para imágenes, videos y otros archivos multimedia.

## Nunca es uso justo

- Servicios de VPN y Proxy.
- Minería de criptomonedas.
- Servicios de scraping y crawling.
- Actividades ilegales.
- Agregadores de torrent.
- Bots y scripts automatizados.

## Directrices de uso

Esperamos que todos nuestros usuarios puedan usar nuestros servicios de manera justa, y nos reservamos el derecho de suspender o terminar cualquier cuenta que viole estas directrices.

- Para usuarios del plan Serverless, dado que no están verificados con un plan de pago, si detectamos que su uso no es justo, terminaremos sus servicios inmediatamente, y si continúan violando las directrices, suspenderemos su cuenta permanentemente.
- Para usuarios del plan Developer y del plan Team, les notificaremos primero por correo electrónico si detectamos que su uso no es justo, y habrá un período de gracia de 3 días para que ajusten su uso.

## Uso comercial

Los **planes Serverless** están restringidos solo para uso no comercial. Todo uso comercial requiere un plan Developer o Team.

El uso comercial se define como cualquier uso con fines comerciales, incluyendo pero no limitándose a:

- Solicitar pago por el acceso a sus servicios.
- Publicitar o promover un producto o servicio.
- Recibir pago por el desarrollo y mantenimiento del servicio.
- Con el propósito principal de marketing de afiliados.

> Las donaciones no se consideran uso comercial.

Si no estás seguro de si tu uso se considera comercial, por favor [contáctanos](mailto:contact@zeabur.com).

## Reporte de abuso

Zeabur trabaja con organizaciones de ciberseguridad para combatir el crimen y el uso malicioso. Te animamos a reportar cualquier comportamiento que viole las Directrices de Uso Justo. Si encuentras algún sitio web o actividad que viole estas directrices, por favor repórtalo inmediatamente.

Por favor consulta nuestras [Directrices de Reporte de Abuso](/legal/abuse-report) para aprender cómo reportar abuso.
