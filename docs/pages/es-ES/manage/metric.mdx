# Uso del Servicio

Zeabur sigue un modelo de facturación por volumen y proporciona un gráfico detallado para mostrar el uso de tu servicio.

Haz clic en tu servicio y selecciona la pestaña **Metrics**, verás los gráficos.

### Uso de Memoria

Muestra la memoria consumida por el servicio en ejecución.

![memory](/manage/metric/Memory_Usage.png)

### Uso de CPU

Muestra el uso de CPU por el servicio en ejecución.

![cpu](/manage/metric/CPU_Usage.png)

### Uso de Red

Muestra el tráfico de red del servicio en ejecución desde cada región de Zeabur.

![network](/manage/metric/Network_Usage.png)

### Uso de Disco

Para despliegues en Google Cloud Platform (GCP), existe un límite de uso de disco de 1 GB. En todas las demás plataformas, el uso de disco funciona con un modelo de pago por uso, lo que permite flexibilidad según tus necesidades de almacenamiento.

![disk](/manage/metric/disk_usage.png)
