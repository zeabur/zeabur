# Presupuesto del Proyecto

Para evitar costos inesperadamente altos, puedes configurar un presupuesto para cada proyecto.

Recibirás una notificación cuando el uso alcance el 80% del presupuesto, y tus servicios se suspenderán para evitar costos excesivos cuando se alcance el presupuesto.

## Cómo Configurar un Presupuesto

1. Ve a la configuración del proyecto haciendo clic en el icono de engranaje en la esquina superior izquierda del dashboard del proyecto.

![Budget Entry](/manage/budget/settings.png)

2. En la página de presupuesto, haz clic en el botón "Update Monthly Budget" para establecer un presupuesto para este proyecto.

![Budget Page](/manage/budget/budget-page.png)

En esta página puedes ver el costo del período actual del proyecto, lo que te ayudará a decidir un presupuesto adecuado.

## Preguntas frecuentes (FAQ)

### ¿Cómo sé si mi proyecto está cerca de alcanzar el presupuesto?

Recibirás una notificación cuando el uso alcance el 80% del presupuesto, y también puedes verificar el uso en la configuración del proyecto.

### ¿Qué sucede cuando se alcanza el presupuesto?

Tus servicios en este proyecto se suspenderán para evitar costos excesivos cuando se alcance el presupuesto.

### ¿Puedo cambiar el presupuesto después de configurarlo?

Sí, puedes cambiar el presupuesto en cualquier momento. El nuevo presupuesto entrará en vigor inmediatamente.

### ¿Cuál es el presupuesto mínimo?

El presupuesto mínimo es $10.00.
