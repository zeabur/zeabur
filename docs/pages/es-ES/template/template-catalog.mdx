# Catálogo de Plantillas

Zeabur ofrece un amplio catálogo de plantillas que te permite desplegar rápidamente diversas aplicaciones y servicios.

Visita el [Catálogo de Plantillas de Zeabur](https://zeabur.com/templates) para explorar todas las plantillas disponibles.

## Cómo utilizar las plantillas

1. Busca la plantilla que necesitas en el catálogo de plantillas
2. Haz clic en el botón "Deploy"
3. Sigue las instrucciones para completar la configuración

También puedes consultar [Formato de Plantilla](/es-ES/template/template-format) para obtener más detalles sobre cómo crear tus propias plantillas.
