# Integraciones

Zeabur proporciona integraciones incorporadas con servicios de terceros, permitiéndole habilitar potentes capacidades de backend directamente en su proyecto — sin necesidad de despliegue separado.

## Integraciones disponibles

### [InsForge](/es-ES/integrations/insforge/overview)

Una alternativa nativa de IA a Supabase que permite a los agentes de IA construir y gestionar aplicaciones full-stack de forma autónoma. Incluye base de datos PostgreSQL, autenticación JWT, almacenamiento de archivos compatible con S3, funciones edge sin servidor y soporte del protocolo MCP para integración con IDEs de IA.

- **Categoría**: Backend / BaaS
- **Facturación**: Pago por uso, facturado por separado
- **Sitio web**: [insforge.dev](https://insforge.dev)
