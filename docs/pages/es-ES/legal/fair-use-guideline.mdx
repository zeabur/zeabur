# Guía de uso justo

Todos los planes de Zeabur están sujetos a la siguiente guía de uso justo. Estas pautas están diseñadas para garantizar que todos los clientes reciban un acceso justo e igualitario a los servicios de Zeabur.

## Ejemplos de uso justo

- Sitios estáticos como blogs, documentación y sitios web personales.
- Bases de datos como MySQL, PostgreSQL y MongoDB.
- APIs de backend para aplicaciones móviles, aplicaciones web y dispositivos IoT.
- Almacenamiento de objetos para imágenes, videos y otros archivos multimedia.

## Usos que nunca se consideran justos

- Servicios VPN y proxy.
- Minería de criptomonedas.
- Servicios de scraping y crawling.
- Actividades ilegales.
- Agregadores de torrents.
- Bots y scripts automatizados.

## Pautas de uso

Esperamos que todos nuestros usuarios utilicen nuestros servicios de forma justa, y nos reservamos el derecho de suspender o cancelar cualquier cuenta que infrinja estas pautas.

- Para usuarios del plan Serverless, como no se verifica mediante un plan de pago, si detectamos un uso no justo, finalizaremos el servicio de inmediato. Si continúas infringiendo las pautas, suspenderemos tu cuenta de forma permanente.
- Para usuarios del plan Developer y del plan Team, si detectamos un uso no justo, primero te notificaremos por correo electrónico y tendrás un período de gracia de 3 días para ajustar tu uso.

## Uso comercial

Los **planes Serverless** están restringidos exclusivamente a uso no comercial. Cualquier uso comercial requiere un plan Developer o Team.

Se considera uso comercial cualquier uso con fines comerciales, incluidos, entre otros:

- Solicitar pagos para acceder a tus servicios.
- Anunciar o promocionar un producto o servicio.
- Cobrar por el desarrollo y el mantenimiento del servicio.
- Tener como objetivo principal el marketing de afiliados.

> Las donaciones no se consideran uso comercial.

Si no estás seguro de si tu uso se considera comercial, por favor [contáctanos](mailto:contact@zeabur.com).

## Reporte de abuso

Zeabur colabora con organizaciones de ciberseguridad para combatir actividades delictivas y usos maliciosos. Te animamos a reportarnos cualquier comportamiento que infrinja esta guía. Si detectas alguna actividad que infrinja estas pautas, repórtalo de inmediato.

Consulta nuestra [Guía para reportar abusos](/legal/abuse-report) para saber cómo reportar abusos.
