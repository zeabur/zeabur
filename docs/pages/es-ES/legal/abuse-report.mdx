# Guía para reportar abusos

Zeabur colabora con organizaciones de ciberseguridad para combatir el crimen y el uso malicioso. Te animamos a reportar cualquier comportamiento que viole las [Directrices de Uso Justo](https://zeabur.com/docs/legal/fair-use-guideline). Si encuentras algún sitio web o actividad que infrinja las [Directrices de Uso Justo](https://zeabur.com/docs/legal/fair-use-guideline), por favor repórtalo inmediatamente.

Si eres una agencia gubernamental o una organización de ciberseguridad, también puedes contactarnos para integrar un sistema de reporte automatizado.

## **Contacto**

Ofrecemos los siguientes métodos de contacto:

- Correo electrónico: [contact@zeabur.com](mailto:contact@zeabur.com)
