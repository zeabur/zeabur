# Empezar según tu Necesidad

¿No sabes por dónde empezar? Elige el camino que mejor se adapte a tu situación.

## 🧪 Solo Quiero Probarlo

Despliega desde una plantilla con un clic — sin necesidad de código.

1. Explora el [Marketplace de Plantillas](https://zeabur.com/templates) y elige algo interesante.
2. Haz clic en "Deploy", selecciona una región y espera unos segundos.
3. Una vez que el servicio esté activo, ve a la pestaña "Domains" para generar una URL gratuita de prueba.

**Lectura recomendada:** [Inicio Rápido](./quick-start) · [Plantillas](/template)

---

## 💻 Tengo un Proyecto para Desplegar

Envía tu código desde GitHub a Zeabur — la mayoría de lenguajes y frameworks son compatibles.

1. En tu proyecto, haz clic en **Add Service** → **GitHub**, autoriza y selecciona tu repositorio.
2. Zeabur detecta automáticamente el lenguaje, framework y estrategia de compilación.
3. Configura variables de entorno en la pestaña "Configuration" si es necesario.
4. Vincula un dominio y publica.

**Lectura recomendada:** [Cómo Funcionan los Despliegues](/deploy/how-deploys-work) · [Integración con GitHub](/deploy/github-integration) · [Variables de Entorno](/deploy/environment-variables)

## 🤖 Quiero Usar Funciones de IA

Zeabur ofrece AI Hub, dándote acceso directo a APIs de varios modelos de lenguaje.

1. Ve a [AI Hub](/ai-hub) para obtener tu endpoint de API y clave.
2. Usa cualquier SDK compatible con OpenAI en tu aplicación para hacer llamadas.
3. Compatible con GPT-4o, Claude, Gemini y muchos más.

**Lectura recomendada:** [Visión General de AI Hub](/ai-hub) · [Modelos Soportados](/ai-hub/model-list)

---

## 📧 Necesito Envío/Recepción de Email

Zeabur Email te permite enviar y recibir correos usando tu propio dominio.

1. Ve a [Zeabur Email](/email) para crear una cuenta y verificar tu dominio.
2. Configura los registros DNS (MX, SPF, DKIM).
3. Envía y recibe correos vía SMTP o REST API.

**Lectura recomendada:** [Servicio de Email](/email) · [Claves API de Email](/email/email-api-keys) · [REST API de Email](/email/email-rest-api)

---

## 🖥️ Tengo Mi Propio Servidor

Trae tu propia máquina (BYO Server) y gestiona los despliegues a través de la interfaz de Zeabur.

1. [Registra un servidor](/server) en el Dashboard.
2. Ejecuta el comando de instalación que Zeabur proporciona en tu servidor.
3. Selecciona tu servidor al crear un nuevo proyecto.

**Lectura recomendada:** [Servidor](/server)

---

## 🔌 Quiero Automatizar vía API / CLI

Integra Zeabur en tu pipeline de CI/CD.

- **CLI**: Ejecuta `npx zeabur@latest deploy` para desplegar desde tu terminal.
- **GraphQL API**: Controla proyectos, servicios y despliegues programáticamente.
- **Upload API**: Sube directamente una aplicación empaquetada en ZIP.

**Lectura recomendada:** [Despliegue con CLI](/developer/cli) · [Public API](/developer/public-api) · [Claves API](/developer/api-keys)
