# Visión General de la Plataforma

**Zeabur** es una plataforma de despliegue en la nube impulsada por IA que pone tus aplicaciones en línea en minutos — sin gestión de servidores, configuración de CI/CD ni complicaciones de infraestructura.

## Capacidades Principales

### Despliegue Automatizado

- **Detección automática de lenguajes y frameworks** — Sube tu código y Zeabur identifica automáticamente Node.js, Python, Go, Java, .NET, Rust y más, eligiendo la estrategia de compilación óptima.
- **CI/CD listo para usar** — Cada `git push` activa automáticamente una compilación y despliegue, sin configuración.
- **Marketplace de Plantillas** — Cientos de plantillas preconfiguradas para desplegar WordPress, n8n, PostgreSQL y otros servicios populares con un clic.

### Asistente IA

Zeabur tiene un Asistente IA (Agent) integrado que te permite describir lo que necesitas en lenguaje natural. Automáticamente crea proyectos, despliega servicios, configura variables de entorno y vincula dominios.

### AI Hub

A través de [AI Hub](/ai-hub), puedes llamar directamente a las APIs de GPT-4o, Claude, Gemini y otros LLMs líderes usando un formato compatible con OpenAI — sin necesidad de registrarte en cada proveedor por separado.

### Integración Todo-en-Uno

Despliega tu frontend, backend, bases de datos, colas de mensajes y almacenamiento de objetos en un solo proyecto. Los servicios pueden comunicarse vía red privada y las variables de entorno pueden referenciarse entre sí.

### Aislamiento Multi-Entorno

Cada proyecto soporta múltiples entornos (ej. Producción, Staging), completamente aislados entre sí para flujos de trabajo de desarrollo y pruebas optimizados.

### Infraestructura Flexible

- **Clúster Compartido** (en proceso de eliminación) — Ideal para empezar rápido, sin gestión de servidores. Ya no se pueden crear nuevos proyectos en clústeres compartidos.
- **Servidor** — Trae tu propia máquina (BYO Server) con control total. Ver [Servidor](/dedicated-server).
