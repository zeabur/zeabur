# Obtener Ayuda

A veces puedes encontrar problemas al usar Zeabur. Puedes contactar con [Soporte de Zeabur](https://zeabur.com/support) para recibir ayuda del equipo.

![Página de Soporte de Zeabur](/community/help/support-home.png)

## Contactar Soporte

Abre [Soporte de Zeabur](https://zeabur.com/support) y envía tu solicitud con suficiente contexto para que nuestro equipo pueda investigarla.

![Formulario para enviar un ticket](/community/help/support-submit-ticket.png)

### Formato de solicitud

Seguir este formato nos ayuda a identificar y resolver tu problema más rápido.

- Describe brevemente tu problema en el título.
- Describe los detalles del problema, el comportamiento esperado y lo que ya intentaste.
- Si el problema está relacionado con un proyecto o servicio específico, incluye tu `projectID`, `serviceID` o `deploymentID`. Puedes encontrarlos en la URL de tu Dashboard de Zeabur.

## Mejora tu Plan

Si necesitas tiempos de respuesta más rápidos y soporte dedicado, considera mejorar tu plan. Los planes de nivel superior incluyen tiempos de respuesta garantizados más cortos. Consulta la página de [Planes](../billing/plans) para más detalles.

## Correo Electrónico

También puedes contactarnos por correo electrónico para obtener soporte o realizar consultas:

- **Correo electrónico:** [contact@zeabur.com](mailto:contact@zeabur.com)

Para obtener tiempos de respuesta más rápidos, recomendamos usar [Soporte de Zeabur](https://zeabur.com/support). Sin embargo, si prefieres la comunicación por correo electrónico, no dudes en enviarnos un mensaje y te responderemos lo antes posible.
