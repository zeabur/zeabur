# Recompensas y Referidos

Zeabur ofrece múltiples programas de recompensas para que puedas obtener beneficios mientras usas y promueves la plataforma.

- [Programa de Referidos](/rewards/referral) — Comparte tu código de referido para dar descuentos a amigos y ganar comisiones
- [Patrocinio de Código Abierto](/rewards/sponsor) — Solicita el patrocinio de Zeabur para tu proyecto de código abierto
- [Canjear Tarjeta Prepago](/rewards/redeem-card) — Canjea una tarjeta prepago a tu saldo de cuenta
