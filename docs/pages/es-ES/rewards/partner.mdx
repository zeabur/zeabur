# Programa de Socios

El Zeabur Partner Program ofrece oportunidades de colaboración para instructores, organizadores de eventos y consultores de ventas para ganar recompensas promoviendo Zeabur.

## Dos Modelos de Colaboración

### Promoción Educativa

Para **instructores, instituciones de capacitación y organizadores de eventos**. Promueve Zeabur a través de cursos y eventos técnicos usando Event Codes que dan a los asistentes acceso gratuito de prueba, y gana recompensas basadas en usuarios de pago retenidos.

- Recibe un Event Code dedicado para tu evento
- Un mes después, gana US$5 por cada usuario de pago retenido
- Ideal para talleres, cursos en línea y hackathons

[Más información →](/partner/education)

### Sales Partner

Para **consultores empresariales, integradores de sistemas y empresas de consultoría tecnológica**. Ayuda a clientes empresariales a adoptar Zeabur y gana una comisión recurrente del 15% mientras el cliente continúe pagando.

- Condición: Contrato individual > US$1,000 o compra institucional
- Modelo de comisión: 15% recurring commission sobre los pagos mensuales del cliente
- Ideal para incorporación empresarial y adquisición institucional

[Más información →](/partner/sales-partner)

## Diferencias con el Programa de Referidos

Zeabur también ofrece un [Programa de Referidos](/rewards/referral) que permite a cualquier usuario ganar recompensas a través de códigos de referencia. Diferencias clave:

| Aspecto | Programa de Referidos | Programa de Socios |
| --- | --- | --- |
| Quién | Cualquier usuario | Instructores / Organizadores / Consultores |
| Cómo | Flujo automático de código de referencia | Revisión manual y contrato |
| Recompensas | Comisiones por referencia (20%–100% según nivel) | Recompensas por retención de Event Code / 15% recurring commission |
| Umbral | Ninguno | Sales Partner: transacción individual > US$1,000 |

<Callout type="info">
Una sola transacción solo puede ir por un camino: código de referencia o programa de socios. No se acumulan.
</Callout>
