# Presupuesto del Proyecto

Para evitar costos inesperadamente altos, puedes configurar un presupuesto para cada proyecto.

Recibirás una notificación cuando el uso alcance el 80% del presupuesto, y tus servicios serán suspendidos para evitar costos excesivos cuando se alcance el presupuesto.

## Cómo Configurar un Presupuesto

1. Ve a la configuración del proyecto haciendo clic en el icono de engranaje en la esquina superior izquierda del panel del proyecto.

![Entrada de Presupuesto](/manage/budget/budget-entry.png)

2. En la página de presupuesto, haz clic en el botón "Actualizar Presupuesto Mensual" para establecer un presupuesto para este proyecto.

![Página de Presupuesto](/manage/budget/budget-page.png)

Puedes ver el costo del período actual del proyecto en esta página, para que puedas decidir un presupuesto adecuado para el proyecto.

## Preguntas Frecuentes

### ¿Cómo sé si mi proyecto está cerca de alcanzar el presupuesto?

Recibirás una notificación cuando el uso alcance el 80% del presupuesto, y también puedes verificar el uso en la configuración del proyecto.

### ¿Qué sucede cuando se alcanza el presupuesto?

Tus servicios en este proyecto serán suspendidos para evitar costos excesivos cuando se alcance el presupuesto.

### ¿Puedo cambiar el presupuesto después de establecerlo?

Sí, puedes cambiar el presupuesto en cualquier momento. El nuevo presupuesto entrará en vigor inmediatamente.

### ¿Cuál es el presupuesto mínimo?

El presupuesto mínimo es de $10.00.
