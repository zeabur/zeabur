# Uso del Servicio

Zeabur sigue un modelo de facturación basado en volumen y también proporciona un gráfico detallado para mostrar el uso de tu servicio.

Haz clic en tu servicio y elige la pestaña **Métricas**, donde verás los gráficos.

### Uso de Memoria

Muestra la memoria consumida por el servicio en ejecución.

![memoria](/manage/metric/memory.png)

### Uso de CPU

Muestra el CPU utilizado por el servicio en ejecución.

![cpu](/manage/metric/CPU.png)

### Uso de Red

Muestra el tráfico de red del servicio en ejecución desde cada región en Zeabur.

![red](/manage/metric/network.png)
