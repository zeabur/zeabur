# **Desplegar un servicio con Zeabur CLI**

Desplegar servicios con un solo clic en el panel es sencillo, pero si prefieres desplegar servicios desde tu terminal con un único comando, también lo tenemos cubierto.

## Instalar Zeabur CLI

Zeabur CLI es una herramienta de línea de comandos que te permite desplegar servicios directamente desde tu terminal.

```bash
# HomeBrew (Linux y macOS)
brew install zeabur/tap/cli

# Mediante script (Linux y macOS)
curl -sSL https://dub.sh/zeabur | bash

# Usuarios de Windows
iwr -useb https://dub.sh/zb-win | iex
```

También puedes descargar el archivo binario desde la [página de lanzamientos en GitHub](https://github.com/zeabur/cli/releases).

Puedes verificar que la CLI se ha instalado correctamente ejecutando el comando `zeabur --help`.

```bash
zeabur --help
```

![CLI help](/deploy/cli/help.png)

## Iniciar sesión en tu cuenta

Una vez completada la instalación, debes iniciar sesión en tu cuenta para poder desplegar servicios.

```bash
zeabur auth login
```

Pulsa `Enter` para abrir la página de inicio de sesión en tu navegador, haz clic en el botón `Confirmar`, y serás redirigido de nuevo al terminal donde verás el mensaje de inicio de sesión correcto.

## Desplegar un servicio

Ahora ya estás listo para desplegar tu servicio. Simplemente ejecuta el siguiente comando dentro del directorio de tu servicio.

```bash
zeabur
```

A continuación, sigue las instrucciones para seleccionar o crear un proyecto donde se desplegará tu servicio.

Después, verás el progreso del despliegue en tu terminal. Espera unos segundos y tu servicio se desplegará automáticamente; obtendrás la URL del servicio directamente en el terminal.

![Deploy in CLI](/deploy/cli/deploy.png)
```
