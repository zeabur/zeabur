# **Crear Proyecto**

Para desplegar servicios en Zeabur, primero necesitas crear un proyecto.

Puedes seguir los pasos a continuación para crear un proyecto.

## **¿Qué es un proyecto?**

Un proyecto es una colección de servicios que se despliegan juntos; los servicios en el mismo proyecto pueden comunicarse entre sí a través de la red interna. Puedes crear múltiples proyectos en diferentes regiones y desplegar diversos tipos de servicios en cada proyecto.

Además, puedes [establecer el presupuesto y gestionar los usos](https://zeabur.com/docs/es-ES/manage/budget) del proyecto, [invitar a otros miembros del equipo](https://zeabur.com/docs/es-ES/manage/invite) para gestionar los servicios, y más.

## **Cómo crear un proyecto**

Primero, debes navegar a la [página de proyectos del dashboard](https://zeabur.com/projects).

Si eres nuevo en Zeabur y no tienes ningún proyecto, verás una página de onboarding para crear tu primer proyecto.

![Crear proyecto onboarding](/deploy/create-project/Deployment_Services_Create_Project.png)

Si ya has creado proyectos anteriormente, simplemente haz clic en el botón **Crear Proyecto** para navegar a la página **Crear Nuevo Proyecto**, seleccionar el **Tipo de Despliegue**, y elegir entre **servidor dedicado** o **cluster compartido**.

![Elegir tipo de despliegue](/deploy/create-project/choose_Deployment_Type_2.png)

Luego selecciona la región según tu plan de suscripción y la región deseada, y el proyecto se creará automáticamente con nombres como untitled-0, untitled-1, etc.

![Seleccionar una región](/deploy/create-project/select_a_region_2.png)

## **¿Qué sigue?**

Verás un proyecto vacío después de seleccionar la región:

![Proyecto vacío](/deploy/create-project/Deployment_Services_empty_project.png)

Puedes hacer clic en el botón **Desplegar Nuevo Servicio** para [desplegar un servicio](https://zeabur.com/docs/es-ES/deploy/create-service) en este proyecto, o hacer clic en el botón de configuración para gestionar los ajustes del proyecto e [invitar a otros miembros del equipo](https://zeabur.com/docs/es-ES/manage/invite).
