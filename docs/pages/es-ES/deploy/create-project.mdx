# Crear Proyecto

Para desplegar servicios en Zeabur, primero necesitas crear un proyecto.

## Pasos

1. Inicia sesión en el [Panel de Zeabur](https://dash.zeabur.com).
2. Haz clic en el botón **Crear Proyecto**.
3. Selecciona la región donde deseas desplegar tu proyecto.

<Callout>
  Elige la región más cercana a tus usuarios para obtener la menor latencia.
</Callout>

Una vez creado el proyecto, serás redirigido a la página del proyecto donde puedes comenzar a agregar servicios.
