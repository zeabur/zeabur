# Usar el complemento de Chrome para desplegar servicios

Zeabur ofrece varios complementos que te permiten desplegar tu código local con un solo clic, sin salir de tu entorno de desarrollo.

Este artículo explica cómo desplegar servicios usando el complemento de Zeabur para Chrome.

## Complemento de Chrome

El complemento de Zeabur para Chrome te permite desplegar código directamente desde plataformas de generación de código con IA compatibles, como Lovable, Bolt.new o v0.dev.

### Instalación del complemento

Ve a la [Chrome Web Store](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem) y haz clic en “Añadir a Chrome”.

### Cómo usarlo

Cuando escribas código en una plataforma de IA compatible, aparecerá un botón “Deploy with Zeabur” en la pantalla. Haz clic para iniciar el proceso de despliegue.
