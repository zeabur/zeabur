# Implementar servicios con extensiones

Zeabur ofrece múltiples extensiones que te permiten implementar tu código local con un solo clic, sin salir de tu entorno de desarrollo.

Este artículo te mostrará cómo implementar tus servicios utilizando las extensiones de Zeabur para VS Code, Cursor, Chrome y Raycast.

## Extensión para VS Code / Cursor

Puedes utilizar la extensión de Zeabur tanto en Visual Studio Code como en el editor Cursor para implementar directamente tus proyectos locales en Zeabur.

### Instalación

1. Abre VS Code o Cursor.
2. Ve al panel de extensiones y busca "Zeabur".
3. Haz clic en "Instalar", o visita directamente el [Marketplace de Visual Studio](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) para instalarla.

### Cómo usarla

1. Abre la carpeta del proyecto que deseas implementar.
2. Haz clic en el icono de Zeabur en la barra lateral.
3. Haz clic en "Deploy" (Implementar); se abrirá automáticamente una ventana del navegador.
4. Elige una región de implementación.
5. En unos minutos, tu proyecto estará implementado.

## Extensión para Chrome

La extensión de Zeabur para Chrome te permite implementar código directamente desde plataformas compatibles de generación de código con IA como Lovable, Bolt.new o v0.dev.

### Instalación

Dirígete a la [Chrome Web Store](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem) y haz clic en "Añadir a Chrome".

### Cómo usarla

Cuando trabajes en una plataforma compatible, aparecerá un botón llamado "Deploy with Zeabur" (Implementar con Zeabur) en pantalla. Haz clic para comenzar el proceso de implementación.

## Extensión para Raycast

Con la extensión de Zeabur para Raycast, puedes implementar proyectos rápidamente, buscar documentación o explorar plantillas.

### Instalación

Abre Raycast Store, busca la extensión de Zeabur e instálala. También puedes visitar directamente la [página de la extensión en Raycast](https://www.raycast.com/ridemountainpig/zeabur) y hacer clic en "Install Extension" (Instalar extensión).

### Cómo usarla

1. Una vez instalada, abre Raycast y escribe `Zeabur`.
2. Selecciona el comando "Deploy Project" (Implementar proyecto).
3. Elige la carpeta del proyecto que deseas implementar y presiona `⌘ + Enter`.
4. Selecciona la región de implementación. En unos minutos, el proyecto estará disponible.
