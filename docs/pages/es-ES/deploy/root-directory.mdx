# **Directorio Raíz Personalizado**

Para algunos repositorios Git, como proyectos que incluyen tanto frontend como backend, es necesario especificar manualmente el directorio raíz del proyecto.

### **Paso 1: Desplegar el servicio**

Sigue la guía de [Desplegar el servicio](https://zeabur.com/docs/es-ES/get-started) para desplegar tu aplicación.

## **Paso 2: Cambiar el directorio raíz del proyecto**

Haz clic en el servicio desplegado y entra en la pestaña de ajustes. Desplázate hacia abajo hasta la sección **Root Directory**.

![Root Directory](/deploy/root-directory/root_directory.png)

Configura aquí el directorio raíz de tu proyecto.
