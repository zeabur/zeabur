# Botón de Despliegue

Con nuestro botón de despliegue, es fácil desplegar servicios en Zeabur con un solo clic. El botón se puede incrustar en cualquier archivo markdown o HTML. Puede transformar cualquier [plantilla de Zeabur](https://zeabur.com/templates) en un proyecto desplegable.

![button-demo](/deploy/button/button-demo.png)

## ¿Qué es un Botón de Despliegue?

El botón de despliegue es un simple fragmento de código HTML. Puedes copiar el fragmento y pegarlo en cualquier archivo markdown o HTML. Cada botón redirigirá a los usuarios a una plantilla deseada. La plantilla puede ser un servicio individual o un grupo de servicios definidos por el autor de la plantilla.

Los botones también pueden incluir el código de referido del autor. Cada nuevo usuario que se registre con el código de referido y actualice a un plan de pago, el autor recibirá US$ 5 en créditos.

## Copiar el Botón

Como el botón está conectado a una plantilla, solo el autor de la plantilla puede generar el botón.

Para copiar el botón, ve a la página de cuenta en el [Panel de Zeabur](https://dash.zeabur.com/account), selecciona la pestaña **Template** y verás una lista de plantillas que has creado.

![created-templates](/deploy/button/created-templates.png)

Elige la plantilla deseada y haz clic en el botón de compartir. Aparecerá un modal con el código del botón.

![popup](/deploy/button/popup.png)

Si seleccionaste habilitar el código de referido, el código del botón se añadirá con el código de referido, que es tu nombre de usuario en Zeabur.

## Casos de Uso

Es la mejor manera de compartir tu maravilloso proyecto desplegado en Zeabur con el mundo. Cualquiera puede desplegar fácilmente tu proyecto con un solo clic.

Por ejemplo, puedes pegar el botón en estos lugares:

- Archivos README de GitHub
- Blogs
- Documentación y tutoriales
- Perfiles en redes sociales

## Requisitos

Primero necesitas crear una plantilla para obtener un botón de despliegue. Puedes crear una plantilla en el [Panel de Zeabur](https://dash.zeabur.com/create-template).

Como se muestra en la imagen a continuación, necesitas completar el nombre de la plantilla, la descripción y seleccionar los servicios que deseas incluir en la plantilla. Es fácil crear una plantilla, como llenar un formulario.

![create-template](/deploy/button/create-template.png)

## Ganancias

Cuando alguien usa tu código de referido al comprar un Servidor o recargar AI Hub, ganas una comisión escalonada (20%–100%) según tu nivel de referidos. Las comisiones se depositan en tu billetera y pueden convertirse en créditos de Zeabur para pagar tus servicios.

Puedes leer una explicación más detallada en nuestro [documento de referidos](../community/referral).
