# Botón de Despliegue

Con nuestro botón de despliegue, es fácil desplegar servicios en Zeabur con un solo clic. El botón se puede usar en cualquier archivo markdown o html. Puede transformar cualquier [plantilla de Zeabur](https://zeabur.com/templates) en un proyecto desplegable.

![button-demo](/deploy/button/button-demo.png)

## ¿Qué es un Botón de Despliegue?

El botón de despliegue es un simple fragmento de código HTML. Puedes copiar el fragmento y pegarlo en cualquier archivo markdown o html. Cada botón redirigirá a los usuarios a una plantilla deseada. La plantilla puede ser un solo servicio o un grupo de servicios, decidido por el autor de la plantilla.

Los botones también pueden incluir el código de referido del autor; cada nuevo usuario que se registre con el código de referido y actualice a un plan de pago, el autor recibirá 5 US$ en créditos.

## Copiar el Botón

Dado que el botón está conectado a una plantilla, solo el autor de la plantilla puede generar el botón.

Para copiar el botón, ve a la página de cuenta en el [Dashboard de Zeabur](https://zeabur.com/account), selecciona la pestaña **Template**, y verás una lista de plantillas que has creado.

![created-templates](/deploy/button/created-template.png)

Elige la plantilla deseada y haz clic en el botón de compartir. Aparecerá un modal con el código del botón.

![popup](/deploy/button/popup.png)

Si seleccionaste habilitar el código de referido, el código del botón se añadirá con el código de referido, que es tu nombre de usuario en Zeabur.

## Casos de Uso

Es la mejor forma de compartir tu maravilloso proyecto desplegado en Zeabur con el mundo. Cualquiera puede desplegar tu proyecto fácilmente con un solo clic.

Por ejemplo, puedes pegar el botón en estos lugares:

- Archivos README de GitHub
- Blogs
- Documentaciones y tutoriales
- Perfiles sociales

## Requisitos

Necesitas crear una plantilla primero para obtener un botón de despliegue. Puedes crear una plantilla con esta [instrucción](https://zeabur.com/docs/es-ES/template/template-in-code).

## Ganancias

Como se mencionó antes, puedes ganar 5 US$ en créditos por cada nuevo usuario que se registre con tu código de referido y actualice a un plan de pago. Puedes usar los créditos para pagar tus propios servicios.

Puedes leer una explicación más detallada en nuestro [documento de referidos](https://zeabur.com/docs/es-ES/billing/referral).
