# Usar el complemento de VS Code para desplegar servicios

Zeabur ofrece varios complementos que te permiten desplegar tu código local con un solo clic, sin salir de tu entorno de desarrollo.

Este artículo explica cómo desplegar servicios usando el complemento de Zeabur para VS Code.

## Complemento para VS Code / Cursor

Puedes utilizar el complemento de Zeabur tanto en Visual Studio Code como en el editor Cursor para desplegar directamente proyectos locales en Zeabur.

### Instalación del complemento

1. Abre Visual Studio Code o el editor Cursor.
2. Ve al panel de Extensiones y busca “Zeabur”.
3. Haz clic en “Instalar” o dirígete directamente a [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) para instalarlo.

### Cómo usarlo

1. Abre la carpeta del proyecto que deseas desplegar.
2. Haz clic en el icono del complemento de Zeabur en la barra lateral.
3. Haz clic en “Deploy” y se abrirá automáticamente una página del navegador.
4. Selecciona la región de despliegue.
5. En unos minutos, el despliegue estará completo.
