# Zeabur Mesh

<Callout type="info">

Zeabur Mesh es una solución de comunicación entre servicios y gestión de tráfico que permite conectividad y enrutamiento seguro y confiable entre tus servicios en Zeabur.

La documentación detallada estará disponible cuando se lance la función. Si tienes alguna pregunta, no dudes en contactarnos en el [foro de la comunidad](https://zeabur.com/forum).
