# **快速入门**

## **第一步：创建账户**

您需要创建一个账户才能开始在 Zeabur 上部署服务。

如果是 Zeabur 新用户，请使用邮箱注册（第一个选项）。在此输入您的邮箱地址，然后点击“使用邮箱继续”。

![SignUp.png](/start/get-started/SignUp.png)

随后会收到提示“邮件发送成功”。

![SignUp Part 2.png](/start/get-started/SignUp%20Part%202.png)

打开您的邮箱收件箱（例如 Gmail、Outlook、Apple Mail），您会看到来自 Zeabur 的邮件。点击邮件中的“登录”按钮。

![Sign Up Callback.png](/start/get-started/Sign%20Up%20Callback.png)

如果登录过程中打开的浏览器不是您偏好的默认浏览器（例如打开了 Safari 而不是 Chrome），您可以手动复制“登录”按钮下方的链接，并粘贴到您想要的浏览器中。

出现此登录界面即表示您已成功登录！

![Dashboard.png](/start/get-started/Dashboard.png)

请记住，您可以在左侧面板切换界面语言！我们目前支持繁体中文、简体中文、英语、日语、西班牙语和印尼语。

![Languages.png](/start/get-started/Languages.png)

## **第二步：创建项目**

在 Zeabur 中，项目是一个单一容器，用于存放构成真实世界应用的所有相关服务。例如，如果您的应用包含前端网站、后端 API 和数据库，它们都应放在同一个项目中。

![Project structure](/start/project-structure.webp)

要开始创建项目，请点击中央的**创建项目**按钮。

![Create New Project.png](/start/get-started/Create%20New%20Project.png)

如果是第一次创建项目，您可能会看到“账户验证”提示。

![Toast if not verified.png](/start/get-started/Toast%20if%20not%20verified.png)

您可以选择：

1. 使用**手机号**验证。大多数国家和地区都支持此方式。如果您的国家/地区不支持，或需要绑定多个 Zeabur 账户，除[联系我们](https://zeabur.com/forum)外，还可参考第二和第三个选项。
2. 预存一些**使用额度**。只要您未订阅 Dev 计划、未购买专用服务器，且未涉及严重滥用（例如违反 ToS 发起高流量 DoS 攻击），我们不会扣除余额。
3. 我们需要**信用卡**来验证账户，以确保操作顺畅。请注意，除非您主动发起订阅相关交易，否则不会产生任何扣款。

选择您偏好的方式，然后点击对应部分的“验证”按钮。

验证完成后，您可以重新创建项目，并看到成功创建的项目。

![After Verification.png](/start/get-started/After%20Verification.png)

## **第三步：部署服务**

Zeabur 支持部署多种服务，包括数据库（PostgreSQL、MySQL、Redis）、开源项目（n8n、WordPress）或您自己的自定义应用（从 GitHub 导入或从文件夹上传）。

- 若要部署数据库，点击“数据库”直接部署常见数据库。
    
    ![Databases.png](/start/get-started/Databases.png)
    
- 若要部署开源项目，Zeabur 的“模板”提供了许多预配置的常见开源项目，选择“模板”即可一键部署。
    
    ![templates.png](/start/get-started/templates.png)
    
- 若要部署自己的自定义应用，可点击“GitHub”或“本地项目”上传代码，或点击“Cursor”直接部署当前项目。
- 若您想部署的开源项目不在“模板”中，也可使用“Docker 镜像”自行设置（属于高级操作）。

以 GitHub 部署为例。首先将 Zeabur 账户绑定到 GitHub，等待约 1 分钟，然后搜索您想部署的仓库：

![Github.png](/start/get-started/Github.png)

点击仓库后，Zeabur 代理会自动分析您的代码并为您选择最适合的部署方式。

![Build Plan.png](/start/get-started/Build%20Plan.png)

点击“部署”即可一键部署项目。如有需要，可前往“配置”调整环境变量。

在配置中，将根目录设置为您的应用文件夹（例如“/”或子文件夹），通过键/值或编辑原始变量添加所需环境变量，然后点击下一步确认。

![Configure Build Plan  .png](/start/get-started/Configure%20Build%20Plan%20%20.png)

## **使用 Zeabur AI 代理部署任意服务**

Zeabur 的 AI 代理允许您使用自然语言快速部署所需服务。

![Postgresql Examples.png](/start/get-started/Postgresql%20Examples.png)

点击右侧面板中的“Zeabur AI”进入 AI 助手界面。如果遇到任何部署问题，也可以向它求助！

## **第四步：启动您的网站**

要让您的网站可供他人访问，可在“域名”中绑定自己的域名：

![Bind domain](/start/bind-domain.webp)

等待约 30 秒后即可点击域名。Zeabur 会为您处理 HTTPS 等设置。全部确认后，即可分享您的网站。

如需绑定自己的域名，可查看[域名绑定](https://zeabur.com/docs/deploy/domain-binding)文档获取更多信息。

## **下一步做什么？**

若希望使用 Google 或 GitHub 账户登录，可在“设置” > “集成”中绑定 GitHub 和 Google 账户。

![Account integration](/start/account-integration.webp)

接下来，您可以继续探索 Zeabur 丰富的模板库以及各种功能（例如文件管理、日志）。如遇技术问题，可在 [Zeabur 官方论坛](https://zeabur.com/forum) 提交开发者工单。如需社区讨论和互助，请加入我们的开发者社区 [Zeabur Discord 服务器](https://zeabur.com/dc)。
