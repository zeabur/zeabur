# 退款政策

## 订阅方案退订

针对 **开发者方案（Dev Plan）**、**专业方案（Pro Plan）** 或 **团队方案（Team Plan）**：

- 取消订阅后，您仍可继续使用目前方案的功能直到当期账单周期结束。周期结束后，账号将自动切换至免费方案。

### 如何退订

<Steps>

### 前往 Dashboard 账单页面

打开 [Dashboard 账单页面](https://dash.zeabur.com/account/billing)。

### 找到「**取消订阅**」按钮

下滑至订阅区块。

### 确认退订

检查确认提示并完成退订。

</Steps>

![取消订阅](/billing/subscription/unsubscribe.png)

<Callout type="warning">
  降级到 Free Plan
  后，你目前部署的容器化服务将会被系统自动暂停。请在取消前确认已备份重要数据。
</Callout>

## 服务器退款

- **首次退款**：可在 Server 页面自助取消，无条件退款至 **Zeabur Credits**。
- **第二次及后续退款**：需到 [Zeabur Forum](https://zeabur.com/forum) 提交退款申请，由客服审核处理。

### 退款注意事项

- 为避免在下一个计费周期产生费用，可提前关闭自动续约设置。

![关闭自动续约](/billing/pricing/auto-renewal.png)

服务器退款的到账时间与手续费取决于退款去向：

| 退款方式           | 处理时间        | 手续费                                |
| ------------------ | --------------- | ------------------------------------- |
| **Zeabur Credits** | 核准后立即到账  | 无                                    |
| **原付款方式退回** | 最多 7 个工作天 | 约退款金额的 30%（Stripe 交易手续费） |

<Callout type="info">
  选择退为 Zeabur Credits 可以避免 Stripe 手续费，且到账更快。Credits
  可用于支付未来的订阅费或资源使用费。
</Callout>

<Callout type="warning">
  退款前请务必确认服务器内所有数据已备份或确认不需保留——**主机销毁后数据无法恢复**。
</Callout>

## 充值 Credits 退款

已充值的 Credits 目前**不支持退款**。请依实际需求充值。
