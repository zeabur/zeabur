# Free Plan

Free Plan 是 Zeabur 的免费方案，专为个人探索与学习而设计。无需绑定信用卡，注册即可开始部署。

## 方案详情

| 项目 | Free Plan |
| --- | --- |
| 月费 | US$0 |
| 可绑定外部服务器数量 | 1 |
| 编译 CI 规格 | 2C4G |
| 日志保留时间 | 48 小时 |
| 容器文件管理 | 单文件上传 |
| 数据库查看 | 支持 |
| 持久存储空间 | 支持 |
| 自定义域名绑定 | 支持 |
| 域名购买（注册） | 不支持 |
| 数据库备份（导出／导入） | 不支持 |
| 日志转发 | 不支持 |
| Zeabur AI | 不支持 |
| Zeabur Email — 每日配额 | 0 |
| Zeabur Email — 每月配额 | 0 |
| 域名 | 0 |
| API 密钥 | 0 |
| Webhooks | 0 |
| 论坛发帖 | 受限（基础） |
| 优先支持 | 不包含 |
| SLA | 不包含 |

## 限制

- **自动休眠** — Free Plan 上的服务在一段时间无流量后会自动休眠，下次有请求时唤醒，可能产生数秒的冷启动延迟。
- **无 SLA** — Free Plan 不包含服务等级协议。
- **无 Zeabur Email** — Free Plan 不支持通过 [Zeabur Email](/email) 发送邮件。
- **无自动数据库备份／导入或日志转发** — 这些进阶运维功能从 Dev Plan 开始提供。

<Callout type="info">
Free Plan 上的服务在空闲时会自动休眠以节省资源。如需服务持续运行，请考虑升级至 [Dev Plan](/pricing/dev-plan)。
</Callout>

## 升级路径

当需求超出 Free Plan 时，可升级至付费方案：

- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 天免费试用），适合有个人生产项目的独立开发者，新增 Zeabur AI、数据库备份、域名购买及 Zeabur Email。
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 天免费试用），推荐用于生产工作负载，新增日志转发、进阶配额、自定义服务图标及优先支持。
- **[Team Plan](/pricing/team-plan)** — US$79/月（含 3 个席位，每增加一个席位 US$24/月），专为团队协作设计，支持角色与权限管理。
- **Enterprise** — 定制方案，含专属支持与 SLA — 详情请[联系我们](https://zeabur.com/forum)。

访问 [zeabur.com/pricing](https://zeabur.com/pricing) 比较所有方案。
