# Free Plan（Serverless 方案）

Serverless 方案是 Zeabur 的免费方案，适合个人探索与学习使用。无需绑定信用卡，注册即可开始部署。

## 包含资源

| 资源 | 限制 |
| --- | --- |
| 项目数量 | 1 个 |
| 服务数量 | 每项目最多 2 个 |
| 部署区域 | 共享集群区域 |
| 免费额度 | 每月重置（详见[免费额度说明](/subscription/free-quota)） |
| 自定义域名 | 不支持 |
| 持久存储 | 不支持 |

### 服务费率

Serverless 方案使用与付费方案相同的费率计算，只是扣减免费额度：

- **vCPU** — 不计费（$0.0 / vCPU-分钟）
- **内存** — US$0.00025 / GB-分钟（约 US$10.8 / GB-月）
- **网络输出** — US$0.1 / GB
- **持久存储** — US$0.2 / GB-月

## 限制与注意事项

- **服务自动暂停** — Serverless 方案的服务在一段时间无流量后会自动暂停（sleep），下次收到请求时自动唤醒。这可能导致首次请求有数秒延迟。
- **容器化服务限制** — 无法部署需要持久存储的服务（如数据库）。
- **无 SLA 保障** — 免费方案不提供服务等级协议。
- **额度用尽** — 当月免费额度用尽后，服务会被暂停，下月重置后恢复。

<Callout type="info">
Serverless 方案的服务会在没有流量时自动暂停以节省资源。如果你需要服务持续运行，建议升级到 [Dev Plan](/pricing/dev-plan)。
</Callout>

## 升级路径

当你的需求超过 Serverless 方案的范围，可以升级到付费方案：

- **[Dev Plan](/pricing/dev-plan)** — US$5/月，适合个人开发者的正式项目，包含自定义域名、持久存储和更多项目配额。
- **[Team Plan](/pricing/team-plan)** — 依用量计费，适合团队协作，支持成员管理和发票功能。
- **Enterprise** — 定制化方案，专属支持与 SLA，详情请[联系我们](https://zeabur.com/dc)。

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 比较各方案的完整功能。
