# Dev Plan（开发者方案）

Dev Plan 是 Zeabur 面向独立开发者的入门付费方案。**US$5/月**，含 **14 天免费试用**，开放 Zeabur AI、自动数据库备份、域名购买及 Zeabur Email 服务——适合个人生产项目与副业项目。

## 适用场景

- **个人副业项目** — 部署个人网站、博客、API 服务等。
- **开发与测试** — 需要持久存储空间（如数据库）和自动备份的环境。
- **小型生产服务** — 需要全天候运行的低流量服务。
- **模板开发** — 使用[一键部署按钮](/deploy/deploy-button)创建并维护 Zeabur 模板，推广你的项目。

## 何时升级至 Pro

如果需要**日志转发**、**更高的 Email／域名／API 密钥／Webhook 配额**、**自定义服务图标**或**优先支持**，[Pro Plan](/pricing/pro-plan)（US$19/月，14 天免费试用）是下一步。详情请参阅 [Pro Plan](/pricing/pro-plan) 页面。

## 如何订阅

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 选择 Dev Plan，或在 [Dashboard 账单页面](https://dash.zeabur.com/account/billing) 进行升级。支持信用卡及支付宝付款——详情请参阅[付款方式](/subscription/payment-methods)。
