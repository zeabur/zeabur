# Dev Plan（开发者方案）

开发者方案（Developer Plan）是 Zeabur 针对个人开发者设计的付费方案，每月 **US$5**，解锁完整的部署功能，适合个人正式项目与 side project。

## 方案内容

| 项目 | 说明 |
| --- | --- |
| 月费 | US$5 / 月 |
| 包含额度 | US$5 / 月（可抵扣资源费用） |
| 项目数量 | 无限制 |
| 服务数量 | 无限制 |
| 自定义域名 | 支持 |
| 持久存储 | 支持 |
| 部署区域 | 所有可用区域 |
| 成员数 | 仅限 1 人（账号拥有者） |

### 费率

Dev Plan 的资源费率与 [方案总览](/pricing/pricing-plans) 中列出的费率一致：

- **vCPU** — 不计费
- **内存** — US$0.00025 / GB-分钟（约 US$10.8 / GB-月）
- **网络输出** — US$0.1 / GB
- **持久存储** — US$0.2 / GB-月

每月包含的 US$5 额度会自动抵扣资源使用费。超出额度的部分会从你的信用卡或充值余额扣款。

## 与 Free Plan 的差异

| 功能 | Free Plan | Dev Plan |
| --- | --- | --- |
| 月费 | 免费 | US$5 |
| 项目数量 | 1 | 无限制 |
| 服务数量 | 每项目 2 个 | 无限制 |
| 自定义域名 | 不支持 | 支持 |
| 持久存储 | 不支持 | 支持 |
| 服务休眠 | 无流量自动暂停 | 持续运行 |
| SLA | 无 | 标准 |

## 适用场景

- **个人 Side Project** — 部署个人网站、Blog、API 服务等。
- **开发与测试** — 需要持久存储（如数据库）的开发环境。
- **小型生产服务** — 流量不大但需要 24/7 运行的服务。
- **模板开发** — 创建和维护 Zeabur 模板，搭配[一键部署按钮](/deploy/deploy-button)推广项目。

## 如何订阅

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 选择 Dev Plan，或在 [Dashboard 账单页面](https://dash.zeabur.com/account/billing) 升级。支持信用卡和支付宝付款，详见[付款方式](/subscription/payment-methods)。
