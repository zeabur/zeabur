# Team Plan（团队方案）

团队方案（Team Plan）是 Zeabur 为团队协作设计的方案，支持多成员管理和统一账单，适合需要多人共同管理项目的团队。

## 方案内容

| 项目 | 说明 |
| --- | --- |
| 计费方式 | 依用量计费 |
| 项目数量 | 无限制 |
| 服务数量 | 无限制 |
| 自定义域名 | 支持 |
| 持久存储 | 支持 |
| 部署区域 | 所有可用区域 |
| 成员数 | 多人（依 seat 数） |
| 发票 / 税号 | 支持 |

### Seat 机制

Team Plan 支持多成员协作。团队成员数依 seat 计算，具体费率请参考 [zeabur.com/pricing](https://zeabur.com/pricing)。

### 费率

Team Plan 的资源费率与 Dev Plan 相同，请参考[方案总览](/pricing/pricing-plans)。

## 发票与税号

Team Plan 支持开立发票及设置税号，适合需要报账的企业和组织。详细说明请参考[发票与税号](/subscription/invoicing)。

## 与 Dev Plan 的差异

| 功能 | Dev Plan | Team Plan |
| --- | --- | --- |
| 计费方式 | US$5/月 + 用量 | 依用量计费 |
| 成员数 | 1 人 | 多人（依 seat） |
| 成员权限管理 | 不适用 | 支持 |
| 发票 / 税号 | 不支持 | 支持 |
| 适用对象 | 个人开发者 | 团队 / 企业 |

<Callout type="info">
如需更大规模的定制化方案、专属支持与 SLA，请参考 **Enterprise 方案**，详情请[联系我们](https://zeabur.com/dc)。
</Callout>

## 适用场景

- **初创团队** — 多位开发者共同管理部署环境。
- **企业内部项目** — 需要统一账单和发票的团队。
- **接案公司** — 为不同客户创建独立项目，团队统一管理。
- **开源社区** — 多位维护者协同管理服务部署。

## 如何订阅

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 选择 Team Plan，或在 [Dashboard 账单页面](https://dash.zeabur.com/account/billing) 升级。支持信用卡和支付宝付款，详见[付款方式](/subscription/payment-methods)。
