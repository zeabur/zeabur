# Team Plan（团队方案）

Team Plan 是 Zeabur 为需要 **协作、无限绑定服务器与优先支持** 的小团队设计的方案。起价 **US$79 / 月**，内含 3 个席位，每额外席位 **US$24 / 月**。

<Callout type="warning">
Team 方案**不**包含免费试用。若想先评估平台，建议先试用 Dev 或 Pro 方案。
</Callout>

### 席位机制

Team Plan 支持多成员协作，每个席位皆拥有 Pro 等级功能。基础订阅包含 **3 席**，额外席位为每席每月 **US$24**。

示例：5 人团队支付 `$79 + 2 × $24 = $127 / 月`。

## 角色与权限

Team Plan 的席位分为 **Viewer**、**Editor**、**Admin** 三种角色。依据每位成员实际需要的权限分配角色 — 从只读查看、日常开发操作，到完整的管理控制。

| 具体功能 | Viewer | Editor（即将推出） | Admin |
| --- | :---: | :---: | :---: |
| 查看 Team 成员 | ✅ | ✅ | ✅ |
| 邀请新成员 | ❌ | ❌ | ✅ |
| 修改成员角色 | ❌ | ❌ | ✅ |
| 移除成员 | ❌ | ❌ | ✅ |

| 创建 Project / Service | ❌ | ✅ | ✅ |
| 修改 Service 设置 | ❌ | ✅ | ✅ |
| 删除 Project / Service | ❌ | ❌ | ✅ |
| 查看部署记录 | ✅ | ✅ | ✅ |
| 手动触发 Deploy | ❌ | ✅ | ✅ |
| Rollback 部署 | ❌ | ✅ | ✅ |
| 查看 Logs | ✅ | ✅ | ✅ |
| 搜索 Logs | ❌ | ✅ | ✅ |

| 修改订阅方案 | ❌ | ❌ | ✅ |

- **Viewer** — 对团队资源的只读权限。适合相关方、审计人员或新进同事的上手期。
- **Editor** _（即将推出）_ — 可构建与部署：创建／修改 Project、触发部署、Rollback、搜索与下载 Logs。但无法删除资源，也不能管理成员或账务。
- **Admin** — 完整的管理权限，包含成员管理、破坏性操作与账务。任何影响 Team 本身或订阅的操作都需要 Admin 角色。

## 相较 Pro Plan 新增

包含 Pro Plan 所有功能，并额外提供：

- **无限** 绑定外部服务器（Pro：10 台）
- 日志保留 **90 天**（Pro：30 天）
- **内含 3 个席位**（每席 Pro 等级）
- **团队项目协作**
- **角色与权限管理**
- **优先客服与技术支持**
- 容器**批量文件操作**
- **企业采购与发票开立**

## 发票与税号

Team Plan 支持开具正式发票及设置税号，适合需要报账的企业和组织。详细说明请参考 [发票与税号](/subscription/invoicing)。

## 适用场景

- **初创团队** — 多位开发者共同管理部署环境。
- **企业内部项目** — 需要统一账单和正式发票的团队。
- **接案公司** — 为不同客户创建独立项目，团队统一管理。
- **开源社区** — 多位维护者协同管理服务部署。
