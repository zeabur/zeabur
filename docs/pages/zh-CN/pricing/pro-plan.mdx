# Pro Plan（专业方案）

Pro Plan 是 Zeabur 为需要 **更高构建规格、更长日志保留与强化可观测性** 的开发者设计的方案，每月 **US$19**，在 Dev Plan 之上新增生产级能力。

<Callout type="info">
支持 **14 天免费试用** — 功能与付费版本完全一致，试用结束前不会收费。详见 [14 天免费试用](/billing/subscription#14-天免费试用)。
</Callout>

## 方案内容

| 项目 | 说明 |
| --- | --- |
| 月费 | US$19 / 月 |
| 绑定外部服务器 | 最多 10 台 |
| 构建 CI 规格 | 4C8G |

| 日志转发（Log Drain） | 支持 |
| 自定义监控与告警 | 支持 |

| 容器文件管理 | 单文件上传 |
| HA 服务部署 | 支持 |
| 域名与 IP 访问控制 | 支持 |
| 成员数 | 仅限 1 人（账号拥有者） |

## 相较 Dev Plan 新增

包含 Dev Plan 所有功能，并额外提供：

- 最多 **10 台绑定外部服务器**（Dev：3 台）
- **构建 CI 4C8G**（Dev：2C4G）— 更快、更强的构建能力
- 日志保留 **30 天**（Dev：7 天）
- **高级日志搜索** — 筛选与查询历史日志
- **日志转发（Log Drain）** — 将日志导向第三方系统
- **高可用服务部署**
- **域名与 IP 访问控制** — 限制谁可以访问你的服务
- 更高的文件上传限制

## 适用场景

- **有流量的生产服务** — 更高 CI、HA 部署与更长日志保留，保持生产环境稳定。
- **需要可观测性的一人团队** — 高级日志搜索、日志转发与自定义告警，让排错更轻松。
- **安全敏感服务** — 域名与 IP 访问控制，限制谁可以连接你的端点。
- **长时间构建** — 4C8G 构建 CI 更快，特别适合 monorepo。
