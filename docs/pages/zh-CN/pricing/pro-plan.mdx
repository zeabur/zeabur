# Pro Plan

Pro Plan 是 Zeabur 推荐用于生产工作负载的方案——**US$19/月**，含 **14 天免费试用**。在 Dev Plan 基础上新增**日志转发**、**更大的 CI 规格**、**更长的日志保留时间**、**更高的 Email／域名／API 密钥配额**、**自定义服务图标**及**优先支持**。

## 适用场景

- **小型生产部署** — 受益于更长日志保留时间、更大 CI 编译规格及专属支持渠道的服务。
- **高交易量邮件应用** — 每月 50,000 封的邮件配额可满足大多数早期 SaaS 及电商应用。
- **运维密集型工作负载** — 将日志转发至现有可观测性平台（Datadog、Grafana 等），使 Pro 层服务与中央监控保持一致。
- **多域名前端应用** — 10 个域名槽位和 100 个 API 密钥为 Pro 提供充足空间。
