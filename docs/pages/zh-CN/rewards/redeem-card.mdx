# 兑换预付卡

有些时候，你可能会在某些与 Zeabur 合作的第三方合作伙伴处购买或取得 Zeabur 预付卡。这些预付卡可以用来兑换 Zeabur 服务的余额，你可以用这些兑换到的余额升级 Zeabur 付费会员、购买专用服务器或支付共享集群的使用费。

本篇文章将教你如何将你购买或取得的 Zeabur 预付卡兑换至余额。

## 如何兑换

1. 登录你的 Zeabur 帐号。如果你还没有 Zeabur 帐号，请先[注册](https://zeabur.com/login)一个。
2. 访问 Zeabur 控制台的账单管理页面 [https://zeabur.com/account/billing](https://zeabur.com/account/billing)。
3. 你可以在账单管理页面找到「帐号余额」区块，点击右上角的「兑换积分」按钮。
4. 在弹出的对话框中输入你的预付卡代码，然后点击「兑换」按钮。
5. 如果兑换成功，你会在画面右下角弹出的通知中看到兑换成功的提示。此时你的帐户余额将会增加相应的金额。

## 如果遇到问题

如果你在兑换预付卡时遇到问题，请到 [Zeabur 社区论坛](https://zeabur.com/forum) 与我们的客服团队联系，我们将竭诚为你提供帮助。
