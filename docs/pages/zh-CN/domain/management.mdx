# 域名管理


域名管理页面列出了你账户下注册的所有域名。

## 域名状态

每个域名具有以下状态之一：

| 状态 | 说明 |
|------|------|
| **已激活** | 域名已注册且正常运行。 |
| **待处理** | 注册正在处理中。 |
| **已过期** | 域名已过期。 |
| **已暂停** | 域名被暂停（例如注册人未完成验证）。 |
| **失败** | 注册失败。 |

域名较多时，可以使用状态筛选下拉框缩小列表范围。

将在 30 天内到期的域名会在到期日期列显示警告。

## 自动续费

管理表格中每个域名都有自动续费开关。开启后，Zeabur 会在域名到期前使用你绑定的支付方式自动续费。

你可以直接在表格行中切换自动续费，无需打开详情页。

## 手动续费

手动续费域名的步骤：

1. 点击域名打开详情页。
2. 点击标题栏中的 **续费** 按钮，按钮上会显示续费价格。
3. 完成结账流程。

<Callout type="info">
  详情页同时显示原始购买价格和续费价格，供你参考。
</Callout>

## Zeabur Email 集成

在域名详情页中，你可以一键将域名接入 **Zeabur Email**。选择发送区域后，所需的 DNS 记录（DKIM、SPF、DMARC）会自动创建。

更多关于邮件域名验证的信息，请参阅 [Zeabur Email 域名管理](/email/domains)文档。
