# 域名管理

域名管理页面列出你账户下所有已注册的域名。

## 域名状态

每个域名具有以下状态之一：

| 状态 | 说明 |
|------|------|
| **Active** | 域名已注册且正常运行。 |
| **Pending** | 注册正在处理中。 |
| **Expired** | 域名已过期。 |
| **Suspended** | 域名被暂停（例如注册人未验证）。 |
| **Failed** | 注册失败。 |

域名较多时，使用状态筛选下拉菜单缩小列表范围。

30 天内即将到期的域名会在到期日列显示警告。

## 自动续费

管理表格中每个域名都有自动续费开关。启用后，Zeabur 会在域名到期前使用已绑定的支付方式自动续费。

直接在表格行中切换自动续费，无需打开详情页。

## 手动续费

手动续费域名：

1. 点击域名进入详情页。
2. 点击页头的 **续费** 按钮，按钮上显示续费价格。
3. 完成结账流程。

<Callout type="info">
  详情页同时显示原始购买价格和续费价格，供参考。
</Callout>

## Zeabur Email 集成

在域名详情页，可以一键将域名接入 **Zeabur Email**。选择发送区域后，所需的 DNS 记录（DKIM、SPF、DMARC）会自动创建。

详情参见 [Zeabur Email 域名管理](/email/domains) 文档。
