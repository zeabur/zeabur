# 注册域名

import { Callout, Steps } from 'nextra/components'

<Callout type="info">
  域名购买需要 **Developer** 或 **Team** 方案。Free 方案用户会看到升级提示。
</Callout>

## 搜索域名

在侧边栏打开 **域名** 页面，输入想要的域名。Zeabur 会同时查询 9 个后缀（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）的可用性。

结果逐步加载——热门后缀（`.com`、`.dev`、`.app`、`.io`）优先显示，其余随后出现。每个结果旁实时显示价格。

输入完整域名如 `example.com` 时，页面顶部会出现**最佳匹配**卡片。

## 购买域名

<Steps>

### 选择域名

在搜索结果中找到可用的域名，点击**加入购物车**。

### 选择注册人资料

在购买弹窗中选择已有的[注册人资料](/domain/profiles)，或创建新的资料。如果设置了默认资料，会自动选中。

### 完成结账

点击**下一步**进入 Zeabur 结账页面。结账流程与其他 Zeabur 产品相同，使用已绑定的支付方式。

付款后域名进入 **Pending** 状态，等待注册处理。大多数域名会在几分钟内变为 **Active**。

</Steps>

## 注册人验证

部分后缀在购买后要求注册人验证邮箱地址：

1. 验证邮件发送到注册人资料中的邮箱地址。
2. 管理页面的域名卡片上显示 **验证待处理** 标签。
3. 点击标签或进入域名详情页，可重新发送验证邮件。

<Callout type="warning">
  长期未验证的域名可能被注册局暂停。购买后请及时完成验证。
</Callout>
