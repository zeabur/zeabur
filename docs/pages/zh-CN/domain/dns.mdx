# DNS 记录

每个已注册域名的详情页内置 DNS 编辑器。在管理列表中点击任意域名即可进入。

## 支持的记录类型

| 类型 | 说明 | 代理支持 |
|------|------|----------|
| **A** | 将域名映射到 IPv4 地址 | 是 |
| **AAAA** | 将域名映射到 IPv6 地址 | 是 |
| **CNAME** | 别名指向另一个域名 | 是 |
| **MX** | 邮件交换服务器 | 否 |
| **TXT** | 任意文本（SPF、DKIM、验证等） | 否 |
| **SRV** | 服务定位 | 否 |
| **CAA** | 证书颁发机构授权 | 否 |
| **NS** | 域名服务器委派 | 否 |

## 添加记录

<Steps>

### 打开 DNS 编辑器

进入域名详情页，点击 DNS 记录表格上方的 **添加记录**。

### 填写记录

- **类型**：从下拉菜单选择记录类型。
- **名称**：子域名或 `@`（代表根域名）。
- **内容**：记录值（IP 地址、主机名、文本等）。
- **TTL**：存活时间。从预设值中选择（auto、60s、300s、3600s、86400s 等）。开启代理时，TTL 自动设为 auto。
- **优先级**：MX 和 SRV 记录必填。
- **代理**：仅 A、AAAA 和 CNAME 记录可用。详见 [Cloudflare 代理](#cloudflare-代理)。

### 保存

点击 **保存** 创建记录。更改通常在几分钟内生效。

</Steps>

## 编辑和删除记录

- 点击记录行的**编辑图标**，可修改内容、TTL、优先级或代理设置。记录类型和名称在创建后不可更改——如需修改，请删除后重新创建。
- 点击**删除图标**移除记录。

## Cloudflare 代理

A、AAAA 和 CNAME 记录有代理开关。启用后：

- 流量经过 Cloudflare 路由，提供 DDoS 防护、缓存和 SSL 终结。
- TTL 自动设为 **Auto**，不可手动更改。
- DNS 查询中隐藏实际源站 IP 地址。

<Callout type="info">
  代理默认关闭。当你需要 Cloudflare 的防护和性能功能时，为对应记录启用它。
</Callout>
