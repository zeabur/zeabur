# 注册人资料


域名注册需要提供 WHOIS 联系信息。注册人资料让你保存一次信息，在多次域名购买中复用。

## 资料字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 名 | 是 | 注册人的名 |
| 姓 | 是 | 注册人的姓 |
| 邮箱 | 是 | 联系邮箱（也用于注册人验证） |
| 电话 | 是 | 联系电话 |
| 地址行 1 | 是 | 街道地址 |
| 地址行 2 | 否 | 公寓、套房等 |
| 城市 | 是 | 城市 |
| 州 / 省 | 是 | 州或省 |
| 国家 | 是 | 国家（从下拉列表选择） |
| 邮政编码 | 是 | 邮政编码 |
| 组织 | 否 | 公司或组织名称 |

## 创建资料

<Steps>

### 打开资料页

在侧边栏进入 **域名** 页面，选择 **资料** 标签。

### 填写表单

点击 **创建资料**，填写必填字段。从可搜索的下拉列表中选择国家。

### 保存

点击 **保存**。新资料立即可用于域名注册。

</Steps>

## 默认资料

点击星标图标将某个资料设为 **默认**。默认资料会在域名购买时自动预选到注册人资料选择器中。你仍然可以在结账时切换到其他资料。

## 编辑和删除

- 点击资料卡片上的 **编辑图标** 更新字段。
- 点击 **删除图标** 移除资料。

<Callout type="warning">
  删除资料不会影响已使用该资料注册的域名。已注册的域名会保留注册时提交的联系信息。
</Callout>

## 内联创建资料

你也可以在域名购买对话框中直接创建新资料，无需跳转到资料页。新资料会保存到你的账户，可用于未来的注册。
