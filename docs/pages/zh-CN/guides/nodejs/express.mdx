# 部署 Express 应用

[Express](https://expressjs.com/) 是一个一个轻便灵活的 Node.js 网络应用框架，为网络和移动应用提供了一套强大的功能。

在 Zeabur 上一键部署你的 Express 应用。

### 第一步：创建 Express 应用（可选）

- 电脑上安装了 Node.js
- 使用 express-generator 或手动创建

```bash
# 使用 express-generator
npx express-generator
```

你可以在 Express 的[官方文档](https://expressjs.com/en/starter/installing.html)看到更详细的教学。

创建完毕后，为其初始化一个 Git 仓库并部署到你的 GitHub 上。

### 第二步：部署

在你的项目中，点击 **Add new service** 按钮，并选择 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜索想要部署的 Express 应用仓库，并点击导入，你的 Express 应用就将自动开始部署。
