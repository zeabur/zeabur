# 部署 Svelte Kit 项目

在 Zeabur 上部署你的 Svelte Kit 项目。

### 第一步: 建立 Svelte Kit 项目（可选）

打开终端，输入

```bash
npm create svelte@latest
```

接下来，你需要安装依赖：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash
    pnpm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    npm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    yarn
    ```
  </Tabs.Tab>
</Tabs>

创建完成后，为其初始化一个 Git 仓库并部署到你的 GitHub 上。

### 第二步：设置 `@zeabur/svelte-adapter`

接下来，你需要安装 `@zeabur/svelte-adapter`：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash
    pnpm add -D @zeabur/svelte-adapter
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    npm install -D @zeabur/svelte-adapter
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    yarn add -D @zeabur/svelte-adapter
    ```
  </Tabs.Tab>
</Tabs>

然后，打开 `svelte.config.js`，并将其内容改为：

```js
import adapter from '@zeabur/svelte-adapter'; // <-- 注意这里

export default {
  kit: {
    adapter: adapter()
  }
}
```

### 第三步：部署 Svelte Kit 项目

在你的项目中，点击 **Deploy service** 或者 **Add new service** 按钮，并选择 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜索想要部署的 Svelte Kit 代码仓库，并点击导入，你的 Svelte Kit 应用就将自动开始部署。

## 第四步：绑定域名

部署完成后，你可能需要为你的网站配置域名。

打开服务页面的「网络」标签页，然后点击「生成域名」或「自定义域名」。

关于如何为你的服务绑定域名的更多信息，请参考 [绑定域名](/deploy/domain-binding)。
