# 在 Zeabur 上部署 Flutter

[Flutter](https://flutter.dev) 是由 Google 开发的开源 UI 软件开发工具包，用于构建跨平台的移动应用程序。

本文将帮助你在 Zeabur 上部署 Flutter 网站。

## 步骤一：创建一个 Flutter 项目

### 方法一：使用 `flutter create` 命令创建项目

请先根据 [Flutter 官方文档](https://docs.flutter.dev/get-started/install) 安装 Flutter。

你可以输入下面的命令来检查 Flutter 是否已经安装好。

```bash
flutter doctor
```

如果你看到所有的检查项都是绿色的，那么恭喜你，你已经安装好了 Flutter。

然后，使用 `flutter create` 命令创建一个新的 Flutter 项目。

```bash
flutter create myapp
cd myapp
flutter run -d chrome
```

### 方法二：使用模板创建项目

我们为你准备了一个模板来帮助你快速开始 Flutter 项目： [zeabur/flutter-template](https://github.com/zeabur/flutter-template)。

你可以在 GitHub 页面上点击 `Use this template` 按钮来使用模板。

## 步骤二：部署到 Zeabur

### 方法一：使用 Zeabur CLI

首先，确保你已经按照 [Zeabur CLI 部署指南](/deploy/deploy-in-cli) 安装好了 Zeabur CLI。

然后，在你刚初始化好的 Flutter 项目中，将尚未提交到版本控制的文件加入到 Git 中。

```bash
git init
git add -A
git commit -m "Initial commit"
```

接着，使用下面的命令部署你的服务。

```bash
zeabur
```

### 方法二：使用 Zeabur 控制台

前往 [Zeabur 控制台](https://dash.zeabur.com) 并创建一个新项目。

然后，点击 `Deploy New Service` 按钮，选择从 GitHub 部署。

<Callout>
    如果你是第一次在 Zeabur 上从 GitHub 部署项目，你需要先配置并安装 GitHub App。
    你可以在 [这里](/deploy/github) 查看更详细的指南。
</Callout>

当你选择好仓库和分支后，Zeabur 将会自动开始构建你的服务。

Zeabur 会自动检测到你的服务是由 Flutter 构建的，所以你不需要做任何额外的配置。你的部署将会在几分钟内完成。

## 下一步

部署完成后，你需要为你的 Flutter 网站配置域名。

打开服务页面的 "Domain" 分页，然后点击 "Generate Domain" 或 "Custom Domain"。

关于如何为你的服务绑定域名的更多信息，请参考 [绑定域名](/deploy/domain-binding)。
