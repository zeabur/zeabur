# 定价

Zeabur 为你的项目提供按量计费的定价模式，您无需为未使用的资源买单，只需为您的服务在这一分钟实际使用的资源付费。

## 服务费率

Zeabur 有两种服务器类型：**共享集群** 和 [_专用服务器_](/dedicated-server)。这个费率表只适用于 **共享集群**，我们不会收取专用服务器的使用费用。

在 **共享集群** 中使用的费率如下：

- vCPU - 不计费 ($0.0 / vCPU-分钟)

- 内存 - **US$0.00025** / GB-分钟 (约 US$10.8 / GB-月)
  - 我们只收取您服务实际使用的内存，服务 crash 或暂停时将不再收费。

- 网络输出费用 - **US$0.1** / GB (US$10 / 100GB)
  - 1GB 约等于 1,000 名独立访客，在没有 CDN 和压缩的情况下，浏览您 1MiB 的网页时会产生的流量。
  - [私有网络](/networking/private) 不算作输出流量。

- 持久存储空间 - **US$0.2** / GB-月

## 退款

如有任何退款相关问题，请至 [我们的 Discord 服务器](https://zeabur.com/dc) 联系我们。

### 服务器与订阅方案

针对通过 **信用卡或其他非余额付款方式** 购买的 **专属服务器** 和 **订阅方案**（开发者方案、团队方案），可以申请退款。您可以选择：

- 将退款转为 **Zeabur 余额** — 核准后立即完成。
- 退回 **原付款方式** — 可能需要最多 7 天处理时间，且您需要支付交易手续费。

### 使用余额购买的项目

对于使用 **Zeabur 余额** 购买的服务器或订阅方案，退款只能退回至您的 Zeabur 余额。

### 已购买的余额

**已购买的余额无法退款。**
