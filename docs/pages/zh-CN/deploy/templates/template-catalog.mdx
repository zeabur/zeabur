# Zeabur 模板目录

Zeabur 提供了丰富的模板目录，让你可以快速部署各种应用和服务。

访问 [Zeabur 模板目录](https://zeabur.com/templates) 浏览所有可用的模板。

## 如何使用模板

1. 在模板目录中找到你需要的模板
2. 点击「部署」按钮
3. 按照提示完成配置

你也可以查看 [模板格式](/zh-CN/template/template-format) 了解更多关于创建自定义模板的详细信息。
