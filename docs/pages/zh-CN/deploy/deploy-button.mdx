# 部署按钮

使用我们的部署按钮，可以一键轻松将服务部署到 Zeabur。该按钮适用于任何 Markdown 文件或 HTML 文件。它可以将任何 [Zeabur 模板](https://zeabur.com/templates) 转换为可部署的项目。

![button-demo](/deploy/button/button-demo.png)

## 什么是部署按钮

部署按钮是一个简单的 HTML 代码片段。你可以复制该片段并粘贴到任何 Markdown 或 HTML 文件中。每个按钮会将用户重定向到指定的模板。模板可以是单个服务或由模板作者决定的服务组。

按钮还可附加作者的推荐码；每位使用推荐码注册并升级到付费计划的新用户，将为作者带来 5 美元的信用额度。

## 复制按钮

按钮与模板关联，因此只有模板作者才能生成按钮。

前往 [Zeabur 仪表板](https://zeabur.com/account) 的账户页面，选择 **Template** 标签，即可看到你创建的模板列表。

![created-templates](/deploy/button/created-template.png)

选择所需的模板，点击分享按钮。将弹出模态框显示按钮代码。

![popup](/deploy/button/popup.png)

若启用推荐码，按钮代码将附加推荐码（即你在 Zeabur 的用户名）。

## 使用场景

这是向全世界分享你在 Zeabur 上部署的精彩项目的最佳方式。任何人只需一键即可部署你的项目。

例如，你可以将按钮粘贴到以下位置：

- GitHub README 文件
- 博客
- 文档和教程
- 社交资料

## 要求

需先创建模板才能获取部署按钮。按照此[说明](https://zeabur.com/docs/zh-CN/template/template-in-code)创建模板。

## 收益

如上所述，每位使用你的推荐码注册并升级到付费计划的新用户，将为你带来 5 美元信用额度。这些信用可用于支付你自己的服务费用。

更多详情见我们的[推荐文档](https://zeabur.com/docs/zh-CN/billing/referral)。
