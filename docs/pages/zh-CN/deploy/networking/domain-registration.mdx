## 域名注册

<LastUpdated />

Zeabur 让你可以直接在 Dashboard 或 CLI 搜索、购买和管理域名，无需使用其他域名注册商。

购买的域名会自动配置 Cloudflare DNS，并且可以一键绑定到你的服务。

### 支持的顶级域名

| 顶级域名 | 示例 |
|---------|------|
| `.com` | `example.com` |
| `.net` | `example.net` |
| `.org` | `example.org` |
| `.io` | `example.io` |
| `.dev` | `example.dev` |
| `.app` | `example.app` |
| `.co` | `example.co` |
| `.me` | `example.me` |
| `.xyz` | `example.xyz` |

### 购买域名

<Steps>

### 创建注册人资料

在购买域名之前，你需要一份注册人资料（Registrant Profile）— 这是 ICANN 要求的域名注册联系信息（WHOIS 数据）。

前往 **Settings** → **Registrant Profiles**，填写你的姓名、电子邮件、电话和地址。

<Callout type="info">
注册人资料只需要创建一次，之后所有域名购买都可以重复使用。
</Callout>

### 搜索域名

前往 **Domains** → **Register Domain**，输入你想要的域名。Zeabur 会检查可用性并显示价格。

你也可以使用 CLI 搜索：

```bash copy
npx zeabur@latest domain search --domain example.com
```

### 购买

点击 **Purchase** 并确认付款。域名会被注册，Cloudflare DNS zone 也会自动创建。

使用 CLI：

```bash copy
npx zeabur@latest domain purchase --domain example.com --registrant-profile-id <profile-id>
```

<Callout type="warning">
域名购买不可退款，请在确认前确保输入了正确的域名。
</Callout>

### 绑定到服务

购买完成后，你可以将域名绑定到任何服务。前往服务的 **Domains** 选项卡，选择 **Custom Domain**，然后输入你购买的域名。

通过 Zeabur 购买的域名会自动配置 DNS 记录，不需要手动设置 CNAME。

</Steps>

### 管理 DNS 记录

通过 Zeabur 购买的域名，DNS 由 Cloudflare 管理。你可以在 Dashboard 或 CLI 中添加、修改和删除记录。

支持的记录类型：`A`、`AAAA`、`CNAME`、`MX`、`TXT`、`SRV`、`CAA`、`NS`。

使用 CLI：

```bash copy
# 列出记录
npx zeabur@latest domain dns list --id <domain-id>

# 添加记录
npx zeabur@latest domain dns create --id <domain-id> --type A --name @ --content 1.2.3.4

# 删除记录
npx zeabur@latest domain dns delete --id <domain-id> --record-id <record-id>
```

### 续约

域名注册期限为 1 年。你可以在 Dashboard 启用**自动续约**，或手动续约。

```bash copy
# 启用自动续约
npx zeabur@latest domain auto-renew --id <domain-id> --enable

# 手动续约
npx zeabur@latest domain renew --id <domain-id>
```

### ICANN 注册人验证

购买域名后，ICANN 要求你验证注册人电子邮件地址。你会收到一封验证邮件 — **请在 15 天内点击链接**，否则域名可能会被暂停。

如果没有收到验证邮件或需要重新发送：

```bash copy
npx zeabur@latest domain verification resend --id <domain-id>
```

如果注册时填错了电子邮件，你可以更新注册人联系信息：

```bash copy
npx zeabur@latest domain verification update-contact --id <domain-id> --email correct@example.com --first-name John --last-name Doe --phone "+1.5551234567" --address1 "123 Main St" --city "San Francisco" --state CA --country US --postal-code 94105
```

<Callout type="warning">
更改注册人电子邮件会触发新的验证流程，并可能对域名施加 60 天的转移锁定。
</Callout>

查看当前的验证状态：

```bash copy
npx zeabur@latest domain verification status --id <domain-id>
```
