# **自定义根目录**

对于某些 Git 仓库（如同时包含前端和后端的项目），需要手动指定项目的根目录。

### **步骤 1：部署服务**

按照[部署服务](https://zeabur.com/docs/zh-CN/get-started)的指南部署你的应用。

## **步骤 2：更改项目根目录**

点击已部署的服务，进入设置标签，向下滚动至 **Root Directory** 栏目。

![Root Directory](/deploy/root-directory/root_directory.png)

在此处设置你的项目根目录。
