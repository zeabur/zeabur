# Chrome 扩展

Zeabur 提供多种扩展，让你无需离开开发环境，即可一键部署本地代码。

本文将介绍如何通过 Zeabur 的 Chrome 扩展来部署服务。

## Chrome 扩展

Zeabur 的 Chrome 扩展可以让你在支持的 AI 代码生成平台（如 Lovable、Bolt.new 或 v0.dev）中直接部署代码。

### 安装扩展

前往 [Chrome 网上应用店](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem)，点击「添加至 Chrome」。

### 使用方式

在支持的 AI 平台中编写代码时，界面上会出现「Deploy with Zeabur」按钮。点击即可开始部署流程。
