# 执行命令

你可以在 Zeabur 服务内执行命令，用于安装包、运行程序、查看信息，甚至备份特定数据。

## 打开命令执行界面

打开服务的 **Overview** 标签页，点击 **Command** 即可显示“Execute Command”界面。

![打开命令执行界面](/deploy/command-execution/Command_part1.png)

命令执行界面：

- 命令输出区域：显示命令执行结果。成功时显示输出内容，失败时显示错误码。

![命令执行界面](/deploy/command-execution/terminal.png)

## 执行命令

以简单 `echo` 为例，输入 `echo Hello, World!`，应看到输出 `Hello, World!`。

![执行 hello world 命令](/deploy/command-execution/terminal_2.png)

输入不存在的命令会显示错误码。

![命令执行失败](/deploy/command-execution/terminal_3.png)

你还可以在 MySQL 或 PostgreSQL 等数据库容器中执行 SQL 语句、使用 cURL 上传日志或备份文件，甚至运行迁移程序（如 `prisma migrate`、`node seeder.js`）。

对于 MySQL，可复制“Command to connect to your MySQL”命令，删除 `host` 和 `port` 参数（容器内无需远程连接），将 `mysqlsh` 改为 `mysql`（容器内无 `mysqlsh`），最后加上 `-e` 执行 SQL 语句。

```bash
mysql --user=root --password=<password> --database=zeabur -e "SHOW DATABASES;"

![执行 MySQL 命令](/deploy/command-execution/MySQL_terminal.png)
