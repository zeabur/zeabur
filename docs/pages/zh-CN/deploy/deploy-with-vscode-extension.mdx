# 使用 VS Code 插件部署服务

Zeabur 提供多种插件，让你无需离开开发环境，即可一键部署本地代码。

本文将介绍如何通过 Zeabur 的 VS Code 插件来部署服务。

## VS Code / Cursor 插件

你可以在 Visual Studio Code 和 Cursor 编辑器中使用 Zeabur 插件，直接将本地项目部署到 Zeabur。

### 安装插件

1. 打开 VS Code 或 Cursor 编辑器。
2. 前往扩展面板，搜索「Zeabur」。
3. 点击「安装」，或直接前往 [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) 安装。

### 使用方式

1. 打开要部署的项目文件夹。
2. 点击侧边栏的 Zeabur 插件图标。
3. 点击「Deploy」后将自动打开浏览器页面。
4. 选择部署地区。
5. 几分钟后即可完成部署。
