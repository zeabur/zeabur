# 使用 Raycast 插件部署服务

Zeabur 提供多种插件，让你无需离开开发环境，即可一键部署本地代码。

本文将介绍如何通过 Zeabur 的 Raycast 插件来部署服务。

## Raycast 插件

使用 Zeabur 的 Raycast 插件，你可以快速部署项目、搜索文档或查找模板。

### 安装插件

打开 Raycast Store，搜索 Zeabur 插件并安装。你也可以直接前往 [Raycast Zeabur 插件页面](https://www.raycast.com/ridemountainpig/zeabur)，点击「Install Extension」安装插件。

### 使用方式

1. 安装完成后，打开 Raycast 并输入 `Zeabur`。
2. 选择「Deploy Project」指令。
3. 选择你想部署的项目文件夹，按下 `⌘ + Enter`。
4. 选择部署地区，几分钟后即可完成部署。
