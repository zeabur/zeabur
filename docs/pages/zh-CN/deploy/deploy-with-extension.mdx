# 使用插件部署服务

Zeabur 提供多种插件，让你无需离开开发环境，即可一键部署本地代码。

本文将介绍如何通过 Zeabur 的 VS Code、Cursor、Chrome 和 Raycast 插件来部署服务。

## VS Code / Cursor 插件

你可以在 Visual Studio Code 和 Cursor 编辑器中使用 Zeabur 插件，直接将本地项目部署到 Zeabur。

### 安装插件

1. 打开 VS Code 或 Cursor 编辑器。
2. 前往扩展面板，搜索「Zeabur」。
3. 点击「安装」，或直接前往 [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) 安装。

### 使用方式

1. 打开要部署的项目文件夹。
2. 点击侧边栏的 Zeabur 插件图标。
3. 点击「Deploy」后将自动打开浏览器页面。
4. 选择部署地区。
5. 几分钟后即可完成部署。

## Chrome 插件

Zeabur 的 Chrome 插件可以让你在支持的 AI 代码生成平台（如 Lovable、Bolt.new 或 v0.dev）中直接部署代码。

### 安装插件

前往 [Chrome 网上应用店](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem)，点击「添加至 Chrome」。

### 使用方式

在支持的 AI 平台中编写代码时，界面上会出现「Deploy with Zeabur」按钮。点击即可开始部署流程。

## Raycast 插件

使用 Zeabur 的 Raycast 插件，你可以快速部署项目、搜索文档或查找模板。

### 安装插件

打开 Raycast Store，搜索 Zeabur 插件并安装。你也可以直接前往 [Raycast Zeabur 插件页面](https://www.raycast.com/ridemountainpig/zeabur)，点击「Install Extension」安装插件。

### 使用方式

1. 安装完成后，打开 Raycast 并输入 `Zeabur`。
2. 选择「Deploy Project」指令。
3. 选择你想部署的项目文件夹，按下 `⌘ + Enter`。
4. 选择部署地区，几分钟后即可完成部署。
