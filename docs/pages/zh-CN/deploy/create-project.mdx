# 创建项目

在 Zeabur 上部署服务之前，你需要先创建一个项目。

你可以按照以下步骤创建一个项目。

## 什么是项目

一个项目是一组一起部署的服务，同一项目中的服务可以通过内部网络相互通信。
你可以在不同的区域创建多个项目，并在每个项目中部署各种服务。

除此之外，你可以为项目[设置预算和管理用量](../manage/budget)，[邀请其他团队成员](../manage/invite)来管理服务等。

## 如何创建项目

<Callout type="warning">
  Zeabur 正在逐步淘汰共享集群。新项目只能创建在专用服务器上。详情请参阅 [changelog](https://zeabur.com/changelogs/phasing-out-shared-cluster)。
</Callout>

首先，你需要导航到[控制台的项目页面](https://zeabur.com/projects)。

如果你是第一次使用 Zeabur，你会看到一个创建项目的引导页面。

![Create project onboarding](/deploy/create-project/onboarding.png)

如果你之前创建过项目，你可以直接点击 **Create Project** 按钮，或者按下 `CMD + K` 或 `Ctrl + K` 打开命令面板选择 **Create Project**。选择你已有的专用服务器，或选择 **Buy New Server** 购买新服务器，或选择 **Bind External Server** 通过 SSH 连接你自己的服务器。然后点击 **Create Project** 完成创建。

![Select Region](/deploy/create-project/select-region.png)

## 下一步

创建完成后，你将看到一个空项目：

![Empty Project](/deploy/create-project/empty-project.png)

你可以点击 **部署新服务** 按钮在这个项目中部署服务，或者点击配置按钮管理项目设置和邀请团队成员。
