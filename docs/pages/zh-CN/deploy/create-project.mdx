# **创建项目**

要在 Zeabur 上部署服务，首先需要创建一个项目。

你可以按照以下步骤创建项目。

## **什么是项目**

项目是一组一起部署的服务的集合，同一项目中的服务可以通过内部网络相互通信。你可以在不同区域创建多个项目，并在每个项目中部署各种服务。

此外，你可以为项目[设置预算并管理用量](https://zeabur.com/docs/zh-CN/manage/budget)，[邀请其他团队成员](https://zeabur.com/docs/zh-CN/manage/invite)来管理服务等。

## **如何创建项目**

首先，导航到[仪表板的 projects 页面](https://zeabur.com/projects)。

如果你是 Zeabur 新用户且没有任何项目，将看到用于创建第一个项目的引导页面。

![创建项目引导](/deploy/create-project/Deployment_Services_Create_Project.png)

如果你之前已创建过项目，只需点击 **Create Project** 按钮，进入 **Create New Project** 页面，选择 **Deployment Type**，选择 **dedicated server** 或 **share cluster**。

![选择部署类型](/deploy/create-project/choose_Deployment_Type_2.png)

然后根据订阅计划和期望区域选择区域，项目将自动创建，名称为 untitled-0、untitled-1 等。

![选择区域](/deploy/create-project/select_a_region_2.png)

## **后续步骤**

选择区域后，你将看到一个空项目：

![空项目](/deploy/create-project/Deployment_Services_empty_project.png)

你可以点击 **Deploy New Service** 按钮在此项目中[部署服务](https://zeabur.com/docs/zh-CN/deploy/create-service)，或点击配置按钮管理项目设置并[邀请其他团队成员](https://zeabur.com/docs/zh-CN/manage/invite)。
