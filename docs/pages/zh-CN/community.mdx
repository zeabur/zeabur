# 社区

Zeabur 是跟在上面部署、分享、实际在跑东西的人一起长出来的。这里是你接入的入口。

- [大使计划](/zh-CN/community/ambassador-programme) — 加入创始团队：Pro 方案、每月额度、抢先试用，以及一条直通团队的线
- [社区行为准则](/zh-CN/community/code-of-conduct) — 我们在 Forum、Discord 以及每个 Zeabur 空间共同遵守的规范
- [社区论坛](/zh-CN/community/forum) — 提问、分享你做的东西、帮其他开发者
