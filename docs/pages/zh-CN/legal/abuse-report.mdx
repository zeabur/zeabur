# 滥用举报指引

Zeabur 与网络安全组织合作打击犯罪和恶意使用。我们鼓励您报告任何违反[《公平使用指南》](https://zeabur.com/docs/legal/fair-use-guideline)的行为。如果您发现任何网站或活动违反[《公平使用指南》](https://zeabur.com/docs/legal/fair-use-guideline)，请立即向我们报告。

如果您是政府机构或网络安全组织，也可以联系我们讨论集成自动化报告系统。
