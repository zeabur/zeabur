# 按需求开始

不确定从哪里开始？选择最适合你的路径。

## 🧪 我只是想试试看

一键从模板部署——无需编写代码。

1. 浏览[模板市场](https://zeabur.com/templates)，挑选感兴趣的内容。
2. 点击"Deploy"，选择区域，等待几秒钟。
3. 服务上线后，前往"Domains"标签生成免费的测试 URL。

**推荐阅读：** [快速开始](./quick-start) · [模板](/template)

---

## 💻 我有项目要部署

从 GitHub 推送代码到 Zeabur——支持大多数语言和框架。

1. 在项目中，点击 **Add Service** → **GitHub**，授权并选择你的仓库。
2. Zeabur 自动检测语言、框架和构建策略。
3. 如需要，在"Configuration"标签中设置环境变量。
4. 绑定域名并上线。

**推荐阅读：** [部署原理](/deploy/how-deploys-work) · [GitHub 集成](/deploy/github-integration) · [环境变量](/deploy/environment-variables)

---

## 🤖 我想使用 AI 功能

Zeabur 提供 AI Hub，让你直接访问各种大语言模型的 API。

1. 前往 [AI Hub](/ai-hub) 获取 API 端点和密钥。
2. 在应用中使用任何 OpenAI 兼容的 SDK 进行调用。
3. 支持 GPT-4o、Claude、Gemini 等众多模型。

**推荐阅读：** [AI Hub 概览](/ai-hub) · [支持的模型](/ai-hub/model-list)

---

## 📧 我需要邮件收发功能

Zeabur Email 让你使用自己的域名收发邮件。

1. 前往 [Zeabur Email](/email) 创建账户并验证域名。
2. 配置 DNS 记录（MX、SPF、DKIM）。
3. 通过 SMTP 或 REST API 收发邮件。

**推荐阅读：** [邮件服务](/email) · [邮件 API 密钥](/email/email-api-keys) · [邮件 REST API](/email/email-rest-api)

---

## 🖥️ 我有自己的服务器

自带服务器（BYO Server），通过 Zeabur 界面管理部署。

1. 在控制面板中[注册服务器](/dedicated-server)。
2. 在服务器上运行 Zeabur 提供的安装命令。
3. 创建新项目时选择你的服务器。

**推荐阅读：** [服务器](/dedicated-server)

---

## 🔌 我想通过 API / CLI 自动化

将 Zeabur 集成到你的 CI/CD 流水线中。

- **CLI**：运行 `npx zeabur@latest deploy` 从终端部署。
- **GraphQL API**：以编程方式控制项目、服务和部署。
- **Upload API**：直接上传 ZIP 打包的应用。

**推荐阅读：** [CLI 部署](/developer/cli) · [Public API](/developer/public-api) · [API 密钥](/developer/api-keys)
