# 常见问题与支持

## 获取帮助

如果你在使用 Zeabur 时遇到问题，有以下几种方式获取帮助：

- [社区论坛](https://zeabur.com/forum) — 访问我们的社区论坛与社区互动
- [讨论区](https://github.com/orgs/zeabur/discussions) — 在 GitHub Discussions 中提问
- [联系支持](/get-started/faq-support/help) — 了解更多支持渠道
- [社区认证](/get-started/faq-support/verify) — 了解如何验证你的社区身份

## 常见问题

### 如何重置密码？

Zeabur 使用第三方登录（GitHub、Google 等），没有单独的密码需要重置。如果无法登录，请尝试使用注册时相同的登录方式，或通过[社区论坛](https://zeabur.com/forum)联系支持团队。

### 为什么我的部署失败了？

部署失败通常由以下原因造成：

- **构建错误** — 确保你的代码在本地能成功构建。
- **缺少环境变量** — 确认所有必需的环境变量已正确配置。
- **端口配置错误** — 确保应用监听正确的端口（Zeabur 自动注入 `PORT` 环境变量）。
- **内存不足** — 构建或运行时可能超出了计划的资源限制。

在控制面板的服务日志中查看详细错误信息。

### 计费如何运作？

Zeabur 采用按使用量计费模式，根据 CPU、内存和网络使用量收费。详细说明请参见[定价计划](/pricing/pricing-plans)。

### 可以使用自定义域名吗？

可以。Zeabur 支持绑定自己的域名。你需要在 DNS 设置中添加相应的 CNAME 记录。详细步骤请参见[公网访问](/deploy/public-networking)。

### 如何联系支持？

你可以通过以下方式联系我们：

- 加入[社区论坛](https://zeabur.com/forum)并在支持频道提问
- 在 [GitHub Discussions](https://github.com/orgs/zeabur/discussions) 提交问题
- 更多详情请参见[支持页面](/get-started/faq-support/help)

### 支持哪些编程语言？

Zeabur 支持广泛的编程语言和框架，包括 Node.js、Python、Go、Rust、Java、.NET、PHP、Ruby 等。完整列表和部署指南请参见[指南](/guides)。

### 如何升级计划？

你可以在 Zeabur 控制面板的账户设置中升级或更改订阅计划。详细步骤请参见[订阅](/subscription)。

### 有免费套餐吗？

有。Zeabur 提供免费计划，让你免费试用和部署服务。免费计划有一定的资源限制。详情请参见[免费计划](/pricing/free-plan)。
