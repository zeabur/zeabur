# 寻求帮助

有些时候你可能会在使用 Zeabur 的过程中遇到一些问题，你可以联系 [Zeabur 支持](https://zeabur.com/support) 来获取团队帮助。

![Zeabur 支持页面](/community/help/support-home.png)

## 如何得到帮助

打开 [Zeabur 支持](https://zeabur.com/support) 并提交你的请求。请提供足够的背景信息，方便我们的团队调查问题。

![提交工单表单](/community/help/support-submit-ticket.png)

### 请求格式

遵循以下格式来描述你的问题可以帮助我们更快地定位和解决你的问题。

- 在标题中简要描述你的问题
- 描述问题细节、预期行为，以及你已经尝试过的处理方式
- 如果问题与特定项目或服务有关，请提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在 Zeabur 控制面板的 URL 中找到它们。

### 优先支持

优先支持通过 Express pass 处理。付费方案每月会包含一定数量的 pass，可让你的账户提交的支持请求进入优先队列：

| 方案 | Express passes / 月 |
| ---- | ------------------- |
| Developer | 2 |
| Pro | 5 |
| Team | 10 |

Express pass 影响的是首次响应优先级：它可以让你的请求进入团队首次回复的优先队列，但不保证问题解决时间。我们仍会尽力尽快帮助每位用户解决问题。你提供的上下文越完整，例如预期行为、已经尝试过的处理方式、相关 ID、日志和截图，我们需要来回确认的次数就越少，也能更快帮助你。如果你的账户没有剩余 Express pass，仍然可以提交支持请求，但不会进入优先队列。验证账户本身不会授予优先支持；它可以帮助我们更快识别与你的请求关联的账户。请参考[这篇文档](./verify)来进行验证。

## 其他联系方式

你也可以通过电子邮件联系我们以获得支持或询问：

- **电子邮件：** [contact@zeabur.com](mailto:contact@zeabur.com)

为了获得更快的响应时间，我们建议使用 [Zeabur 支持](https://zeabur.com/support)。不过，如果你偏好使用电子邮件沟通，欢迎随时发送消息给我们，我们会尽快回复。
