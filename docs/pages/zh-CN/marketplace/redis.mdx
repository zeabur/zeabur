# 在 Zeabur 部署 Redis

Zeabur 提供了一键部署 Redis 服务的功能，允许你可以快速地在 Zeabur 上部署 Redis 数据库。

## 部署 Redis 服务

<CreateProject />

在你的项目中，点击 **创建服务**，选择 Marketplace 之后搜索 `Redis`。

![部署 "Redis"](/databases/redis/deploy-redis.gif)

点击「Redis」后，Zeabur 会自动帮你部署 Redis 服务。

## 环境变量

成功部署 Redis 服务后，Zeabur 会自动注入环境变量到服务中。目前有以下变量：

- `REDIS_CONNECTION_STRING`
- `REDIS_HOST`
- `REDIS_PASSWORD`
- `REDIS_PORT`
- `REDIS_URI`

## 连接数据库

- [Redis cli](https://docs.redis.com/latest/rs/references/cli-utilities/redis-cli/)

### redis-cli

[redis-cli](https://www.mongodb.com/docs/mongodb-shell/) 是一个命令行工具，让你可以与 Redis 数据库互动。 使用 redis-cli，你可以在命令行终端或交互模式下执行 Redis 命令。

根据文档下载并安装完毕后，回到 Zeabur 中。点击 Redis 服务中 Instructions 页签，点击 `Command to connect to your Redis` 的复制按钮来复制连接地址。

![connect-path](/databases/redis/redis-connect-path.png)

将复制后的指令贴到终端机，并输入 `PING` 测试连接是否成功：

![test-connection](/databases/redis/redis-test-connection.png)

此时，即可开始操作 Redis。
