# 合作伙伴计划

Zeabur Partner Program 为讲师、活动主办方和导入顾问提供合作机会，通过推广 Zeabur 获得奖励回馈。

## 两种合作模式

### 教育推广

适合**讲师、培训机构、活动主办方**。通过教学课程或技术活动推广 Zeabur，使用 Event Code 让学员免费体验，并依留存付费用户数获得奖励。

- 提供专属 Event Code 供学员使用
- 一个月后统计留存付费用户，每人奖励 US$5
- 适合工作坊、线上课程、Hackathon 等场景

[了解更多 →](/partner/education)

### 导入顾问（Sales Partner）

适合**企业顾问、系统集成商、技术咨询公司**。协助企业客户导入 Zeabur，完成签约并开始付费后获得返现奖励。

- 触发条件：单笔金额 > US$1,000 或机构统一采购
- 返现计算：合约总额 ÷ 合约月数（即第一个月等比金额）
- 适合企业导入、机构采购等大型合作

[了解更多 →](/partner/sales-partner)

## 与推荐计划的区隔

Zeabur 同时提供[推荐计划](/rewards/referral)（Referral Program），让所有用户通过推荐码赚取回馈。两者的主要差异：

| 项目 | 推荐计划 | 合作伙伴计划 |
| --- | --- | --- |
| 适用对象 | 所有用户 | 讲师 / 活动主办 / 企业顾问 |
| 触发方式 | 推荐码自动流程 | 人工审核与合约 |
| 奖励机制 | 推荐码佣金 | Event Code 留存奖励 / 合约返现 |
| 金额门槛 | 无 | Sales Partner：单笔 > US$1,000 |

<Callout type="info">
同一笔交易只能走一条路——推荐码或合作伙伴计划，两者不叠加。
</Callout>
