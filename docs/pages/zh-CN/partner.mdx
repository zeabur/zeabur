# 合作伙伴计划

Zeabur Partner Program 为讲师、活动主办方、企业顾问与系统集成商提供合作机会，通过推广或导入 Zeabur 获得奖励回馈。

## 两种合作模式

### 教育伙伴

适合**讲师、社群领袖、Hackathon 主办方与教育机构**。通过教学课程或技术活动推广 Zeabur，使用活动码（Event Code）让参与者免费体验，并根据留存付费用户数获得奖励。

- 获取专属活动码供参与者使用
- 活动后一个月统计留存付费用户并发放奖励
- 适合工作坊、在线课程、Hackathon、社群活动等场景

[了解更多 →](/partner/event-partner)

### 销售伙伴（Sales Partner）

适合**企业顾问、系统集成商、技术顾问公司**。协助企业客户或机构导入 Zeabur，根据实际参与的合作阶段获得经常性分润。

- 适合大型合同、机构采购、技术顾问导入等情境
- 两层分润架构：交易层累进费率 × 贡献阶段权重
- 具体费率、门槛与条件在合同中个别约定，不对外公开

[了解更多 →](/partner/sales-partner)

## 与 Referral Program 的区隔

Zeabur 同时提供 [Referral Program](/rewards/referral)（推荐计划），让所有用户通过推荐码赚取回馈。两者的主要差异：

| 项目 | Referral Program | 合作伙伴计划 |
| --- | --- | --- |
| 适用对象 | 所有用户 | 讲师 / 活动主办 / 企业顾问 / 系统集成商 |
| 触发方式 | 推荐码自动流程 | 人工审核与合同 |
| 奖励机制 | 推荐码佣金（依等级） | 活动留存奖励 / Sales Partner 分润 |
| 合同条件 | 平台条款统一 | 每份合作合同个别约定 |

<Callout type="info">
同一笔交易只能走 Referral Program 或合作伙伴计划其中一条路径，**两者不叠加**。
</Callout>

## 如何申请

前往 [partner.zeabur.com](https://partner.zeabur.com) 创建账号并提交申请，根据你的合作类型填写对应资料：

- **教育伙伴** — 填写活动名称、预计人数、时间与形式。
- **销售伙伴** — 填写公司信息、预估客户类型与合作规模。

Zeabur 团队会在收到申请后联系你，讨论合作条件。
