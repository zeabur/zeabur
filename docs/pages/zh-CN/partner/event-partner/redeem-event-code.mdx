# 兑换活动码

活动码（Event Code）由 Zeabur 教育伙伴（Education Partner）在讲座、课程、工作坊或 Hackathon 发放给参与者，可在 Zeabur 兑换为体验额度，用于升级订阅方案、部署项目与其他付费功能。

本页介绍活动码的两种兑换方法。活动主办方可将本页链接直接分享给参与者，避免重复说明。

<Callout type="info">
活动码都有使用期限，太早或太晚均无法兑换成功。请留意主办方告知的有效期间。
</Callout>

## 方法一：活动注册链接（最简单，适合尚未注册的人）

如果你还没有 Zeabur 账号，可以直接使用活动注册链接。将你的活动码加到以下网址后方：

```
https://zeabur.com/events?code=你的活动码
```

例如活动码为 `i_love_zeabur`，就打开：

```
https://zeabur.com/events?code=i_love_zeabur
```

打开后跟着页面引导完成注册，活动额度会自动兑换到你的新账户。

## 方法二：在账单页面兑换（适合已有 Zeabur 账号的人）

<Steps>

### 登录 Zeabur 账号

前往 [zeabur.com/login](https://zeabur.com/login) 登录。

### 进入账号设置

点击左上角你的名字打开菜单，再点 **Settings**。

![打开账号设置](/partner/event-partner/redeem-event-code/step-2-open-settings.png)

### 切换到 Billing，点击「Redeem Credits」

在左侧 sidebar 点击 **Billing**，在「Account Credit」区块找到 **Redeem Credits** 按钮并点击。

![切到 Billing 并点击 Redeem Credits](/partner/event-partner/redeem-event-code/step-3-billing-redeem-credits.png)

### 输入活动码

在弹出的对话框中输入你的活动码，点击 **Redeem**。兑换成功后会在右下角显示通知，账户余额立即增加。

![输入活动码](/partner/event-partner/redeem-event-code/step-4-enter-code.png)

</Steps>

## 兑换成功后

活动额度会直接进入你的**账户余额**，可用于 Zeabur 上的任何服务——部署项目、购买服务器、使用 AI Hub 等，使用方式与自行充值的储值金相同。

![活动额度可用于部署、服务器、AI Hub 等服务](/partner/event-partner/redeem-event-code/after-redeem-uses.png)

<Callout type="info">
**想额外升级至 Pro Plan？** 前往 [Pricing 页面](https://zeabur.com/pricing#upgrade) 点击升级即可（见 [Pro Plan](/pricing/pro-plan)）。

- **活动额度足以覆盖 Pro 月费**：直接以额度扣抵升级，免绑卡。
- **活动额度不足**：启用 Zeabur Pro 的 **14 天免费试用**，到期后系统会以活动额度优先抵扣月费。
</Callout>

## 常见问题

### 活动额度可以用在哪？

任何 Zeabur 上的付费项目——部署、数据库、服务器、AI Hub 等。它就是一般的账户余额，使用方式与自行充值的储值金相同。

### 我需要绑信用卡吗？

兑换活动码**不需要**绑卡。升级 Pro 方案时，若活动额度足以覆盖首月费用就不需绑卡；不足时需绑卡补齐差额。后续月份想继续使用 Pro 也需绑卡以便扣款。

### 活动结束后还能继续用 Zeabur 吗？

可以。想继续用 Pro 的话，绑定信用卡后就会自动续订；不想继续付费就维持[免费方案](/pricing)即可，随时也能回到 [Pricing 页面](https://zeabur.com/pricing) 重新升级。

### 活动额度用完后会怎样？

- **已在 Pro 方案**：下次结算会从绑定的信用卡扣款。
- **仍在免费方案**：超过免费配额的资源无法使用，可以[充值](https://zeabur.com/account/billing)或升级方案继续使用。

### 活动码过期 / 无效 / 其他问题

- **过期** — 过期后无法补救，请确认主办方告知的有效期间。
- **无效** — 确认输入是否完整、大小写是否正确。
- **其他** — 前往 [Zeabur 社群论坛](https://zeabur.com/forum) 联系客服团队。
