# 兑换活动码

活动码（Event Code）由 Zeabur 教育伙伴（Education Partner）在讲座、课程、工作坊或 Hackathon 发放给参与者，可在 Zeabur 兑换为体验额度，用于升级订阅方案、部署项目与其他付费功能。

本页介绍活动码的两种兑换方法。活动主办方可将本页链接直接分享给参与者，避免重复说明。

<Callout type="info">
活动码都有使用期限，太早或太晚均无法兑换成功。请留意主办方告知的有效期间。
</Callout>

## 方法一：活动注册链接（最简单，适合尚未注册的人）

如果你还没有 Zeabur 账号，可以直接使用活动注册链接。将你的活动码加到以下网址后方：

```
https://zeabur.com/events?code=你的活动码
```

例如活动码为 `i_love_zeabur`，就打开：

```
https://zeabur.com/events?code=i_love_zeabur
```

打开后跟着页面引导完成注册，活动额度会自动兑换到你的新账户。

## 方法二：在账单页面兑换（适合已有 Zeabur 账号的人）

<Steps>

### 登录 Zeabur 账号

前往 [zeabur.com/login](https://zeabur.com/login) 登录。

### 进入账单管理页

点击右上角头像菜单中的「账单」，或直接前往 [zeabur.com/account/billing](https://zeabur.com/account/billing)。

### 点击「兑换积分」

在「账户余额」区块右上角找到「兑换积分」按钮并点击。

### 输入活动码

在弹出的对话框中输入你的活动码，点击「兑换」。兑换成功后会在右下角显示通知，账户余额立即增加。

</Steps>

## 升级至付费方案（可选）

若你需要部署到特定地区的机房、解锁订阅方案功能，兑换后可前往 [Pricing 页面升级](https://zeabur.com/pricing#upgrade)。升级后的费用会优先从账户余额扣除。

<Callout type="info">
活动码兑换后通常足以升级订阅方案，实际可用方案依活动额度金额决定。若有疑问请与活动主办方确认。
</Callout>

## 常见问题

- **活动码过期** — 过期后无法补救，请确认主办方告知的有效期间。
- **活动码无效** — 确认输入是否完整、大小写是否正确。
- **其他问题** — 前往 [Zeabur 社群论坛](https://zeabur.com/forum) 联系客服团队。
