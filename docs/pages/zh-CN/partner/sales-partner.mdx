# 销售伙伴（Sales Partner）

销售伙伴计划面向**企业顾问、系统集成商、技术顾问公司**。当你协助企业客户、机构或组织导入 Zeabur 时，可根据实际参与的合作阶段获得经常性分润。

## 适用场景

销售伙伴计划适合下列合作类型：

- **企业云端迁移** — 协助企业将现有服务迁移至 Zeabur。
- **机构统一采购** — 教育机构、政府单位或企业 IT 部门的批量采购、合同采购，含 Voucher、需开发票、需合同等情境。
- **技术顾问导入** — 作为技术顾问或系统集成商，将 Zeabur 纳入你为客户设计的部署方案。
- **长期服务合作** — 替客户提供部署、运维或资金代收等服务，并以 Zeabur 作为底层平台。

若金额未达门槛且为个人自助付款，请改走 [Referral Program](/rewards/referral) 的自动流程。具体适用门槛与条件在合同中个别约定。

<Callout type="info">
同一笔交易只能走 Referral Program 或销售伙伴其中一条路径，**两者不叠加**。
</Callout>

## 分润架构

销售伙伴的分润由两个层级组成，根据你实际参与的范围计算。

### 第一层：交易费率

每笔交易独立计算，费率随交易金额采用累进制。费率根据伙伴类型、市场定位与合作规模**个别约定**，不对外公开标准值。

### 第二层：贡献阶段权重

根据你在该笔交易中实际负责的阶段，四阶段权重加总为 100%。Zeabur 自行负责的阶段不外付。

| 阶段 | 定义 | 认定基准 |
| --- | --- | --- |
| Lead（引荐与开发） | 找到客户 | 你促成双方首次建立联系（含成立专属沟通群组）之时间 |
| Close（成单与签约） | 谈成签约 | 以双方签署之报价单为准 |
| Deliver（服务与维护） | 服务交付 | 客户端问题回复与操作协助皆由你完成，无需 Zeabur 工程团队介入 |
| Collect（资金与发票） | 催收资金 | 由你完成收款、开具发票，无需 Zeabur 提醒或追账 |

### 常见参与组合

| 情境 | Lead | Close | Deliver | Collect |
| --- | :---: | :---: | :---: | :---: |
| 全包 | ✓ | ✓ | ✓ | ✓ |
| 纯介绍 | ✓ |   |   |   |
| 介绍 + 服务 | ✓ |   | ✓ |   |
| 服务 + 收款 |   |   | ✓ | ✓ |
| 签约 + 代操（无 Lead） |   | ✓ | ✓ | ✓ |
| 介绍 + 服务 + 收款 | ✓ |   | ✓ | ✓ |

### 多方拆账

同一笔交易可由不同合作伙伴负责不同阶段。交易层封闭计算、不跨池，各方独立结算、互不影响。

## 配套规则

- **结算币别**：以 USD 为基准结算，汇兑成本由销售伙伴自行承担。
- **结算周期**：分润以月为单位计算，经双方确认后入账至 Partner Portal。
- **提现等待期**：分润入账后需经过 holding 期方可提现。等待期内客户若 churn 会自动扣除，期满后已提现的部分不予追回。具体天数依合同约定。
- **最低销售承诺**：合作期间设有最低销售承诺，连续未达时 Zeabur 有权调整费率或终止合同。具体金额依合同约定。
- **Lead 认定**：Lead 阶段需经 Zeabur 确认后方能锁定。
- **阶段锁定**：Deal 登记时锁定各阶段责任归属，变更需双方书面同意。
- **分润存续**：客户持续付费期间持续计算，客户 churn 即停。

<Callout type="warning">
销售伙伴路径与 Referral Program 不可套用于同一笔交易——每笔交易择一，以避免双重利益冲突。
</Callout>

## 合作流程

1. **联系 Zeabur** — 前往 [partner.zeabur.com](https://partner.zeabur.com) 申请，或通过 [Zeabur 社群论坛](https://zeabur.com/forum) 联系团队，说明你的客户背景与预估合作规模。
2. **洽谈合同** — Zeabur 团队评估后与你洽谈具体费率、阶段权重与配套条件。个别合同内容保密。
3. **签署合同** — 确认条款后正式签约，开始追踪 Deal。
4. **客户导入** — 协助客户完成评估、测试与上线。
5. **月结分润** — 客户付款成功后，根据合同条件进行月结分润。

## 相关附约

部分合作会搭配独立合同或附约处理，例如：

- **资金代收代付服务费**：若你担任代收代付角色，相关手续费走独立合同。
- **教育补助**：若合作包含学校或教育机构的教育补助，以独立附约/MOU 处理。

## 如何申请

- 也可在 [Zeabur 社群论坛](https://zeabur.com/forum) 联系 Zeabur 团队说明合作意愿。
