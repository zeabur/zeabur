# 教育推广

教育推广计划面向**讲师、培训机构和活动主办方**，通过教学和技术活动将 Zeabur 介绍给学员，并依据留存转换为付费用户的数量获得奖励。

## 合作模式

1. **申请合作** — 联系 Zeabur 团队，说明你的课程或活动计划。
2. **取得 Event Code** — Zeabur 为你的活动创建专属 Event Code，学员使用此 Code 可免费体验 Zeabur 服务。
3. **进行教学 / 活动** — 在课程或活动中带领学员使用 Zeabur 部署项目。
4. **统计留存用户** — 活动结束一个月后，Zeabur 统计通过你的 Event Code 注册且持续付费的用户数。
5. **领取奖励** — 每位留存付费用户奖励 US$5。

## Event Code 机制

Event Code 是一组专属代码，提供学员在活动期间使用 Zeabur 的体验额度：

- 每个 Event Code 与特定活动绑定。
- 学员在 Zeabur 注册时输入 Event Code 即可获得体验额度。
- Event Code 有效期依活动安排而定，通常涵盖活动期间及之后一段缓冲时间。

<Callout type="info">
Event Code 的兑换不会触发推荐计划的佣金机制——因为 Event Code 提供的是体验额度，而非真实金流。推荐佣金仅在用户实际付款成功时才会计算。
</Callout>

## 奖励计算

| 项目 | 说明 |
| --- | --- |
| 统计时间 | Event Code 发放后一个月 |
| 留存标准 | 学员通过 Event Code 注册，且在统计期间内升级为付费用户并持续使用 |
| 奖励金额 | 每位留存付费用户 US$5 |
| 发放方式 | Zeabur Credits 或其他约定方式 |

### 范例

你在一场 50 人的工作坊中发放 Event Code。一个月后，有 8 位学员持续使用 Zeabur 并升级为付费用户。你的奖励为 8 × US$5 = **US$40**。

## 适用场景

- **程序设计课程** — 带学员部署课程项目到云端。
- **技术工作坊** — 现场实作活动，学员即时体验部署流程。
- **Hackathon** — 参赛团队使用 Zeabur 快速部署作品。
- **线上课程** — 提供 Event Code 让远端学员实操练习。

## 申请方式

请通过以下方式联系 Zeabur 团队申请教育推广合作：

- **Discord**：前往 [Zeabur Discord 服务器](https://zeabur.com/dc) 开启 ticket，说明你的活动计划。
- 提供活动名称、预计人数、时间和形式，Zeabur 团队将为你安排 Event Code。
