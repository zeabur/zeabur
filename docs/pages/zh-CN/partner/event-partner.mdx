# 教育伙伴（Education Partner）

教育伙伴计划面向**讲师、社群领袖、Hackathon 主办方与教育机构**。当你举办技术活动、开设教学课程或带领工作坊时，可以通过活动码（Event Code）让参与者免费体验 Zeabur，并根据活动后留存的付费用户数量获得奖励。

## 合作模式

<Steps>

### 申请合作

前往 [partner.zeabur.com](https://partner.zeabur.com) 申请，说明你的课程或活动计划。

### 获取活动码

Zeabur 为你的活动创建专属活动码，参与者输入后可获得体验额度。

### 开展活动

在课程、工作坊或 Hackathon 中带领参与者使用 Zeabur 部署项目。

### 统计留存

活动码发放期结束后一个月，统计通过你活动码注册且持续付费的用户。

### 领取奖励

根据留存付费用户人数发放奖励。

</Steps>

## 活动码机制

活动码是一组专属代码，提供参与者在活动期间使用 Zeabur 的体验额度：

- 每个活动码与特定活动绑定。
- 参与者可通过活动注册链接（`https://zeabur.com/events?code=...`）完成注册后自动兑换，或在登录后于账单页面手动兑换以获得体验额度。
- 活动码有使用期限，通常涵盖活动期间及之后一段缓冲时间。
- 参与者兑换活动码的完整步骤请参考 [兑换活动码](/partner/event-partner/redeem-event-code)。

<Callout type="info">
活动码兑换不会触发 Referral Program 的佣金机制——活动码提供的是体验额度而非真实付款，推荐佣金仅在用户实际付款成功时才会计算。
</Callout>

## 奖励计算

| 项目 | 说明 |
| --- | --- |
| 统计时间 | 活动码发放期结束后一个月 |
| 留存标准 | 参与者通过活动码注册，且在统计期间内升级为付费用户并持续使用 |
| 基本奖励 | 每位留存付费用户 US$5 |
| 发放方式 | Zeabur Credits 或其他约定方式 |

### 示例

你在一场 50 人的工作坊中发放活动码，活动后一个月有 8 位参与者持续使用 Zeabur 并升级为付费方案。你的奖励为 8 × US$5 = **US$40**。

### 大型活动另案计算

针对大型 Hackathon、长期课程或影响力较高的系列活动，Zeabur 可根据活动规模、主办方背景与预估转化，将单位奖励调整至 **US$10 per 留存付费用户**。此类合作需事先与 Zeabur 团队讨论，并在申请时注明。

## 适用场景

- **编程课程** — 带领学员将课程项目部署到云端。
- **技术工作坊** — 现场实操活动，参与者实时体验部署流程。
- **Hackathon** — 参赛团队使用 Zeabur 快速部署作品。
- **在线课程** — 提供活动码让远程学员实操练习。
- **社群活动** — 技术社群聚会、分享会搭配 demo 环境。

## 如何申请

前往 [partner.zeabur.com](https://partner.zeabur.com) 创建账号并提交活动合作申请，填写活动名称、预计人数、时间与形式。Zeabur 团队审核后会协助建立专属活动码。

如有其他疑问，欢迎在 [Zeabur 社群论坛](https://zeabur.com/forum) 提问。
