# 集成服务

Zeabur 提供与第三方服务的内置集成，让您可以直接在项目中启用强大的后端功能 — 无需另外部署。

## 可用的集成服务

### [InsForge](/zh-CN/integrations/insforge/overview)

AI 原生的 Supabase 替代方案，让 AI 代理能够自主构建和管理全栈应用程序。包含 PostgreSQL 数据库、JWT 身份验证、S3 兼容文件存储、无服务器边缘函数，以及用于 AI IDE 集成的 MCP 协议支持。

- **类别**：后端 / BaaS
- **计费**：按用量计费，独立计费
- **网站**：[insforge.dev](https://insforge.dev)
