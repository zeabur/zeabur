# 用量与账单

InsForge 提供详细的使用量追踪，让您监控资源消耗和费用。

## 查看使用量

1. 在 Zeabur 面板中导航到您项目的 InsForge 集成页面
2. 向下滚动到 **使用量概览** 区域

### 概览选项卡

概览选项卡显示所选时间范围的使用量摘要：

- **最大数据库大小** — 数据库存储的峰值使用量
- **最大文件存储** — 文件存储的峰值使用量
- **总流量** — 消耗的总出站带宽
- **EC2 计算** — 使用的总计算时数
- **AI Credits** — 消耗的 LLM AI 点数
- **AI Tokens** — 处理的 AI token 总数
- **MCP 调用** — MCP 协议调用总数
- **电子邮件请求** — 电子邮件 API 请求总数
- **函数调用** — 无服务器函数执行总数

### 每日使用量选项卡

每日使用量选项卡提供按日期分类的资源使用量明细，以分页表格显示每天的所有指标。

### 时间范围筛选

您可以按以下范围筛选使用量数据：
- **1 天** — 过去 24 小时
- **7 天** — 过去一周
- **30 天** — 过去一个月
- **全部** — 所有可用数据

## 账单

<Callout type="warning">
  InsForge 为独立计费服务，不包含在 Zeabur 免费额度中。由于 InsForge 是云端托管服务，即使您的 Zeabur 项目部署在专用服务器上仍会产生费用。
</Callout>

### 定价

| 项目 | 费率 |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / 小时 |
| Database Size | $0.125 / GB / 月 |
| Storage | $0.021 / GB / 月 |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / credit |

### 账单重要信息

- **没有人工限制** — 不存在 API rate limits、Row limits 或 Max User 上限。您只需为实际使用的计算和存储付费。
- **统一账单** — 所有 InsForge 成本会直接汇总到您既有的 Zeabur 项目账单中。一个平台，一张账单。
- **按用量计费** — 您只需要为后端实际消耗的资源付费。

### 账单问题

如果您的 InsForge 实例因账单问题而暂停：

1. 在 Zeabur 面板中前往 **账单**
2. 绑定付款方式或充值余额
3. 返回 InsForge 集成页面，点击 **恢复** 以重新扣款并恢复服务

<Callout type="info">
  如果您的实例因账单问题而暂停，数据将被保留。解决账单问题后恢复实例即可恢复服务。
</Callout>
