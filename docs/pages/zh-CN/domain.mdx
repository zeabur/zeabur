# 域名注册

import { Callout } from 'nextra/components'

域名注册服务让你在 Zeabur 控制台中直接注册、管理和配置域名，无需第三方注册商。

## 功能

- **域名搜索** — 同时查询 9 个后缀（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）的可用性和实时价格
- **DNS 管理** — 完整的 DNS 编辑器，支持 A、AAAA、CNAME、MX、TXT、SRV、CAA、NS 记录及 Cloudflare 代理
- **自动和手动续费** — 为每个域名开关自动续费，或随时手动续费
- **注册人资料** — 保存 WHOIS 联系信息，在多次注册中复用
- **注册人验证** — 处理部分后缀购买后要求的邮件验证
- **Zeabur Email 集成** — 一键将域名接入 Zeabur Email，DNS 记录自动创建

<Callout type="info">
  域名购买需要 **Developer** 或 **Team** 方案。
</Callout>

## 开始使用

1. 在侧边栏打开 **域名** 页面
2. [注册域名](/domain/register) — 搜索、比较价格并购买
3. [管理域名](/domain/manage) — 查看状态、切换续费、接入 Zeabur Email
4. [配置 DNS 记录](/domain/dns) — 在详情页添加和编辑 DNS 记录
5. [设置注册人资料](/domain/profiles) — 保存联系信息以加快注册
