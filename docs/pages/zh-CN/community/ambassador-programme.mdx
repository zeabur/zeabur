# Zeabur 大使计划

跟我们一起，让 Zeabur 社群活起来。

大使是这个生态圈里让事情发生的一群人 — 你的每一个部署、每一篇分享、每一场活动，都让社群更活络、团队跟你靠得更近。

## 你会做的事

形式不限，社群里最常见的产出形式有：

| | |
| -- | -- |
| **内容创作** | 写文章、做视频、录 podcast 分享你用 Zeabur 做的事 |
| **活动主办** | 在你的城市办社群聚会、工作坊、黑客松 |
| **深度使用** | 把 Zeabur 用得深、回报问题、把真实案例分享出来 |
| **模板贡献** | 贡献高品质模板到模板市场以及社群内 |

## 你会拿到的

* 更进阶的产品使用
  * Pro 方案资格
  * 每月 30 美元 Zeabur 额度（含 Pro 方案）
  * 抢先使用未发布与测试版功能
* 更直接与团队交流
  * 大使专属交流空间 — 直通 Zeabur 团队
  * AMA — 跟 Zeabur 团队直接对话
  * 开发者咨询会 — 你的反馈直接影响产品路线图
* 更多品牌资源协助
  * 活动赞助
  * 大规模使用 / 高影响力的个案，跟 Zeabur 一对一洽谈加码合作空间

## 我们在找的人

不需要符合特定身份 — 但下列几种开发者最自然契合：

* **建立者 / 创作者** — 用 Zeabur 部署自己的作品、把过程记录下来分享
* **社群领袖** — 你已经在某个语区、Discord、Forum、或线下活动是节点
* **重度使用者** — 把 Zeabur 当正式环境的基础设施、深度地用

我们的要求：过去 90 天有部署记录、身份可被验证、同意[社群行为准则](/zh-CN/community/code-of-conduct)。

## 我们对你的期待

有额度，就有最轻量的门槛 — 形式不限，不订绩效指标。

半年内至少一次可公开验证的贡献，下列任一即可：

* 一篇文章、一支视频、一集 podcast
* 一场社群聚会、工作坊、讲座
* 一个上架的模板
* 一个公开的部署案例
* Forum / Discord 高品质深度解答累积五篇以上
* 被产品团队采纳的问题回报或功能建议
* 参与或共同设计一场社群活动并产出成果

## 常见问题

**大使计划跟推荐计划是什么关系？**

两者解耦 — 推荐计划对所有 Zeabur 用户平等开放，不是大使福利。高影响力大使若带来大量推荐，可在加码合作的框架下另案洽谈个案条件。

**可以同时担任其他平台的大使吗？**

可以。我们不要求独家 — 多元身份对开发者社群是有机而非冲突。

**没办法每场 AMA 都参加怎么办？**

没关系 — AMA 全程录影、可事后补看，鼓励出席但不强制。

**半年没做出贡献会发生什么？**

我们会主动联系了解 — 月度额度暂停、身份保留，恢复活动即复原；没有羞辱，没有强制退出。

**我可以提名朋友吗？**

可以。白名单制度期间由 Zeabur 主动邀请；正式上线后开放公开申请。

**对 Zeabur 的批评可以公开讲吗？**

可以。大使身份不是公关代言 — 开发者社群最值得保留的气息就是真实，批评、竞品比较、善意的提问与抱怨都欢迎。

## 申请

大使计划目前为白名单制度，由 Zeabur 主动邀请首批创始团队；正式上线后开放公开申请 — 届时此处更新申请入口。
