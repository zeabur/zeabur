# 寻求帮助

有些时候你可能会在使用 Zeabur 的过程中遇到一些问题，你可以通过以下渠道获得帮助。

## 社区论坛

[Zeabur 社区论坛](https://zeabur.com/forum) 是获取帮助的主要渠道。你可以创建帖子描述你的问题，我们的团队会根据你的方案支持优先级进行回复。详细的帖子创建方式和注意事项，请参考[社区论坛](./forum)页面。

![new post](/community/forum/new-post.png)

## Discord

你也可以加入[我们的官方 Discord 服务器](https://zeabur.com/dc)与其他用户和 Zeabur 团队进行讨论。

<Callout>
  Discord 是一个社区讨论平台。虽然 Zeabur 工作人员可能会参与并回答问题，但我们不负责回复那里的所有消息。如需有保证响应时间的官方支持，请改用[社区论坛](./forum)。
</Callout>

![server](/community/help/server.png)

## 升级方案

如果你需要更快的响应时间和专属支持，可以考虑升级你的方案。较高等级的方案在[社区论坛](./forum)上享有更短的保证响应时间。详情请参阅[方案](../billing/plans)页面。

## 电子邮件

你也可以通过电子邮件联系我们以获得支持或询问：

- **电子邮件：** [contact@zeabur.com](mailto:contact@zeabur.com)

为了获得更快的响应时间，我们建议使用[社区论坛](./forum)。不过，如果你偏好使用电子邮件沟通，欢迎随时发送消息给我们，我们会尽快回复。
