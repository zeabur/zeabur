# **寻求帮助**

在使用 Zeabur 时有时可能会遇到问题。您可以在我们的 [Zeabur 论坛](https://zeabur.com/forum) 中从 Zeabur 团队获得帮助。

![Zeabur Forum](/community/get-help/Zeabur-Forum.png)

## **如何获取帮助**

在我们的论坛中，有一个名为 `New Post` 的按钮，供大家创建问题工单。

![Zeabur Forum Overview](/community/get-help/zeabur-forum-overview.gif)

### **发帖格式**

遵循以下格式清晰描述问题，能帮助我们更快定位并解决问题。

1. 选择 **Issue Type**（问题类型）
2. 在 **Title**（标题）中简要描述问题。
3. 选择 **Project**（项目）或 **Server**（服务器） - 会列出您 Zeabur 账户中的所有项目/服务器
4. 在服务中选择出现问题的服务
5. 然后贴上日志中的错误消息。
6. 最后，添加内容上下文，帮助支持团队全面了解问题。

## 其他

![Others](/community/get-help/other-issues.png)

选择“Others”标签，尽量详细描述以获得 Zeabur 团队最高效的响应

### **优先支持**

- Team 计划：半个工作日内回复
- Dev 计划：3 个工作日内回复
- Free 计划：5 个工作日内回复

**其他联系方式**

虽然 Zeabur 论坛是我们的官方支持平台，但您也可以通过邮件联系我们进行支持或咨询：

- **邮箱：** [contact@zeabur.com](mailto:contact@zeabur.com)

为获得更快的响应时间和社区资源访问，我们推荐使用 Discord。但如果您更喜欢邮件沟通，随时发送消息，我们会尽快回复。
