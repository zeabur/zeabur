# 定价

Zeabur 提供四个订阅方案，以及可自定义的企业级选项：

- **[Free Plan](/pricing/free-plan)** — US$0/月
- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 天免费试用）
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 天免费试用，推荐用于生产环境）
- **[Team Plan](/pricing/team-plan)** — US$79/月含 3 个席位（每增加一个席位 +US$24/月）
- **Enterprise** — 自定义定价，[联系我们](https://zeabur.com/forum)了解详情
