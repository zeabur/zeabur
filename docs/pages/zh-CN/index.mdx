**开始使用**

**Zeabur** 是一个可以帮助你一键部署服务的平台，无论你使用什么编程语言或开发框架，都可以轻松完成部署。

你可以加入我们的 [Zeabur 论坛](https://zeabur.com/forum) 了解更多并获得优先支持！

你也可以加入我们的 [官方 Discord 服务器](https://zeabur.com/dc) 与社区交流想法并获得帮助。

### **Zeabur 的优势**

### **方便快捷**

与传统部署方式相比，Zeabur 让你无需花时间学习复杂的部署知识，就能在几分钟内完成部署，把时间与精力专注在产品开发上。

- AI Agent 自动识别项目类型
- 多家厂商、多地区的共享集群 / 专用服务器
- 友好直观的 UI
- 开箱即用的 CI/CD
- 自动生成域名 / 支持自带域名（BYOD）
- 多环境隔离
- Zeabur AI Hub
- 以及更多

### **一站式整合**

Zeabur 提供强大的一站式整合能力。

你可以在同一个项目中部署前端、后端、数据库、消息队列、对象存储等各类服务，无需分别部署到不同平台。

### 一键部署模板

Zeabur 在 [模板市场](https://zeabur.com/templates) 提供了大量模板。只需点击 `Deploy` 按钮，模板就会立即开始部署。几分钟内，无论是部署数据库还是像 n8n 这样的应用，都能准备就绪并开始使用。

更多详情可以查看 [Zeabur 模板页面](https://zeabur.com/templates)。

### Zeabur AI Hub

Zeabur AI Hub 是一个统一平台，旨在帮助开发者更轻松地使用强大的 AI 模型。你无需为不同供应商分别管理账号与 API Key，即可通过一个界面访问行业内最流行的模型。

- **统一接入**：在一个平台中使用 OpenAI（GPT 系列）、Anthropic（Claude 系列）、Google（Gemini）、xAI（Grok）以及开源模型（Llama、DeepSeek、Qwen）等主流模型。
- **单一 API Key**：使用一个简单的 API Key 即可调用任意支持的模型，省去管理多套凭据的复杂度。
- **兼容 OpenAI**：API 与 OpenAI API 格式完全兼容，可轻松集成到现有应用或工具（如 n8n、SillyTavern）中，无需修改代码。
- **全球低延迟**：在东京、旧金山等多个地区部署 API Endpoint，可选择最近的服务器以获得最佳性能。

你可以在 [Zeabur AI Hub 页面](https://zeabur.com/ai-hub) 了解更多并开始使用。

### **按量计费**

Zeabur 采用按量计费模式，这意味着你只需为服务实际使用到的资源付费，而不是为整台服务器付费。

你可以在[定价文档](https://zeabur.com/docs/zh-CN/billing/pricing)中了解更详细的内容。
