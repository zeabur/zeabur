# 服务使用量

Zeabur 采用按量计费模式，并提供详细图表展示你的服务使用情况。

点击你的服务，选择 **Metrics** 标签页，即可查看图表。

### 内存使用量

显示服务运行时消耗的内存。

![memory](/manage/metric/Memory_Usage.png)

### CPU 使用量

显示服务运行时调用的 CPU。

![cpu](/manage/metric/CPU_Usage.png)

### 网络使用量

显示 Zeabur 各区域的服务运行网络流量。

![network](/manage/metric/Network_Usage.png)

### 磁盘使用量

对于 Google Cloud Platform (GCP) 部署，磁盘使用量限制为 1 GB。对于其他所有平台，磁盘使用量采用按量付费模式，可根据你的存储需求灵活使用。

![disk](/manage/metric/disk_usage.png)
