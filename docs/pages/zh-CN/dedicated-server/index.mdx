# 专用服务器

除了共享集群，你也可以将自己的服务器**注册到 Zeabur**，让 Zeabur 替你管理服务器上的服务部署、防火墙规则和资源分配——无需 SSH 进去手动操作。

## 为什么使用专用服务器

- **完整资源独享**：服务不与其他用户共享硬件，资源稳定可预测。
- **自带主机（BYOH）**：已在 AWS、GCP、Hetzner 或自建机房购买服务器？一键挂载到 Zeabur，立即享用 Zeabur 的部署管理能力。
- **地理位置灵活**：在共享集群尚未覆盖的区域，用自己的服务器补足覆盖范围。
- **高稳定性生产环境**：对 SLA 与安全性有更高要求的大型系统。

## Zeabur 帮你管理什么

一旦服务器注册，你就能通过 Zeabur Dashboard 完成以下工作，无需直接操作主机：

- 部署和更新服务
- 查看实时日志与性能指标
- 设置环境变量
- 管理持久存储空间与网络设置

<Callout type="info">
  专用服务器按月计费（固定费用），而非按资源使用量计费。适合硬件需求稳定的生产工作负载。
</Callout>

## 开始使用

1. **[购买服务器](/zh-CN/dedicated-server/purchase)** — 从 Zeabur 购买专用服务器，享有优惠价格
2. **[新增服务器](/zh-CN/dedicated-server/register)** — 将现有服务器挂载到 Zeabur
3. **[管理服务器](/zh-CN/dedicated-server/operate)** — 设置防火墙、监控资源、调整设置

## 常见问题

### 使用专用服务器需要订阅方案吗？

不需要，你不需要订阅方案即可使用专用服务器。不过，我们建议订阅开发者方案以发挥 Zeabur 的完整潜力——备份等功能仅提供给开发者方案的订阅者。开发者方案用户还可获得来自工作人员的优先支持，而免费用户可以通过 [Zeabur 论坛](https://zeabur.com/forum)获得帮助，但没有保证的回复时间。详情请参阅[订阅方案](/billing/plans)。

### 我只想架设小型网站，需要同时购买方案和服务器吗？

你可以在没有订阅方案的情况下，在专用服务器上部署服务。不过，备份等特定功能仅限开发者方案的用户使用。如果你想完整使用 Zeabur 的平台功能和优先支持，我们建议在服务器之外订阅开发者方案。更多信息请参阅[社区与支持](/community/forum)。
