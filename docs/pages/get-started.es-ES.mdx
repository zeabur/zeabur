# Empezar

## Paso 1: Registrarse

Para empezar, necesitas **crear una cuenta** en Zeabur.
Necesitarás una **cuenta de GitHub** para registrarte. Si no tienes una, puedes crear una [aquí](https://github.com/join).

Ve a la [página de inicio de sesión](https://zeabur.com/login), verás un botón para registrarte con GitHub.
Haz clic en el botón, y serás redirigido a la página de autorización de GitHub. Haz clic en el botón "Autorizar" para autorizar a Zeabur a acceder a tu cuenta de GitHub.

![iniciar sesión](/start/sign-in.png)

## Paso 2: Crear un Nuevo Proyecto

Un proyecto se presenta como un grupo de servicios. Puedes implementar múltiples servicios en un proyecto para que puedan comunicarse entre sí fácilmente.

Después de la autorización, serás redirigido automáticamente al panel de control de Zeabur.
Haz clic en el botón **Crear Proyecto** para crear un nuevo proyecto. También puedes escribir `CMD + k` para abrir un menú de acciones rápidas, donde puedes encontrar `Crear Proyecto` para continuar.

Verás un modal a continuación para seleccionar la región de tu proyecto. Zeabur ahora admite 2 regiones, incluyendo las regiones `ap-east`, `us-west`, `eu-central`, `ap-northeast` en AWS. La región `asia-east` en GCP solo está disponible para usuarios del Plan Team.

Elige la región donde deseas implementar tu servicio, y crearemos un proyecto para ti con un nombre aleatorio, que puedes modificar más tarde en la configuración del proyecto.

![nuevo proyecto](/start/create-project.png)

<Callout>
  Si es la primera vez que creas un proyecto en Zeabur, es posible que necesites seguir
  las instrucciones para leer y aceptar nuestros términos de servicio para continuar.
</Callout>

## Paso 3: Implementar Servicios

Haz clic en el proyecto que acabas de crear, Zeabur abrirá una página de proyecto para ti.

![proyecto](/start/project.png)

Para implementar un servicio, simplemente haz clic en el botón **Agregar Servicio**. Y verás un modal que te permite seleccionar el tipo de servicio.

![seleccionar tipo de servicio](/start/select-service-type.png)

Zeabur admite 3 tipos de servicios, incluyendo **Servicios de Git**, **Servicios Preconstruidos** y **Servicios Once**.

- **Servicios de Git** te permite implementar tu código fuente desde GitHub. Zeabur construirá y implementará automáticamente tu código fuente por ti.
  Debes configurar tu acceso a GitHub la primera vez para que Zeabur pueda acceder a tus repositorios.

- **Servicios Preconstruidos** te permite implementar un servicio desde un mercado preconstruido de servicios comunes (como una base de datos o una herramienta de código abierto), puedes ver el mercado de código abierto [aquí](https://github.com/zeabur/zeabur/tree/main/prebuilts).

- **Servicios Once** te permite implementar un servicio que compraste en [Once](https://once.com), como [Campfire](https://once.com/campfire).

Simplemente elige el tipo de servicio que deseas, y puedes buscar el servicio que deseas implementar.
