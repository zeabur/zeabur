# 部署 Express 應用程式

[Express](https://expressjs.com/) 是一個一個輕便靈活的 Node.js 網路應用程式框架，為網路和行動應用程式提供了一套強大的功能。

在 Zeabur 上一鍵部署你的 Express 應用程式。

### 第一步：建立 Express 應用程式（可選）

- 電腦上安裝了 Node.js。
- 使用 express-generator 或手動建立。

```bash
# 使用 express-generator
npx express-generator
```

你可以在 Express 的[官方文件](https://expressjs.com/en/starter/installing.html)看到更詳細的教學。

建立完畢後，為其初始化一個 Git 儲存庫並上傳到你的 GitHub 上。

### 第二步：部署

在你的專案中，點選 **Add new service** 按鈕，並選擇 **Deploy your source code**。

![deploy](/deploy/common/select-service.en-US.png)

搜尋想要部署的 Express 應用程式儲存庫，並點選匯入，你的 Express 應用程式就將自動開始部署。
