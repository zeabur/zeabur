# 部署 Svelte Kit 專案

在 Zeabur 上部署你的 Svelte Kit 專案。

### 第一步: 建立 Svelte Kit 專案（可選）

打開終端機，輸入：

```bash
npm create svelte@latest
```

接下來，你需要安裝套件：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install
    ```
  </Tab>
  <Tab>
    ```bash
    npm install
    ```
  </Tab>
  <Tab>
    ```bash
    yarn
    ```
  </Tab>
</Tabs>

建立完畢後，為其初始化一個 Git 儲存庫並部署到你的 GitHub 上。

### 第二步：設定 `@sveltejs/adapter-node`

接下來，你需要安裝 `@sveltejs/adapter-node`：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D @sveltejs/adapter-node
    ```
  </Tab>
</Tabs>

然後，打開 `svelte.config.js`，並將其內容改為：

```js
import adapter from '@sveltejs/adapter-node'; // <-- 注意這裡

export default {
  kit: {
    adapter: adapter({ out: 'build' })
  }
}
```

### 第三步：設定 `package.json`

打開 `package.json`，在 `scripts` 中加入一個 `start` 指令：

```json
{
  "scripts": {
    "start": "node ./build/index.js"
  }
}
```

### 第四步：部署 Svelte Kit 專案

在你的專案中，點擊 **Deploy service** 或者 **Add new service** 按鈕，並選擇 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜尋想要部署的 Svelte Kit 專案儲存庫，並點擊匯入，你的 Svelte Kit 應用程式就將自動開始部署。
