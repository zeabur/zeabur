# 部署 Remix 專案

在 Zeabur 上部署你的 Remix 專案。

### 第一步: 建立 Remix 專案（可選）

環境要求：

- 確保 node 版本是 14 或以上

打開終端機，輸入

```bash
npx create-remix@latest
```

建立完畢後，為其初始化一個 Git 儲存庫並部署到你的 Github 上。

### 第二步：部署 Remix 專案

在你的專案中，點選 **Deploy service** 或者 **Add new service** 按鈕，並選擇 **Deploy your source code**.

![deploy](/framework/deploy.png)

搜尋想要部署的 Remix 專案程式碼儲存庫，並點選 `import` ，你的 Remix 專案就將自動開始部署。
