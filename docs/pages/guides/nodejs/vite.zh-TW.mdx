# 部署 Vite

在 Zeabur 上部署你的 Vite 專案。

### 第一步: 建立 Vite 專案（可選）

環境要求：

- Vite 需要 Node.js 版本 14.18+、16+。然而，有些模板需要依賴更高的 Node.js 版本才能正常運作，當你的套件管理工具發出警告時，請注意升級你的 Node.js 版本。

打開終端機，輸入：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
```bash
pnpm create vite
```
  </Tab>
  <Tab>
```bash
npx create vite@latest
```
  </Tab>
  <Tab>
```bash
yarn create vite
```
  </Tab>
</Tabs>

建立完畢後，為其初始化一個 Git 儲存庫並部署到你的 GitHub 上。

### 第二步：部署 Vite 專案

在你的專案中，點擊 **Deploy service** 或者 **Add new service** 按鈕，並選擇 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜尋想要部署的 Vite 專案儲存庫，並點擊匯入，你的 Vite 應用程式就將自動開始部署。
