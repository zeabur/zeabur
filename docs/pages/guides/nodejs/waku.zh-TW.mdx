# 在 Zeabur 上部署 Waku

[Waku](https://waku.gg/) 是一個支援 React Server Components 的最小化 React 框架。

Waku 在日語中的意思是「框架」，而 Waku-Waku 則意味著「興奮」。

本文將幫助你在 Zeabur 上部署 Waku 網站。

## 步驟一：建立一個 Waku 專案

### 方法一：使用 Waku CLI 建立專案

你可以使用下面的命令建立一個 Waku 專案並跟隨提示進行配置。

```bash
# npm
npm create waku@latest
# yarn
yarn create waku 
# pnpm
pnpm create waku
```

### 方法二：使用模板建立專案

我們為你準備了一個模板來幫助你快速開始 Waku 的開發：[Waku 模板](https://github.com/zeabur/waku-template)

你可以在 GitHub 頁面上點擊 **Use this template** 按鈕來使用模板。

## 步驟二：部署到 Zeabur

前往 [Zeabur 控制台](https://dash.zeabur.com) 並建立一個新專案。

然後，點選 **Deploy New Service** 按鈕，選擇從 GitHub 部署。

<Callout>
    如果你是第一次在 Zeabur 上從 GitHub 部署專案，你需要先配置並安裝 GitHub App。
    你可以在 [這裡](/deploy/github) 查看更詳細的指南。
</Callout>

當你選擇好儲存庫和分支後，Zeabur 將會自動開始建置你的服務。

Zeabur 會自動檢測到你的服務是由 Waku 建置的，所以你不需要做任何額外的配置。你的部署將會在幾分鐘內完成。

## 下一步

部署完成後，你可能需要為你的 Waku 網站配置網域。

打開服務頁面的 Domain 分頁，然後點選 Generate Domain 或 Custom Domain。

關於如何為你的服務綁定網域的更多資訊，請參考 [綁定網域](/deploy/domain-binding)。
