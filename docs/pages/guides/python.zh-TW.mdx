# 部署 Node.js 專案

Zeabur 支援眾多類型的 Node.js 專案，包含但不限於這些：

- [Flask](./python/flask)
- [Django](./python/django)
- FastAPI

如果你的原始碼底下有 `app.py`, `main.py` 等 entrypoint，Zeabur 也會自動啟動。

## Serverless

Zeabur 支援將 Python 部署成 serverless 專案。不過目前需要手動 opt-in。
請參考 [啟用 Serverless](/deploy/serverless) 頁面啟用 serverless 支援。
