# 部署 Fresh 專案

[Fresh](https://fresh.deno.dev/) 是一個現代全端網頁框架，適用於 JavaScript 和 TypeScript。

本文將教你如何在 Zeabur 一鍵部署 Fresh 應用程式。

## 建立 Fresh 應用程式（可選）

- 請先在你的電腦上安裝 Deno。
- 你可以在 [Deno 官方文件](https://deno.land/manual/getting_started/installation)中找到更詳細的說明。

利用此命令初始化一個 Deno 專案並且使用 Fresh 框架。

```bash
deno run -A -r https://fresh.deno.dev my-project
```

在 `main.ts` 中，將 `port: Deno.env.get("PORT")` 加入到 `start` 函數中，使應用程式能夠正確監聽 Zeabur 給定的 `PORT` 環境變數的埠。

```typescript
// in main.ts
await start(manifest, { plugins: [twindPlugin(twindConfig)], port: Deno.env.get("PORT")}, );
```

## 開發專案

執行 `deno task start` 以開始開發伺服器。

```bash
deno task start
```

接下來，你就可以開始修改程式碼來開發你的 Fresh 專案了。

當你完成時，為其初始化一個 GitHub 儲存庫，並上傳到你的 GitHub。

## 部署專案

在你的專案中，點選 **部署新服務**  按鈕，並選擇 **部署你的程式碼**。

![deploy](/deploy/common/select-service.en-US.png)

搜尋想要部署的 Fresh 專案儲存庫，並點選匯入，你的 Fresh 應用程式就將自動開始部署。

部署完成後，你可以參考[〈綁定網域〉](../../deploy/domain-binding)瞭解如何為你的 Fresh 專案綁定一個能夠讓外部造訪的公開網域。
