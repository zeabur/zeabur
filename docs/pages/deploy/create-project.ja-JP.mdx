# プロジェクトを作成

Zeaburでサービスをデプロイするには、まずプロジェクトを作成する必要があります。

以下の手順に従ってプロジェクトを作成できます。

## プロジェクトとは

プロジェクトは、一緒にデプロイされるサービスのコレクションです。同じプロジェクト内のサービスは内部ネットワークでお互いに通信できます。
異なる地域に複数のプロジェクトを作成し、各プロジェクトにさまざまな種類のサービスをデプロイできます。

さらに、プロジェクトの[予算を設定して使用量を管理](../manage/budget)したり、[他のチームメンバーを招待](../manage/invite)してサービスを管理したりすることができます。

## プロジェクトの作成方法

まず、[ダッシュボードのプロジェクトページ](https://dash.zeabur.com/projects)に移動する必要があります。

Zeaburを初めて使用する場合、プロジェクトがないと、最初のプロジェクトを作成するためのオンボーディングページが表示されます。

![プロジェクト作成のオンボーディング](/deploy/create-project/onboarding.png)

以前にプロジェクトを作成したことがある場合は、**プロジェクトを作成**ボタンをクリックするか、`CMD + K`または`Ctrl + K`を押してコマンドパレットを開き、**プロジェクトを作成**を選択します。
次に、プロジェクトを作成したい地域を選択すると、ランダムな名前でプロジェクトが自動的に作成されます。

![地域を選択](/deploy/create-project/select-region.png)

## 次のステップ

地域を選択すると、空のプロジェクトが表示されます：

![空のプロジェクト](/deploy/create-project/empty-project.png)

このプロジェクトにサービスをデプロイするには、**新しいサービスをデプロイ**ボタンをクリックするか、設定ボタンをクリックしてプロジェクト設定を管理し、チームメンバーを招待することができます。
