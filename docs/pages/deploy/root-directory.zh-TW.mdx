# 自訂根目錄

對於某些 Git 儲存庫，例如同時具有前後端的專案，你需要手動為其指明專案根目錄。

### 第一步：部署服務

參照[部署服務](/zh-TW/get-started)來部署你的應用程式。

### 第二步：更改專案根目錄

選擇部署好的服務，並進入設定頁面，下拉至 **Root Directory** 一欄。
