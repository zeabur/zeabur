# Template Catalog

Zeabur provides a rich template catalog that allows you to quickly deploy various applications and services.

Visit the [Zeabur Template Catalog](https://zeabur.com/templates) to browse all available templates.

## How to Use Templates

1. Find the template you need in the template catalog
2. Click the "Deploy" button
3. Follow the instructions to complete the configuration

You can also check out [Template Format](/template/template-format) for more details on creating your own templates.
