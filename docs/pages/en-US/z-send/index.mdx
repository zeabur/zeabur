# Zeabur Email

Zeabur Email is an enterprise-grade email sending API service provided by Zeabur, offering developers a simple, reliable, and high-performance email delivery solution.

## Core Features

- **Simple REST API** - Integrate email functionality with just a few lines of code
- **Instant & Scheduled Sending** - Send emails immediately or schedule them for later
- **Batch Sending** - Send large volumes of emails with automatic queue management
- **Attachment Support** - Support for various email attachment formats
- **Webhook Notifications** - Real-time delivery status updates (delivery, bounces, complaints)
- **Domain Management** - Support for custom sender domains
- **API Key Management** - Flexible permission control (read-only, send-only, full access)

## Key Features

### High Reliability
Built on AWS SES with 99.9% uptime guarantee, ensuring stable email delivery.

### Real-time Monitoring
Receive real-time email status updates via Webhooks:
- Send confirmation
- Delivery confirmation
- Bounce notifications
- Spam complaints

### Security
- HMAC-SHA256 signature verification
- API key permission isolation
- Domain DKIM/SPF verification
