# Domain Registration

import { Callout } from 'nextra/components'

Domain Registration lets you register, manage, and configure domains directly from the Zeabur dashboard — no third-party registrar needed.

## Features

- **Domain Search** — Check availability across 9 TLDs (`.com`, `.net`, `.org`, `.io`, `.dev`, `.app`, `.co`, `.me`, `.xyz`) with real-time pricing
- **DNS Management** — Full DNS editor supporting A, AAAA, CNAME, MX, TXT, SRV, CAA, NS records with Cloudflare proxy support
- **Auto & Manual Renewal** — Toggle auto-renewal per domain or renew manually at any time
- **Registrant Profiles** — Save WHOIS contact info and reuse across multiple registrations
- **Registrant Verification** — Handle post-purchase email verification for TLDs that require it
- **Zeabur Email Integration** — One-click setup to connect your domain to Zeabur Email with automatic DNS record creation

<Callout type="info">
  Domain purchase requires a **Developer** or **Team** plan.
</Callout>

## Getting Started

1. Open the **Domains** section from the sidebar
2. [Register a domain](/domain/register) — search, compare pricing, and purchase
3. [Manage your domains](/domain/manage) — view status, toggle renewal, connect to Zeabur Email
4. [Configure DNS records](/domain/dns) — add and edit DNS records from the detail page
5. [Set up registrant profiles](/domain/profiles) — save contact info for faster registration
