# Contribution Reward Program

To encourage developers to help Zeabur build a high-quality [template](https://zeabur.com/templates) ecosystem, we offer the "Contribution Reward Program" to provide free usage credits to template authors whose templates are actively used by the community.

*This program takes effect on June 1, 2026. Only templates created on or after this date are eligible.*

## How It Works

Rewards are based entirely on **unique user deployments** — the number of distinct users who have deployed your template. Creating a template alone does not earn any reward; only real usage counts.

### Reward Tiers

| Milestone | Reward | Cumulative |
|-----------|--------|------------|
| 50+ unique users | $10 | $10 |
| 100+ unique users | $20 | $30 |
| 500+ unique users | $60 | $90 |
| 2,000+ unique users | $150 | $240 |
| 10,000+ unique users | $300 | $540 |

Rewards are cumulative. For example, a template reaching 500+ unique users earns a total of **$90** ($10 + $20 + $60). Each milestone is paid out once per template. Deleting and recreating a template (including same upstream project, same functionality, or renamed variants) does not reset milestone progress.

### Template Requirements

To be eligible for rewards, your template must meet **all** of the following:

1. The template must not duplicate existing templates unless it is a new version or intended for a different scenario or purpose.
2. The template must include complete information, including the template name, description, cover image, introduction, and usage instructions.
3. The template introduction must have at least an English version.
4. The template must be deployable with one click. If extra steps are needed after deployment, those must be documented in the template introduction.

Refer to [Creating Templates from YAML](../template/template-format) to learn how to create templates.

## How to Claim Rewards

Once your template reaches a milestone, please let us know through [Zeabur Support](https://zeabur.com/support) and we will grant the corresponding credits.
