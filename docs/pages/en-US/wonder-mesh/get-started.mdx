# Get Started

Follow these steps to turn your device into a Zeabur server using Wonder Mesh.

## Prerequisites

<Callout type="warning">
  We recommend using a **fresh machine** to avoid conflicts with existing software.  
  If you have Tailscale or K3s already installed, the setup script may interfere with your existing configuration.
</Callout>

Your device must meet the following minimum requirements:

| Resource | Minimum |
|----------|---------|
| CPU | 2 cores |
| Memory | 2 GB |
| Disk | 40 GB |

## Step 1: Create a New Project

Go to the [Zeabur Dashboard](https://zeabur.com/projects) and create a new project.

## Step 2: Bind an External Server with Wonder Mesh

In your project, select **Bind External Server**, then choose **Wonder Mesh**. Configure your device's architecture (amd64 or arm64) and operating system (Linux or macOS).

For macOS, you will also need to specify the virtual machine resource allocation:
- CPU cores
- Memory (GB)
- Disk size (GB)

After confirming, you will be redirected to the server page.

## Step 3: Run the Install Script on Your Device

On the server page, click **View Install Command** to reveal the full install command. Copy it and run it in a terminal on the device you want to register.

![View Install Command](/wonder-mesh/view-install-command.png)

<Callout type="warning">
  On macOS, do **not** use `sudo`. The installer will request elevated permissions when needed.
</Callout>

![Install Script Running](/wonder-mesh/install-script.png)

When the script finishes successfully, you should see output similar to:

``` {13-14}
=== Zeabur Mesh Server Setup ===
Installing Tailscale...
Tailscale installed: 1.94.2
Ensuring tailscaled is running...
Downloading wonder binary...
Joining Wonder Mesh Network...
Connecting to Wonder Mesh Net...
Successfully joined Wonder Mesh Net!
SSH server already running, skipping installation...
Enabling SSH password authentication...
Configuring SSH user...
Completing registration...
=== Zeabur Mesh Server Setup Complete ===
Mesh IP: 100.x.x.x
```

<Callout type="info">
  Want to know exactly what the install script changes on your machine?  
  See [What Does the Install Script Do?](./wonder-mesh/install-script-details)
</Callout>

## Step 4: Wait for the Server to Connect

After the installation completes, your device will appear as connected in the console.

![Server Connected](/wonder-mesh/server-connected.png)

## Step 5: Install K3s

Once the device has joined the mesh network, click **Install K3s** in the console. K3s is a lightweight Kubernetes distribution — after it is installed, your device is ready to receive and run service deployments.

## Step 6: Start Deploying

When the server shows a healthy status, you can deploy services to it just like any other Zeabur server.

![Server Healthy](/wonder-mesh/server-healthy.png)

## Reinstallation

If you need to re-setup a device, go to the server's page in the console, regenerate the install command, and run it again on the device.
