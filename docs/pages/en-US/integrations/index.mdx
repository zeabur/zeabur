# Integrations

Zeabur provides built-in integrations with third-party services, allowing you to enable powerful backend capabilities directly within your project — no separate deployment required.

## Available Integrations

### [InsForge](/en-US/integrations/insforge/overview)

An agent-native Supabase alternative that lets AI agents build and manage full-stack applications autonomously. Includes PostgreSQL database, JWT authentication, S3-compatible file storage, serverless edge functions, and MCP protocol support for AI IDE integration.

- **Category**: Backend / BaaS
- **Billing**: Pay-as-you-go, billed separately
- **Website**: [insforge.dev](https://insforge.dev)
