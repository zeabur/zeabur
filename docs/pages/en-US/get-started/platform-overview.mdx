# Platform Overview

**Zeabur** is an AI-powered cloud deployment platform that gets your applications live in minutes — no server management, CI/CD configuration, or infrastructure hassle required.

## Core Capabilities

### Automated Deployment

- **Auto-detect languages and frameworks** — Push your code and Zeabur automatically identifies Node.js, Python, Go, Java, .NET, Rust, and more, choosing the optimal build strategy.
- **Out-of-the-box CI/CD** — Every `git push` triggers a build and deployment automatically, zero configuration needed.
- **Template Marketplace** — Hundreds of pre-configured templates to one-click deploy WordPress, n8n, PostgreSQL, and other popular services.

### AI Assistant

Zeabur has a built-in AI Assistant (Agent) that lets you describe what you need in plain language. It will automatically create projects, deploy services, configure environment variables, and bind domains.

### AI Hub

Through [AI Hub](/ai-hub), you can directly call APIs for GPT-4o, Claude, Gemini, and other leading LLMs using an OpenAI-compatible format — no need to sign up for each provider separately.

### All-in-One Integration

Deploy your frontend, backend, databases, message queues, and object storage in a single project — no need to scatter services across different platforms. Services can communicate via private networking, and environment variables can reference each other.

### Multi-Environment Isolation

Each project supports multiple environments (e.g., Production, Staging), fully isolated from each other for streamlined development and testing workflows.

### Flexible Infrastructure

- **Shared Cluster** (being phased out) — Great for getting started quickly, no server management needed. New projects can no longer be created on shared clusters.
- **Dedicated Server** — Bring your own machine (BYO Server) with full control. See [Dedicated Server](/dedicated-server).
