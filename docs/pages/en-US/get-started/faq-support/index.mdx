# FAQ & Support

## Get Help

If you encounter issues while using Zeabur, there are several ways to get help:

- [Discord Community](https://zeabur.com/dc) - Join our Discord server to interact with the community
- [Forum](https://github.com/orgs/zeabur/discussions) - Ask questions in GitHub Discussions
- [Community Verification](/get-started/faq-support/verify) - Learn how to verify your community identity

## Frequently Asked Questions

We are collecting and organizing frequently asked questions. If you have specific questions, please contact our support team.
