# Team Plan

The Team Plan is Zeabur's collaborative plan designed for teams. It supports multi-member management and unified billing, ideal for teams that need multiple people managing projects together.

## Plan Details

| Item | Details |
| --- | --- |
| Billing | Usage-based |
| Projects | Unlimited |
| Services | Unlimited |
| Custom domains | Supported |
| Persistent storage | Supported |
| Deployment regions | All available regions |
| Members | Multiple (by seat count) |
| Invoicing / Tax ID | Supported |

### Seat Model

The Team Plan supports multi-member collaboration. Member count is based on seats — see [zeabur.com/pricing](https://zeabur.com/pricing) for specific rates.

### Rates

Team Plan resource rates are the same as the Dev Plan. See [Plans Overview](/pricing/pricing-plans) for details.

## Invoicing & Tax ID

The Team Plan supports invoice issuance and tax ID (統一編號) configuration, making it suitable for businesses and organizations that need receipts for accounting. See [Invoicing](/subscription/invoicing) for details.

## Dev Plan vs. Team Plan

| Feature | Dev Plan | Team Plan |
| --- | --- | --- |
| Billing | US$5/month + usage | Usage-based |
| Members | 1 | Multiple (by seat) |
| Member permissions | N/A | Supported |
| Invoicing / Tax ID | Not supported | Supported |
| Best for | Individual developers | Teams / Organizations |

## Use Cases

- **Startup teams** — Multiple developers jointly managing deployment environments.
- **Enterprise internal projects** — Teams that need unified billing and invoicing.
- **Agencies** — Create separate projects for different clients under unified team management.
- **Open-source communities** — Multiple maintainers collaborating on service deployments.

## How to Subscribe

Go to [zeabur.com/pricing](https://zeabur.com/pricing) and select Team Plan, or upgrade from the [Dashboard billing page](https://dash.zeabur.com/account/billing). Credit card and Alipay are supported — see [Payment Methods](/subscription/payment-methods) for details.
