# Dev Plan (Developer Plan)

The Dev Plan is Zeabur's paid plan built for solo developers who want AI Agent, database backups, and domain management. At **US$5/month**, it unlocks the core productivity features for individual builders.

<Callout type="info">
Eligible for the **14-day free trial** — same features as paid, no charge until the trial ends. See [14-Day Free Trial](/billing/subscription#14-day-free-trial).
</Callout>

## Plan Details

| Item | Details |
| --- | --- |
| Monthly fee | US$5 / month |
| Bound external servers | Up to 3 |
| Build CI spec | 2C4G |
| Log retention | 7 days |
| File upload limit | Medium |
| Container file management | Single file upload |
| Zeabur AI (AI Agent) | Included |
| Purchase domain | Supported |
| Email system | Included |
| Database backup export | Supported |
| Members | 1 (account owner only) |
| Deployment regions | All available regions |

## What's included (vs. Free)

Everything in Free, plus:

- Up to **3 bound external servers** (Free: 1)
- **7 days** of log retention (Free: 48 hours)
- **Zeabur AI** — AI-powered agent for managing your services
- **Purchase domain** directly through Zeabur
- **Email system** — send transactional emails
- Higher file upload limits

## Resource Rates

Shared cluster usage is billed per-minute. See [Billing / Pricing](/billing/pricing) for the full table.

- **vCPU** — Free
- **Memory** — US$0.00025 / GB-minute (~US$10.8 / GB-month)
- **Network egress** — US$0.1 / GB
- **Persistent storage** — US$0.2 / GB-month

## Use Cases

- **Personal side projects** — Deploy personal websites, blogs, API services.
- **AI-assisted workflows** — Let Zeabur AI manage and troubleshoot your services.
- **Small production services** — Low-traffic services that need to run 24/7.
- **Template development** — Build and maintain Zeabur templates with [Deploy Buttons](/deploy/deploy-button).

## How to Subscribe

Go to [zeabur.com/pricing](https://zeabur.com/pricing) and select Dev Plan, or upgrade from the [Dashboard billing page](https://dash.zeabur.com/account/billing). Credit card and Alipay are supported — see [Payment Methods](/subscription/payment-methods) for details.
