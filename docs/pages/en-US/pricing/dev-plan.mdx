# Dev Plan (Developer Plan)

The Dev Plan is Zeabur's entry-level paid tier, designed for individual developers. At **US$5/month** with a **14-day free trial**, it unlocks Zeabur AI, automated database backups, domain purchase, and the Zeabur Email service — ideal for personal production projects and side projects.

For the full feature list and the latest quotas, see the [comparison chart](/pricing).

## Use Cases

- **Personal side projects** — Deploy personal websites, blogs, API services, etc.
- **Development & testing** — Environments that need persistent storage (e.g., databases) and automated backups.
- **Small production services** — Low-traffic services that need to run 24/7.
- **Template development** — Create and maintain Zeabur templates with [Deploy Buttons](/deploy/deploy-button) to promote your projects.

## When to Upgrade to Pro

If you need **log forwarding**, **larger Email/Domain/API Key/Webhook quotas**, **custom service icons**, or **priority support**, the [Pro Plan](/pricing/pro-plan) (US$19/month, 14-day free trial) is the next step. See the [Pro Plan](/pricing/pro-plan) page for details.

## How to Subscribe

Go to [zeabur.com/pricing](https://zeabur.com/pricing) and select Dev Plan, or upgrade from the [Dashboard billing page](https://dash.zeabur.com/account/billing). Credit card and Alipay are supported — see [Payment Methods](/subscription/payment-methods) for details.
