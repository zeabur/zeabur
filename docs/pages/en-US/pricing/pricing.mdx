# Pricing

Zeabur offers four subscription tiers plus a custom Enterprise option:

- **[Free Plan](/pricing/free-plan)** — US$0/month
- **[Dev Plan](/pricing/dev-plan)** — US$5/month (14-day free trial)
- **[Pro Plan](/pricing/pro-plan)** — US$19/month (14-day free trial)
- **[Team Plan](/pricing/team-plan)** — US$79/month for 3 seats (+US$24/seat)
- **Enterprise** — [contact us](https://zeabur.com/forum)

For the full feature comparison and refund policy, see [Pricing Plans](/pricing/pricing-plans). For the latest pricing, visit [zeabur.com/pricing](https://zeabur.com/pricing).
