# Free Plan

The Free Plan is Zeabur's entry-level tier, designed for users trying out the Zeabur deployment flow for the first time. No credit card required — just sign up and start deploying.

## Included Resources

| Resource | Limit |
| --- | --- |
| Monthly fee | $0 |
| Bound external servers | 1 |
| Build CI spec | 2C4G |
| Log retention | 48 hours |
| File upload limit | Low |
| Container file management | Single file upload |
| Database view | Supported |
| AI Agent (Zeabur AI) | Not included |
| Deployment regions | Shared cluster regions |

## Resource Rates

Shared cluster usage is billed per-minute and uses the same rates as paid plans:

- **vCPU** — No cost ($0.0 / vCPU-minute)
- **Memory** — US$0.00025 / GB-minute (~US$10.8 / GB-month)
- **Network egress** — US$0.1 / GB
- **Persistent storage** — US$0.2 / GB-month

See [Billing / Pricing](/billing/pricing) for full details.

## Limitations

- **No Zeabur AI / AI Agent** — The AI-powered operations experience is only available on paid plans.
- **Limited log retention** — Only 48 hours of logs are kept on Free.
- **Limited server binding** — Only 1 external server can be bound on Free.
- **No SLA** — The Free plan does not include a Service Level Agreement.
- **Community support only** — Priority technical support requires the Team plan.

## Upgrade Path

When your needs exceed the Free Plan, pick a paid plan:

- **[Dev Plan](/pricing/dev-plan)** — US$5/month. Solo developers who want AI Agent, database backups, domain management, and email system.
- **[Pro Plan](/pricing/pro-plan)** — US$19/month. Higher build specs, 30-day log retention, advanced observability, HA deployment, and IP access control.
- **[Team Plan](/pricing/team-plan)** — US$79/month (3 seats included). Unlimited bound servers, team collaboration, role management, and priority support.
- **Enterprise** — Custom contracts, dedicated resources, compliance support — [contact us](https://zeabur.com/forum).

Visit [zeabur.com/pricing](https://zeabur.com/pricing) to compare all plans.
