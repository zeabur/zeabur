# Free Plan

The Free Plan is Zeabur's free tier, designed for personal exploration and learning. No credit card required — just sign up and start deploying.

## Plan Details

| Item | Free Plan |
| --- | --- |
| Monthly fee | US$0 |
| External server bindings | 1 |
| Build CI specs | 2C4G |
| Log retention | 48 hours |
| Container file management | Single-file upload |
| Database viewer | Supported |
| Persistent storage | Supported |
| Custom domain binding | Supported |
| Domain purchase (registrar) | Not supported |
| Database backup (export / import) | Not supported |
| Log forwarding | Not supported |
| Zeabur AI | Not supported |
| Zeabur Email — daily quota | 0 |
| Zeabur Email — monthly quota | 0 |
| Domains | 0 |
| API Keys | 0 |
| Webhooks | 0 |
| Forum posting | Limited (basic) |
| Priority support | Not included |
| SLA | Not included |

## Limitations

- **Auto-sleep** — Services on the Free Plan automatically sleep after a period of inactivity. They wake up on the next incoming request, which may cause a few seconds of cold-start latency.
- **No SLA** — The Free Plan does not include a Service Level Agreement.
- **No Zeabur Email** — Sending email through [Zeabur Email](/email) is not available on the Free Plan.
- **No automated database backup/import or log forwarding** — these advanced operational features start at the Dev Plan.

<Callout type="info">
Services on the Free Plan auto-sleep when idle to conserve resources. If you need services to run continuously, consider upgrading to the [Dev Plan](/pricing/dev-plan).
</Callout>

## Upgrade Path

When your needs exceed the Free Plan, you can upgrade to a paid plan:

- **[Dev Plan](/pricing/dev-plan)** — US$5/month (14-day free trial), ideal for individual developers with personal production projects. Adds Zeabur AI, database backup, domain purchase, and Zeabur Email.
- **[Pro Plan](/pricing/pro-plan)** — US$19/month (14-day free trial), recommended for production workloads. Adds log forwarding, advanced quotas, custom service icons, and priority support.
- **[Team Plan](/pricing/team-plan)** — US$79/month for 3 seats (+US$24/seat), designed for team collaboration with role and permission management.
- **Enterprise** — Custom plans with dedicated support and SLAs — [contact us](https://zeabur.com/forum) for details.

Visit [zeabur.com/pricing](https://zeabur.com/pricing) to compare all plans.
