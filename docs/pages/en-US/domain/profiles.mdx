# Registrant Profiles

Domain registration requires WHOIS contact information. Registrant profiles let you save this information once and reuse it across multiple domain purchases.

## Profile Fields

| Field | Required | Description |
|-------|----------|-------------|
| First Name | Yes | Registrant's first name |
| Last Name | Yes | Registrant's last name |
| Email | Yes | Contact email (also used for registrant verification) |
| Phone | Yes | Contact phone number |
| Address Line 1 | Yes | Street address |
| Address Line 2 | No | Apartment, suite, etc. |
| City | Yes | City |
| State / Province | Yes | State or province |
| Country | Yes | Country (select from dropdown) |
| Postal Code | Yes | ZIP or postal code |
| Organization | No | Company or organization name |

## Creating a Profile

<Steps>

### Open the Profiles Tab

Navigate to the **Domains** section in the sidebar and select the **Profiles** tab.

### Fill in the Form

Click **Create Profile** and fill in the required fields. Select a country from the searchable dropdown.

### Save

Click **Save**. The new profile is immediately available for use when registering domains.

</Steps>

## Default Profile

Set one profile as **default** by clicking the star icon. The default profile is automatically pre-selected in the registrant profile selector during domain purchase. You can still switch to a different profile at checkout.

## Editing and Deleting

- Click the **edit icon** on a profile card to update its fields.
- Click the **delete icon** to remove a profile.

<Callout type="warning">
  Deleting a profile does not affect domains already registered with it. Existing domains retain the contact information that was submitted at the time of registration.
</Callout>

## Inline Profile Creation

You can also create a new profile directly from the domain purchase dialog without navigating to the Profiles tab. The new profile is saved to your account and can be reused for future registrations.
