# Education Partner

The Education Partner program is for **instructors, community leaders, hackathon organizers, and educational institutions**. When you host a technical event, teach a course, or lead a workshop, you can use an Event Code to give attendees free Zeabur credits, and earn rewards based on how many attendees retain as paying users after the event.

## How It Works

<Steps>

### Apply

Sign up at [partner.zeabur.com](https://partner.zeabur.com) and describe your course or event plan.

### Receive an Event Code

Zeabur creates a dedicated Event Code for your event, which you can distribute to attendees.

### Run your event

Guide attendees through deploying projects on Zeabur during your course, workshop, or hackathon.

### Retention count

One month after the Event Code distribution period ends, Zeabur counts users who registered with your Event Code and are still paying.

### Claim rewards

Earn rewards based on the number of retained paying users.

</Steps>

## Event Code Mechanism

An Event Code is a unique code that provides attendees with trial credits during an event:

- Each Event Code is tied to a specific event.
- Attendees receive trial credits by visiting the event sign-up link (`https://zeabur.com/events?code=...`) or by redeeming the code on the Billing page after logging in.
- Event Codes have a validity period, typically covering the event and a buffer period afterward.
- See the [Redeem Event Code](/partner/event-partner/redeem-event-code) guide for the full redemption steps you can share with attendees.

<Callout type="info">
Event Code redemptions do not trigger the Referral Program's commission mechanism — Event Codes provide trial credits, not real payments. Referral commissions only apply when a user's actual payment succeeds.
</Callout>

## Reward Calculation

| Item | Details |
| --- | --- |
| Counting period | One month after the Event Code distribution period ends |
| Retention criteria | Attendee registered via Event Code, upgraded to a paid plan, and is still active during the counting period |
| Base reward | US$5 per retained paying user |
| Payout method | Zeabur Credits or other agreed methods |

### Example

You host a workshop for 50 attendees and distribute an Event Code. One month later, 8 attendees are still using Zeabur and have upgraded to a paid plan. Your reward is 8 × US$5 = **US$40**.

### Larger Events, Higher Rewards

For larger hackathons, extended course series, or high-impact events, Zeabur may adjust the reward to **US$10 per retained paying user** based on event scale, organizer reach, and expected conversion. These arrangements are discussed with the Zeabur team in advance and noted on your application.

## Ideal Scenarios

- **Programming courses** — Students deploy course projects to the cloud.
- **Technical workshops** — Hands-on events where attendees experience the deployment flow in real time.
- **Hackathons** — Teams use Zeabur to rapidly deploy their projects.
- **Online courses** — Provide Event Codes for remote students to practice.
- **Community events** — Tech meetups and talks paired with demo environments.

## How to Apply

Sign up at [partner.zeabur.com](https://partner.zeabur.com) and submit an event partnership application with the event name, expected attendance, dates, and format. The Zeabur team will review and set up a dedicated Event Code for you.

For other questions, feel free to post on the [Zeabur community forum](https://zeabur.com/forum).
