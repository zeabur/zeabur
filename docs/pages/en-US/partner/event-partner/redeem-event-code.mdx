# Redeem Event Code

Event Codes are distributed by Zeabur Education Partners at talks, courses, workshops, or hackathons. They can be redeemed for trial credits on Zeabur, which can be used to upgrade a subscription plan, deploy projects, and access other paid features.

This page covers the two ways to redeem an Event Code. Event organizers can share this link directly with attendees to avoid answering the same questions repeatedly.

<Callout type="info">
Event Codes have an expiration date — redeeming too early or too late will fail. Note the validity period given by your event organizer.
</Callout>

## Method 1: Event Sign-up Link (Easiest, for users without a Zeabur account)

If you don't have a Zeabur account yet, use the event sign-up link. Append your Event Code to this URL:

```
https://zeabur.com/events?code=YOUR_EVENT_CODE
```

For example, if your Event Code is `i_love_zeabur`, open:

```
https://zeabur.com/events?code=i_love_zeabur
```

Follow the on-page flow to complete Zeabur registration. The event credits are redeemed into your new account automatically.

## Method 2: Manual Redemption via Billing Page (For users with an existing account)

<Steps>

### Log in to Zeabur

Log in at [zeabur.com/login](https://zeabur.com/login).

### Go to Billing

Click your avatar in the top-right and choose "Billing", or go directly to [zeabur.com/account/billing](https://zeabur.com/account/billing).

### Click "Redeem Credits"

Find the "Redeem Credits" button in the upper-right corner of the "Account Balance" section and click it.

### Enter the Event Code

Enter your Event Code in the dialog, then click "Redeem". On success, you'll see a notification in the bottom-right corner and your account balance will update immediately.

</Steps>

## Upgrade to a Paid Plan (Optional)

If you need to deploy to specific regional servers or unlock subscription plan features, after redeeming you can [upgrade on the Pricing page](https://zeabur.com/pricing#upgrade). Upgrade charges are deducted from your account balance first.

<Callout type="info">
Event Code redemptions are typically sufficient to upgrade a subscription plan; actual available plans depend on the credit amount. Check with your event organizer if unsure.
</Callout>

## Common Issues

- **Expired Event Code** — Expired codes cannot be recovered. Confirm the validity period with your event organizer.
- **Invalid Event Code** — Double-check for typos and correct capitalization.
- **Other issues** — Contact customer support on the [Zeabur community forum](https://zeabur.com/forum).
