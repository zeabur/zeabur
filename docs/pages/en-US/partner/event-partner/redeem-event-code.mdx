# Redeem Event Code

Event Codes are distributed by Zeabur Education Partners at talks, courses, workshops, or hackathons. They can be redeemed for trial credits on Zeabur, which can be used to upgrade a subscription plan, deploy projects, and access other paid features.

This page covers the two ways to redeem an Event Code. Event organizers can share this link directly with attendees to avoid answering the same questions repeatedly.

<Callout type="info">
Event Codes have an expiration date — redeeming too early or too late will fail. Note the validity period given by your event organizer.
</Callout>

## Method 1: Event Sign-up Link (Easiest, for users without a Zeabur account)

If you don't have a Zeabur account yet, use the event sign-up link. Append your Event Code to this URL:

```
https://zeabur.com/events?code=YOUR_EVENT_CODE
```

For example, if your Event Code is `i_love_zeabur`, open:

```
https://zeabur.com/events?code=i_love_zeabur
```

Follow the on-page flow to complete Zeabur registration. The event credits are redeemed into your new account automatically.

## Method 2: Manual Redemption via Billing Page (For users with an existing account)

<Steps>

### Log in to Zeabur

Log in at [zeabur.com/login](https://zeabur.com/login).

### Open account settings

Click your name in the top-left to open the menu, then click **Settings**.

![Open account settings](/partner/event-partner/redeem-event-code/step-2-open-settings.png)

### Switch to Billing and click "Redeem Credits"

Click **Billing** in the left sidebar, then click the **Redeem Credits** button in the "Account Credit" section.

![Switch to Billing and click Redeem Credits](/partner/event-partner/redeem-event-code/step-3-billing-redeem-credits.png)

### Enter the Event Code

Enter your Event Code in the dialog, then click **Redeem**. On success, you'll see a notification in the bottom-right corner and your account balance will update immediately.

![Enter the Event Code](/partner/event-partner/redeem-event-code/step-4-enter-code.png)

</Steps>

## After Redemption

The event credits go straight into your **account balance** and can be used for any paid service on Zeabur — deploying projects, renting servers, AI Hub, and so on. They behave exactly like credits you'd top up yourself.

![Event credits can be spent on deployments, servers, AI Hub, and more](/partner/event-partner/redeem-event-code/after-redeem-uses.png)

<Callout type="info">
**Want to upgrade to Pro Plan as well?** Head to the [Pricing page](https://zeabur.com/pricing#upgrade) and click upgrade (see [Pro Plan](/pricing/pro-plan)).

- **Your event credits cover the Pro monthly fee**: The credits are applied directly to the upgrade — no card required.
- **Your credits don't cover the fee**: Start Zeabur Pro's **14-day free trial** — once it ends, your event credits are applied first to the monthly fee.
</Callout>

## FAQ

### What can I use the event credits for?

Any paid item on Zeabur — deployments, databases, servers, AI Hub, etc. They're just regular account balance, identical to credits you'd top up yourself.

### Do I need to bind a credit card?

Redeeming the Event Code does **not** require a card. When upgrading to Pro, no card is needed as long as your event credits cover the first month; otherwise a card is required to cover the difference. A card is also required from the second month onward if you want to keep using Pro.

### Can I keep using Zeabur after the event?

Yes. To stay on Pro, bind a credit card and billing continues automatically. If you'd rather not keep paying, stay on the [Free plan](/pricing) — you can return to the [Pricing page](https://zeabur.com/pricing) to upgrade again anytime.

### What happens when my event credits run out?

- **On a paid plan**: The next invoice is charged to your bound credit card.
- **On the Free plan**: You won't be able to use paid resources beyond the free quota. You can [top up](https://zeabur.com/account/billing) or upgrade to keep going.

### Expired / invalid Event Code / other issues

- **Expired** — Expired codes cannot be recovered. Confirm the validity period with your event organizer.
- **Invalid** — Double-check for typos and correct capitalization.
- **Other** — Contact customer support on the [Zeabur community forum](https://zeabur.com/forum).
