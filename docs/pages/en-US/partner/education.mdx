# Education Advocacy

The Education Advocacy program is for **instructors, training institutions, and event organizers** who introduce Zeabur to their students through teaching and technical events, earning rewards based on the number of attendees who convert to paying users.

## How It Works

1. **Apply** — Contact the Zeabur team and describe your course or event plan.
2. **Receive an Event Code** — Zeabur creates a dedicated Event Code for your event. Students can use this code to try Zeabur for free.
3. **Run your event** — Guide students through deploying projects on Zeabur during your course or event.
4. **Retention count** — One month after the event, Zeabur counts the users who registered with your Event Code and are still paying.
5. **Claim rewards** — Earn US$5 per retained paying user.

## Event Code Mechanism

An Event Code is a unique code that provides attendees with trial credits during an event:

- Each Event Code is tied to a specific event.
- Students enter the Event Code during Zeabur registration to receive trial credits.
- Event Codes have a validity period based on the event schedule, typically covering the event and a buffer period afterward.

<Callout type="info">
Event Code redemptions do not trigger the referral program's commission mechanism — because Event Codes provide trial credits, not real payments. Referral commissions are only calculated when a user's actual payment succeeds.
</Callout>

## Reward Calculation

| Item | Details |
| --- | --- |
| Counting period | One month after Event Code distribution |
| Retention criteria | Student registered via Event Code, upgraded to a paid plan, and is still active during the counting period |
| Reward amount | US$5 per retained paying user |
| Payout method | Zeabur Credits or other agreed methods |

### Example

You host a workshop for 50 attendees and distribute an Event Code. One month later, 8 students are still using Zeabur and have upgraded to a paid plan. Your reward is 8 × US$5 = **US$40**.

## Ideal Scenarios

- **Programming courses** — Students deploy course projects to the cloud.
- **Technical workshops** — Hands-on events where attendees experience the deployment process in real time.
- **Hackathons** — Teams use Zeabur to rapidly deploy their projects.
- **Online courses** — Provide Event Codes for remote students to practice.

## How to Apply

Contact the Zeabur team to apply for an Education Advocacy partnership:

- **Community Forum**: Create a post on the [Zeabur community forum](https://zeabur.com/forum) describing your event plan.
- Provide the event name, expected attendance, dates, and format. The Zeabur team will set up an Event Code for you.
