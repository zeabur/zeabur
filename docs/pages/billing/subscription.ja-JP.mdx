# 定期購読

## 定期購読とは

Zeabur には 3 種類のプランがあり、それぞれ **Serverless Plan**、**Developer Plan**、**Team Plan**です。
各プランの詳細は[プラン](/billing/plans)ページで確認できます。

どのプランにも加入していない場合、デフォルトで**Serverless Plan**のユーザーとみなされます。

## 購読するには

Zeabur の購読は[Stripe](https://stripe.com/)を基盤としています。

プランを購読するには、**クレジットカード**と**Alipay**の二つの支払い方法があります。

### クレジットカードでのお申し込み

クレジットカードを利用して、米ドルでの定期購読を申し込むことができます。

- 定期購読に加入せずに設定を行う: [課金ページ](https://dash.zeabur.com/account/billing)にアクセスし、**Payment Method**セクションでクレジットカードを設定します。クレジットカード情報を入力した後、**Add**ボタンをクリックしてクレジットカードを保存し、[プランページ](https://dash.zeabur.com/update-plan)の**Subscribe**ボタンをクリックするだけで定期購読を更新できます。
- 定期購読と共に設定を行う: ダッシュボードの[プランページ](https://dash.zeabur.com/update-plan)でプランを選択し、クレジットカード情報を入力して**Subscribe**ボタンをクリックすると、Stripe のチェックアウトページにリダイレクトされます。

### プリペイドクレジットでの加入

Stripeサービスで使えるクレジットカードをお持ちでない場合は、プリペイドクレジットを購入してプランに加入できます。
ダッシュボードの課金ページでGet More Creditsボタンを押してクレジットを購入し、その後プランページで購入したクレジットを使ってプランをアップグレードできます。

![Get more credits](/billing/subscription/get-more-credits.png)

## 無料プランへのダウングレード方法

有料プランに加入している場合、いつでも**無料プラン**にダウングレードすることができます。

[課金ページ](https://dash.zeabur.com/account/billing), 退会するための危険ゾーンまでスクロールしてください。

![Unsubscribe Plan](/billing/subscription/unsubscribe.png)
