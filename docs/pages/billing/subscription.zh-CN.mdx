# 订阅

## 什么是订阅？

Zeabur 有 3 种类型的方案：**Serverless 方案**、**开发者方案** 和 **团队方案**。
你可以在 [方案](/billing/plans) 页面查看每个方案的详情。

如果你没有订阅任何方案，你将默认与 **Serverless 方案** 用户拥有同样的权利与限制。

## 如何订阅？

Zeabur 的订阅是基于 [Stripe](https://stripe.com/) 的。

有两种方式可以订阅方案：**信用卡** 和 **储值**。

### 用信用卡订阅

你可以用信用卡订阅方案，以美元支付。

- 在订阅前设置卡片：在 [账单页面](https://dash.zeabur.com/account/billing)，你可以在 **支付方式** 部分设置你的信用卡。在你输入完信用卡信息后，点击 **添加** 按钮保存你的信用卡，然后你就可以在 [方案页面](https://dash.zeabur.com/update-plan) 点击 **订阅** 按钮来更新你的订阅。
- 在订阅时设置卡片：在 [升级方案页面](https://dash.zeabur.com/update-plan) 选择一个方案，然后输入你的信用卡信息，点击 **订阅** 按钮，你将被重定向到 Stripe 的支付页面，填写表单后，即完成方案的订阅，同时也会存入你的信用卡信息。

### 用支付宝订阅

如果你没有支持 Stripe 的信用卡，你可以用支付宝进行储值，以人民币支付。

前往 [账单页面](https://dash.zeabur.com/account/billing)，点击储值按钮来获取预付积分。

![Get More Credits](/billing/subscription/unsubscribe.png)

## 如何更改支付方式

如果你的信用卡被拒绝或者你想更改支付方式，你可以在 [账单页面](https://dash.zeabur.com/account/billing) 中进行更改。

找到 **解绑支付方式** 按钮来解绑当前信用卡，然后你可以设置一个新的信用卡。

## 如何降级到 Serverless 方案

如果你订阅了付费方案，你也可以随时降级到 **Serverless 方案**。

前往 [账单页面](https://dash.zeabur.com/account/billing)，下滑找到 **取消订阅** 按钮。

![Unsubscribe Plan](/billing/subscription/unsubscribe.png)
