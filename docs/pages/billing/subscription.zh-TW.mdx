# 訂閱

## 什麼是訂閱？

Zeabur 有 3 種方案：**免費方案**、**開發者方案** 和 **團隊方案**。
你可以在 [方案](/billing/plans) 頁面檢視每個方案的詳細資訊。

如果你沒有訂閱任何方案，你將預設和 **免費方案** 使用者一樣，擁有同樣的權利與限制。

## 如何訂閱？

Zeabur 的訂閱以 [Stripe](https://stripe.com/) 為基礎。

有兩種方式可以訂閱方案：**信用卡** 和 **支付寶**。

### 用信用卡訂閱

你可以用信用卡訂閱方案，以美金付款。

- 在訂閱前設定卡片：在[帳單頁面](https://dash.zeabur.com/account/billing)，你可以在 **付款方式** 部分設定你的信用卡。在輸入信用卡資訊後，點擊 **新增** 按鈕儲存你的信用卡，然後你就可以在 [方案頁面](https://dash.zeabur.com/update-plan) 點擊 **訂閱** 按鈕來更新你的訂閱。
- 在訂閱時設定卡片：在[升級方案頁面](https://dash.zeabur.com/update-plan) 選擇一個方案，然後輸入你的信用卡資訊，點擊 **訂閱** 按鈕，即會重新導向至 Stripe 的付款頁面。填寫表單後，即完成方案的訂閱，同時也會存入你的信用卡資訊。

### 用支付寶訂閱

如果你沒有支援 Stripe 的信用卡，你可以用支付寶訂閱方案，以人民幣付款。

前往 [方案頁面](https://dash.zeabur.com/update-plan)，選擇一個方案，點擊 **No Credit Card?** 連結，你將被重新導向至 Stripe 的付款頁面。你可以選擇用支付寶訂閱方案。

> **注意**：一旦你用支付寶訂閱了方案，你便無法再選擇用信用卡訂閱。

## 如何降級到免費方案

如果你訂閱了付費方案，你也可以隨時降級到 **免費方案**。

前往[方案頁面](https://dash.zeabur.com/update-plan)，選擇 **免費方案**，點擊 **訂閱** 按鈕，你將被重新導向到 Stripe 的付款頁面。你可以在這裡選擇取消你的訂閱。
