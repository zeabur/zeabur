# 訂閱

## 什麼是訂閱？

Zeabur 有 3 種類型的方案：**Serverless 方案**、**開發者方案** 和 **團隊方案**。
你可以在 [方案](/billing/plans) 頁面查看每個方案的詳情。

如果你沒有訂閱任何方案，你將預設與 **Serverless 方案** 使用者擁有同樣的權利與限制。

## 如何訂閱？

Zeabur 的訂閱是基於 [Stripe](https://stripe.com/) 的。

有兩種方式可以訂閱方案：**信用卡** 和 **儲值**。

### 用信用卡訂閱

你可以用信用卡訂閱方案，以美元支付。

- 在訂閱前設定卡片：在 [帳單頁面](https://dash.zeabur.com/account/billing)，你可以在 **支付方式** 部分設定你的信用卡。在你輸入完信用卡資訊後，點選 **加入** 按鈕儲存你的信用卡，然後你就可以在 [方案頁面](https://dash.zeabur.com/update-plan) 點選 **訂閱** 按鈕來更新你的訂閱。
- 在訂閱時設定卡片：在 [升級方案頁面](https://dash.zeabur.com/update-plan) 選擇一個方案，然後輸入你的信用卡資訊，點選 **訂閱** 按鈕，你將被重定向到 Stripe 的支付頁面，填寫表單後，即完成方案的訂閱，同時也會存入你的信用卡資訊。

### 用支付寶訂閱

如果你沒有支援 Stripe 的信用卡，你可以用支付寶進行儲值，以人民幣支付。

前往 [帳單頁面](https://dash.zeabur.com/account/billing)，點選儲值按鈕來獲取預付積分。

![Get More Credits](/billing/subscription/unsubscribe.png)

## 如何更改付款方式 

如果你的信用卡被拒絕或者你想更改付款方式，你可以在 [帳單頁面](https://dash.zeabur.com/account/billing) 進行更改。

找到 **解除綁定** 按鈕來解除當前的信用卡，然後你可以設定一個新的信用卡。

## 如何降級到 Serverless 方案

如果你訂閱了付費方案，你也可以隨時降級到 **Serverless 方案**。

前往 [帳單頁面](https://dash.zeabur.com/account/billing)，下滑找到 **取消訂閱** 按鈕。
