# 價格

Zeabur 為你的專案提供按量計費的價格模式，你不會因為未使用的資源而收到任何帳單。

相反，Zeabur 會根據你的服務實際使用的每分鐘 vCPU 和記憶體資源來收費。

## 服務費用

Zeabur 會根據以下公式計算服務費用：
  
`服務總時間` (分鐘) × (`記憶體使用量` (GB) + `vCPU 使用量` (核心))

並且每分鐘的資源使用費用如下：

- 記憶體 - **US$0.00023** / GB-分鐘 (約 US$10 / GB-月)

- vCPU - **US$0.0005** / vCPU-分鐘 (約 US$20 / vCPU-月)

- 出站流量 - 對於 Serverless Plan 的使用者，前 10GB 流量免費，Developer Plan 和 Team Plan 則為 100GB 免費。超出部分的價格則是 **US$10** / 100GB.

沒有最低服務費用，如果你的服務沒有使用資源或崩潰，我們也不會向你收費。
