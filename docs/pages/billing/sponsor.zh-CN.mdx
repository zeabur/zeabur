# 赞助

## 什么是赞助？

Zeabur 的赞助包含以下内容：

- 项目的开发者计划，赞助项目的所有贡献者都可以免费使用一个月。

- 一定的量的 Credits 用来支付在 Zeabur 的用量费用和订阅费用，数量根据项目的规模和贡献者的贡献程度而定。

## 赞助标准

- 项目必须是非商业性的，完全开源的。

- 项目需要在其网站的页脚和 `README.md` 文件中显示 Zeabur 的标志，我们会将资源提供给您。

## 如何申请赞助

- 赞助对符合上述标准的任何项目开放。

- 想要申请赞助，请按照以下步骤进行：

    - 确认您的项目符合所有标准。

    - 发送邮件到 contact@zeabur.com ，确认您的项目符合所有标准，并简要说明您的项目为什么需要赞助。
