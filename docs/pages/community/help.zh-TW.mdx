# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以在[我們的官方 Discord 伺服器](https://discord.gg/DrdGCvXEyY)中尋求幫助。

![server](/community/help/server.png)

## 如何得到幫助

在我們的 Discord 伺服器中，有一個叫做「問題求助」的頻道，你可以在這裡發布你的問題。

![help channel](/community/help/help-channel.png)

### 發文格式

遵循以下格式來描述你的問題可以幫助我們更快地定位和解決你的問題。

- 在貼文標題簡單描述你的問題
- 在內容區塊中描述你的問題細節
- 在貼文中提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在你的 Zeabur 控制台的 URL 中找到它們。
  ![url](/community/help/url.png)
