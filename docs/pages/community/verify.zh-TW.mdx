# 在 Discord 驗證你的帳戶

Zeabur 提供給所有已驗證使用者優先支援。

你的驗證可以幫助我們更快地定位和解決你的問題，你也可以獲得一些積分作為你在我們的 Discord 伺服器中的活動的獎勵。

## 如何驗證

我們使用我們的 Discord 機器人 **Zebra** 來幫助我們進行驗證你的帳戶。

### 步驟一：加入我們的 Discord 伺服器

首先，你需要[加入我們的 Discord 伺服器](https://discord.gg/DrdGCvXEyY)。你可以在裡面與我們的工作人員和其他使用者交流，並發布你的回饋和問題。

### 步驟二：驗證你的帳戶

當你進入我們的 Discord 伺服器時，你會收到 Zebra 的歡迎資訊。你會在資訊內容中找到[「驗證頻道」](https://discord.com/channels/1060209568820494336/1095916895649153094)。

![verify](/community/verify/verify.png)

點擊驗證按鈕，你會看到 Discord 的授權頁面。點擊 **Authorize** 按鈕來將你的 Discord 帳戶與 Zeabur 連結。

![authorize](/community/verify/authorize.png)

當你被重新導向到我們的網站時，你會看到網頁彈出成功的提示訊息，這時就可以關閉這個網頁了。

![verified](/community/verify/verified.png)

## 下一步

當你成功驗證了你的帳號後，你可以獲得我們團隊的優先支援。當你升級到其他的計劃時，**Zebra** 會自動為你在 Discord 伺服器中新增一個特殊的身分組，藉此你將能夠存取優先支援頻道。

![dev-plan](/community/verify/dev-plan.png)
