# プロジェクト予算

予期せぬ高額なコストを避けるために、各プロジェクトに予算を設定することができます。

予算の使用率が80%に達したときに通知を受け取り、予算が達成されると追加料金を回避するためにサービスが一時停止されます。

## 予算の設定方法

プロジェクトダッシュボードの左上隅にある歯車アイコンをクリックして、プロジェクト設定に移動します。

![Budget Entry](/manage/budget/budget-entry.png)

予算ページで、このプロジェクトの予算を設定するために「毎月の予算を更新」ボタンをクリックします。

![Budget Page](/manage/budget/budget-page.png)

このページでは、このプロジェクトの現在の期間のコストを確認できるため、適切な予算を決定することができます。

## よくある質問

### プロジェクトが予算に近づいているかどうかをどのように知ることができますか？

予算の使用率が80%に達したときに通知を受け取り、プロジェクト設定で使用率を確認することもできます。

### 予算が達成されると何が起こりますか？

予算が達成されると、追加料金を回避するためにこのプロジェクト内のサービスが一時停止されます。

### 設定した予算を変更できますか？

はい、いつでも予算を変更することができます。新しい予算はすぐに有効になります。

### 最低予算はいくらですか？

最低予算は$10.00です。
