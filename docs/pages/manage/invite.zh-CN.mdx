# 邀请成员

一个项目中可能有两个或多个成员，项目所有者可以邀请成员加入项目来管理服务。

## 如何邀请成员

### 步骤一：打开项目设置

首先，打开您想要邀请成员的项目。单击项目页面左上角的配置按钮，您将被导航到项目设置页面。

![Project Config](/manage/invite/project-config.png)

### 步骤二：邀请成员

选择侧边栏中的团队面板，您将在团队面板中看到邀请成员按钮。

![Project Settings](/manage/invite/project-settings.png)

填写您想要邀请的成员的电子邮件地址，然后单击邀请按钮。成员将收到加入项目的电子邮件邀请。

你也可以复制邀请链接并与您想要邀请的成员分享，然后成员只需打开链接即可加入项目。

![Invite Member](/manage/invite/invite.png)

## 常见问题

- **Q: 我能邀请一个未在 zeabur 上注册的成员吗？**
    - A: 目前，您只能邀请已在 zeabur 上注册的成员。

- **Q: 成员加入项目后有哪些权限？**
    - A: 成员将拥有除了删除项目之外的所有权限。

- **Q: 我可以从项目中删除成员吗？**
    - A: 是的，您可以在项目设置页面的团队面板中删除项目中的成员。
