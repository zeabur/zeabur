# 邀請成員

一個專案往往會有多個成員，專案擁有者可以邀請成員進入自己的專案。

### 第一步：輸入受邀方的電子郵件

點選專案左上角的邀請按鈕，輸入受邀方的電子郵件，點選邀請按鈕。

![email](/manage/invite/email.zh-TW.png)

### 第二步：分享連結（可選）

按下邀請按鈕後，團隊成員將會收到一封邀請電子郵件，透過點選電子郵件裡的同意即可進入專案。

除此之外，你也可以選擇分享邀請後生成的專屬邀請連結，並將連結直接分享給對應的受邀方，他透過點選連結，即可進入你的專案。

![email](/manage/invite/success.zh-TW.png)
