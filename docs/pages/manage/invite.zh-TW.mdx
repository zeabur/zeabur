# 邀請成員

一個專案往往會有多個成員，專案擁有者可以邀請成員進入自己的專案。

### 第一步：輸入受邀者的電子郵件地址

點擊專案左上角的邀請按鈕，輸入受邀者的電子郵件地址，然後點擊 **邀請**。

![email](/manage/invite/email.zh-TW.png)

### 第二步：分享連結（可選）

按下邀請按鈕後，團隊成員將會收到一封邀請電子郵件，透過點擊電子郵件裡的同意即可進入專案。

除此之外，你也可以選擇分享邀請後生成的專屬邀請連結，並將連結直接分享給對應的受邀者，他可以透過連結進入你的專案。

![email](/manage/invite/success.zh-TW.png)
