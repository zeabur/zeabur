# Flowise のデプロイ

[Flowise](https://github.com/FlowiseAI/Flowise) は `ドラッグ＆ドロップの UI でカスタマイズされた LLM フローを構築する` オープンソースツールです。

この記事では、Zeabur 上で Flowise サービスをデプロイする方法を教えます。

## 方法 1: テンプレートからのデプロイ

あなたのためにテンプレートを用意しました。以下のボタンをクリックするだけで、実行中の Flowise サービスを入手できます。

[![Zeabur 上でデプロイ](https://zeabur.com/button.svg)](https://zeabur.com/templates/2JYZTR)

## 方法 2: 事前構築済みマーケットプレイスからのデプロイ

### ステップ 1: プロジェクトの作成

<CreateProject />

### ステップ 2: Flowise のデプロイ

Zeabur ダッシュボードで「サービスを作成」をクリックし、「事前構築済み」を選択すると、リストに Flowise が表示されます。

![Flowise のデプロイ](/marketplace/flowise/prebuilt.png)

Flowise を選択すると、Flowise サービスが自動的にデプロイされます。

### ステップ 3: ドメイン名のバインド

Flowise の起動を待っている間に、それにドメインをバインドできます。

サービスページの「ネットワーキング」タブを開き、「ドメイン生成」または「カスタムドメイン」をクリックします。

サービスにドメインをバインドする方法の詳細については、[ドメインのバインド](../deploy/domain-binding) を参照してください。
