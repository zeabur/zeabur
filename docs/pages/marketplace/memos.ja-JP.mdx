# メモのデプロイ

[memos](https://github.com/usememos/memos) プライバシーを最優先にした軽量なメモサービス。素晴らしいアイデアを簡単にキャプチャして共有することができます。

以下はZeaburでホストされているメモのライブデモです：[デモ](https://demo.usememos.com/explore)

この記事では、Zeabur上でメモをデプロイする方法を紹介します。

## 方法1: テンプレートからのデプロイ（推奨）

私たちがテンプレートを用意しました。以下のボタンをクリックするだけで、稼働中のメモサービスを手に入れることができます。

[![Zeabur上でデプロイ](https://zeabur.com/button.svg)](https://dash.zeabur.com/templates/LPCVAH)

## 方法2: マーケットプレイスからのデプロイ

### ステップ1: プロジェクトの作成

<CreateProject />

### ステップ2: マーケットプレイスからのデプロイ

プロジェクトを作成した後、プロジェクトページにリダイレクトされます。

「サービスの追加」ボタンをクリックし、「マーケットプレイス」を選択します。一覧に**memos**が表示されます。

![メモのデプロイ](/marketplace/memos/deploy.png)

それをクリックするか、単にEnterキーを押すと、メモのサービスがデプロイされます。

### ステップ3: ドメインのバインド

このサービスを稼働待ちの間に、それにドメインをバインドすることができます。

サービスページの「ドメイン」タブを開き、「ドメインの生成」または「カスタムドメイン」をクリックします。

サービスにドメインをバインドする方法の詳細は、[ドメインのバインド](../deploy/domain-binding)を参照してください。

![ドメインのバインド](/marketplace/memos/domain.png)
