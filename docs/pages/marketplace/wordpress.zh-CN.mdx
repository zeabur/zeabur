# 部署 WordPress

[WordPress](https://wordpress.org/) 是一个开源软件，让任何人都能轻松构建漂亮的网站、博客或应用程序。

本文将向您展示如何通过 Marketplace 在 Zeabur 上使用您想要的域名部署自己的 WordPress 服务。

## 从模板部署 WordPress

### 第一步：打开 WordPress 模板页面

由于 WordPress 需要一个 MySQL 数据库，因此您不能直接从镜像部署它。

我们为您制作了一个模板，可以一键部署包含 MySQL 的 WordPress。

[![在Zeabur上部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/CV344X)

### 第二步：部署模板

![模板](/marketplace/wordpress/template.png)

点击部署按钮，然后选择一个区域来部署您的服务。

![选择区域](/marketplace/wordpress/region.png)

选择后，您将自动重定向到项目页面，在那里您可以看到部署进度。

再等几秒钟，您的 WordPress 服务将成功部署并开始运行。

### 第三步：绑定域名

部署完成后，您需要绑定一个域名以访问您的 WordPress 服务。

您可以按照[绑定域名的指南](../deploy/domain-binding)了解更多详情。

### 第四步：登录并设置 WordPress

完成域名绑定后，您可以点击域名进入 WordPress 安装界面。首先，选择您想要的语言并点击“继续”（或类似的内容）：

![WordPress安装，选择语言](/marketplace/wordpress/wordpress-inst-step1.png)

之后，按照步骤进行安装：

![WordPress安装，设置密码和站点信息](/marketplace/wordpress/wordpress-inst-step2.png)

当您看到这个界面时，安装就完成了。

![WordPress安装，完成安装](/marketplace/wordpress/wordpress-inst-step3.png)
