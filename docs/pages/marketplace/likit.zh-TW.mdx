# 在 Zeabur 部署 Likit

[Likit](https://github.com/CorrectRoadH/Likit) Likit 是一個 Backend as a Service，提供點讚（投票、點讚、計數）功能。

這篇文章將會向你展示如何在 Zeabur 上部署 Likit。

## 方法 1: 從模板部署 (推薦)

我們已經為 Likit 準備了一個模板，你只需要點選下面的按鈕，就能夠一鍵部署好屬於你的 Likit 服務。

[![在 Zeabur 部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/KZOLHA?referralCode=CorrectRoadH)

## 步驟二：部署 Postgres 服務

在建立 Likit 服務之前，需要先部署一個 Postgres 服務。

請點選「建立服務」並選擇「Prebuilt」，你將在列表中找到 PostgreSQL。

![部署 Postgres](/marketplace/likit/likit-postgres.png)

## 步驟三：部署 Redis 服務

在建立 Likit 服務之前，需要先部署一個 Redis 服務。

請點選「建立服務」並選擇「Prebuilt」，你將在列表中找到 Redis。

![部署 Redis](/marketplace/likit/likit-redis.png)

### 步驟四: 從 Prebuilt Marketplace 部署

點選「建立服務」並選擇「Prebuilt」，你將在列表中找到 Likit。

點選它或者按下確認鍵，我們將會開始為你部署 Likit。

### 步驟三: 綁定網域

當你在等待服務啟動的時候，你可以綁定一個網域到這個服務上。
這個網域可以是你自己的網域，也可以是 Zeabur 提供的免費網域。

打開服務頁面的「網域」分頁，然後點選「產生網域」或者「自訂網域」。

關於如何綁定網域的更多資訊，請參考 [綁定網域](/deploy/domain-binding)。

### 步驟四: 開始使用 Likit

在綁定好網域之後，你可以點選網域進入 Likit 的控制面板，開始使用 Likit。

![likit 預覽](/marketplace/likit/likit.png)
