# 部署 Ghost 

[Ghost](https://ghost.org/) 是一个强大的应用程序，适用于新媒体创作者发布、分享和发展其内容业务。

本文将教你如何在几分钟内在 Zeabur 上部署你自己的 Ghost 服务。

## 从模板部署 Ghost

Ghost 需要 MySQL 作为数据库，因此你不能直接从镜像部署 Ghost 服务。除非你在同一个项目中运行 MySQL 服务，否则它将无法工作。

我们已经制作了一个包含 MySQL 服务的 Ghost 模板，因此你可以通过一次点击同时部署这两个服务。

### 步骤 1：打开 Ghost 模板页面

首先，访问 [Ghost 模板页面](https://https://zeabur.com/templates/SHEU50)。

你会看到模板已经配置了 MySQL 服务，因此你不需要做任何事情。

![模板页面](/marketplace/ghost/template.png)

### 步骤 2：部署模板

点击 **Deploy** 按钮，然后选择一个区域来部署你的服务。

![选择区域](/marketplace/ghost/region.png)

选择后，你将自动重定向到项目页面，在那里你可以看到部署进度。

![部署进度](/marketplace/ghost/deployment.png)

再等待几秒钟，你的 Ghost 服务将成功部署并开始运行。

### 步骤 3：访问 Ghost

部署完成后，你需要绑定一个域名来访问你的 Ghost 服务。

你可以按照 [绑定域名指南](../deploy/domain-binding) 获取更多详情。

### 步骤 4：设置 Ghost

域名绑定后，你可以点击域名访问 Ghost 首页。在域名后添加 `/ghost` 以访问 Ghost 管理设置页面。

![设置 Ghost](/marketplace/ghost/setup.png)
