# Desplegar WordPress

[WordPress](https://wordpress.org/) es un software de código abierto que permite a cualquier persona crear fácilmente sitios web, blogs o aplicaciones hermosas.

Este artículo te mostrará cómo desplegar tu propio servicio de WordPress en Zeabur con el dominio que desees a través del Marketplace.

## Desplegar WordPress desde una plantilla

### Paso 1: Abrir la página de la plantilla de WordPress

Dado que WordPress requiere una base de datos MySQL, no puedes desplegarlo directamente desde la imagen.

Hemos creado una plantilla para que puedas desplegar WordPress con MySQL con un solo clic.

[![Desplegar en Zeabur](https://zeabur.com/button.svg)](https://zeabur.com/templates/CV344X)

### Paso 2: Desplegar la plantilla

![Plantilla](/marketplace/wordpress/template.png)

Haz clic en el botón Desplegar y luego selecciona una región para desplegar tus servicios.

![Seleccionar Región](/marketplace/wordpress/region.png)

Después de la selección, serás redirigido automáticamente a la página del proyecto donde podrás ver el progreso del despliegue.

Espera unos segundos más y tu servicio de WordPress se desplegará correctamente y comenzará a funcionar.

### Paso 3: Vincular dominio

Necesitas vincular un dominio para acceder a tu servicio de WordPress después del despliegue.

Puedes seguir [la guía para vincular un dominio](/deploy/domain-binding) para más detalles.

### Paso 4: Iniciar sesión y configurar WordPress

Después de completar la vinculación del dominio, puedes hacer clic en el nombre del dominio para ingresar a la interfaz de instalación de WordPress. Primero, selecciona el idioma deseado y haz clic en "Continuar" (o algo similar en tu idioma):

![Instalación de WordPress, Seleccionar Idioma](/marketplace/wordpress/wordpress-inst-step1.png)

Después de esto, sigue los pasos para instalar:

![Instalación de WordPress, Configurar Contraseña e Información del Sitio](/marketplace/wordpress/wordpress-inst-step2.png)

Cuando veas esto, la instalación estará completa.

![Instalación de WordPress, Instalación Completa](/marketplace/wordpress/wordpress-inst-step3.png)
