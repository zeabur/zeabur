# Zeabur で Linkwarden をデプロイ

[Linkwarden](https://github.com/linkwarden/linkwarden) は、ウェブページを収集、整理、アーカイブするためのセルフホスト型オープンソースの協調ブックマークマネージャーです。

この記事では、Zeabur で Linkwardenを1分でデプロイする方法を紹介します。

![プレビュー](https://github.com/linkwarden/linkwarden/raw/main/assets/dashboard.png)

## テンプレートから Linkwarden をデプロイ

Linkwarden はデータベースとして PostgreSQL を必要とするため、イメージから直接デプロイすることはできません。

PostgreSQL を含む Linkwarden をワンクリックでデプロイできるテンプレートを用意しました。

[![Zeaburでデプロイ](https://zeabur.com/button.svg)](https://zeabur.com/templates/1E5ABT)

### ステップ1: Linkwarden テンプレートページを開く

まず、Linkwarden の[テンプレートページ](https://zeabur.com/templates/1E5ABT)を訪問します。

テンプレートにはすでに PostgreSQL サービスが設定されているので、特に何もする必要はありません。

![テンプレートページ](/marketplace/linkwarden/template.png)

### ステップ2: テンプレートをデプロイ

デプロイボタンをクリックし、サービスをデプロイする地域を選択します。

![地域を選択](/marketplace/linkwarden/region.png)

選択後、プロジェクトページに自動的にリダイレクトされ、デプロイの進捗状況を確認できます。

![デプロイの進捗](/marketplace/linkwarden/deployment.png)

数秒待つと、Linkwarden サービスが正常にデプロイされ、稼働を開始します。

### ステップ3: ドメインをバインド

デプロイ後、Linkwarden サービスにアクセスするためにドメインをバインドする必要があります。

詳細については、[ドメインバインドのガイド](../deploy/domain-binding)を参照してください。
