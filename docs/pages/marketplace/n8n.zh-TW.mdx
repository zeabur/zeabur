# 部署 n8n

[n8n](https://github.com/n8n-io/n8n) 是一款可擴展的工作流自動化工具。採用公平程式碼分發模型，n8n 始終具有可見的原始碼，可用於自主託管，並允許您添加自己的自訂函數、邏輯和應用程式。n8n 的基於節點的方法使其高度靈活，使您能夠將任何東西連接到任何東西。

本文將向你展示如何在 Zeabur 上使用 PostgreSQL 部署 n8n。

## 從模板部署 n8n

由於 n8n 需要 PostgreSQL 作為資料庫，因此無法直接從鏡像部署。

我們已經為您製作了一個模板，可以一鍵部署帶有 PostgreSQL 的 n8n。

[![在 Zeabur 上部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/W2H4RW)

### 步驟一：打開 n8n 模板頁面

首先，訪問 [n8n 的模板頁面](https://zeabur.com/templates/W2H4RW)。

你可以看到模板已經配置了 PostgreSQL 服務，因此你無需做任何額外的配置。

![模板頁面](/marketplace/n8n/template.png)

### 步驟二：部署模板

點擊部署按鈕，然後選擇一個區域來部署你的服務。

![選擇區域](/marketplace/n8n/region.png)

選擇後，你將自動重定向到項目頁面，在那裡你可以看到部署進度。  

![部署進度](/marketplace/n8n/deployment.png)

再等待幾秒鐘，你的 n8n 服務將成功部署並開始執行。

### 步驟三：綁定域名

部署完成後，你需要綁定一個域名以訪問你的 n8n 服務。

你可以按照[綁定域名的指南](../deploy/domain-binding)獲取更多詳細資訊。

### 步驟四：設定帳戶

部署完成後，你可以透過你綁定的域名訪問你的 n8n 服務。你將會看到如下頁面：

![n8n Setup](/marketplace/n8n/setup.png)

跟隨頁面上的說明設定你的 n8n 帳戶。設定完成後，你就可以正常使用 n8n 了。

![finish](/marketplace/n8n/finish.png)
