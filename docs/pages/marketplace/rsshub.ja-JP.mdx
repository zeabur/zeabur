# RSSHubサービスのデプロイ

[RSSHub](https://rsshub.app)はオープンソースで使いやすく、拡張可能なRSSフィードアグリゲータであり、ほとんどすべてからRSSフィードを生成することができます。

![RSSHub](/marketplace/rsshub/intro.png)

この記事では、サービスマーケットを介してZeaburに独自のRSSHubサービスをデプロイし、独自のドメイン名をバインドする方法について説明します。

## ステップ1: プロジェクトの作成

<CreateProject />

## ステップ2: RSSHubサービスのデプロイ

マーケットプレイスで簡単にRSSHubサービスをデプロイできます。

プロジェクトページで "Add Service" ボタンをクリックするか、"cmd + k" を押して "Marketplace" を選択します。

![Select Marketplace](/deploy/common/select-service.en-US.png)

リストからRSSHubを見つけてクリックし、デプロイを開始します。

## ステップ3: ドメインのバインド

ZeaburダッシュボードのRSSHubサービスの "Domain" ブロックで、ドメインをバインドすることができます。

このドメインは独自のドメインか、Zeaburが提供する無料のドメインです。

サービスにドメインをバインドする方法の詳細については、[ドメインのバインド](../deploy/domain-binding) を参照してください。

## ステップ4: 楽しむ

デプロイが完了すると、バインドしたドメイン名を使用してRSSHubサービスにアクセスできます。

お気に入りのRSSフィードを選択し、RSSリーダーで購読できます。
