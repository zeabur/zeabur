# 在 Zeabur 上部署 Linkwarden

[Linkwarden](https://github.com/linkwarden/linkwarden) 是一個自託管、開源的協作書籤管理器，用於收集、組織和存檔網頁。

本文將向您展示如何在一分鐘內在 Zeabur 上部署 Linkwarden。

![預覽](https://github.com/linkwarden/linkwarden/raw/main/assets/dashboard.png)

## 從模板部署 Linkwarden

由於 Linkwarden 需要 PostgreSQL 作為資料庫，因此無法直接從鏡像部署。

我們已經為您製作了一個模板，可以一鍵部署帶有 PostgreSQL 的 Linkwarden。

[![在 Zeabur 上部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/1E5ABT)

### 第一步：打開 Linkwarden 模板頁面

首先，訪問 [Linkwarden 的模板頁面](https://zeabur.com/templates/1E5ABT)。

您將看到模板已經配置了 PostgreSQL 服務，因此您無需做任何額外的配置。

![模板頁面](/marketplace/linkwarden/template.png)

### 第二步：部署模板

點擊部署按鈕，然後選擇一個區域來部署您的服務。

![選擇區域](/marketplace/linkwarden/region.png)

選擇後，您將自動重定向到項目頁面，在那裡您可以看到部署進度。

![部署進度](/marketplace/linkwarden/deployment.png)

再等待幾秒鐘，您的 Linkwarden 服務將成功部署並開始執行。

### 第三步：綁定域名

部署完成後，您需要綁定一個域名以訪問您的 Linkwarden 服務。

您可以按照[綁定域名的指南](../deploy/domain-binding)獲取更多詳細資訊。
