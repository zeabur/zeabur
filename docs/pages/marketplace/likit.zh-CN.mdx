# 在 Zeabur 部署 Likit

[Likit](https://github.com/CorrectRoadH/Likit) Likit 是一个 Backend as a Service ，提供点赞（投票、点赞、计数）功能。

这篇文章将会向你展示如何在 Zeabur 上部署 Likit。

## 方法 1: 从模板部署(推荐)

我们已经为 Likit 准备了一个模板，你只需要点击下面的按钮，就能够一键部署好属于你的 Likit 服务。

[![Deploy on Zeabur](https://zeabur.com/button.svg)](https://zeabur.com/templates/KZOLHA?referralCode=CorrectRoadH)

## 方法 2: 从 Marketplace 部署

### 步骤一: 步骤一：创建项目

<CreateProject />

## 步骤二：部署 Postgres 服务

在创建 Likit 服务之前，需要先部署一个 Postgres 服务。

请点击“创建服务”并选择“Prebuilt”，你将在列表中找到 PostgreSQL。

![部署 Postgres](/marketplace/likit/likit-postgres.png)

## 步骤三：部署 Redis 服务

在创建 Likit 服务之前，需要先部署一个 Redis 服务。

请点击“创建服务”并选择”Prebuilt”，你将在列表中找到 Redis。

![部署 Redis](/marketplace/likit/likit-redis.png)

### 步骤四: 从 Prebuilt Marketplace 部署

点击“创建服务”并选择“Prebuilt”，你将在列表中找到 Likit。

点击它或者按下确认键，我们将会开始为你部署 Likit。

### 步骤五：绑定域名

当你在等待服务启动的时候，你可以绑定一个域名到这个服务上。
这个域名可以是你自己的域名，也可以是 Zeabur 提供的免费域名。

打开服务页面的 "网络" 分页，然后点选 "生成域名" 或者 "自定义域名"。

关于如何绑定域名的更多信息，请参考 [绑定网域](/deploy/domain-binding)。

### 步骤六：开始使用 Likit

在绑定好域名之后，你可以点选域名进入 Likit 的控制面板，开始使用 Likit。

![likit Preview](/marketplace/likit/likit.png)
