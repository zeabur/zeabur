# 在 Zeabur 部署 Redis

Zeabur 提供了一鍵部署 Redis 服務的功能，讓你可以快速的在 Zeabur 上部署 Redis 資料庫。

## 部署 Redis 服務

<CreateProject />

在你的專案中，點擊 **建立服務**，選擇 **Marketplace** 之後搜尋 `Redis`。

![部署 "Redis"](/databases/redis/deploy-redis.gif)

點選「Redis」後，Zeabur 會自動幫你部署 Redis 服務。

## 環境變數

成功部署 Redis 服務後，Zeabur 會自動注入環境變數到服務中。目前有以下變數：

- `REDIS_CONNECTION_STRING`
- `REDIS_HOST`
- `REDIS_PASSWORD`
- `REDIS_PORT`
- `REDIS_URI`

## 連結資料庫

- [Redis cli](https://docs.redis.com/latest/rs/references/cli-utilities/redis-cli/)

### redis-cli

[redis-cli](https://www.mongodb.com/docs/mongodb-shell/) 是一個命令列工具，讓你可以與 Redis 資料庫互動。 使用 redis-cli，你可以在命令列終端或互動模式下執行 Redis 命令。

根據文件下載且安裝完畢後，回到 Zeabur 中。點選 Redis 服務中 Instructions 頁籤，點選 `Command to connect to your Redis` 的複製按鈕來複製連結地址。

![connect-path](/databases/redis/redis-connect-path.png)

將複製後的指令貼到終端機，並輸入 `PING` 測試連結是否成功：

![test-connection](/databases/redis/redis-test-connection.png)

此時，即可開始操作 Redis。
