# Ghost をデプロイする 

[Ghost](https://ghost.org/) は、新しいメディアクリエイターがコンテンツビジネスを公開、共有、発展させるための強力なアプリケーションです。

この記事では、数分で Zeabur 上に独自の Ghost サービスをデプロイする方法を教えます。

## テンプレートから Ghost をデプロイする

Ghost にはデータベースとして MySQL が必要なため、イメージから直接 Ghost サービスをデプロイすることはできません。同じプロジェクト内で MySQL サービスが実行されていないと機能しません。

私たちは MySQL サービスを含む Ghost テンプレートを作成しましたので、ワンクリックで両方のサービスを同時にデプロイできます。

### ステップ 1：Ghost テンプレートページを開く

まず、[Ghost のテンプレートページ](https://https://zeabur.com/templates/SHEU50)にアクセスします。

テンプレートには既に MySQL サービスが設定されているので、何もする必要はありません。

![テンプレートページ](/marketplace/ghost/template.png)

### ステップ 2：テンプレートをデプロイする

**Deploy** ボタンをクリックし、サービスをデプロイする地域を選択します。

![地域を選択](/marketplace/ghost/region.png)

選択後、デプロイの進行状況を確認できるプロジェクトページに自動的にリダイレクトされます。

![デプロイ進行状況](/marketplace/ghost/deployment.png)

数秒待つと、Ghost サービスが正常にデプロイされ、実行を開始します。

### ステップ 3：Ghost にアクセスする

デプロイが完了したら、Ghost サービスにアクセスするためにドメインをバインドする必要があります。

詳細については、[ドメインをバインドするガイド](../deploy/domain-binding)を参照してください。

### ステップ 4：Ghost を設定する

ドメインがバインドされたら、ドメインをクリックして Ghost ホームページにアクセスできます。ドメインの後に `/ghost` を追加して Ghost 管理設定ページにアクセスします。

![Ghost を設定する](/marketplace/ghost/setup.png)
