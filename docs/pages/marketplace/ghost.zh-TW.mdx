# 部署 Ghost

[Ghost](https://ghost.org/) 是一個強大的應用程式，適用於新媒體創作者發布、分享和發展其內容業務。

本文將教你如何在幾分鐘內在 Zeabur 上部署你自己的 Ghost 服務。

## 從模板部署 Ghost

Ghost 需要 MySQL 作為資料庫，因此你不能直接從鏡像部署 Ghost 服務。除非你在同一個項目中執行 MySQL 服務，否則它將無法工作。

我們已經製作了一個包含 MySQL 服務的 Ghost 模板，因此你可以透過一次點擊同時部署這兩個服務。

### 步驟 1：打開 Ghost 模板頁面

首先，訪問 [Ghost 模板頁面](https://https://zeabur.com/templates/SHEU50)。

你會看到模板已經配置了 MySQL 服務，因此你不需要做任何事情。

![模板頁面](/marketplace/ghost/template.png)

### 步驟 2：部署模板

點擊 **Deploy** 按鈕，然後選擇一個區域來部署你的服務。

![選擇區域](/marketplace/ghost/region.png)

選擇後，你將自動重定向到項目頁面，在那裡你可以看到部署進度。

![部署進度](/marketplace/ghost/deployment.png)

再等待幾秒鐘，你的 Ghost 服務將成功部署並開始執行。

### 步驟 3：訪問 Ghost

部署完成後，你需要綁定一個域名來訪問你的 Ghost 服務。

你可以按照 [綁定域名指南](/deploy/domain-binding) 獲取更多詳情。

### 步驟 4：設定 Ghost

域名綁定後，你可以點擊域名訪問 Ghost 首頁。在域名後添加 `/ghost` 以訪問 Ghost 管理設定頁面。

![設定 Ghost](/marketplace/ghost/setup.png)
