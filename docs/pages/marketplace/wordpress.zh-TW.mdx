# 部署 WordPress

[WordPress](https://wordpress.org/) 是一個開源軟體，讓任何人都能輕鬆建置漂亮的網站、部落格或應用程式。

本文將向您展示如何透過 Marketplace 在 Zeabur 上使用您想要的網域部署自己的 WordPress 服務。

## 從模板部署 WordPress

### 第一步：打開 WordPress 模板頁面

由於 WordPress 需要一個 MySQL 資料庫，因此您不能直接從鏡像部署它。

我們為您製作了一個模板，可以一鍵部署包含 MySQL 的 WordPress。

[![在Zeabur上部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/CV344X)

### 第二步：部署模板

![模板](/marketplace/wordpress/template.png)

點選部署按鈕，然後選擇一個區域來部署您的服務。

![選擇區域](/marketplace/wordpress/region.png)

選擇後，您將自動重定向到項目頁面，在那裡您可以看到部署進度。

再等幾秒鐘，您的 WordPress 服務將成功部署並開始運作。

### 第三步：綁定網域

部署完成後，您需要綁定一個網域以造訪您的 WordPress 服務。

您可以按照[綁定網域的指南](/deploy/domain-binding)了解更多詳情。

### 第四步：登入並設定 WordPress

完成網域綁定後，您可以點選網域進入 WordPress 安裝介面。首先，選擇您想要的語言並點選“繼續”（或類似的內容）：

![WordPress安裝，選擇語言](/marketplace/wordpress/wordpress-inst-step1.png)

之後，按照步驟進行安裝：

![WordPress安裝，設定密碼和站點資訊](/marketplace/wordpress/wordpress-inst-step2.png)

當您看到這個介面時，安裝就完成了。

![WordPress安裝，完成安裝](/marketplace/wordpress/wordpress-inst-step3.png)
