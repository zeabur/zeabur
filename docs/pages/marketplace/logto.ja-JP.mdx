# Logto サービスをデプロイする

[Logto](https://logto.io/) は、アプリケーションに迅速に認証機能を実装するのに役立つオープンソースの認証サービスで、[Auth0](https://auth0.com/) の最も有名なオープンソース代替品でもあります。

この記事では、サービスマーケットを通じて Zeabur に Logto サービスをデプロイし、独自のドメイン名をバインドして、他のウェブサイトがデプロイした Logto サービスを使用して認証を行う方法を説明します。

## ステップ 1：テンプレートからデプロイ

### 方法 1：新しいプロジェクトでデプロイ

まず、[Logto のテンプレートページ](https://zeabur.com/templates/AV1NZ4) にアクセスします。

テンプレートにはすでに PostgreSQL サービスが設定されているので、特に何もする必要はありません。

![テンプレートページ](/marketplace/logto/template.png)

デプロイボタンをクリックし、サービスをデプロイするリージョンを選択します。

![リージョンを選択](/marketplace/logto/region.png)

選択後、自動的にプロジェクトページにリダイレクトされ、デプロイ進行状況が表示されます。

![デプロイ](/marketplace/logto/deployment.png)

さらに数秒待つと、Logto サービスが正常にデプロイされ、稼働を開始します。

### 方法 2：既存のプロジェクトにデプロイ

すでにプロジェクトがあり、そのプロジェクトに認証システムを追加したい場合は、Logto サービスをワンクリックで既存のプロジェクトにデプロイすることもできます。

プロジェクトページに移動し、**新しいサービスをデプロイ** ボタンをクリックして、テンプレートカテゴリから Logto を選択します。

![プロジェクト内のテンプレート](/marketplace/logto/template-in-project.png)

Logto を選択した後、テンプレート内のサービス（Logto と PostgreSQL）は自動的にプロジェクトにデプロイされます。

## ステップ 2：各ポートにドメイン名をバインドする

Zeabur ダッシュボードでは、**ネットワーキング**タブで Logto サービスの各ポートにドメイン名をバインドできます。

Logto サービスには 2 つのポートがあり、1 つは Web インターフェイス用で、もう 1 つは管理コンソールインターフェイス用です。コンソールで各ポートにドメイン名をバインドできます。

**ドメインを生成** または **カスタムドメイン** をクリックすると、入力ボックスとポートセレクターが表示され、ここで各ポートにドメイン名をバインドできます。

![Logto の Web ドメイン](/marketplace/logto/web-domain.png)

![Logto のコンソールドメイン](/marketplace/logto/console-domain.png)

サービスにドメイン名をバインドする方法については、[ドメインバインディング](../deploy/domain-binding) を参照してください。

## ステップ 3：環境変数の設定

ドメイン名をバインドした後、Logto サービスにバインドしたドメイン名を知らせる必要があります。

Logto サービスには 2 つの環境変数が必要です：

- `ENDPOINT`: Web インターフェイス用にバインドしたドメイン名、例：`https://logto-web.zeabur.app`
- `ADMIN_ENDPOINT`: 管理コンソールインターフェイス用にバインドしたドメイン名、例：`https://logto-console.zeabur.app`

サービスページの **変数** タブを開き、**生の変数を編集** ボタンをクリックして、以下の変数を入力ボックスに入力します。

```json
{
  "ENDPOINT": "https://logto-web.zeabur.app", // Web インターフェイス用のドメイン名
  "ADMIN_ENDPOINT": "https://logto-console.zeabur.app" // 管理コンソールインターフェイス用のドメイン名
}
```

変数を追加した後、**再起動** ボタンを押して約 30 秒待つと、サービスが再起動して有効になります。

## ステップ 4：Logto にログインして設定する

これで Logto サービスの準備が整いました。設定することができます。管理コンソールインターフェイス用にバインドしたドメイン名を開くと、例：`https://logto-console.zeabur.app`、Logto サービスのログインページが表示されます。

![Logto 管理](/marketplace/logto/admin.png)

その後、アカウントを作成して Logto サービスの使用を開始できます。Logto サービスが提供する指示に従ってください。
