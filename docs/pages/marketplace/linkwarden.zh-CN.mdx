# 在 Zeabur 上部署 Linkwarden

[Linkwarden](https://github.com/linkwarden/linkwarden) 是一个自托管、开源的协作书签管理器，用于收集、组织和存档网页。

本文将向您展示如何在一分钟内在 Zeabur 上部署 Linkwarden。

![预览](https://github.com/linkwarden/linkwarden/raw/main/assets/dashboard.png)

## 从模板部署 Linkwarden

由于 Linkwarden 需要 PostgreSQL 作为数据库，因此无法直接从镜像部署。

我们已经为您制作了一个模板，可以一键部署带有 PostgreSQL 的 Linkwarden。

[![在 Zeabur 上部署](https://zeabur.com/button.svg)](https://zeabur.com/templates/1E5ABT)

### 第一步：打开 Linkwarden 模板页面

首先，访问 [Linkwarden 的模板页面](https://zeabur.com/templates/1E5ABT)。

您将看到模板已经配置了 PostgreSQL 服务，因此您无需做任何额外的配置。

![模板页面](/marketplace/linkwarden/template.png)

### 第二步：部署模板

点击部署按钮，然后选择一个区域来部署您的服务。

![选择区域](/marketplace/linkwarden/region.png)

选择后，您将自动重定向到项目页面，在那里您可以看到部署进度。

![部署进度](/marketplace/linkwarden/deployment.png)

再等待几秒钟，您的 Linkwarden 服务将成功部署并开始运行。

### 第三步：绑定域名

部署完成后，您需要绑定一个域名以访问您的 Linkwarden 服务。

您可以按照[绑定域名的指南](../deploy/domain-binding)获取更多详细信息。
