# パートナープログラム

Zeabur Partner Program は、講師、イベント主催者、企業コンサルタント、システムインテグレーターに対して協業の機会を提供し、Zeabur の推進や導入を通じて報酬を獲得できる仕組みです。

## 2 種類の協業モデル

### 教育パートナー

**講師、コミュニティリーダー、ハッカソン主催者、教育機関**に適しています。講座やテクニカルイベントを通じて Zeabur を推進し、イベントコード (Event Code) で参加者に無料体験を提供し、継続した有料ユーザー数に応じて報酬を獲得します。

- イベント参加者向けに専用のイベントコードを取得
- イベント終了から 1 ヶ月後に継続した有料ユーザーを集計し報酬を支払い
- ワークショップ、オンライン講座、ハッカソン、コミュニティイベントなどに最適

[詳しく見る →](/partner/event-partner)

### セールスパートナー (Sales Partner)

**企業コンサルタント、システムインテグレーター、テクノロジーコンサルティング企業**に適しています。企業顧客や機関の Zeabur 導入を支援し、実際に関与した協業フェーズに応じて継続的なレベニューシェアを獲得します。

- 大型契約、機関調達、テクニカルコンサルティング導入などのケースに適合
- 2 階層のレベニューシェア構造：取引レート（累進制）× 貢献フェーズの重み
- 具体的なレート、閾値、条件は個別の契約で定め、対外的には公開しません

[詳しく見る →](/partner/sales-partner)

## Referral Program との違い

Zeabur では [Referral Program](/rewards/referral)（リファラルプログラム）も提供しており、すべてのユーザーが推薦コードを通じて報酬を得ることができます。両者の主な違いは以下のとおりです：

| 項目 | Referral Program | パートナープログラム |
| --- | --- | --- |
| 対象 | すべてのユーザー | 講師 / イベント主催者 / 企業コンサルタント / システムインテグレーター |
| トリガー | 推薦コードによる自動フロー | 人手による審査と契約 |
| 報酬メカニズム | 推薦コードのコミッション（ランクに応じる） | イベント継続報酬 / セールスパートナーのレベニューシェア |
| 契約条件 | プラットフォーム利用規約で一律 | 協業契約ごとに個別に交渉 |

<Callout type="info">
同一の取引は Referral Program またはパートナープログラムのいずれか 1 つの経路のみを通ることができ、**両者の併用はできません**。
</Callout>

## 応募方法

[partner.zeabur.com](https://partner.zeabur.com) でアカウントを作成し、協業タイプに応じて必要情報を記入のうえ応募してください：

- **教育パートナー** — イベント名、想定参加人数、日程、形式を記入。
- **セールスパートナー** — 会社情報、想定される顧客タイプと協業規模を記入。

Zeabur チームは応募を受領後、ご連絡のうえ協業条件について協議いたします。
