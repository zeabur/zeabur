# 不正利用の報告ガイドライン

Zeabur は各種のサイバーセキュリティ機関と連携し、犯罪行為および悪意ある利用の撲滅に取り組んでいます。[フェアユースガイドライン](/legal/fair-use-guideline) に違反する行為を発見した場合は、ぜひご報告ください。[フェアユースガイドライン](/legal/fair-use-guideline) に違反するウェブサイトや行為を見つけた場合は、速やかにご連絡ください。

政府機関またはサイバーセキュリティ機関の方は、自動化された報告システムへの接続についてもご相談いただけます。

## 連絡方法

以下の方法でご連絡いただけます：

- メール：[contact@zeabur.com](mailto:contact@zeabur.com)
- Discord コミュニティ：[Zeabur Discord](https://zeabur.com/dc)
- 郵送または公文書送付先住所：[105臺北市松山區民生東路四段133號6樓](https://maps.app.goo.gl/BkQC63MjtC74aDu89)

いただいたご報告は 2 営業日以内に返信し、対応状況をご案内します。2 営業日以内に返信がない場合は、再度ご連絡ください。
