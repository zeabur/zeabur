# 不正利用の報告ガイドライン

Zeaburはサイバーセキュリティ組織と協力して犯罪や悪用を防いでいます。[Fair Use Guidelines](https://zeabur.com/docs/legal/fair-use-guideline)に違反する行為があれば報告をお願いします。ウェブサイトや活動が[Fair Use Guidelines](https://zeabur.com/docs/legal/fair-use-guideline)に違反している場合、すぐにご報告ください。

政府機関やサイバーセキュリティ組織の方は、自動報告システムとの統合についてもお問い合わせいただけます。

## **連絡先**

以下の連絡方法をご利用ください：

- メール: [contact@zeabur.com](mailto:contact@zeabur.com)
- Zeaburフォーラム: [Zeabur Forum](https://zeabur.com/forum)
- 郵送先住所: [6F, No. 133, Section 4, Minsheng E Rd, Songshan District, Taipei City, 105, Taiwan](https://maps.app.goo.gl/BkQC63MjtC74aDu89)

報告を受け付けてから2営業日以内に返信し、対応状況をお知らせします。2営業日以内に返信がない場合は、再度ご連絡ください。
