# ICP申請前サブドメイン

中国本土の法律・法規により、中国本土内のサーバーで運用されているすべてのサービスはICP申請を完了しなければアクセスできません。これは、まだ開発段階にあるプロジェクトにとっては不便な場合があります。

従来の方法では、中国本土のサーバーを購入した場合、ICP申請なしでは IP + ポート番号の形式でサービスにアクセスするしかありませんでした。しかしこの方法ではドメインを使ったアクセスができず、SSL証明書の発行も行えません。

このような状況に対して、Zeabur は ICP申請前サブドメイン機能を提供しています。実名認証を完了することを前提に、Zeabur が提供するICP申請済みドメインを一時的に使用してデプロイ済みサービスにアクセスできます。開発段階や個人的なテスト用途の一時サービスとして非常に便利であり、完全に合法・適法です。

## 利用条件

- サーバーはZeaburから購入したものであること。
- サーバーが中国本土内にあること。
- ZeaburアカウントがすでにICP申請に必要な中国本土の実名認証を完了していること。
- サーバープロバイダーがZeaburでICP申請前サブドメインに対応していること:

## 使用方法

1. 中国本土にデプロイしているZeabur管理サーバーのプロジェクトを選択し、ICP申請前サブドメインを紐付けたいサービスを選択します。
2. 「ネットワーク」タブを選択します。
3. 「ドメインを生成」をクリックし、希望するサブドメインを入力します。

![](https://cdn.zeabur.com/pre-icp-1.png)

4. 「ドメインを作成」をクリックします。実名認証が未完了の場合、Zeaburが実名認証の完了を促します。
5. 実名認証ウィンドウで氏名と身分証番号を入力し、送信をクリックするとAlibaba Cloudの実名認証サービスに自動的にリダイレクトされます。

![](https://cdn.zeabur.com/pre-icp-2.png)

6. 実名認証完了後、再度「ドメインを生成」をクリックすると、ICP申請済みドメインが自動生成・紐付けされます。

## 注意事項

- ICP申請前サブドメインを使用することで、対応するクラウドプロバイダーの利用規約に同意したものとみなされます。
- Zeaburは、ICP申請前サブドメインを使用して現地の法律・法規に違反する行為を厳しく禁止しています。違反が発覚した場合、Zeaburは一方的にサービスを終了し、実名情報をユーザーの所在国・地域の執行機関に提供する権利を有します。
- ICP申請前サブドメインを商用サービスの正式なドメインとして使用することは推奨しません。テスト・開発が完了したら、正式なICP申請手続きを経てご自身のドメインを取得することをお勧めします。ZeaburはICP申請サポートを無料で提供しています。
