# スポンサー

## スポンサーシップとは何ですか？

Zeabur のスポンサーシップには次のような特典があります：

- すべてのコントリビュータにデベロッパープランを 1 ヶ月間無料で提供

- プロジェクトの規模や個人の貢献度によって異なる、コントリビュータのためのクレジット

## スポンサーシップの基準

- プロジェクトは、非商用で完全なオープンソースでなければならない

- プロジェクトの Web サイトのフッターと `README.md` ファイルに Zeabur バナーを表示する必要がある

リソースはダッシュボードの[紹介ページ](https://dash.zeabur.com/account/referral)で確認できます。

[![Zeabur でのデプロイ](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://dash.zeabur.com/account/referral)

## スポンサーシップの申請方法

- スポンサーシップは、上記の条件を満たすプロジェクトであれば、どのようなプロジェクトでも参加できます。

- 申請するには、以下の手順に従ってください：

  - あなたのプロジェクトがリストにある基準をすべて満たしていることを確認する

  - [スポンサーシップフォーム](https://forms.gle/EmpRN7rnjkxQB72Y6) にご記入ください
