# 定価

Zeabur はプロジェクトに従量課金制の料金モデルを提供しており、未使用のリソースには支払い不要です。ご利用中のリソース分だけ、実際の使用時間に基づいてお支払いいただけます。

## サービス料金

Zeabur には **共有クラスター** と [**専用サーバー**](/dedicated-server) の2種類のサーバータイプがあります。この料金表は **共有クラスター** にのみ適用され、専用サーバーの使用料金は発生しません。

**共有クラスター** の料金は以下の通りです：

- vCPU - 無料 ($0.0 / vCPU-分)

- メモリ - **US$0.00025** / GB-分 (約 US$10.8 / GB-月)
  - サービスで実際に使用したメモリに対してのみ料金が発生します。サービスがクラッシュまたは一時停止した場合は料金が発生しません。

- ネットワーク出力費用 - **US$0.1** / GB (US$10 / 100GB)
  - 1GB は約 1,000 人のユニーク訪問者に相当します。CDN や圧縮を利用しない場合、1MiB のページを閲覧する際に発生するトラフィック量です。
  - [プライベートネットワーク](/networking/private) は出力トラフィックに含まれません。

- 永続ストレージ - **US$0.2** / GB-月

## 返金

返金に関するご質問がある場合は、[私たちの Discord サーバー](https://zeabur.com/dc) にてお問い合わせください。

### サーバーとサブスクリプションプラン

**クレジットカードまたはその他のクレジット以外の支払い方法** で購入した **専用サーバー** および **サブスクリプションプラン**（開発者プラン、チームプラン）については、返金が可能です。以下からお選びいただけます：

- **Zeabur クレジット** として返金 — 承認後すぐに処理されます。
- **元の支払い方法** への返金 — 処理に最大 7 日間かかる場合があり、取引手数料をお支払いいただく必要があります。

### クレジットで購入した場合

**Zeabur クレジット** で購入したサーバーまたはサブスクリプションプランの返金は、Zeabur クレジット残高への返金のみ可能です。

### 購入済みクレジット

**購入済みクレジットの返金はできません。**
