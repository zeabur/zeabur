# リミックスをデプロイ

Zeaburでリミックスアプリをデプロイします。

### ステップ1：リミックスアプリを作成（オプション）

- Node.js v14以上のバージョン。

ターミナルを開いて、次のコマンドを実行します。
```bash
npx create-remix@latest
```

それに対応するGitHubリポジトリを初期化し、GitHubにアップロードします。

### ステップ2：リミックスアプリをデプロイ

**デプロイサービス**または**新しいサービスを追加**ボタンをクリックし、「ソースコードをデプロイ」を選択します。

![deploy](/deploy/common/select-service.ja-JP.png)

リミックスアプリリポジトリを検索し、インポートをクリックして、リミックスアプリがデプロイされます。
