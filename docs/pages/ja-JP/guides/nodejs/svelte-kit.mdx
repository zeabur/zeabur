# Svelte Kitのデプロイ

Svelte KitプロジェクトをZeaburにデプロイします。

### ステップ1：Svelte Kitプロジェクトを作成する（オプション）

ターミナルを開き、次のコマンドを実行します。

```bash
npm create svelte@latest
```

次に、依存関係をインストールする必要があります：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install
    ```
  </Tab>
  <Tab>
    ```bash
    npm install
    ```
  </Tab>
  <Tab>
    ```bash
    yarn
    ```
  </Tab>
</Tabs>

その後、Gitリポジトリを初期化し、GitHubにデプロイします。

### ステップ2：`@zeabur/svelte-adapter`をセットアップする

次に、`@zeabur/svelte-adapter`をインストールする必要があります：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D @zeabur/svelte-adapter
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -D @zeabur/svelte-adapter
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D @zeabur/svelte-adapter
    ```
  </Tab>
</Tabs>

その後、`svelte.config.js`を開き、以下のように変更します：

```js
import adapter from '@zeabur/svelte-adapter'; // <-- この行に注意

export default {
  kit: {
    adapter: adapter()
  }
}
```

### ステップ3：Svelte Kitプロジェクトをデプロイする

プロジェクトで**デプロイサービス**または**新しいサービスを追加**をクリックし、**ソースコードをデプロイ**を選択します。

![deploy](/deploy/common/select-service.en-US.png)

デプロイしたいSvelte Kitコードリポジトリを検索し、インポートをクリックします。Svelte Kitアプリは自動的にデプロイを開始します。

## ステップ4：ドメインをバインドする

デプロイが完了したら、Svelte Kitウェブサイトにドメインを設定することが求められるかもしれません。

サービスページの「ドメイン」タブを開き、「Generate Domain」または「Custom Domain」をクリックします。

サービスにドメインをバインドする方法についての詳細は、[ドメインのバインド](/deploy/domain-binding)を参照してください。
