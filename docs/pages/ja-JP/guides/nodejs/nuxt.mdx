# Nuxtアプリのデプロイ

ZeaburでNuxtアプリをデプロイしてください。

### ステップ1：Nuxtアプリの作成（任意）

端末を開いて、以下のコマンドを実行して新しいNuxtアプリを作成します。

<Tabs items={['pnpm', 'npx']}>
  <Tab>
    ```bash copy
    pnpm dlx nuxi init <project-name>
    ```
  </Tab>
  <Tab>
    ```bash copy
    npx nuxi init <project-name>
    ```
  </Tab>
</Tabs>

その後、依存関係をインストールします。

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm install --shamefully-hoist
    ```
  </Tab>
  <Tab>
    ```bash copy
    npm install
    ```
  </Tab>
  <Tab>
    ```bash copy
    yarn
    ```
  </Tab>
</Tabs>

<Callout type="info" emoji="ℹ️">
  pnpmは別の方法で依存関係を管理するため、依存関係をインストールする際に `--shamefully-hoist` フラグを使用する必要があります。
  pnpmを使用する場合は、デプロイ前に `.npmrc` ファイルに `shamefully-hoist=true` を追加していることを確認してください。
</Callout>

````.npmrc filename=".npmrc" copy
shamefully-hoist=true
````

### ステップ2：デプロイ

**新しいサービスを追加** ボタンをクリックし、**ソースコードをデプロイ** を選択してください。

![deploy](/deploy/common/select-service.en-US.png)

Nuxtアプリのリポジトリを検索し、インポートをクリックして、Nuxtアプリのデプロイを開始します。
