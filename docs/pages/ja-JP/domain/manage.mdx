# ドメイン管理

ドメイン管理ページには、アカウントに登録されたすべてのドメインが一覧表示されます。

## ドメインステータス

各ドメインは以下のいずれかのステータスを持ちます：

| ステータス | 説明 |
|-----------|------|
| **Active** | ドメインは登録済みで正常に動作しています。 |
| **Pending** | 登録処理中です。 |
| **Expired** | ドメインの有効期限が切れています。 |
| **Suspended** | ドメインが停止されています（例：登録者未認証）。 |
| **Failed** | 登録に失敗しました。 |

ドメインが多い場合は、ステータスフィルターのドロップダウンでリストを絞り込めます。

30 日以内に期限切れとなるドメインには、有効期限列に警告が表示されます。

## 自動更新

管理テーブルの各ドメインに自動更新トグルがあります。有効にすると、Zeabur が登録済みの支払い方法を使って有効期限前に自動更新します。

テーブルの行から直接切り替えられます。詳細ページを開く必要はありません。

## 手動更新

ドメインを手動で更新するには：

1. ドメインをクリックして詳細ページを開きます。
2. ヘッダーの **更新** ボタンをクリックします。ボタンに更新価格が表示されます。
3. チェックアウトフローを完了します。

<Callout type="info">
  詳細ページには、元の購入価格と更新価格の両方が参考として表示されます。
</Callout>

## Zeabur Email 統合

ドメイン詳細ページから、ワンクリックでドメインを **Zeabur Email** に接続できます。送信リージョンを選択すると、必要な DNS レコード（DKIM、SPF、DMARC）が自動的に作成されます。

詳細は [Zeabur Email ドメイン管理](/email/domains) ドキュメントを参照してください。
