# ドメイン管理


ドメイン管理ページには、アカウントに登録されたすべてのドメインが一覧表示されます。

## ドメインステータス

各ドメインには以下のいずれかのステータスがあります：

| ステータス | 説明 |
|-----------|------|
| **アクティブ** | ドメインは登録済みで正常に動作しています。 |
| **保留中** | 登録処理中です。 |
| **期限切れ** | ドメインの有効期限が過ぎています。 |
| **停止中** | ドメインが停止されています（例：登録者の認証未完了）。 |
| **失敗** | 登録に失敗しました。 |

ドメインが多い場合は、ステータスフィルターのドロップダウンで一覧を絞り込めます。

有効期限まで 30 日以内のドメインには、有効期限列に警告が表示されます。

## 自動更新

管理テーブルの各ドメインには自動更新トグルがあります。有効にすると、Zeabur が登録済みの支払い方法を使用してドメインの期限前に自動更新します。

テーブルの行から直接自動更新を切り替えられます。詳細ページを開く必要はありません。

## 手動更新

ドメインを手動で更新するには：

1. ドメインをクリックして詳細ページを開きます。
2. ヘッダーの **更新** ボタンをクリックします。ボタンには更新価格が表示されます。
3. チェックアウトフローを完了します。

<Callout type="info">
  詳細ページには、参考として購入時の価格と更新価格の両方が表示されます。
</Callout>

## Zeabur Email 統合

ドメイン詳細ページから、ワンクリックでドメインを **Zeabur Email** に接続できます。送信リージョンを選択すると、必要な DNS レコード（DKIM、SPF、DMARC）が自動的に作成されます。

メールドメインの認証の詳細については、[Zeabur Email ドメイン管理](/email/domains)のドキュメントをご覧ください。
