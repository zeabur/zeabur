# ドメイン登録

import { Callout, Steps } from 'nextra/components'

<Callout type="info">
  ドメインの購入には **Developer** または **Team** プランが必要です。Free プランのユーザーにはアップグレードの案内が表示されます。
</Callout>

## ドメインの検索

サイドバーから **ドメイン** ページを開き、希望のドメイン名を入力します。Zeabur が 9 つの TLD（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）の空き状況を同時にチェックします。

結果は段階的に読み込まれ、人気の TLD（`.com`、`.dev`、`.app`、`.io`）が最初に表示されます。各結果の横にリアルタイムで価格が表示されます。

`example.com` のような完全なドメインを入力すると、ページ上部に**ベストマッチ**カードが表示されます。

## ドメインの購入

<Steps>

### ドメインを選択

検索結果から利用可能なドメインを見つけ、**カートに追加** をクリックします。

### 登録者プロファイルを選択

購入ダイアログで既存の[登録者プロファイル](/domain/profiles)を選択するか、新しいプロファイルを作成します。デフォルトプロファイルが設定されている場合は自動的に選択されます。

### チェックアウトを完了

**次へ** をクリックして Zeabur のチェックアウトページに進みます。チェックアウトフローは他の Zeabur 製品と同じで、登録済みの支払い方法が使用されます。

支払い後、ドメインは **Pending** ステータスになり、登録処理を待ちます。ほとんどのドメインは数分以内に **Active** になります。

</Steps>

## 登録者認証

一部の TLD では購入後に登録者のメールアドレス認証が必要です：

1. 登録者プロファイルのメールアドレスに認証メールが送信されます。
2. 管理ページのドメインカードに **認証待ち** バッジが表示されます。
3. バッジをクリックするか、ドメイン詳細ページから認証メールを再送信できます。

<Callout type="warning">
  未認証のままのドメインはレジストリによって停止される場合があります。購入後は速やかに認証を完了してください。
</Callout>
