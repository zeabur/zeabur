# 登録者プロファイル


ドメイン登録には WHOIS 連絡先情報が必要です。登録者プロファイルを使えば、情報を一度保存して複数のドメイン購入で再利用できます。

## プロファイルフィールド

| フィールド | 必須 | 説明 |
|-----------|------|------|
| 名 | はい | 登録者の名前 |
| 姓 | はい | 登録者の姓 |
| メールアドレス | はい | 連絡先メール（登録者認証にも使用） |
| 電話番号 | はい | 連絡先電話番号 |
| 住所 1 | はい | 番地 |
| 住所 2 | いいえ | マンション名、部屋番号など |
| 市区町村 | はい | 市区町村 |
| 都道府県 | はい | 都道府県 |
| 国 | はい | 国（ドロップダウンから選択） |
| 郵便番号 | はい | 郵便番号 |
| 組織 | いいえ | 会社または組織名 |

## プロファイルの作成

<Steps>

### プロファイルタブを開く

サイドバーから **ドメイン** セクションに移動し、**プロファイル** タブを選択します。

### フォームに入力

**プロファイルを作成** をクリックし、必須フィールドを入力します。検索可能なドロップダウンから国を選択します。

### 保存

**保存** をクリックします。新しいプロファイルはすぐにドメイン登録で使用できます。

</Steps>

## デフォルトプロファイル

星アイコンをクリックしてプロファイルを **デフォルト** に設定します。デフォルトプロファイルは、ドメイン購入時の登録者プロファイル選択で自動的に選択されます。チェックアウト時に別のプロファイルに切り替えることもできます。

## 編集と削除

- プロファイルカードの **編集アイコン** をクリックしてフィールドを更新します。
- **削除アイコン** をクリックしてプロファイルを削除します。

<Callout type="warning">
  プロファイルを削除しても、そのプロファイルで登録済みのドメインには影響しません。既存のドメインは登録時に送信された連絡先情報を保持します。
</Callout>

## インライン作成

ドメイン購入ダイアログから直接新しいプロファイルを作成することもできます。プロファイルタブに移動する必要はありません。新しいプロファイルはアカウントに保存され、今後の登録でも再利用できます。
