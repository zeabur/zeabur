# FAQ とサポート

## ヘルプを得る

Zeabur の使用中に問題が発生した場合、以下の方法でヘルプを受けられます：

- [コミュニティフォーラム](https://zeabur.com/forum) — コミュニティフォーラムで交流
- [ディスカッション](https://github.com/orgs/zeabur/discussions) — GitHub Discussions で質問
- [サポートに連絡](/get-started/faq-support/help) — その他のサポートチャネルについて
- [コミュニティ認証](/get-started/faq-support/verify) — コミュニティ ID の認証方法

## よくある質問

### パスワードをリセットするには？

Zeabur はサードパーティログイン（GitHub、Google など）を使用しているため、リセットすべき個別のパスワードはありません。ログインできない場合は、サインアップ時と同じログイン方法を試すか、[コミュニティフォーラム](https://zeabur.com/forum)でサポートチームに連絡してください。

### デプロイが失敗するのはなぜ？

デプロイの失敗は通常、以下の原因で発生します：

- **ビルドエラー** — コードがローカルで正常にビルドされることを確認してください。
- **環境変数の不足** — 必要な環境変数がすべて正しく設定されていることを確認してください。
- **ポート設定の誤り** — アプリケーションが正しいポートでリッスンしていることを確認してください（Zeabur は `PORT` 環境変数を自動注入します）。
- **メモリ不足** — ビルドまたはランタイムがプランのリソース制限を超えている可能性があります。

ダッシュボードのサービスログで詳細なエラーメッセージを確認してください。

### 課金はどのように機能しますか？

Zeabur は使用量ベースの課金モデルを採用しており、CPU、メモリ、ネットワーク使用量に基づいて課金されます。詳細は[料金プラン](/pricing/pricing-plans)をご覧ください。

### カスタムドメインは使えますか？

はい。Zeabur は独自ドメインのバインドに対応しています。DNS 設定に適切な CNAME レコードを追加する必要があります。詳しい手順は[パブリックネットワーキング](/deploy/public-networking)をご覧ください。

### サポートに連絡するには？

以下の方法でお問い合わせいただけます：

- [コミュニティフォーラム](https://zeabur.com/forum)に参加してサポートチャネルで質問
- [GitHub Discussions](https://github.com/orgs/zeabur/discussions) で質問を投稿
- 詳細は[サポートページ](/get-started/faq-support/help)をご覧ください

### どのプログラミング言語に対応していますか？

Zeabur は Node.js、Python、Go、Rust、Java、.NET、PHP、Ruby など、幅広いプログラミング言語とフレームワークに対応しています。完全なリストとデプロイガイドは[ガイド](/guides)をご覧ください。

### プランをアップグレードするには？

Zeabur ダッシュボードのアカウント設定でサブスクリプションプランのアップグレードや変更ができます。詳しい手順は[サブスクリプション](/subscription)をご覧ください。

### 無料プランはありますか？

はい。Zeabur は無料プランを提供しており、無料でサービスの試用とデプロイが可能です。無料プランにはリソース制限があります。詳細は[無料プラン](/pricing/free-plan)をご覧ください。
