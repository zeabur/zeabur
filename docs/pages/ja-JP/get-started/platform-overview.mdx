# プラットフォーム概要

**Zeabur** は AI 搭載のクラウドデプロイプラットフォームで、サーバー管理、CI/CD 設定、インフラの手間なしに、数分でアプリケーションを公開できます。

## コア機能

### 自動デプロイ

- **言語とフレームワークの自動検出** — コードをプッシュすると、Zeabur が Node.js、Python、Go、Java、.NET、Rust などを自動識別し、最適なビルド戦略を選択。
- **すぐに使える CI/CD** — `git push` のたびにビルドとデプロイが自動トリガー、設定不要。
- **テンプレートマーケットプレイス** — 数百の事前設定済みテンプレートで、WordPress、n8n、PostgreSQL などをワンクリックデプロイ。

### AI アシスタント

Zeabur には AI アシスタント（Agent）が内蔵されており、自然言語で必要なことを説明できます。プロジェクトの作成、サービスのデプロイ、環境変数の設定、ドメインのバインドを自動的に行います。

### AI Hub

[AI Hub](/ai-hub) を通じて、GPT-4o、Claude、Gemini などの主要な LLM の API を OpenAI 互換形式で直接呼び出せます。各プロバイダーに個別に登録する必要はありません。

### オールインワン統合

フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージを1つのプロジェクトにデプロイ。サービスはプライベートネットワークで通信でき、環境変数は相互参照可能です。

### マルチ環境分離

各プロジェクトは複数の環境（本番、ステージングなど）をサポートし、完全に分離されて開発とテストのワークフローを効率化します。

### 柔軟なインフラストラクチャ

- **共有クラスタ**（段階的に廃止中）— すぐに始められ、サーバー管理不要。共有クラスタでの新規プロジェクト作成は終了。
- **サーバー** — 自分のマシン（BYO Server）で完全制御。[サーバー](/dedicated-server)を参照。
