# メンバーの招待

プロジェクトには2人以上のメンバーが参加できます。プロジェクトオーナーはメンバーをプロジェクトに招待して、サービスを共同で管理できます。

## メンバーを招待する方法

### ステップ1：プロジェクト設定を開く

まず、メンバーを招待したいプロジェクトを開きます。プロジェクトページの左上にある設定ボタンをクリックすると、プロジェクト設定ページに移動します。

![Project Config](/manage/invite/project-config.png)

### ステップ2：メンバーを招待する

サイドバーからチームパネルを選択すると、チームパネルにメンバー招待ボタンが表示されます。

![Project Settings](/manage/invite/project-settings.png)

招待したいメンバーのメールアドレスを入力し、招待ボタンをクリックします。メンバーにはプロジェクトへの参加招待メールが送信されます。

また、招待リンクをコピーして招待したいメンバーに共有することもできます。メンバーはリンクを開くだけでプロジェクトに参加できます。

![Invite Member](/manage/invite/invite.png)

## FAQ

- **Q: Zeabur に未登録のメンバーを招待できますか？**
    - A: 現在、Zeabur に登録済みのメンバーのみ招待可能です。

- **Q: プロジェクトに参加したメンバーにはどのような権限がありますか？**
    - A: メンバーはプロジェクトの削除権限を除き、プロジェクトオーナーと同じ権限を持ちます。

- **Q: プロジェクトからメンバーを削除できますか？**
    - A: はい、プロジェクト設定ページのチームパネルからメンバーを削除できます。
