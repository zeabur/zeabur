# プロジェクト予算

予期しない高額な費用を避けるために、プロジェクトごとに予算を設定できます。

使用量が予算の80%に達すると通知が届き、予算に達するとサービスが一時停止されて超過費用を防ぎます。

## 予算の設定方法

1. プロジェクトダッシュボードの左上にある歯車アイコンをクリックして、プロジェクト設定に移動します。

![Budget Entry](/manage/budget/budget-entry.png)

2. 予算ページで「Update Monthly Budget」ボタンをクリックして、このプロジェクトの予算を設定します。

![Budget Page](/manage/budget/budget-page.png)

このページではプロジェクトの現在の期間のコストを確認できるため、適切な予算を決定する際に役立ちます。

## FAQ

### 予算に近づいているかどうかはどうやってわかりますか？

使用量が予算の80%に達すると通知が届きます。また、プロジェクト設定で使用量を確認することもできます。

### 予算に達するとどうなりますか？

このプロジェクトのサービスは、超過費用を防ぐために一時停止されます。

### 設定後に予算を変更できますか？

はい、いつでも予算を変更できます。新しい予算は即座に有効になります。

### 最低予算はいくらですか？

最低予算は $10.00 です。
