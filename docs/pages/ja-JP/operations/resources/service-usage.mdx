# 使用量分析

Zeabur は従量課金モデルを採用しており、サービスの使用量を表示する詳細なチャートも提供しています。

サービスをクリックして **Metrics** タブを選択すると、チャートが表示されます。

### メモリ使用量

サービス実行中に消費されたメモリを表示します。

![memory](/manage/metric/memory.png)

### CPU 使用量

サービス実行中に使用された CPU を表示します。

![memory](/manage/metric/CPU.png)

### ネットワーク使用量

Zeabur の各リージョンからのサービスのネットワークトラフィックを表示します。

![network](/manage/metric/network.png)
