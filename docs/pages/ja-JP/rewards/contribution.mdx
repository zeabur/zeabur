# コントリビューション報酬プログラム

開発者の皆様に Zeabur の [テンプレート](https://zeabur.com/templates) や [ドキュメント](https://zeabur.com/docs) のコンテンツを充実させていただくため、「コントリビューション報酬プログラム」を開始しました。貢献者には無料利用クレジットを提供します。

以下の貢献方法を参考にして、コミュニティフォーラムで貢献内容をお知らせください。貢献に応じた無料利用クレジットを付与いたします。

このページに記載されている報酬額は固定値ではありません。貢献の内容、品質、難易度などを考慮して報酬額を決定します。最大の報酬を得るために、貢献を始める前にご相談いただくことをお勧めします。

## テンプレートの貢献

テンプレートの貢献方法には、テンプレートの作成、補充、翻訳があります。

### テンプレートの作成

テンプレートを作成することで、他の開発者がオープンソースプロジェクトをワンクリックで Zeabur にデプロイできるようになります。[YAML からテンプレートを作成](../template/template-format) を参照してください。

作成した各テンプレートで最低 **$10** の無料利用クレジットを獲得できます。ただし、以下の**すべての**条件を満たす必要があります：

1. 既存のテンプレートと重複しないこと（新バージョンや異なるシナリオ・用途の場合を除く）。
2. テンプレート名、説明、カバー画像、紹介、使用方法を含む完全な情報が含まれていること。
3. テンプレートの紹介に少なくとも英語版があること。
4. ワンクリックでデプロイ可能であること。デプロイ後に追加操作が必要な場合は、紹介に記載すること。

上記の必須条件に加え、プログラムの悪用を防ぐため、以下の条件の**少なくとも1つ**を満たす必要があります：

1. 対応するオープンソースプロジェクトが GitHub で 500 スター以上。
2. 対応するオープンソースプロジェクトが他の PaaS プラットフォームでワンクリックデプロイボタンまたはテンプレートを持ち、30 回以上使用されている。

ボーナス：テンプレートが他の開発者に 30 回以上使用された場合（デプロイ後も継続稼働している必要あり）、追加で **$20** の無料利用クレジットを獲得できます。

### テンプレートの補充

既存のテンプレートにコンテンツを追加することです。例えば、デプロイ後に追加の設定手順が必要だが紹介に記載されていない場合、その手順を追加できます。

補充した各テンプレートで最低 **$5** の無料利用クレジットを獲得できます。ただし、以下の**すべての**条件を満たす必要があります：

1. デプロイ回数が 30 回以上のテンプレートであること。
2. 補充内容が具体的な助けになること（紹介の追加、使用方法の追加、デプロイ手順の追加など）。

テンプレートの設定自体に問題があり、ワンクリックデプロイが正常に動作しない場合、テンプレートの修正と説明の追加でも **$5** の無料利用クレジットを獲得できます。

### テンプレートの翻訳

既存のテンプレートを翻訳することです。例えば、英語のみの紹介を他の言語に翻訳できます。

翻訳した各テンプレートで最低 **$5** の無料利用クレジットを獲得できます。ただし、以下の**すべての**条件を満たす必要があります：

1. デプロイ回数が 30 回以上のテンプレートであること。
2. テンプレート紹介や変数名・説明などのフィールドを含む完全な翻訳であること。
3. テンプレートページがサポートする言語（英語、繁体中国語、簡体中国語、日本語、スペイン語）に翻訳すること。

## 参加方法

コントリビューション報酬プログラムに参加したい場合は、まず [Zeabur コミュニティフォーラム](https://zeabur.com/forum) に参加し、プログラムへの参加意思と、提出・補充・翻訳したいテンプレートやドキュメントのページをお知らせください。

貢献したいが、どのオープンソースプロジェクトのテンプレートを作成すべきか、どのテンプレートを修正・翻訳すべきかわからない場合も、お気軽にご連絡ください。他のユーザーのニーズをお伝えできます。

貢献の過程で問題が発生した場合も、直接お知らせください。貢献プロセスの改善に役立てます。
