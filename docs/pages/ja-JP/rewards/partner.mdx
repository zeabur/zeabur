# パートナープログラム

Zeabur Partner Program は、講師、イベント主催者、導入コンサルタントに向けた協業プログラムです。Zeabur の推進を通じて報酬を獲得できます。

## 2つの協業モデル

### 教育推進

**講師、研修機関、イベント主催者**向け。教育コースや技術イベントを通じて Zeabur を推進し、Event Code で受講生に無料体験を提供。継続課金ユーザー数に応じて報酬を獲得できます。

- 専用 Event Code を提供
- 1ヶ月後に継続課金ユーザーを集計、1人あたり US$5 の報酬
- ワークショップ、オンラインコース、ハッカソンなどに最適

[詳しく見る →](/partner/education)

### 導入コンサルタント（Sales Partner）

**企業コンサルタント、システムインテグレーター、技術コンサルティング会社**向け。企業クライアントの Zeabur 導入を支援し、クライアントが課金を継続する限り 15% のリカーリングコミッションを獲得できます。

- トリガー条件：単発金額 > US$1,000 または機関一括購入
- コミッションモデル：クライアント課金継続中、毎月 15% recurring commission
- 企業導入、機関調達など大規模案件に最適

[詳しく見る →](/partner/sales-partner)

## リファラルプログラムとの違い

Zeabur は[リファラルプログラム](/rewards/referral)も提供しており、すべてのユーザーが紹介コードで報酬を獲得できます。主な違い：

| 項目 | リファラルプログラム | パートナープログラム |
| --- | --- | --- |
| 対象 | すべてのユーザー | 講師 / イベント主催者 / 企業コンサルタント |
| トリガー | 紹介コード自動フロー | 手動審査と契約 |
| 報酬 | 紹介コードコミッション（ティアにより 20%–100%） | Event Code 継続報酬 / 15% recurring commission |
| 金額閾値 | なし | Sales Partner：単発 > US$1,000 |

<Callout type="info">
1つの取引は1つのパスのみ——紹介コードまたはパートナープログラム。両方を併用することはできません。
</Callout>
