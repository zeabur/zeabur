# 使用量と請求

InsForgeは詳細な使用量追跡を提供し、リソース消費とコストを監視できます。

## 使用量の確認

1. ZeaburダッシュボードでプロジェクトのInsForge統合ページに移動
2. **使用量概要** セクションまでスクロール

### 概要タブ

選択した期間の使用量サマリーを表示：

- **最大データベースサイズ** — データベースストレージのピーク使用量
- **最大ファイルストレージ** — ファイルストレージのピーク使用量
- **合計エグレス** — 消費した総アウトバウンド帯域幅
- **EC2コンピュート** — 使用した総コンピュート時間
- **AI Credits** — 消費したLLM AIクレジット
- **AI Tokens** — 処理したAIトークン総数
- **MCP呼び出し** — MCPプロトコル呼び出し総数
- **メールリクエスト** — メールAPIリクエスト総数
- **ファンクション呼び出し** — サーバーレスファンクション実行総数

### 日別使用量タブ

日別使用量タブは、日付ごとのリソース使用量の内訳をページ付きテーブルで提供します。

### 期間フィルター

- **1日** — 過去24時間
- **7日** — 過去1週間
- **30日** — 過去1ヶ月
- **全期間** — すべてのデータ

## 請求

import { Callout } from 'nextra/components'

<Callout type="warning">
  InsForgeは別途課金され、Zeabur無料クレジットには含まれません。InsForgeはクラウドホスティングサービスのため、Zeaburプロジェクトがサーバー上で稼働している場合でも課金が発生します。
</Callout>

### 料金

| 項目 | 料金 |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / 時間 |
| Database Size | $0.125 / GB / 月 |
| Storage | $0.021 / GB / 月 |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / credit |

### 請求の重要事項

- **人工的な制限なし** — APIレート制限、行数制限、最大ユーザー数上限はありません
- **統一請求** — すべてのInsForgeコストは既存のZeaburプロジェクト請求に統合されます
- **従量課金** — バックエンドが実際に消費するリソースに対してのみ課金

### 請求の問題

InsForgeインスタンスが請求問題で一時停止された場合：

1. Zeaburダッシュボードで **請求** に移動
2. 支払い方法を追加するか残高をチャージ
3. InsForge統合ページに戻り、**復元** をクリックして再課金しサービスを再開

<Callout type="info">
  請求問題で一時停止されたインスタンスのデータは保持されます。請求問題を解決してインスタンスを復元するとサービスが再開されます。
</Callout>
