# InsForge

InsForgeは、Zeaburに統合されたAIネイティブなSupabase代替サービスです。フルサービスをデプロイせずに、シンプルで軽量なバックエンドとデータベースを必要とする開発者を支援します。AIエージェントはMCPプロトコルを通じて、フルスタックアプリケーションを自律的に構築・管理できます。

## コア機能

- **PostgreSQL データベース** - 設定不要でインスタントREST APIを提供
- **JWT 認証** - ソーシャルサインオン対応の組み込み認証
- **S3互換ファイルストレージ** - 馴染みのあるAPIでファイルを保存・提供
- **サーバーレスエッジファンクション** - インフラ管理不要でサーバーサイドロジックをデプロイ
- **MCP プロトコル対応** - AI IDEを接続し、エージェントがバックエンドを自律管理

## 仕組み

import { Callout } from 'nextra/components'

1. Zeaburプロジェクトダッシュボードで**InsForgeを有効化**
2. MCPプロトコルで**AI IDEを接続**（Cursor、Claude Code、Windsurf、Trae、Cline、Roo Code）
3. **AIエージェントに構築を任せる** — テーブル作成、認証管理、ストレージ処理などを自律的に実行

<Callout type="info">
  InsForgeは各Zeaburプロジェクトに専用のバックエンドインスタンスを作成します。各インスタンスには固有のAPI EndpointとAPI Keyがあります。
</Callout>

## 料金

InsForgeは別途課金され、**無料クレジットには含まれません**。バックエンドが実際に消費するリソースに対してのみ課金されます：

| 項目 | 料金 |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / 時間 |
| Database Size | $0.125 / GB / 月 |
| Storage | $0.021 / GB / 月 |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / credit |

- **人工的な制限なし** — APIレート制限、行数制限、最大ユーザー数の上限はありません
- **統一請求** — すべてのコストは既存のZeaburプロジェクト請求に統合されます

<Callout type="warning">
  InsForgeを有効化する前に、クレジットカードの登録と十分なアカウント残高をご確認ください。
</Callout>
