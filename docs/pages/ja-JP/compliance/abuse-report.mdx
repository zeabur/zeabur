# 不正利用報告ガイドライン

Zeabur はサイバーセキュリティ組織と協力して、犯罪や悪意のある利用と戦っています。[公正利用ガイドライン](/legal/fair-use-guideline)に違反する行為の報告を奨励しています。[公正利用ガイドライン](/legal/fair-use-guideline)に違反するウェブサイトや活動を発見した場合は、すぐにご報告ください。

政府機関やサイバーセキュリティ組織の場合は、自動報告システムとの統合についてもお問い合わせいただけます。
