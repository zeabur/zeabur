# クラスターを購入する

Zeabur は Linode クラスターをワンクリックで作成するフローを提供しています。コントロールプレーン、ノードプール、ネットワーク設定を手動でセットアップすることなく、すぐに使える Kubernetes クラスターを求めるチームに最適です。

## 開始前の準備

- Zeabur アカウントに有効な支払い方法または十分な残高があることを確認してください。
- クラスターのリージョンとノードプランを決定してください。

## 購入手順

1. Zeabur にログインし、[サーバー](https://zeabur.com/servers)ページに移動します。
2. 「クラスター」を選択し、右上の「作成」をクリックします。
   ![cluster page](/cluster/cluster-page.webp)
3. 作成フローで「Zeabur からクラスターを購入」を選択します。
   ![create cluster](/cluster/create-cluster.webp)
4. クラスターのリージョンとノードプランを選択し、「今すぐ購入」をクリックします。
   ![select cluster plan](/cluster/cluster-select-plan.webp)
5. コストと設定を確認し、注文を送信します。
6. Zeabur がクラスターの作成と初期化を完了するまで待ちます — 通常 3〜5 分かかります。
7. 完了すると、新しいクラスターがサーバーリストに表示され、Zeabur から直接プロジェクトを作成してサービスをデプロイできます。
   ![cluster information](/cluster/cluster-information.webp)

クラスターが作成された後、Zeabur はオンボーディングに必要な基盤となるセットアップを処理するため、Kubernetes コンポーネントを自分で設定することなく、Dashboard からデプロイを管理できます。

## 次のステップ

- 既存のクラスターを Zeabur に接続するには、[既存のクラスターを接続する](/cluster/bind)を参照してください。
- 単一ノードとマルチノードのセットアップの違いをまだ検討している場合は、[クラスター概要](/cluster)に戻ってください。
