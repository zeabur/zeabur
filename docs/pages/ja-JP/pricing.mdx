# 定価

Zeabur は、カスタムの Enterprise オプションを含む 4 つのサブスクリプションプランを提供しています：

- **[Free Plan](/pricing/free-plan)** — US$0/月
- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 日間無料トライアル）
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 日間無料トライアル、本番ワークロードに推奨）
- **[Team Plan](/pricing/team-plan)** — US$79/月でシート 3 つ（追加シートは +US$24/シート/月）
- **Enterprise** — カスタム定価、[お問い合わせ](https://zeabur.com/forum)ください
