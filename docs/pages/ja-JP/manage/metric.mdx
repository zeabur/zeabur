# サービス使用量

Zeaburはボリュームベースの課金モデルを採用しており、サービスの使用量を詳細なチャートで表示します。

サービスをクリックして **Metrics** タブを選択すると、チャートが表示されます。

### メモリ使用量

サービス実行中のメモリ消費量を表示します。

![memory](/manage/metric/Memory_Usage.png)

### CPU使用量

サービス実行中のCPU使用量を表示します。

![cpu](/manage/metric/CPU_Usage.png)

### ネットワーク使用量

Zeaburの各リージョンからのサービス実行中のネットワークトラフィックを表示します。

![network](/manage/metric/Network_Usage.png)

### ディスク使用量

Google Cloud Platform (GCP) デプロイの場合、ディスク使用量の上限は1 GBです。その他のプラットフォームでは、ディスク使用量は従量課金制で、ストレージニーズに応じた柔軟な利用が可能です。

![disk](/manage/metric/disk_usage.png)

### 課金

より詳細な課金ドキュメントは[こちら](/ja-JP/billing/pricing)を参照してください。
