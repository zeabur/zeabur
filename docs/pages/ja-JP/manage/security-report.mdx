# セキュリティレポート

Zeabur は、AI 分析を使用してアプリケーション内の潜在的なセキュリティ脆弱性を検出する、内蔵のセキュリティレポート機能を提供しています。

セキュリティレポート機能は、AI駆動のインテリジェント分析を採用しており、多くの一般的なセキュリティ脆弱性（OWASP Top 10、CWE Top 25 などの業界標準）をチェックします。これには、Secret Key の漏洩、SQL インジェクション、XSS 攻撃などが含まれます。ワンクリックで、5〜10分以内にスキャンを完了し、レポートを生成できます。

## セキュリティレポートの生成方法

### ステップ1：サービスページに移動

チェックしたいサービスをクリックし、**セキュリティ**タブを選択します。

![セキュリティレポートページ](https://cdn.zeabur.com/security-report/security-report-list.png)

### ステップ2：レポートの生成

**「レポート生成」**ボタンをクリックすると、システムがコードの分析を開始します。

<Callout type="info">
GitHub からデプロイされたコードでも、ローカルからアップロードされたコードでも、セキュリティレポート機能を使用できます。
</Callout>

### ステップ3：レポートの表示

数分待つと、リストに新しく生成されたレポートが表示されます。レポートをクリックすると、検出された問題、リスクレベル、コード内の位置を含む詳細な内容が表示されます。

![セキュリティレポート詳細ページ](https://cdn.zeabur.com/security-report/security-report-detail.png)

## セキュリティ問題の修正方法

レポートの内容を**コピー**または**ダウンロード**して、お好みの AI 開発ツール（Cursor、Claude Code など）に貼り付け、AI にコードの修正を手伝ってもらいます。

修正後、コードを再デプロイしてセキュリティの脆弱性を解決します。

## よくある質問

### セキュリティレポートはどのような脆弱性をチェックしますか？

セキュリティレポート機能は、以下を含む複数の一般的なセキュリティ脆弱性をチェックします：

- OWASP Top 10（SQL インジェクション、XSS 攻撃など）
- CWE Top 25
- Secret Key と機密情報の漏洩
- 安全でない依存関係

### スキャンにはどのくらい時間がかかりますか？

コードの複雑さとサイズによって、スキャンには通常 5〜10 分かかります。

### レポートをダウンロードできますか？

はい。レポートの内容をコピーまたはダウンロードして、チームと共有したり、AI 開発ツールに貼り付けて修正したりできます。

### レポートは保存されますか？

はい。生成されたすべてのレポートはリストに保存され、いつでも履歴を確認できます。
