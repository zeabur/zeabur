# メンバーを招待

プロジェクトには2人以上のメンバーが存在する場合があります。プロジェクトオーナーはメンバーを招待してサービスを管理できます。

## メンバーの招待方法

### ステップ1: プロジェクト設定を開く

まず、メンバーを招待したいプロジェクトを開きます。プロジェクトページの左上隅にある設定ボタンをクリックすると、プロジェクト設定ページに移動します。

![Project Config](/manage/invite/Invite_team_member_step_1.png)

### ステップ2: メンバーを招待

サイドバーからチームパネルを選択すると、チームパネル内にメンバーを招待するボタンが表示されます。

![Project Settings](/manage/invite/Invite_team_member_step_2.png)

招待したいメンバーのメールアドレスを入力し、招待ボタンをクリックします。メンバーはプロジェクトに参加するための招待メールを受け取ります。

招待リンクをコピーして招待したいメンバーに共有することもできます。メンバーはリンクを開くだけでプロジェクトに参加できます。

![Invite Member](/manage/invite/project_team_pages.png)

## よくある質問 (FAQ)

- **Q: Zeaburに未登録のメンバーを招待できますか？**  
  A: 現在、Zeaburに登録済みのメンバーのみ招待可能です。

- **Q: メンバーがプロジェクトに参加した後の権限は？**  
  A: プロジェクトオーナーと同じ権限を持ちますが、プロジェクトの削除権限はありません。

- **Q: プロジェクトからメンバーを削除できますか？**  
  A: はい、プロジェクト設定ページのチームパネルからメンバーを削除できます。
