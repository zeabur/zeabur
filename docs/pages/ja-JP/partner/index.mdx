# パートナープログラム

Zeabur パートナープログラムは、講師、イベント主催者、セールスコンサルタント向けのコラボレーション機会を提供し、Zeabur を推進することで報酬を獲得できます。

## 2 つのパートナーシップモデル

### 教育推進

**講師、研修機関、イベント主催者**向けです。授業やテクニカルイベントを通じて Zeabur を推進し、参加者に無料トライアルアクセスを提供する Event Code を使用して、継続的な有料ユーザー数に基づいて報酬を獲得します。

- イベント専用の Event Code を受け取る
- 1 ヶ月後、継続的に利用する有料ユーザーごとに US$5 を獲得
- ワークショップ、オンラインコース、ハッカソンに最適

[詳細を見る →](/partner/education)

### セールスパートナー

**企業コンサルタント、システムインテグレーター、テクノロジーコンサルティング企業**向けです。企業クライアントの Zeabur 導入を支援します。契約が締結され支払いが開始されると、キャッシュバック報酬を獲得します。

- トリガー：単一契約が US$1,000 超または機関一括購入
- キャッシュバック：契約総額 ÷ 契約月数（1 ヶ月相当額）
- 企業オンボーディングおよび機関調達に最適

[詳細を見る →](/partner/sales-partner)

## リファラルプログラムとの違い

Zeabur は、任意のユーザーがリファラルコードを通じて報酬を獲得できる[リファラルプログラム](/rewards/referral)も提供しています。主な違い：

| 側面 | リファラルプログラム | パートナープログラム |
| --- | --- | --- |
| 対象者 | すべてのユーザー | 講師 / 主催者 / コンサルタント |
| 方法 | 自動リファラルコードフロー | 手動審査と契約 |
| 報酬 | リファラルコードコミッション | Event Code 継続報酬 / 契約キャッシュバック |
| 閾値 | なし | セールスパートナー：単一取引 US$1,000 超 |

<Callout type="info">
1 つの取引は、リファラルコードまたはパートナープログラムのいずれか 1 つの経路のみを通ることができます。両方を併用することはできません。
</Callout>
