# 教育推進

教育推進プログラムは、授業やテクニカルイベントを通じて学生に Zeabur を紹介する**講師、研修機関、イベント主催者**向けで、有料ユーザーに転換した参加者の数に基づいて報酬を獲得します。

## 仕組み

1. **応募** — Zeabur チームに連絡し、コースやイベントの計画を説明します。
2. **Event Code を受け取る** — Zeabur がイベント専用の Event Code を作成します。学生はこのコードを使用して Zeabur を無料で試すことができます。
3. **イベントを実施** — コースやイベント中に、学生が Zeabur でプロジェクトをデプロイできるよう案内します。
4. **継続カウント** — イベント後 1 ヶ月で、Zeabur が Event Code で登録し有料プランを利用しているユーザーをカウントします。
5. **報酬を請求** — 継続的な有料ユーザーごとに US$5 を獲得します。

## Event Code のメカニズム

Event Code は、イベント中に参加者にトライアル Credits を提供するユニークなコードです：

- 各 Event Code は特定のイベントに紐付けられます。
- 学生は Zeabur 登録時に Event Code を入力してトライアル Credits を受け取ります。
- Event Code にはイベントスケジュールに基づく有効期間があり、通常はイベントとその後のバッファ期間をカバーします。

<Callout type="info">
Event Code の利用はリファラルプログラムのコミッションメカニズムをトリガーしません。Event Code はトライアル Credits を提供するものであり、実際の支払いではないためです。リファラルコミッションは、ユーザーの実際の支払いが成功した場合にのみ計算されます。
</Callout>

## 報酬の計算

| 項目 | 詳細 |
| --- | --- |
| カウント期間 | Event Code 配布から 1 ヶ月後 |
| 継続の基準 | Event Code で登録し、有料プランにアップグレードし、カウント期間中にアクティブであること |
| 報酬額 | 継続的な有料ユーザーごとに US$5 |
| 支払い方法 | Zeabur Credits またはその他の合意された方法 |

### 例

50 名の参加者向けにワークショップを開催し、Event Code を配布します。1 ヶ月後、8 名の学生がまだ Zeabur を使用しており、有料プランにアップグレードしています。あなたの報酬は 8 × US$5 = **US$40** です。

## 最適なシナリオ

- **プログラミングコース** — 学生がコースプロジェクトをクラウドにデプロイ。
- **テクニカルワークショップ** — 参加者がリアルタイムでデプロイプロセスを体験するハンズオンイベント。
- **ハッカソン** — チームが Zeabur を使用してプロジェクトを迅速にデプロイ。
- **オンラインコース** — リモートの学生に練習用の Event Code を提供。

## 応募方法

教育推進パートナーシップに応募するには、Zeabur チームにお問い合わせください：

- **コミュニティフォーラム**: [Zeabur コミュニティフォーラム](https://zeabur.com/forum)でイベント計画を説明して投稿を作成してください。
- イベント名、予想参加人数、日程、形式をお伝えください。Zeabur チームが Event Code を設定します。
