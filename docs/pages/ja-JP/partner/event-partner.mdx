# 教育パートナー (Education Partner)

教育パートナープログラムは、**講師、コミュニティリーダー、ハッカソン主催者、教育機関**を対象としています。テクニカルイベントの開催、講座の運営、ワークショップの主導などを行う際、イベントコード (Event Code) を利用することで参加者に無料で Zeabur を体験していただき、イベント終了後に有料ユーザーとして継続した人数に応じて報酬を獲得できます。

## 仕組み

1. **応募** — [partner.zeabur.com](https://partner.zeabur.com) から応募し、講座やイベントの計画を説明します。
2. **イベントコードを受け取る** — Zeabur がイベント専用のイベントコードを作成します。参加者はこのコードを入力することで体験用クレジットを受け取れます。
3. **イベントを実施** — 講座、ワークショップ、ハッカソンなどで、参加者が Zeabur を使ってプロジェクトをデプロイできるよう案内します。
4. **継続状況をカウント** — イベント終了から 1 ヶ月後に、イベントコード経由で登録し、継続して有料プランを利用しているユーザーをカウントします。
5. **報酬を受け取る** — 継続した有料ユーザー数に応じて報酬が支払われます。

## イベントコードの仕組み

イベントコードは、参加者にイベント期間中の体験用クレジットを提供する専用コードです：

- 各イベントコードは特定のイベントに紐付けられます。
- 参加者は Zeabur の登録フローでイベントコードを入力することで体験用クレジットを受け取れます。
- イベントコードには有効期間があり、通常はイベント期間とその後のバッファ期間をカバーします。
- 参加者向けの詳しい引き換え手順は [イベントコードの引き換え](/partner/event-partner/redeem-event-code) をご参照ください。

<Callout type="info">
イベントコードの引き換えは Referral Program のコミッション機構をトリガーしません。イベントコードは体験用クレジットを提供するものであり、実際の支払いではないためです。リファラルコミッションは、ユーザーが実際に支払いを完了した場合にのみ計算されます。
</Callout>

## 報酬の計算

| 項目 | 詳細 |
| --- | --- |
| カウント期間 | イベントコード配布から 1 ヶ月後 |
| 継続基準 | 参加者がイベントコード経由で登録し、有料プランにアップグレードし、カウント期間中も利用を継続していること |
| 基本報酬 | 継続した有料ユーザー 1 名につき US$5 |
| 支払い方法 | Zeabur Credits またはその他の合意された方法 |

### 例

50 名規模のワークショップでイベントコードを配布したとします。イベント終了から 1 ヶ月後、8 名の参加者が Zeabur を使い続け、有料プランにアップグレードしていました。あなたの報酬は 8 × US$5 = **US$40** となります。

### 大型イベントの個別計算

大型のハッカソン、長期講座、影響力の大きいシリーズイベントについては、イベント規模、主催者の実績、想定される転換率などを踏まえて、単位報酬を **継続した有料ユーザー 1 名につき US$10** まで引き上げることがあります。こうした協業は事前に Zeabur チームと協議し、応募時にその旨を明記してください。

## 適したシナリオ

- **プログラミング講座** — 受講生が講座のプロジェクトをクラウドにデプロイ。
- **テクニカルワークショップ** — デプロイフローをその場で体験できるハンズオンイベント。
- **ハッカソン** — 参加チームが Zeabur を使って作品を素早くデプロイ。
- **オンライン講座** — リモート受講生が実習できるようイベントコードを配布。
- **コミュニティイベント** — 技術コミュニティの勉強会や LT 会とデモ環境を組み合わせて実施。

## 応募方法

[partner.zeabur.com](https://partner.zeabur.com) でアカウントを作成し、教育パートナーシップ応募フォームに、イベント名、想定参加人数、日程、形式を記入して送信してください。Zeabur チームが審査のうえ、専用のイベントコードを発行します。

その他のご質問は、[Zeabur コミュニティフォーラム](https://zeabur.com/forum) までお気軽にお問い合わせください。
