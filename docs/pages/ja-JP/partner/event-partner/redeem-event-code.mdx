# イベントコードの引き換え

イベントコード (Event Code) は、Zeabur の教育パートナー (Education Partner) が講演、講座、ワークショップ、ハッカソンなどで参加者に配布するコードです。Zeabur 上で体験用クレジットに引き換えられ、サブスクリプションプランへのアップグレード、プロジェクトのデプロイ、その他の有料機能の利用に使えます。

本ページでは、イベントコードの 2 種類の引き換え方法をご紹介します。イベント主催者の方は、このページのリンクを参加者にそのまま共有することで、同じ説明を繰り返す手間を省けます。

<Callout type="info">
イベントコードには有効期間があり、早すぎる、または遅すぎるタイミングでは引き換えに失敗します。主催者から案内された有効期間にご注意ください。なお、イベントコードはサードパーティのパートナーが販売する**プリペイドカード (Voucher)** とは異なる仕組みです。プリペイドカードをお持ちの場合は [プリペイドカードの引き換え](/partner/sales-partner/redeem-card) をご参照ください。
</Callout>

## 方法 1：イベント登録リンク（最も簡単、まだ登録していない方向け）

Zeabur アカウントをまだお持ちでない場合は、イベント登録リンクを直接ご利用いただけます。以下の URL の末尾にイベントコードを追記してください：

```
https://zeabur.com/events?code=あなたのイベントコード
```

たとえばイベントコードが `i_love_zeabur` であれば、以下の URL を開きます：

```
https://zeabur.com/events?code=i_love_zeabur
```

ページの案内に従って Zeabur の登録を完了すると、イベントクレジットが新しいアカウントに自動で引き換えられます。

## 方法 2：請求ページで手動引き換え（既にアカウントをお持ちの方向け）

<Steps>

### Zeabur アカウントにログイン

[zeabur.com/login](https://zeabur.com/login) からログインしてください。

### 請求管理ページへ移動

右上のアバターメニューから「請求」をクリックするか、[zeabur.com/account/billing](https://zeabur.com/account/billing) に直接アクセスします。

### 「クレジットを引き換え」をクリック

「アカウント残高」セクションの右上にある「クレジットを引き換え」ボタンを探してクリックします。

### イベントコードを入力

表示されたダイアログにイベントコードを入力し、「引き換え」をクリックします。引き換えが成功すると、右下に通知が表示され、アカウント残高が即時に増加します。

</Steps>

## 有料プランへのアップグレード（任意）

特定リージョンのデータセンターへのデプロイや、サブスクリプションプランの機能のアンロックが必要な場合は、引き換え後に [Pricing ページからアップグレード](https://zeabur.com/pricing#upgrade) できます。アップグレード後の料金はアカウント残高から優先的に差し引かれます。

<Callout type="info">
イベントコードの引き換え後のクレジットは通常、サブスクリプションプランへのアップグレードに十分な金額です。実際に利用可能なプランはイベントクレジットの金額によって決まります。ご不明な点があればイベント主催者にご確認ください。
</Callout>

## よくある質問

- **イベントコードの期限切れ** — 期限切れ後は救済できません。主催者から案内された有効期間をご確認ください。
- **イベントコードが無効** — 入力内容が完全であるか、大文字・小文字が正しいかをご確認ください。
- **その他の問題** — [Zeabur コミュニティフォーラム](https://zeabur.com/forum) からカスタマーサポートチームにご連絡ください。
