# イベントコードの引き換え

イベントコード (Event Code) は、Zeabur の教育パートナー (Education Partner) が講演、講座、ワークショップ、ハッカソンなどで参加者に配布するコードです。Zeabur 上で体験用クレジットに引き換えられ、サブスクリプションプランへのアップグレード、プロジェクトのデプロイ、その他の有料機能の利用に使えます。

本ページでは、イベントコードの 2 種類の引き換え方法をご紹介します。イベント主催者の方は、このページのリンクを参加者にそのまま共有することで、同じ説明を繰り返す手間を省けます。

<Callout type="info">
イベントコードには有効期間があり、早すぎる、または遅すぎるタイミングでは引き換えに失敗します。主催者から案内された有効期間にご注意ください。なお、イベントコードはサードパーティのパートナーが販売する**プリペイドカード (Voucher)** とは異なる仕組みです。プリペイドカードをお持ちの場合は [プリペイドカードの引き換え](/partner/sales-partner/redeem-card) をご参照ください。
</Callout>

## 方法 1：イベント登録リンク（最も簡単、まだ登録していない方向け）

Zeabur アカウントをまだお持ちでない場合は、イベント登録リンクを直接ご利用いただけます。以下の URL の末尾にイベントコードを追記してください：

```
https://zeabur.com/events?code=あなたのイベントコード
```

たとえばイベントコードが `i_love_zeabur` であれば、以下の URL を開きます：

```
https://zeabur.com/events?code=i_love_zeabur
```

ページの案内に従って Zeabur の登録を完了すると、イベントクレジットが新しいアカウントに自動で引き換えられます。

## 方法 2：請求ページで手動引き換え（既にアカウントをお持ちの方向け）

<Steps>

### Zeabur アカウントにログイン

[zeabur.com/login](https://zeabur.com/login) からログインしてください。

### アカウント設定を開く

左上のお名前をクリックしてメニューを開き、**Settings** をクリックします。

![アカウント設定を開く](/partner/event-partner/redeem-event-code/step-2-open-settings.png)

### Billing に切り替えて「Redeem Credits」をクリック

左サイドバーの **Billing** をクリックし、「Account Credit」セクションの **Redeem Credits** ボタンをクリックします。

![Billing に切り替えて Redeem Credits をクリック](/partner/event-partner/redeem-event-code/step-3-billing-redeem-credits.png)

### イベントコードを入力

表示されたダイアログにイベントコードを入力し、**Redeem** をクリックします。引き換えが成功すると、右下に通知が表示され、アカウント残高が即時に増加します。

![イベントコードを入力](/partner/event-partner/redeem-event-code/step-4-enter-code.png)

</Steps>

## 引き換え後について

イベントクレジットはそのまま**アカウント残高**に加算され、Zeabur の任意の有料サービス——プロジェクトのデプロイ、サーバーのレンタル、AI Hub などに使用できます。自分でチャージしたクレジットと同じ扱いです。

![イベントクレジットはデプロイ、サーバー、AI Hub などのサービスに利用できます](/partner/event-partner/redeem-event-code/after-redeem-uses.png)

<Callout type="info">
**さらに Pro Plan へアップグレードしたい方へ：**[Pricing ページ](https://zeabur.com/pricing#upgrade) からアップグレードをクリックしてください（[Pro Plan](/pricing/pro-plan) を参照）。

- **イベントクレジットが Pro の月額料金をカバーできる場合**：クレジットがそのままアップグレードに充当され、カード登録は不要です。
- **クレジットが月額料金に満たない場合**：Zeabur Pro の **14 日間無料トライアル** を開始してください——終了時にイベントクレジットが月額料金に優先充当されます。
</Callout>

## よくある質問

### イベントクレジットは何に使えますか？

Zeabur 上の任意の有料サービス——デプロイ、データベース、サーバー、AI Hub などに使えます。一般的なアカウント残高そのもので、自分でチャージしたクレジットと区別はありません。

### クレジットカードの登録は必要ですか？

イベントコードの引き換え自体には**不要**です。Pro プランへのアップグレード時も、イベントクレジットが初月分をカバーできればカード登録は不要。不足分がある場合はカードで補う必要があります。2 か月目以降に Pro を継続利用するにも、請求のためカード登録が必要です。

### イベント終了後も Zeabur を使い続けられますか？

はい。Pro を継続したい場合はクレジットカードを登録すれば自動で継続課金されます。支払いを続けたくない場合は[無料プラン](/pricing)に留まればよく、いつでも [Pricing ページ](https://zeabur.com/pricing) から再度アップグレードできます。

### イベントクレジットを使い切ったらどうなりますか？

- **有料プラン利用中**：次回の請求は登録済みのクレジットカードに課金されます。
- **無料プラン利用中**：無料枠を超える有料リソースは利用できません。[チャージ](https://zeabur.com/account/billing) または有料プランへのアップグレードで継続利用できます。

### イベントコードが期限切れ / 無効 / その他の問題

- **期限切れ** — 期限切れ後は救済できません。主催者から案内された有効期間をご確認ください。
- **無効** — 入力内容が完全であるか、大文字・小文字が正しいかをご確認ください。
- **その他** — [Zeabur コミュニティフォーラム](https://zeabur.com/forum) からカスタマーサポートチームにご連絡ください。
