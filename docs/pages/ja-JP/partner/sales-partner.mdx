# セールスパートナー (Sales Partner)

セールスパートナープログラムは、**企業コンサルタント、システムインテグレーター、テクノロジーコンサルティング企業**を対象としています。企業顧客、機関、組織への Zeabur 導入を支援する際、実際に関与した協業フェーズに応じて継続的なレベニューシェアを獲得できます。

## 適したシナリオ

セールスパートナープログラムは、以下のような協業タイプに適しています：

- **企業のクラウド移行** — 企業の既存サービスを Zeabur へ移行する支援。
- **機関の一括調達** — 教育機関、政府機関、企業の IT 部門などによる大量調達、契約調達。Voucher、請求書発行、契約書対応などのケースを含みます。
- **テクニカルコンサルティング導入** — テクニカルコンサルタントまたはシステムインテグレーターとして、顧客向けのデプロイ設計に Zeabur を組み込む。
- **長期サービス協業** — 顧客にデプロイ、運用保守、決済代行などのサービスを提供し、そのプラットフォームとして Zeabur を採用する。

金額が閾値に達せず、個人がセルフサービスで支払う場合は、[Referral Program](/rewards/referral) の自動フローをご利用ください。具体的な適用閾値や条件は個別の契約で定めます。

<Callout type="info">
同一の取引は Referral Program またはセールスパートナーのいずれか 1 つの経路のみを通ることができ、**両者の併用はできません**。
</Callout>

## レベニューシェアの構造

セールスパートナーのレベニューシェアは 2 つの階層で構成され、実際に関与した範囲に応じて計算されます。

### 第 1 階層：取引レート

取引ごとに独立して計算され、レートは取引金額に応じた累進制です。レートはパートナーの種類、市場でのポジショニング、協業規模などに応じて**個別に交渉**し、標準値は公開していません。

### 第 2 階層：貢献フェーズの重み

その取引で実際に担当したフェーズに応じて決まり、4 つのフェーズの重みは合計 100% になります。Zeabur が自ら担当したフェーズについては外部支払いを行いません。

| フェーズ | 定義 | 判定基準 |
| --- | --- | --- |
| Lead（紹介と開拓） | 顧客を見つける | あなたが双方の初回接点（専用チャットグループの作成を含む）を仲介した時点 |
| Close（成約と契約締結） | 契約を締結する | 双方が署名した見積書を基準とする |
| Deliver（サービス提供と保守） | サービスを提供する | 顧客からの問い合わせ対応や操作サポートをあなたが完結し、Zeabur のエンジニアリングチームの介入が不要であること |
| Collect（回収と請求書発行） | 入金を回収する | 集金と請求書発行をあなたが完遂し、Zeabur からのリマインドや督促が不要であること |

### よくある参加パターン

| シナリオ | Lead | Close | Deliver | Collect |
| --- | :---: | :---: | :---: | :---: |
| フルサポート | ✓ | ✓ | ✓ | ✓ |
| 紹介のみ | ✓ |   |   |   |
| 紹介 + サービス | ✓ |   | ✓ |   |
| サービス + 回収 |   |   | ✓ | ✓ |
| クローズ + 代行運営（Lead なし） |   | ✓ | ✓ | ✓ |
| 紹介 + サービス + 回収 | ✓ |   | ✓ | ✓ |

### 複数パートナーでの按分

同じ取引を複数のパートナーがそれぞれ異なるフェーズで担当することもできます。取引単位でクローズして計算し、プールをまたがず、各パートナーは独立して精算されるため互いに影響しません。

## 付帯ルール

- **決済通貨**：USD を基準として決済し、為替コストはセールスパートナーが負担します。
- **精算サイクル**：レベニューシェアは月次で計算し、双方の確認後に Partner Portal に反映されます。
- **引き出し待機期間**：レベニューシェアが計上された後、holding 期間を経てから引き出せます。待機期間中に顧客が churn した場合は自動で減算されますが、期間満了後に引き出し済みの分は遡って返還請求しません。具体的な日数は契約で定めます。
- **最低販売コミット**：協業期間中には最低販売コミットが設定され、連続して未達の場合は Zeabur がレートの調整または契約の終了を行う権利を有します。具体的な金額は契約で定めます。
- **Lead の認定**：Lead フェーズは Zeabur による確認を経てロックされます。
- **フェーズのロック**：Deal 登録時に各フェーズの責任所在を確定し、変更には双方の書面による同意が必要です。
- **レベニューシェアの存続**：顧客が支払いを継続している期間は計算が続き、顧客が churn した時点で停止します。

<Callout type="warning">
セールスパートナープログラムと Referral Program は、二重の利害関係の衝突を避けるため、同一アカウントに同時適用できません。
</Callout>

## 協業フロー

1. **Zeabur に連絡** — [partner.zeabur.com](https://partner.zeabur.com) から応募するか、[Zeabur コミュニティフォーラム](https://zeabur.com/forum) から Zeabur チームにご連絡いただき、顧客の背景と想定される協業規模をご説明ください。
2. **契約の協議** — Zeabur チームが評価のうえ、具体的なレート、フェーズの重み、付帯条件について協議します。個別契約の内容は機密扱いです。
3. **契約の締結** — 条項を確認したうえで正式に契約を締結し、Deal のトラッキングを開始します。
4. **顧客のオンボーディング** — 顧客の評価、テスト、本番導入を支援します。
5. **月次でのレベニューシェア精算** — 顧客の支払い成功後、契約条件に沿って月次でレベニューシェアを精算します。

## 関連付帯契約

一部の協業では、独立した契約または附属契約を併用して処理します。例：

- **決済代行サービス手数料**：決済代行を担う場合、関連手数料は独立契約で取り扱います。
- **教育補助**：学校や教育機関に対する教育補助を含む協業の場合、独立した附属契約 / MOU で取り扱います。

## 応募方法

- [partner.zeabur.com](https://partner.zeabur.com) でアカウントを作成し、貴社情報、顧客の背景、想定される協業規模を記入して応募してください。
- [Zeabur コミュニティフォーラム](https://zeabur.com/forum) から Zeabur チームに連絡し、協業の意向をお伝えいただくことも可能です。
