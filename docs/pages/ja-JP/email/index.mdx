# Zeabur Email

Zeabur Emailは、Zeaburが提供するエンタープライズグレードのメール送信APIサービスで、開発者にシンプルで信頼性が高く、高性能なメール配信ソリューションを提供します。

## 主な機能

- **シンプルなREST API** - 数行のコードでメール機能を統合
- **即時送信とスケジュール送信** - すぐにメールを送信するか、後で送信するようスケジュール
- **一括送信** - 自動キュー管理で大量のメールを送信
- **添付ファイルサポート** - さまざまなメール添付ファイル形式をサポート
- **Webhook通知** - リアルタイムの配信ステータス更新（配信、バウンス、苦情）
- **ドメイン管理** - カスタム送信者ドメインのサポート
- **APIキー管理** - 柔軟な権限制御（読み取り専用、送信専用、フルアクセス）

## 主な特徴

### 高い信頼性
AWS SES上に構築され、99.9%の稼働率保証により、安定したメール配信を実現。

### リアルタイム監視
Webhookを介してリアルタイムのメールステータス更新を受信：
- 送信確認
- 配信確認
- バウンス通知
- スパム苦情

### セキュリティ
- HMAC-SHA256署名検証
- APIキー権限分離
- ドメインDKIM/SPF検証

## 使用例

- ユーザー登録確認メール
- パスワードリセットメール
- トランザクション通知
- マーケティングキャンペーン
- システムアラート通知
- 請求書と領収書

## 料金

Zeabur Emailは、送信されたメール数に基づく従量課金制の料金モデルを使用しています。詳細な料金については、[請求ドキュメント](/ja-JP/billing)を参照してください。

<Callout type="warning">
  メール送信は、スパム対策規制とベストプラクティスに準拠する必要があります。サービスの乱用はアカウント停止につながる可能性があります。
</Callout>
