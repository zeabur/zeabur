# デプロイボタン

当社のデプロイボタンを使用すると、1クリックでZeaburにサービスをデプロイできます。このボタンは、任意のマークダウンファイルまたはHTMLファイルで使用できます。[Zeabur template](https://zeabur.com/templates) をデプロイ可能なプロジェクトに変換することができます。

![button-demo](/deploy/button/button-demo.png)

## デプロイボタンとは

デプロイボタンはシンプルなHTMLコードスニペットです。このスニペットをコピーして任意のマークダウンファイルまたはHTMLファイルに貼り付けることができます。各ボタンはユーザーを希望のテンプレートにリダイレクトします。テンプレートは、テンプレートの作者によって決定された単一のサービスまたはサービスグループになります。

ボタンには、作者の紹介コードが付加されることもあります。紹介コードを使用して新規ユーザーが登録し、有料プランにアップグレードすると、作者はUS$5のクレジットを受け取ることができます。

## ボタンのコピー

ボタンはテンプレートに接続されているため、ボタンを生成できるのはテンプレートの作者だけです。

ボタンをコピーするには、[Zeabur ダッシュボード](https://dash.zeabur.com/account) のアカウントページに移動し、「**Template**」タブを選択する必要があります。そこでは、作成したテンプレートの一覧を見ることができます。

![created-templates](/deploy/button/created-templates.png)

希望のテンプレートを選択して、共有ボタンをクリックします。すると、ボタンコードが表示されるモーダルがポップアップします。

![popup](/deploy/button/popup.png)

紹介コードの有効化を選択した場合、ボタンコードにはユーザー名である紹介コードが追加されます。

## ユースケース

Zeaburにデプロイされた素晴らしいプロジェクトを世界と簡単に共有する最良の方法です。

たとえば、このボタンを以下の場所に貼り付けることができます：

- GitHubのREADMEファイル
- ブログ
- ドキュメントとチュートリアル
- ソーシャルプロファイル

## 必要条件

デプロイボタンを取得するには、まずテンプレートを作成する必要があります。[Zeabur ダッシュボード](https://dash.zeabur.com/create-template)でテンプレートを作成できます。

画像のように、テンプレート名、説明、およびテンプレートに含めたいサービスを選択する必要があります。テンプレートの作成は、フォーム記入のように簡単です。

![create-template](/deploy/button/create-template.png)

## 収益

前述のとおり、紹介コードを使用して新規ユーザーが登録し、有料プランにアップグレードするごとに、US$5のクレジットを獲得できます。クレジットは自身のサービスの支払いに使用できます。

詳細な説明については、[紹介文書](../billing/referral) をお読みいただけます。
