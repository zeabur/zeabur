# デプロイボタン

デプロイボタンを使えば、Zeabur にサービスをワンクリックで簡単にデプロイできます。このボタンは任意の Markdown ファイルや HTML ファイルで使用可能。[Zeabur テンプレート](https://zeabur.com/templates) をデプロイ可能なプロジェクトに変換します。

![button-demo](/deploy/button/button-demo.png)

## デプロイボタンとは

デプロイボタンはシンプルな HTML コードスニペットです。スニペットをコピーして任意の Markdown または HTML ファイルに貼り付けます。各ボタンはユーザーを希望のテンプレートにリダイレクトします。テンプレートは単一サービスまたはテンプレート作成者が決めた複数サービスのグループです。

ボタンには作成者のリファラルコードを付与可能。新規ユーザーがリファラルコードでサインアップし有料プランにアップグレードすると、作成者に US$5 のクレジットが付与されます。

## ボタンをコピー

ボタンはテンプレートに紐づくため、テンプレート作成者のみ生成可能。

[Zeabur ダッシュボード](https://zeabur.com/account) のアカウントページへ行き、**Template** タブを選択。作成したテンプレート一覧が表示されます。

![created-templates](/deploy/button/created-template.png)

希望のテンプレートを選択し、共有ボタンをクリック。モーダルがポップアップし、ボタンコードが表示されます。

![popup](/deploy/button/popup.png)

リファラルコードを有効にした場合、ボタンコードに Zeabur ユーザー名であるリファラルコードが付与されます。

## ユースケース

Zeabur にデプロイした素晴らしいプロジェクトを世界に共有する最適な方法。誰でもワンクリックであなたのプロジェクトをデプロイ可能。

例として以下の場所に貼り付けられます：

- GitHub README ファイル
- ブログ
- ドキュメント・チュートリアル
- ソーシャルプロフィール

## 要件

デプロイボタンを取得するにはまずテンプレートを作成。この[手順](https://zeabur.com/docs/ja-JP/template/template-in-code)でテンプレートを作成できます。

## 収益

上記の通り、リファラルコードでサインアップし有料プランにアップグレードした新規ユーザー1人につき US$5 のクレジットが得られます。クレジットは自身のサービス支払いに使用可能。

詳細は[リファラルドキュメント](https://zeabur.com/docs/ja-JP/billing/referral)を参照。
