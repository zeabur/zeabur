# CLIを使用したサービスのデプロイ

コンソールでサービスをワンクリックでデプロイすることは簡単ですが、端末でコマンドを使用してサービスをデプロイしたい場合も、このオプションを提供しています。

この記事では、Zeabur CLIを使用して端末でサービスをデプロイする方法について説明します。

## アカウントにログイン

サービスをデプロイするには、アカウントにログインする必要があります。

```bash
npx zeabur auth login
```

`Enter`を押してブラウザのログインページを開き、`Confirm`ボタンをクリックします。すると、端末にリダイレクトされ、ログイン成功のメッセージが表示されます。

## サービスのデプロイ

これでサービスをデプロイする準備が整いました。サービスディレクトリで以下のコマンドを実行します。

```bash
npx zeabur deploy
```

次に、プロジェクトを選択または作成する指示に従います。サービスはこのプロジェクトにデプロイされます。

プロジェクトを選択した後、端末でデプロイプロセスが表示されます。数秒待つと、サービスが自動的にデプロイされ、サービスのURLが端末に表示されます。
