# Raycast 拡張機能を使ってサービスをデプロイする

Zeabur は複数の拡張機能を提供しており、開発環境を離れることなく、ローカルコードをワンクリックでデプロイできます。

この記事では、Zeabur の Raycast 拡張機能を使ってサービスをデプロイする方法を紹介します。

## Raycast 拡張機能

Zeabur の Raycast 拡張機能を使えば、プロジェクトのデプロイ、ドキュメントやテンプレートの検索をすばやく行えます。

### 拡張機能のインストール

Raycast Store を開き、「Zeabur」と検索して拡張機能をインストールします。または、[Raycast Zeabur 拡張機能ページ](https://www.raycast.com/ridemountainpig/zeabur)にアクセスし、「Install Extension」をクリックしてインストールすることもできます。

### 使い方

1. インストール後、Raycast を開いて `Zeabur` と入力します。
2. 「Deploy Project」コマンドを選択します。
3. デプロイしたいプロジェクトフォルダを選び、`⌘ + Enter` を押します。
4. デプロイ先のリージョンを選択し、数分後にデプロイが完了します。
