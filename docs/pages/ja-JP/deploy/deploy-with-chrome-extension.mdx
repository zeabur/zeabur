# Chrome 拡張機能

Zeabur は複数の拡張機能を提供しており、開発環境を離れることなく、ローカルコードをワンクリックでデプロイできます。

この記事では、Zeabur の Chrome 拡張機能を使ってサービスをデプロイする方法を紹介します。

## Chrome 拡張機能

Zeabur の Chrome 拡張機能を使えば、Lovable、Bolt.new、v0.dev などの対応する AI コード生成プラットフォームから直接コードをデプロイできます。

### 拡張機能のインストール

[Chrome ウェブストア](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem) にアクセスし、「Chrome に追加」をクリックしてください。

### 使い方

対応する AI プラットフォームでコードを書くと、「Deploy with Zeabur」ボタンが画面に表示されます。ボタンをクリックすることで、デプロイプロセスが開始されます。
