# **サービスを作成**

[プロジェクトを作成](https://zeabur.com/docs/ja-JP/deploy/create-project)した後、サービスのデプロイを開始できます。**Deploy New Service**ボタンをクリックするか（または`CMD + K` / `Ctrl + K`を押して**Deploy New Service**を選択）、さまざまなデプロイオプションのメニューが表示されます：

![Create Service](/deploy/Create-Services/add-service.png)

## **テンプレート**

Zeaburは、WordPress、n8n、Ghostなど人気のオープンソースプロジェクト向けに豊富な事前設定テンプレートライブラリを提供しています。これらのテンプレートはワンクリックでデプロイ可能です。

1. **Template**をクリックして利用可能なテンプレートを閲覧。
2. 必要なテンプレートを検索または閲覧。
3. テンプレートをクリックし、指示に従ってデプロイ。

詳細は[テンプレートからデプロイ](https://zeabur.com/docs/ja-JP/template/template-in-code)を参照。

![choose template](/deploy/Create-Services/choose-template.gif)

## **GitHub**

GitHubリポジトリから直接デプロイ。これは独自アプリケーションをデプロイする最も一般的な方法です。

1. 未接続の場合、**GitHub**をクリックしてGitHubアカウントを接続。
2. 接続後、デプロイしたいリポジトリを検索。
3. リポジトリをクリックすると、Zeaburがコードを自動解析し、最適なビルド方法を選択。
4. **Deploy**をクリックしてデプロイ開始。

![Search repository](https://zeabur.com/docs/_next/image?url=%2Fdocs%2F_next%2Fstatic%2Fmedia%2Fsearch-repository.db02d750.webp&w=3840&q=75)

詳細は[GitHubとの統合](https://zeabur.com/docs/ja-JP/deploy/github)を参照。

## **データベース**

設定なしで一般的なデータベースを素早くデプロイ。

1. **Databases**をクリックして利用可能なデータベース一覧を表示。
2. 必要なデータベースを選択（PostgreSQL、MySQL、MongoDB、Redisなど）。
3. データベースが即座にデプロイされ、使用可能に。

![Database](/deploy/Create-Services/database.png)

## **Functions**

Zeabur Functionsでコードを即座に書き、実行、イテレーション。クイックスクリプト、API、サーバーレス関数に最適。

1. **Functions**をクリックして新しい関数を作成。
2. 関数言語として**JavaScript**または**Python**を選択。
3. エディタに直接コードを記述。
4. 関数がデプロイされ、ダッシュボードからアクセス可能。

![Functions](/deploy/Create-Services/functions.png)

## **ローカルプロジェクト**

Gitリポジトリ不要でローカルソースコードを直接Zeaburにアップロード。

1. **Local Project**をクリック。
2. プロジェクトフォルダをドラッグ＆ドロップ、または閲覧して選択。
3. Zeaburが解析してプロジェクトをデプロイ。

![Local Project](/deploy/Create-Services/local-project.png)

リポジトリにプッシュせずに素早くデプロイしたい場合に便利。

アップロード時に`/dist/*`ディレクトリは自動的に無視されます。

## **Dockerイメージ**

Docker Hubや他のコンテナレジストリから事前ビルド済みのDockerイメージをデプロイ。

1. **Docker Image**をクリック。
2. イメージ名を入力（例：`nginx:latest` または `ghcr.io/owner/image:tag`）。
3. 必要な環境変数やポートを設定。
4. **Deploy**をクリック。

詳細は[Dockerイメージでデプロイ](https://zeabur.com/docs/ja-JP/deploy/customize-prebuilt)を参照。

## **Cursor**

Zeabur拡張機能を使ってCursor IDEから直接デプロイ。

1. **Cursor**をクリックしてCursor IDEをプロジェクトで開く。
2. Zeabur拡張機能が未インストールの場合、インストールを促されます。
3. 拡張機能を使って現在のプロジェクトをCursorから直接デプロイ。

CursorをメインIDEとする開発者にシームレスな体験を提供。
