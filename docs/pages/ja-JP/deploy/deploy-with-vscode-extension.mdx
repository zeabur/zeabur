# VS Code 拡張機能を使ってサービスをデプロイする

Zeabur は複数の拡張機能を提供しており、開発環境を離れることなく、ローカルコードをワンクリックでデプロイできます。

この記事では、Zeabur の VS Code 拡張機能を使ってサービスをデプロイする方法を紹介します。

## VS Code / Cursor 拡張機能

Visual Studio Code や Cursor エディタで Zeabur 拡張機能を使えば、ローカルプロジェクトを Zeabur に直接デプロイできます。

### 拡張機能のインストール

1. VS Code または Cursor エディタを開きます。  
2. 拡張機能パネルで「Zeabur」と検索します。  
3. 「インストール」をクリックするか、[Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) にアクセスしてインストールします。

### 使い方

1. デプロイしたいプロジェクトフォルダを開きます。  
2. サイドバーの Zeabur アイコンをクリックします。  
3. 「Deploy」をクリックすると、ブラウザが自動的に開きます。  
4. デプロイ先のリージョンを選択します。  
5. 数分後にデプロイが完了します。
