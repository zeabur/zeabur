# **カスタムルートディレクトリ**

フロントエンドとバックエンドの両方を含むプロジェクトなど、一部のGitリポジトリでは、プロジェクトのルートディレクトリを手動で指定する必要があります。

### **ステップ1: サービスをデプロイ**

[サービスをデプロイ](https://zeabur.com/docs/ja-JP/get-started)に従ってアプリケーションをデプロイしてください。

## **ステップ2: プロジェクトのルートディレクトリを変更**

デプロイしたサービスをクリックし、設定タブに入ります。下にスクロールして**Root Directory**の項目を探します。

![Root Directory](/deploy/root-directory/root_directory.png)

ここでプロジェクトのルートディレクトリを設定します。
