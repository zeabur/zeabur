# 拡張機能でサービスをデプロイする

Zeabur では、複数の拡張機能を提供しており、開発環境から離れることなく、ローカルのコードをワンクリックでデプロイできます。

このドキュメントでは、Zeabur の VS Code、Cursor、Chrome、Raycast 向け拡張機能を使用してサービスをデプロイする方法をご紹介します。

## VS Code / Cursor 拡張機能

Visual Studio Code および Cursor エディターで Zeabur 拡張機能を使うことで、ローカルプロジェクトを直接 Zeabur にデプロイできます。

### インストール方法

1. VS Code または Cursor を開きます。
2. 拡張機能パネルを開き、「Zeabur」と検索します。
3. 「インストール」をクリックするか、[Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) にアクセスしてインストールします。

### 使い方

1. デプロイしたいプロジェクトのフォルダを開きます。
2. サイドバーの Zeabur アイコンをクリックします。
3. 「Deploy」をクリックすると、ブラウザが自動的に開きます。
4. デプロイするリージョンを選択します。
5. 数分でプロジェクトのデプロイが完了します。

## Chrome 拡張機能

Zeabur の Chrome 拡張機能を使えば、Lovable、Bolt.new、v0.dev などの AI コード生成プラットフォームから直接コードをデプロイできます。

### インストール方法

[Chrome ウェブストア](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem) にアクセスし、「Chrome に追加」をクリックします。

### 使い方

対応する AI プラットフォームでコードを作成していると、「Deploy with Zeabur」ボタンが画面に表示されます。クリックするだけでデプロイが開始されます。

## Raycast 拡張機能

Zeabur の Raycast 拡張機能を使用すると、プロジェクトのデプロイ、ドキュメント検索、テンプレートの参照がすばやく行えます。

### インストール方法

Raycast Store を開き、「Zeabur」で検索して拡張機能をインストールします。もしくは、[Raycast Zeabur 拡張ページ](https://www.raycast.com/ridemountainpig/zeabur)にアクセスし、「Install Extension」をクリックします。

### 使い方

1. インストール後、Raycast を開き「Zeabur」と入力します。
2. 「Deploy Project」コマンドを選択します。
3. デプロイしたいプロジェクトフォルダを選択し、`⌘ + Enter` を押します。
4. リージョンを選択すると、数分でデプロイが完了します。
