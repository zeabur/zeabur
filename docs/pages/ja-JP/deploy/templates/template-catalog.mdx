# Zeabur テンプレートカタログ

Zeabur は豊富なテンプレートカタログを提供しており、さまざまなアプリケーションやサービスをすばやくデプロイできます。

[Zeabur テンプレートカタログ](https://zeabur.com/templates)にアクセスして、利用可能なすべてのテンプレートを閲覧してください。

## テンプレートの使い方

1. テンプレートカタログから必要なテンプレートを見つける
2. 「Deploy」ボタンをクリックする
3. 指示に従って設定を完了する

独自のテンプレートを作成する方法については、[テンプレート形式](/ja-JP/template/template-format)もご覧ください。
