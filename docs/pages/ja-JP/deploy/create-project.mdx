# **プロジェクトを作成**

Zeaburでサービスをデプロイするには、まずプロジェクトを作成する必要があります。

以下の手順に従ってプロジェクトを作成できます。

## **プロジェクトとは**

プロジェクトは、一緒にデプロイされるサービスのコレクションです。同じプロジェクト内のサービスは内部ネットワークで相互に通信できます。異なるリージョンに複数のプロジェクトを作成し、各プロジェクトでさまざまな種類のサービスをデプロイできます。

また、プロジェクトに対して[予算を設定し、使用量を管理](https://zeabur.com/docs/ja-JP/manage/budget)したり、[他のチームメンバーを招待](https://zeabur.com/docs/ja-JP/manage/invite)してサービスを管理したりできます。

## **プロジェクトの作成方法**

まず、[ダッシュボードのプロジェクトページ](https://zeabur.com/projects)に移動します。

Zeaburに初めてでプロジェクトがない場合、最初のプロジェクトを作成するためのオンボーディングページが表示されます。

![プロジェクト作成オンボーディング](/deploy/create-project/Deployment_Services_Create_Project.png)

以前にプロジェクトを作成したことがある場合は、**Create Project**ボタンをクリックして**Create New Project**ページに移動し、**Deployment Type**を選択します。**dedicated server**または**share cluster**のいずれかを選択してください。

![デプロイメントタイプを選択](/deploy/create-project/choose_Deployment_Type_2.png)

次に、サブスクリプションプランと希望するリージョンに基づいてリージョンを選択すると、プロジェクトがuntitled-0、untitled-1などの名前で自動的に作成されます。

![リージョンを選択](/deploy/create-project/select_a_region_2.png)

## **次に何をするか**

リージョン選択後、空のプロジェクトが表示されます：

![空のプロジェクト](/deploy/create-project/Deployment_Services_empty_project.png)

**Deploy New Service**ボタンをクリックしてこのプロジェクトに[サービスをデプロイ](https://zeabur.com/docs/ja-JP/deploy/create-service)するか、設定ボタンをクリックしてプロジェクト設定を管理し、[他のチームメンバーを招待](https://zeabur.com/docs/ja-JP/manage/invite)できます。
