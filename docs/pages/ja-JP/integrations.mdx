# インテグレーション

Zeaburはサードパーティサービスとの組み込み統合を提供し、プロジェクト内で直接強力なバックエンド機能を有効化できます — 別途デプロイは不要です。

## 利用可能なインテグレーション

### [InsForge](/ja-JP/integrations/insforge/overview)

AIネイティブなSupabase代替サービスで、AIエージェントがフルスタックアプリケーションを自律的に構築・管理できます。PostgreSQLデータベース、JWT認証、S3互換ファイルストレージ、サーバーレスエッジファンクション、AI IDE統合のためのMCPプロトコルサポートを含みます。

- **カテゴリ**：バックエンド / BaaS
- **課金**：従量課金、別途請求
- **ウェブサイト**：[insforge.dev](https://insforge.dev)
