**はじめよう**

**Zeabur** は、使用するプログラミング言語やフレームワークに関係なく、ワンクリックでサービスをデプロイできるプラットフォームです。

さらに詳しい情報を学んだり、優先サポートを受けたりするには、[Zeabur Forum](https://zeabur.com/forum) に参加できます。

また、[公式 Discord サーバー](https://zeabur.com/dc) に参加して、アイデアを共有したり、コミュニティからサポートを受けたりすることもできます。

### **Zeabur の特長**

### **手軽で高速**

従来のデプロイ方法と比べて、Zeabur は複雑なデプロイ知識を学ぶ時間をかけずに、わずか数分でデプロイを完了できます。これにより、プロダクト開発に時間とエネルギーを集中できます。

- AI エージェントによるプロジェクト種別の自動認識
- 複数ベンダー／複数リージョンの共有クラスター／専用サーバー
- 分かりやすく直感的な UI
- すぐに使える CI/CD
- 自動ドメイン生成／独自ドメイン持ち込み（BYOD）対応
- 複数環境の分離
- Zeabur AI Hub
- ほかにも多数

### **オールインワン統合**

Zeabur は強力なオールインワン統合機能を提供します。

フロントエンド、バックエンド、データベース、メッセージキュー、オブジェクトストレージなど、さまざまなサービスを 1 つのプロジェクト内でデプロイできます。別々のプラットフォームに個別にデプロイする必要はありません。

### ワンクリックデプロイテンプレート

Zeabur は [テンプレートマーケットプレイス](https://zeabur.com/templates) に多数のテンプレートを用意しています。`Deploy` ボタンをクリックするだけで、テンプレートはすぐにデプロイを開始します。データベースでも、n8n のようなアプリでも、数分で利用できる状態になります。

詳しくは、[Zeabur Templates ページ](https://zeabur.com/templates) をご覧ください。

### Zeabur AI Hub

Zeabur AI Hub は、開発者が強力な AI モデルへ簡単にアクセスできるように設計された統合プラットフォームです。プロバイダーごとに別々のアカウントや API キーを管理する代わりに、1 つのインターフェースから業界で最も人気のあるモデルにアクセスできます。

- **統合アクセス**: OpenAI（GPT シリーズ）、Anthropic（Claude シリーズ）、Google（Gemini）、xAI（Grok）、オープンソースモデル（Llama、DeepSeek、Qwen）など主要モデルを 1 か所で利用できます。
- **単一 API キー**: 1 つのシンプルな API キーで対応モデルを呼び出せるため、複数の資格情報管理が不要になります。
- **OpenAI 互換**: API は OpenAI の API 形式と完全互換で、n8n や SillyTavern など既存ツールにもコード変更なしで簡単に統合できます。
- **グローバル低遅延**: 東京やサンフランシスコなど複数リージョンに API エンドポイントを展開しており、最寄りのサーバーを選んで最適なパフォーマンスを得られます。

詳しくは [Zeabur AI Hub ページ](https://zeabur.com/ai-hub) をご覧ください。

### **従量課金**

Zeabur は従量課金制を採用しており、サーバー全体ではなく、サービスが実際に使用したリソースに対してのみ支払います。

料金の詳細は [料金ドキュメント](https://zeabur.com/docs/ja-JP/billing/pricing) をご覧ください。
