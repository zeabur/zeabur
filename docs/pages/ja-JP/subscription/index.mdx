# サブスクリプション

## サブスクリプションとは？

Zeabur には 3 つのプランがあります：**Serverless プラン**、**Developer プラン**、**Team プラン**。
各プランの詳細は[プラン一覧](/pricing/pricing-plans)ページをご確認ください。

いずれのプランにも加入していない場合、デフォルトで **Serverless プラン**ユーザーとなります。

## サブスクリプションの登録方法

Zeabur のサブスクリプションは [Stripe](https://stripe.com/) を基盤としています。

プランへの登録方法は 2 つあります：**クレジットカード** と **プリペイドクレジット**。

### クレジットカードで登録

クレジットカードで USD 支払いにてプランに登録できます。

- カード登録のみ：[請求ページ](https://dash.zeabur.com/account/billing)の **Payment Method** セクションでクレジットカードを設定し、**Add** ボタンをクリックして保存した後、[プランページ](https://zeabur.com/pricing#upgrade)で **Subscribe** ボタンをクリックしてプランを更新できます。
- 登録と同時にカード設定：[プランページ](https://zeabur.com/pricing#upgrade)でプランを選択し、クレジットカード情報を入力して **Subscribe** ボタンをクリックすると、Stripe のチェックアウトページに遷移します。

### プリペイドクレジットで登録

Stripe に対応したクレジットカードをお持ちでない場合は、クレジットを購入してプランに登録できます。

[請求ページ](https://dash.zeabur.com/account/billing)の **Get More Credits** ボタンからクレジットを購入し、[プランページ](https://zeabur.com/pricing#upgrade)でプランをアップグレードできます。

![Get More Credits](/billing/subscription/get-more-credits.png)

## 支払い方法の変更

カードが拒否された場合や支払い方法を変更したい場合は、[請求ページ](https://dash.zeabur.com/account/billing)から変更できます。

**Unbind Payment Method** までスクロールして現在のクレジットカードを解除し、新しいカードを設定してください。

## Serverless プランへのダウングレード

有料プランに加入している場合、いつでも **Serverless プラン**にダウングレードできます。

[Dashboard の請求ページ](https://dash.zeabur.com/account/billing)にアクセスし、下部の解約セクションからダウングレードを実行してください。

![Unsubscribe Plan](/billing/subscription/unsubscribe.png)

<Callout>
    Serverless プランにダウングレードすると、以降の課金は発生しません。ただし、現在デプロイ中のコンテナサービスはシステムにより自動的に停止されます。
</Callout>
