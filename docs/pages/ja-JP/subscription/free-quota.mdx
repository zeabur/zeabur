# 無料クォータ

Zeabur はすべてのユーザーに無料 Credits を提供しており、お支払いなしでプラットフォームをお試しいただけます。

## Credits の計算方法

無料 Credits は**毎月リセットされる仮想クレジット**で、共有クラスターでのリソース使用量を相殺するために使用されます。金額はプランによって異なります：

| プラン | 月間無料 Credits |
| --- | --- |
| Free Plan | 制限付き（アカウントの状態により変動） |
| Dev プラン | US$5（月額料金に含まれる） |
| Team プラン | US$5（月額料金に含まれる） |

有料プランの場合、「無料 Credits」は月額料金に含まれる Credits を指します。例えば、Dev プランの月額 US$5 には US$5 の Credits が含まれており、リソース使用量が US$5 未満であれば追加料金は発生しません。

## 対象となるもの

無料 Credits は**共有クラスターのリソース使用量にのみ適用**されます。以下が含まれます：

- メモリ使用量
- ネットワーク送信
- 永続ストレージ

### 無料 Credits の対象外

以下は無料 Credits から**差し引かれず**、別途支払いが必要です：

- **サーバー料金** — サーバーの月額レンタルは別途請求されます。
- **AI Hub Credits** — AI Hub は独自のクレジットシステムを使用し、別途チャージが必要です。

## Credits がなくなった場合

### Free Plan

月間無料 Credits が枯渇した場合：

1. すべてのサービスが**自動的に停止**されます。
2. Credits 枯渇の通知が届きます。
3. 翌月の Credits リセット時にサービスは自動的に再開されます。
4. すぐにサービスを復旧するには、[Dev プラン](/pricing/dev-plan)にアップグレードするか、Credits をチャージしてください。

### 有料プラン

含まれる月間 Credits が枯渇した場合：

1. 超過分は**クレジットカードまたはプリペイド残高**に自動的に請求されます。
2. 請求に失敗した場合、サービスは直ちに停止されませんが、支払いリマインダーが届きます。
3. 長期間の未払いはサービス停止につながる場合があります。

<Callout type="info">
当月の Credits 使用状況は [Dashboard の請求ページ](https://dash.zeabur.com/account/billing)で確認できます。
</Callout>

## よくある質問

### 未使用の Credits は繰り越されますか？

いいえ。無料 Credits は毎月リセットされます。未使用の Credits は翌月に繰り越されません。

### 残りの Credits を確認するにはどうすればよいですか？

[Dashboard の請求ページ](https://dash.zeabur.com/account/billing)で当月の使用状況と残りの Credits を確認できます。
