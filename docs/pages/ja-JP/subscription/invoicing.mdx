# 請求書

Zeabur は [Team プラン](/pricing/team-plan)以上のプランで請求書発行に対応しており、企業や団体の経理処理を容易にします。

<Callout type="info">
請求書発行は **Team プラン**以上でのみ利用可能です。[Dev プラン](/pricing/dev-plan)および Free プランのユーザーで請求書が必要な場合は、まず Team プランにアップグレードしてください。
</Callout>

## 請求書のリクエスト方法

1. [Dashboard の請求ページ](https://dash.zeabur.com/account/billing)に移動します。
2. 請求書設定セクションで、会社名と税務 ID を入力します。
3. 保存後、以降の請求に請求書情報が自動的に含まれます。

## 税務 ID の設定

請求ページの請求書設定で、以下を設定できます：

- **会社名** — 請求書に表示される会社または組織の正式名称。
- **税務 ID** — 税務識別番号（例：台湾の統一編號、VAT 番号、その他の地域の税務 ID）。
- **請求先住所** — 会社の請求先住所。

## 対応国・地域

Zeabur は Stripe を通じて請求書を発行しており、以下の国・地域の税務識別番号に対応しています：

- **台湾** — 統一編號（Tax ID）
- **アメリカ合衆国** — EIN（Employer Identification Number）
- **欧州連合** — VAT Number
- **日本** — 法人番号（Corporate Number）
- **その他の Stripe 対応国・地域**

上記に該当しない国・地域の場合は、[Discord サーバー](https://zeabur.com/dc)でお問い合わせいただければ、設定をお手伝いします。

## 請求書の内容

Zeabur の請求書には以下が含まれます：

- 請求期間
- サブスクリプションプラン料金
- リソース使用量の内訳（メモリ、ネットワーク、ストレージなど）
- 専用サーバー料金（該当する場合）
- 税金（現地の規制に基づく）

## よくある質問

### 過去の月の請求書を発行できますか？

はい。[Discord サーバー](https://zeabur.com/dc)で必要な月とアカウント情報をお知らせください。

### 請求書はどの通貨で発行されますか？

請求書は Zeabur の請求通貨に合わせて米ドル (USD) で発行されます。

### Dev プランのユーザーは請求書を発行できますか？

Dev プランでは現在、請求書発行に対応していません。請求書が必要な場合は、[Team プラン](/pricing/team-plan)にアップグレードしてください。
