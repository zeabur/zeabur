# お支払い方法

Zeabur の決済システムは [Stripe](https://stripe.com/) で運用されており、複数の支払い方法に対応しています。

## 対応している支払い方法

### クレジットカード / デビットカード

Zeabur は Stripe が対応するすべてのクレジットカードとデビットカードを受け付けており、**米ドル (USD)** で請求されます：

- Visa
- Mastercard
- American Express
- JCB
- UnionPay

### Alipay

Stripe 対応のクレジットカードをお持ちでない場合、Alipay を使用して **Credits をチャージ**し、人民元 (CNY) でお支払いいただけます。

Alipay でチャージした Credits は Zeabur Credits に変換され、サブスクリプション料金やリソース使用量の相殺に使用されます。

## 支払いフロー

### 有料プランへのサブスクリプション

1. [Zeabur 料金ページ](https://zeabur.com/pricing)または [Dashboard の請求ページ](https://dash.zeabur.com/account/billing)に移動します。
2. 希望するプラン（[Dev プラン](/pricing/dev-plan)または [Team プラン](/pricing/team-plan)）を選択します。
3. クレジットカード情報を入力し（または既にリンク済みのカードを使用し）、支払いを完了します。
4. 支払い成功後、プランは即座に有効になります。

### Credits のチャージ

1. [Dashboard の請求ページ](https://dash.zeabur.com/account/billing)に移動します。
2. 「Credits」セクションのチャージボタンをクリックします。
3. 金額を選択し、クレジットカードまたは Alipay で支払いを完了します。
4. Credits はアカウント残高に即座に追加されます。

![Credits をチャージ](/billing/subscription/get-more-credits.png)

## 支払い方法の管理

### クレジットカードのリンク

[請求ページ](https://dash.zeabur.com/account/billing)の「お支払い方法」セクションでクレジットカード情報を入力し、「追加」をクリックします。

### クレジットカードの変更

1. 請求ページの「**解除**」ボタンを見つけて、現在のカードを解除します。
2. 新しいクレジットカードをリンクします。

### 解除

すべての有料サブスクリプションをキャンセルした場合、クレジットカードを解除できます。

<Callout type="warning">
有効な有料サブスクリプションがある場合、クレジットカードを解除すると次回の請求に失敗する可能性があります。サブスクリプションをキャンセルするか、代替の支払い方法を設定してから解除してください。
</Callout>

## よくある質問

### クレジットカードが拒否された場合はどうすればよいですか？

- カードに十分な残高があることを確認してください。
- カードが海外取引（USD）に対応していることを確認してください。
- カード発行会社に連絡して、ブロックされた取引がないか確認してください。
- 別のクレジットカードを試すか、Alipay で Credits をチャージしてください。

### PayPal で支払えますか？

PayPal は現在サポートされていません。クレジットカードまたは Alipay をご利用ください。

### どの通貨で請求されますか？

すべての請求は米ドル (USD) です。クレジットカード発行会社が外貨取引手数料を請求する場合があります — 詳細は銀行にお問い合わせください。

その他の支払いに関するご質問は、[コミュニティフォーラム](https://zeabur.com/forum)でお問い合わせください。
