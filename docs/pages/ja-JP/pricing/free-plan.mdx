# Free プラン

Free プランは Zeabur のエントリープランで、Zeabur のデプロイフローを初めて試すユーザー向けに設計されています。クレジットカード不要 — サインアップするだけですぐにデプロイを開始できます。

## 含まれるリソース

| リソース | 制限 |
| --- | --- |
| 月額 | $0 |
| 外部サーバーバインド | 1 台 |
| ビルド CI スペック | 2C4G |
| ログ保持 | 48 時間 |
| ファイルアップロード上限 | 低 |
| コンテナファイル管理 | 単一ファイルアップロード |
| データベースビュー | サポート |
| AI Agent（Zeabur AI） | 含まれない |
| デプロイリージョン | 共有クラスターリージョン |

## リソース料金

共有クラスターは分単位で課金され、有料プランと同じ料金体系です：

- **vCPU** — 無料（$0.0 / vCPU-分）
- **メモリ** — US$0.00025 / GB-分（約 US$10.8 / GB-月）
- **ネットワーク送信** — US$0.1 / GB
- **永続ストレージ** — US$0.2 / GB-月

詳細は [Billing / Pricing](/billing/pricing) をご覧ください。

## 制限事項

- **Zeabur AI / AI Agent なし** — AI による運用エクスペリエンスは有料プランでのみ利用可能です。
- **ログ保持期間が短い** — Free では 48 時間のみ保持されます。
- **サーバーバインドが制限** — Free では 1 台の外部サーバーのみバインド可能です。
- **SLA なし** — Free プランにはサービスレベルアグリーメントが含まれません。
- **コミュニティサポートのみ** — 優先テクニカルサポートには Team プランが必要です。

<Callout type="info">
AI Agent、バックアップ、カスタムドメイン、長いログ保持をお試しになりたい場合は、[14 日間無料トライアル](/billing/subscription#14-日間無料トライアル) で Dev または Pro を試用できます — 有料版と同じ機能、トライアル終了まで課金されません。
</Callout>

## アップグレードパス

Free プランの範囲を超えるニーズがある場合は、有料プランにアップグレードできます：

- **[Dev プラン](/pricing/dev-plan)** — US$5 / 月。AI Agent、データベースバックアップ、ドメイン管理、メールシステムが必要な個人開発者向け。
- **[Pro プラン](/pricing/pro-plan)** — US$19 / 月。高いビルドスペック、30 日間のログ保持、高度なオブザーバビリティ、HA デプロイ、IP アクセス制御。
- **[Team プラン](/pricing/team-plan)** — US$79 / 月（3 席含む）。無制限サーバーバインド、チームコラボレーション、ロール管理、優先サポート。
- **Enterprise** — カスタム契約、専用リソース、コンプライアンスサポート — 詳細は [お問い合わせ](https://zeabur.com/forum) ください。

すべてのプランの比較は [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。
