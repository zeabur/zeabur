# Free プラン（Serverless プラン）

Serverless プランは Zeabur の無料ティアで、個人の探索や学習向けに設計されています。クレジットカード不要 — サインアップするだけですぐにデプロイを開始できます。

## 含まれるリソース

| リソース | 制限 |
| --- | --- |
| プロジェクト | 1 |
| サービス | プロジェクトあたり最大 2 |
| デプロイリージョン | 共有クラスターリージョン |
| 無料 Credits | 毎月リセット（[無料クォータ](/subscription/free-quota)を参照） |
| カスタムドメイン | サポートなし |
| 永続ストレージ | サポートなし |

### サービス料金

Serverless プランは有料プランと同じ料金を使用し、無料 Credits から差し引かれます：

- **vCPU** — 無料（$0.0 / vCPU 分）
- **メモリ** — US$0.00025 / GB 分（約 US$10.8 / GB 月）
- **ネットワーク送信** — US$0.1 / GB
- **永続ストレージ** — US$0.2 / GB 月

## 制限事項

- **自動スリープ** — Serverless プランのサービスは、一定期間アクティビティがないと自動的にスリープします。次のリクエストで起動しますが、数秒のコールドスタートレイテンシが発生する場合があります。
- **永続ストレージなし** — 永続ボリュームが必要なサービス（データベースなど）はデプロイできません。
- **SLA なし** — 無料プランにはサービスレベルアグリーメントが含まれません。
- **Credits 枯渇** — 月間の無料 Credits がなくなると、翌月のリセットまでサービスが停止されます。

<Callout type="info">
Serverless プランのサービスは、リソースを節約するためにアイドル時に自動スリープします。サービスを継続的に実行する必要がある場合は、[Dev プラン](/pricing/dev-plan)へのアップグレードをご検討ください。
</Callout>

## アップグレードパス

Serverless プランの範囲を超えるニーズがある場合は、有料プランにアップグレードできます：

- **[Dev プラン](/pricing/dev-plan)** — US$5/月、本番プロジェクトを持つ個人開発者に最適です。カスタムドメイン、永続ストレージ、無制限のプロジェクトが含まれます。
- **[Team プラン](/pricing/team-plan)** — 従量制課金、メンバー管理と請求書発行に対応したチームコラボレーション向けです。
- **Enterprise** — 専用サポートと SLA 付きのカスタムプラン — 詳細は[お問い合わせ](https://zeabur.com/forum)ください。

すべてのプランの比較は [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。
