# Free Plan

Free Plan は Zeabur の無料ティアで、個人的な探索と学習を目的として設計されています。クレジットカード不要 — サインアップしてすぐにデプロイを始められます。

## プランの詳細

| 項目 | Free Plan |
| --- | --- |
| 月額料金 | US$0 |
| 外部サーバーのバインド数 | 1 |
| ビルド CI スペック | 2C4G |
| ログ保持期間 | 48 時間 |
| コンテナファイル管理 | 単一ファイルアップロード |
| データベースビューア | サポート |
| 永続ストレージ | サポート |
| カスタムドメインバインディング | サポート |
| ドメイン購入（レジストラ） | 非対応 |
| データベースバックアップ（エクスポート / インポート） | 非対応 |
| ログ転送 | 非対応 |
| Zeabur AI | 非対応 |
| Zeabur Email — 日次クォータ | 0 |
| Zeabur Email — 月次クォータ | 0 |
| ドメイン | 0 |
| API キー | 0 |
| Webhooks | 0 |
| フォーラム投稿 | 制限あり（基本） |
| 優先サポート | 含まない |
| SLA | 含まない |

## 制限事項

- **自動スリープ** — Free Plan のサービスは、一定時間操作がないと自動的にスリープ状態になります。次のリクエストが来た際に再起動しますが、コールドスタートにより数秒の遅延が発生する場合があります。
- **SLA なし** — Free Plan にはサービスレベルアグリーメントが含まれていません。
- **Zeabur Email なし** — Free Plan では [Zeabur Email](/email) によるメール送信は利用できません。
- **自動データベースバックアップ / インポートおよびログ転送なし** — これらの高度な運用機能は Dev Plan 以上で利用可能です。

<Callout type="info">
Free Plan のサービスはアイドル時に自動スリープしてリソースを節約します。サービスを継続的に稼働させる必要がある場合は、[Dev Plan](/pricing/dev-plan) へのアップグレードをご検討ください。
</Callout>

## アップグレードパス

Free Plan の制限を超えた場合は、有料プランにアップグレードできます：

- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 日間の無料トライアル）、個人の本番プロジェクトを持つ個人開発者に最適。Zeabur AI、データベースバックアップ、ドメイン購入、Zeabur Email が追加されます。
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 日間の無料トライアル）、本番ワークロードに推奨。ログ転送、高度なクォータ、カスタムサービスアイコン、優先サポートが追加されます。
- **[Team Plan](/pricing/team-plan)** — 3 シートで US$79/月（+US$24/シート）、ロールと権限管理を含むチームコラボレーション向け。
- **Enterprise** — 専任サポートと SLA を含むカスタムプラン — 詳細は[お問い合わせ](https://zeabur.com/forum)ください。

すべてのプランを比較するには [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。
