# プラン

Zeabur は 4 つのサブスクリプションプランとカスタム Enterprise オプションを提供しています：

- **[Free Plan](/pricing/free-plan)** — US$0/月、個人的な探索向け。
- **[Dev Plan](/pricing/dev-plan)** — US$5/月、14 日間の無料トライアル。個人開発者向け。
- **[Pro Plan](/pricing/pro-plan)** — US$19/月、14 日間の無料トライアル。本番環境に推奨。
- **[Team Plan](/pricing/team-plan)** — 3 シートで US$79/月、+US$24/シート。チーム向け。
- **Enterprise Plan** — 使用シナリオに合わせたプランについては[お問い合わせ](mailto:contact@zeabur.com)ください。

全機能の比較と返金ポリシーについては[定価プラン](/pricing/pricing-plans)をご覧ください。最新の料金については [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。
