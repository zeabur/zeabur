# プリペイドカードの交換

Zeabur と提携するサードパーティパートナーから、Zeabur プリペイドカードを購入または受け取ることがあります。これらのプリペイドカードは Zeabur サービスの残高に交換でき、Zeabur 有料メンバーシップへのアップグレード、サーバーの購入、共有クラスタの利用料金の支払いに使用できます。

この記事では、購入または受け取った Zeabur プリペイドカードを残高に交換する方法を説明します。

## 交換方法

1. Zeabur アカウントにログインします。まだ Zeabur アカウントをお持ちでない場合は、まず[登録](https://zeabur.com/login)してください。
2. Zeabur コンソールの課金管理ページ [https://zeabur.com/account/billing](https://zeabur.com/account/billing) にアクセスします。
3. 課金管理ページで「Account Balance」セクションを見つけます。右上の「Redeem Points」ボタンをクリックします。
4. ポップアップダイアログにプリペイドカードコードを入力し、「Redeem」ボタンをクリックします。
5. 交換が成功すると、画面の右下に成功通知が表示されます。アカウント残高は対応する金額が増加します。

## 問題が発生した場合

プリペイドカードの交換中に問題が発生した場合は、[Zeabur コミュニティフォーラム](https://zeabur.com/forum)でカスタマーサービスチームにお問い合わせください。全力でサポートいたします。
