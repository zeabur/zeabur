# スポンサー

## スポンサーシップとは？

Zeabur からのスポンサーシップは、以下の特典を提供します：

- すべてのコントリビューターに1か月間の Developer プラン無料利用。

- プロジェクトの規模と個人の貢献度に応じたコントリビューター向けクレジット。

## スポンサーシップの基準

- プロジェクトは非商用で完全にオープンソースである必要があります。

- プロジェクトのウェブサイトのフッターと `README.md` ファイルに Zeabur バナーを表示する必要があります。

リソースはダッシュボードの[紹介ページ](https://dash.zeabur.com/account/referral)で確認できます。

[![Deploy on Zeabur](https://zeabur.com/deployed-on-zeabur-dark.svg)](https://dash.zeabur.com/account/referral)

## スポンサーシップの申請方法

- 上記の基準を満たすすべてのプロジェクトがスポンサーシップの対象となります。

- 申請するには、以下の手順に従ってください：

    - プロジェクトがすべての基準を満たしていることを確認してください。

    - [コミュニティフォーラム](https://zeabur.com/forum)の `help` チャンネルに、プロジェクトのリンクと簡単な説明を含む投稿を作成してください。
