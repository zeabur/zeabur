# Pro プラン

Pro プランは、**より高いビルドスペック、長いログ保持、強化されたオブザーバビリティ** を必要とする開発者向けの Zeabur プランです。**US$19/月** で、Dev プランの上に本番レベルの機能を追加します。

<Callout type="info">
**14 日間無料トライアル** 対象 — 有料版と同じ機能、トライアル終了まで課金されません。詳細は [14 日間無料トライアル](/billing/subscription#14-日間無料トライアル) をご覧ください。
</Callout>

| ドメイン & IP アクセス制御 | サポートあり |
| メンバー | 1（アカウントオーナーのみ） |

## Dev プランに対する追加機能

Dev プランのすべての機能に加えて：

- 最大 **10 台の外部サーバー** バインド（Dev：3 台）
- **ビルド CI 4C8G**（Dev：2C4G）— より速く、より強力なビルド
- ログ保持 **30 日間**（Dev：7 日間）
- **高度なログ検索** — 履歴ログのフィルタリングとクエリ
- **ログ転送（Log Drain）** — サードパーティシステムへのログストリーミング
- **高可用性サービスデプロイ**
- **ドメイン & IP アクセス制御** — サービスへのアクセス元を制限
- より高いファイルアップロード上限

## ユースケース

- **トラフィックのある本番サービス** — 高い CI、HA デプロイ、長いログ保持で本番環境を健全に保ちます。
- **オブザーバビリティが必要な個人開発者** — 高度なログ検索、ログ転送、カスタムアラートで大規模なデバッグが可能。
- **セキュリティ重視のサービス** — ドメイン & IP アクセス制御でエンドポイントへのアクセスを制限。
- **長時間のビルド** — 4C8G ビルド CI でより速くデプロイ、特に monorepo に最適。
