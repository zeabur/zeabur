# Pro Plan

Pro Plan は Zeabur の本番ワークロード向け推奨ティアです — **14 日間の無料トライアル**付きで **US$19/月**。Dev Plan をベースに、**ログ転送**、**より大きな CI スペック**、**長いログ保持期間**、**より高い Email / ドメイン / API キーのクォータ**、**カスタムサービスアイコン**、**優先サポート**が追加されます。

## ユースケース

- **小規模本番デプロイメント** — 長いログ保持期間、大きな CI ビルド、直接サポートチャネルが有益なサービス。
- **トランザクションメール量が多いアプリ** — 月 50,000 通のクォータは、初期段階の SaaS や EC アプリのほとんどをカバーします。
- **運用負荷の高いワークロード** — 既存のオブザーバビリティスタック（Datadog、Grafana など）へのログ転送により、Pro ティアのサービスを中央監視に統合できます。
- **多くのドメインを持つフロントエンド重視のアプリ** — 10 のドメインスロットと 100 の API キーが十分な余裕を提供します。

## アップグレードのタイミング（Team へ）

**同じプロジェクトで複数のメンバーが共同作業する**、**ロールと権限の管理**、**請求書発行 / 税 ID のサポート**、または**無制限の外部サーバーバインド**が必要な場合は、[Team Plan](/pricing/team-plan)（3 シートで US$79/月、+US$24/シート）が次のステップです。

<Callout type="info">
大規模なカスタムプラン、専任サポート、SLA については、**Enterprise Plan** をご確認ください — 詳細は[こちら](https://zeabur.com/forum)からお問い合わせください。
</Callout>
