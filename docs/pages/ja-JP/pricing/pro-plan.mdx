# Pro Plan

Pro Plan は Zeabur の本番ワークロード向け推奨ティアです — **14 日間の無料トライアル**付きで **US$19/月**。Dev Plan をベースに、**ログ転送**、**より大きな CI スペック**、**長いログ保持期間**、**より高い Email / ドメイン / API キー / Webhook のクォータ**、**カスタムサービスアイコン**、**優先サポート**が追加されます。

全機能の一覧と最新のクォータは[プラン比較表](/pricing)をご覧ください。

## ユースケース

- **小規模本番デプロイメント** — 長いログ保持期間、大きな CI ビルド、直接サポートチャネルが有益なサービス。
- **トランザクションメール量が多いアプリ** — Pro Plan の月次メールクォータは、初期段階の SaaS や EC アプリのほとんどをカバーします。
- **運用負荷の高いワークロード** — 既存のオブザーバビリティスタック（Datadog、Grafana など）へのログ転送により、Pro ティアのサービスを中央監視に統合できます。
- **多くのドメインを持つフロントエンド重視のアプリ** — Pro のドメインと API キーのクォータは、ほとんどのマルチ環境構成をカバーします。

## Team へのアップグレードのタイミング

**同じプロジェクトで複数のメンバーが共同作業する**、**ロールと権限の管理**、**請求書発行 / 税 ID のサポート**、または**無制限の外部サーバーバインド**が必要な場合は、[Team Plan](/pricing/team-plan)（3 シートで US$79/月、+US$24/シート）が次のステップです。

<Callout type="info">
大規模なカスタムプラン、専任サポート、SLA については、**Enterprise Plan** をご確認ください — 詳細は[こちら](https://zeabur.com/forum)からお問い合わせください。
</Callout>

## サブスクリプション方法

[zeabur.com/pricing](https://zeabur.com/pricing) にアクセスして Pro Plan を選択するか、[Dashboard の請求ページ](https://dash.zeabur.com/account/billing) からアップグレードしてください。クレジットカードと Alipay に対応しています。詳細は[支払い方法](/subscription/payment-methods)をご覧ください。
