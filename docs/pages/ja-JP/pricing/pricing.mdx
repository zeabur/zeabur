# 定価

Zeabur は 4 つのサブスクリプションティアとカスタム Enterprise オプションを提供しています：

- **[Free Plan](/pricing/free-plan)** — US$0/月
- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 日間の無料トライアル）
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 日間の無料トライアル）
- **[Team Plan](/pricing/team-plan)** — 3 シートで US$79/月（+US$24/シート）
- **Enterprise** — [お問い合わせ](https://zeabur.com/forum)

全機能の比較と返金ポリシーについては[定価プラン](/pricing/pricing-plans)をご覧ください。最新の料金については [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。
