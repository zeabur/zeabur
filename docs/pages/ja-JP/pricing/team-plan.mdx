# Team プラン

Team プランは、チーム向けに設計された Zeabur のコラボレーションプランです。複数メンバーの管理と統一請求に対応し、複数人でプロジェクトを管理するチームに最適です。

## プランの詳細

| 項目 | 詳細 |
| --- | --- |
| 課金方式 | 従量制 |
| プロジェクト | 無制限 |
| サービス | 無制限 |
| カスタムドメイン | サポートあり |
| 永続ストレージ | サポートあり |
| デプロイリージョン | すべての利用可能なリージョン |
| メンバー | 複数（シート数に応じて） |
| 請求書発行 / 税務 ID | サポートあり |

### シートモデル

Team プランは複数メンバーのコラボレーションに対応しています。メンバー数はシート数に基づきます — 具体的な料金は [zeabur.com/pricing](https://zeabur.com/pricing) をご覧ください。

### 料金

Team プランのリソース料金は Dev プランと同じです。詳細は[プラン概要](/pricing/pricing-plans)をご覧ください。

## 請求書発行と税務 ID

Team プランは請求書の発行と税務 ID（統一編號）の設定に対応しており、経理処理に領収書が必要な企業や団体に適しています。詳細は[請求書](/subscription/invoicing)をご覧ください。

## Dev プラン vs. Team プラン

| 機能 | Dev プラン | Team プラン |
| --- | --- | --- |
| 課金方式 | US$5/月 + 従量制 | 従量制 |
| メンバー | 1 | 複数（シート数に応じて） |
| メンバー権限 | なし | サポートあり |
| 請求書発行 / 税務 ID | サポートなし | サポートあり |
| 最適な用途 | 個人開発者 | チーム / 組織 |

<Callout type="info">
より大規模なカスタムプラン、専用サポート、SLA については、**Enterprise プラン**をご確認ください — 詳細は[お問い合わせ](https://zeabur.com/dc)ください。
</Callout>

## ユースケース

- **スタートアップチーム** — 複数の開発者がデプロイ環境を共同管理。
- **企業内部プロジェクト** — 統一請求と請求書発行が必要なチーム。
- **エージェンシー** — チーム統一管理の下、異なるクライアントに個別のプロジェクトを作成。
- **オープンソースコミュニティ** — 複数のメンテナーがサービスデプロイで協力。

## サブスクリプション方法

[zeabur.com/pricing](https://zeabur.com/pricing) で Team プランを選択するか、[Dashboard の請求ページ](https://dash.zeabur.com/account/billing)からアップグレードしてください。クレジットカードと Alipay に対応しています — 詳細は[お支払い方法](/subscription/payment-methods)をご覧ください。
