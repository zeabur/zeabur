# Dev プラン（Developer プラン）

Dev プランは、AI Agent、データベースバックアップ、ドメイン管理を必要とする個人開発者向けに設計された Zeabur の有料プランです。**US$5/月** で、個人開発者が最も必要とする生産性機能をアンロックします。

<Callout type="info">
**14 日間無料トライアル** 対象 — 有料版と同じ機能、トライアル終了まで課金されません。詳細は [14 日間無料トライアル](/billing/subscription#14-日間無料トライアル) をご覧ください。
</Callout>

## プランの詳細

| 項目 | 詳細 |
| --- | --- |
| 月額料金 | US$5 / 月 |
| 外部サーバーバインド | 最大 3 台 |
| ビルド CI スペック | 2C4G |
| ログ保持 | 7 日間 |
| ファイルアップロード上限 | 中 |
| コンテナファイル管理 | 単一ファイルアップロード |
| Zeabur AI（AI Agent） | 含まれる |
| ドメイン購入 | サポートあり |
| メールシステム | 含まれる |
| データベースバックアップエクスポート | サポートあり |
| メンバー | 1（アカウントオーナーのみ） |
| デプロイリージョン | すべての利用可能なリージョン |

## Free プランに対する追加機能

Free プランのすべての機能に加えて：

- 最大 **3 台の外部サーバー** バインド（Free：1 台）
- ログ保持 **7 日間**（Free：48 時間）
- **Zeabur AI** — サービスを管理する AI Agent
- **Zeabur でのドメイン購入**
- **メールシステム** — トランザクションメールの送信
- より高いファイルアップロード上限

## リソース料金

共有クラスターは分単位で課金されます。完全な料金表は [Billing / Pricing](/billing/pricing) をご覧ください。

- **vCPU** — 無料
- **メモリ** — US$0.00025 / GB-分（約 US$10.8 / GB-月）
- **ネットワーク送信** — US$0.1 / GB
- **永続ストレージ** — US$0.2 / GB-月

## Free プラン vs. Dev プラン

| 機能 | Free プラン | Dev プラン |
| --- | --- | --- |
| 月額料金 | $0 | US$5 |
| 外部サーバーバインド | 1 台 | 3 台 |
| ログ保持 | 48 時間 | 7 日間 |
| Zeabur AI | — | 含まれる |
| ドメイン購入 | — | サポートあり |
| メールシステム | — | 含まれる |
| ファイルアップロード上限 | 低 | 中 |

## ユースケース

- **個人のサイドプロジェクト** — 個人の Web サイト、ブログ、API サービスなどのデプロイ。
- **AI 支援ワークフロー** — Zeabur AI にサービスの管理とトラブルシューティングを任せる。
- **小規模本番サービス** — 24 時間 365 日稼働が必要な低トラフィックサービス。
- **テンプレート開発** — [デプロイボタン](/deploy/deploy-button) を使用して Zeabur テンプレートを作成・メンテナンス。

## サブスクリプション方法

[zeabur.com/pricing](https://zeabur.com/pricing) で Dev プランを選択するか、[Dashboard の請求ページ](https://dash.zeabur.com/account/billing) からアップグレードしてください。クレジットカードと Alipay に対応しています — 詳細は [お支払い方法](/subscription/payment-methods) をご覧ください。
