# Dev Plan (Developer Plan)

Dev Plan は Zeabur の入門有料ティアで、個人開発者向けに設計されています。**14 日間の無料トライアル**付きで **US$5/月**。Zeabur AI、自動データベースバックアップ、ドメイン購入、Zeabur Email サービスが利用可能になり、個人の本番プロジェクトやサイドプロジェクトに最適です。

全機能の一覧と最新のクォータは[プラン比較表](/pricing)をご覧ください。

## ユースケース

- **個人サイドプロジェクト** — 個人ウェブサイト、ブログ、API サービスなどのデプロイ。
- **開発およびテスト** — 永続ストレージ（データベースなど）と自動バックアップが必要な環境。
- **小規模本番サービス** — 24 時間 365 日稼働が必要な低トラフィックサービス。
- **テンプレート開発** — [デプロイボタン](/deploy/deploy-button)を使って Zeabur テンプレートを作成・維持し、プロジェクトを広める。

## Pro へのアップグレードのタイミング

**ログ転送**、**より大きな Email / ドメイン / API キー / Webhook のクォータ**、**カスタムサービスアイコン**、または**優先サポート**が必要な場合は、[Pro Plan](/pricing/pro-plan)（US$19/月、14 日間の無料トライアル）が次のステップです。詳細は [Pro Plan](/pricing/pro-plan) ページをご覧ください。

## サブスクリプション方法

[zeabur.com/pricing](https://zeabur.com/pricing) にアクセスして Dev Plan を選択するか、[Dashboard の請求ページ](https://dash.zeabur.com/account/billing) からアップグレードしてください。クレジットカードと Alipay に対応しています。詳細は[支払い方法](/subscription/payment-methods)をご覧ください。
