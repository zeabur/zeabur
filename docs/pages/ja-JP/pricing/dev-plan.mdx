# Dev プラン（Developer プラン）

Developer プランは、個人開発者向けに設計された Zeabur の有料プランです。**US$5/月**で、すべてのデプロイ機能がアンロックされ、個人の本番プロジェクトやサイドプロジェクトに最適です。

## プランの詳細

| 項目 | 詳細 |
| --- | --- |
| 月額料金 | US$5 / 月 |
| 含まれる Credits | US$5 / 月（リソースコストを相殺） |
| プロジェクト | 無制限 |
| サービス | 無制限 |
| カスタムドメイン | サポートあり |
| 永続ストレージ | サポートあり |
| デプロイリージョン | すべての利用可能なリージョン |
| メンバー | 1（アカウントオーナーのみ） |

### 料金

Dev プランのリソース料金は[プラン概要](/pricing/pricing-plans)に記載されているものと同じです：

- **vCPU** — 無料
- **メモリ** — US$0.00025 / GB 分（約 US$10.8 / GB 月）
- **ネットワーク送信** — US$0.1 / GB
- **永続ストレージ** — US$0.2 / GB 月

含まれる US$5 の月間 Credits がリソース使用量を自動的に相殺します。超過分はクレジットカードまたはプリペイド残高に請求されます。

## Free プラン vs. Dev プラン

| 機能 | Free プラン | Dev プラン |
| --- | --- | --- |
| 月額料金 | 無料 | US$5 |
| プロジェクト | 1 | 無制限 |
| サービス | プロジェクトあたり 2 | 無制限 |
| カスタムドメイン | サポートなし | サポートあり |
| 永続ストレージ | サポートなし | サポートあり |
| 自動スリープ | アイドル時にスリープ | 常時稼働 |
| SLA | なし | スタンダード |

## ユースケース

- **個人のサイドプロジェクト** — 個人の Web サイト、ブログ、API サービスなどのデプロイ。
- **開発とテスト** — 永続ストレージが必要な環境（データベースなど）。
- **小規模本番サービス** — 24 時間 365 日稼働が必要な低トラフィックサービス。
- **テンプレート開発** — [デプロイボタン](/deploy/deploy-button)を使用して Zeabur テンプレートを作成・メンテナンスし、プロジェクトを宣伝。

## サブスクリプション方法

[zeabur.com/pricing](https://zeabur.com/pricing) で Dev プランを選択するか、[Dashboard の請求ページ](https://dash.zeabur.com/account/billing)からアップグレードしてください。クレジットカードと Alipay に対応しています — 詳細は[お支払い方法](/subscription/payment-methods)をご覧ください。
