# 料金プラン

Zeabur はプロジェクトに対して従量課金制の料金モデルを提供しています。使用していないリソースに対して料金が発生することはありません。

Zeabur はサービスが実際に使用したリソースに対して、分単位で課金します。

## 料金レート

Zeabur には2種類のサーバーがあります：_共有クラスター_ と [_専用サーバー_](/dedicated-server)。この料金表は**共有クラスター**にのみ適用され、専用サーバーには使用料金はかかりません。

**共有クラスター**での使用料金は以下の通りです：

- vCPU - 無料（$0.00 / vCPU-分）

- メモリ - **US$0.00025** / GB-分（約 US$10.80 / GB-月）
  - サービスが実際に使用したメモリのみ課金されます。サービスがクラッシュまたは停止した場合、メモリ使用料は発生しません。

- ネットワーク送信 - **US$0.10** / GB（US$10.00 / 100GB）
  - 1GB は CDN や圧縮なしで 1MiB のページにアクセスする約 1,000 ユニークビジターに相当します。
  - [プライベートネットワーク](/deploy/private-networking)は送信帯域幅としてカウントされません。

- 永続ボリューム - **US$0.20** / GB-月

## 返金

返金に関するお問い合わせは、[Discord サーバー](https://zeabur.com/dc)までご連絡ください。

**Developer プラン**または **Team プラン**の場合、サブスクリプションをキャンセルすると残りの期間分の返金を申請できます。例えば、Developer プランを10日間使用後にキャンセルした場合、残り20日分の返金を受けられます。24時間以内に返金を申請した場合は、全額返金されます。

**専用サーバー**ユーザーの場合、2025年7月21日以降に作成または更新されたサーバーについては返金に対応していません。次の請求サイクルでの課金を避けるため、自動更新設定を無効にしてください。

> この制限は 2025年7月21日以降に作成または更新されたサーバーにのみ適用されます。

![自動更新を無効にする](/billing/pricing/auto-renewal.png)

返金は Zeabur クレジットまたは元の支払い方法で受け取ることができます。Zeabur クレジットを選択した場合、承認後すぐに処理されます。元の支払い方法への返金を選択した場合、処理に最大7日かかり、返金額の約30%の Stripe 手数料が発生します。
