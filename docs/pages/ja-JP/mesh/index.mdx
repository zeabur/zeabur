# Zeabur Mesh

<Callout type="info">
Zeabur Mesh は近日公開予定です。お楽しみに。
</Callout>

Zeabur Mesh は、Zeabur 上のサービス間で安全で信頼性の高い接続とルーティングを実現する、サービス間通信およびトラフィック管理ソリューションです。

機能のローンチ後に詳細なドキュメントが公開されます。ご質問がある場合は、[コミュニティフォーラム](https://zeabur.com/forum)でお気軽にお問い合わせください。
