# リワードと紹介

Zeabur では、プラットフォームを利用・宣伝しながら特典を獲得できる複数のリワードプログラムを提供しています。

- [紹介プログラム](/rewards/referral) — 紹介コードを共有して、友人に割引を提供し、コミッションを獲得
- [オープンソーススポンサー](/rewards/sponsor) — オープンソースプロジェクトへの Zeabur スポンサーシップを申請
- [プリペイドカードの交換](/partner/sales-partner/redeem-card) — プリペイドカードをアカウント残高に交換
