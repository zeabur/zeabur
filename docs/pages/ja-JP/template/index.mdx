# テンプレート

Zeabur は豊富な**ワンクリックデプロイテンプレート**を提供しており、WordPress、Postgres、Redis、n8n、Ghost など、数百の人気サービスを手動設定なしで起動できます。

## テンプレートを探す

[zeabur.com/templates](https://zeabur.com/templates) にアクセスして、完全なカタログをご覧ください。任意のテンプレートをクリックして、プロジェクトに直接デプロイできます。

## テンプレートの仕組み

各テンプレートは以下を事前定義しています：

- 必要なサービスとバージョン
- 環境変数とデフォルト設定
- サービス間の接続関係

デプロイ時、Zeabur はテンプレートで定義されたすべてのサービスを自動的に作成し、相互に接続して、アプリケーションをすぐに使える状態にします。

## デプロイフローでテンプレートを使用する

サービスを作成する際に「テンプレートからデプロイ」を選択すると、デプロイページから直接テンプレートを検索して適用できます。

<Callout type="info">
  テンプレートをデプロイすると、定義されたすべてのサービスを含む完全なプロジェクトが作成されます。デプロイ後、環境変数や設定を自由に変更できます。
</Callout>
