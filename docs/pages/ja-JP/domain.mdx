# ドメイン登録

import { Callout } from 'nextra/components'

ドメイン登録サービスにより、Zeabur ダッシュボードから直接ドメインを登録・管理・設定できます。サードパーティのレジストラは不要です。

## 機能

- **ドメイン検索** — 9 つの TLD（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）の空き状況とリアルタイム価格を一括チェック
- **DNS 管理** — A、AAAA、CNAME、MX、TXT、SRV、CAA、NS レコード対応の DNS エディタ、Cloudflare プロキシサポート付き
- **自動・手動更新** — ドメインごとに自動更新を切り替え、いつでも手動更新も可能
- **登録者プロファイル** — WHOIS 連絡先情報を保存し、複数回の登録で再利用
- **登録者認証** — 一部の TLD で購入後に必要なメール認証に対応
- **Zeabur Email 統合** — ワンクリックでドメインを Zeabur Email に接続、DNS レコードが自動作成

<Callout type="info">
  ドメインの購入には **Developer** または **Team** プランが必要です。
</Callout>

## はじめに

1. サイドバーから **ドメイン** セクションを開く
2. [ドメイン登録](/domain/register) — 検索、価格比較、購入
3. [ドメイン管理](/domain/manage) — ステータス確認、更新切り替え、Zeabur Email 接続
4. [DNS レコード設定](/domain/dns) — 詳細ページから DNS レコードを追加・編集
5. [登録者プロファイル設定](/domain/profiles) — 連絡先情報を保存して登録を高速化
