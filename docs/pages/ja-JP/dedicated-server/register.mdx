# 専用サーバーの登録

この記事では、Zeabur に [専用サーバー](../dedicated-server) を登録する方法を説明します。

## 前提条件

- すでにサーバーを所有しており、そのサーバーには接続可能な少なくとも1つのパブリックIPアドレスが必要です。
- サーバーには少なくとも1つのCPUコアと少なくとも2GBのメモリが必要です。
- サーバーは `22`, `80`, `443`, `4222`, `6443`, `30000-32767` のポートを開放している必要があります。
- サーバーはSSHログインを許可し、ユーザー名は `root` である必要があります。
- このサーバーでは、`root`ユーザーまたは`sudo`権限を持つ他のユーザーが、パスワードまたはSSH鍵を使用してログインできる必要があります。

<Callout type="default">
    `root`ユーザーを使用していない場合、Zeaburは非対話環境でマシンを設定するため（パスワードを入力できません）、そのユーザーが属する`sudo`グループに[`NOPASSWD`](https://linux.die.net/man/5/sudoers)フラグが付与されていることを確認してください。
</Callout>

## 登録手順

![add machine to zeabur](/dedicated-server/add-machine-to-zeabur.webp)

1. Zeabur にログインした後、トップナビゲーションバーまたはプロジェクトリストページの上部にある「マイサーバー」をクリックするか、直接 [https://dash.zeabur.com/servers](https://dash.zeabur.com/servers) にアクセスします。
2. ページの右上にある「Create」（作成）ボタンをクリックし、「Connect your own Server」（自分のサーバーに接続）を選択します。
3. サーバーのIPアドレスとSSHポート（デフォルトは `22`）を入力します。
4. ユーザー名を入力してください（例：`root`）、その後ログイン方式（パスワードまたはSSH鍵）を選択してください。
5. パスワードを選択した場合は、下にパスワードを入力し、SSHキーを選択した場合は、下にSSHキーを入力します。
6. 「サーバーを追加」ボタンをクリックし、サーバーの登録が完了するのを待ちます（このプロセスには約30秒かかります）。
7. 登録が成功すると、サーバーリストに新しく登録されたサーバーが表示されます。
8. ステータスが「オンライン」と表示され、CPUとメモリの使用状況が正常に表示されるまで約1分待ちます。

登録に失敗した場合は、[Discord サーバー](https://zeabur.com/dc) で報告してください。
