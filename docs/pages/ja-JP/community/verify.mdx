# アカウントを確認してください

Zeaburは、すべての確認済みユーザーに優先サポートを提供しています。

あなたの確認は、私たちがあなたの問題を特定し、より速く解決するのに役立ちます。また、Discordサーバーでの活動に対する報酬としていくつかのクレジットをもらうことができます。

## 確認方法

確認プロセスには、弊社のDiscordボット**Zebra**を使用しています。

### ステップ1: Discordサーバーに参加する

まず最初に、[Discordサーバーに参加](https://zeabur.com/dc)する必要があります。そこではスタッフや他のユーザーとコミュニケーションを取ったり、フィードバックや問題を投稿したりすることができます。

### ステップ2: アカウントをリンクする

Discordサーバーに参加すると、Zebraからウェルカムメッセージが表示されます。メッセージの内容には、[確認チャンネル](https://discord.com/channels/1060209568820494336/1095916895649153094)があります。

![verify](/community/verify/verify.png)

確認ボタンをクリックすると、Discordから認証ページが表示されます。**Authorize**ボタンをクリックして、Discordアカウントと当社のウェブサイトをリンクしてください。

![authorize](/community/verify/authorize.png)

ウェブサイトにリダイレクトされると、トースターに成功メッセージが表示され、ウェブサイトを閉じることができます。

![verified](/community/verify/verified.png)

## 次は

確認が完了すると、私たちのチームから優先サポートを受けることができます。また、別のプランにアップグレードすると、**Zebra**が自動的にDiscordサーバーで特別な役割を付与します。その役割で優先チャンネルに参加することができます。

![dev-plan](/community/verify/dev-plan.png)
