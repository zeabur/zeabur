# **ヘルプの取得**

Zeaburを使用中に問題が発生することがあります。Zeaburチームからのヘルプは、[Zeaburフォーラム](https://zeabur.com/forum)で受けられます。

![Zeabur Forum](/community/get-help/Zeabur-Forum.png)

## **ヘルプの取得方法**

フォーラムには誰でもissueチケットを作成できる`New Post`ボタンがあります。

![Zeabur Forum Overview](/community/get-help/zeabur-forum-overview.gif)

### **投稿フォーマット**

問題をより明確に記述するために以下のフォーマットに従うと、迅速に特定・解決できます。

1. **Issue Type** を選択 
2. **Title** に問題を簡潔に記述。
3. **Project** または **Server** を選択 - Zeaburアカウント内の全プロジェクト/サーバーが一覧表示されます
4. サービス内で問題のあるものを選択 
5. ログからのエラーメッセージを投稿。
6. 最後に、問題の全体像を理解できるようコンテキストを追加。

![Ticket System](/community/get-help/Ticket-System.png)

### 電話認証

![Phone Verification](/community/get-help/phone-verification.png)

スタッフのみが電話番号を確認できるプライベート投稿が作成されます。

### 返金リクエスト

返金リクエストテンプレートを記入すると、スムーズにサポートが受けられます。

![refund requests page](/community/get-help/refund-requests.png)

## その他

![Others](/community/get-help/other-issues.png)

「Others」タブを選択し、できるだけ詳細に記述するとZeaburチームから効果的な対応が得られます

### **優先サポート**

- Teamプラン: 半営業日以内に回答
- Devプラン: 3営業日以内に回答
- Freeプラン: 5営業日以内に回答

**その他の連絡方法**

Zeaburフォーラムが公式サポートプラットフォームですが、メールでもサポートや問い合わせが可能です：

- **メール:** [contact@zeabur.com](mailto:contact@zeabur.com)

より迅速な対応とコミュニティリソースへのアクセスにはDiscordをおすすめします。メールを希望する場合はメッセージをお送りください。できるだけ早く返信します。
