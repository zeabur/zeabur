# ヘルプを取得する

Zeaburを使用しているときに、問題に遭遇することがあります。以下のチャンネルを通じてサポートを受けられます。

## コミュニティフォーラム

[Zeabur コミュニティフォーラム](https://zeabur.com/forum) はサポートを受けるための主要な場所です。問題を説明する投稿を作成すると、プランのサポート優先順位に基づいてチームが対応します。投稿の作成方法と含めるべき内容については、[コミュニティフォーラム](./forum)のページをご覧ください。

![new post](/community/forum/new-post.png)

## Discord

[公式Discordサーバー](https://zeabur.com/dc)に参加して、他のユーザーやZeaburチームと議論することもできます。

<Callout>
  Discordはコミュニティディスカッションプラットフォームです。Zeaburスタッフが参加して質問に回答することはありますが、そこでのすべてのメッセージに対応する責任は負いません。保証された応答時間での公式サポートが必要な場合は、[コミュニティフォーラム](./forum)をご利用ください。
</Callout>

![server](/community/help/server.png)

## プランのアップグレード

より迅速な応答時間と専任のサポートが必要な場合は、プランのアップグレードをご検討ください。上位プランでは、[コミュニティフォーラム](./forum)での保証応答時間が短くなります。詳細は[プラン](../billing/plans)ページをご覧ください。

## メール

メールでもサポートやお問い合わせを受け付けています：

- **メール：** [contact@zeabur.com](mailto:contact@zeabur.com)

より迅速な対応のために、[コミュニティフォーラム](./forum)の使用をお勧めします。ただし、メールでのコミュニケーションを希望される場合は、お気軽にメッセージをお送りください。できるだけ早く返信いたします。
