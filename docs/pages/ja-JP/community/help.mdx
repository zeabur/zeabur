# ヘルプを取得する

Zeaburを使用しているときに、問題に遭遇することがあります。[公式Discordサーバー](https://zeabur.com/dc)でZeaburチームからヘルプを受けることができます。

![server](/community/help/server.png)

## ヘルプの受け方

Discordサーバーには、誰もが自分の問題を投稿できる「help」というチャンネルがあります。

![help channel](/community/help/help-channel.png)

### 投稿フォーマット

問題をより明確に説明するために、フォーマットに従って問題を説明すると、問題を効果的に解決することができます。

- 投稿タイトルで問題を簡潔に説明します。
- 投稿本文で問題の詳細を説明します。
- 投稿にあなたの`projectID`、`serviceID`、または`deploymentID`を提供します。これらはZeaburダッシュボードのURLから見つけることができます。
  ![url](/community/help/url.png)
