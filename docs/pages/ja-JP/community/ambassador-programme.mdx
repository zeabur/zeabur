# Zeabur アンバサダープログラム

私たちと一緒に、Zeabur コミュニティを盛り上げましょう。

アンバサダーは、このエコシステムで物事を動かす人たちです — あなたの一つひとつのデプロイ、一つひとつのシェア、一つひとつのイベントが、コミュニティをより活発にし、チームをあなたに近づけます。

## あなたがすること

形式は自由です。コミュニティで一般的なアウトプット形式：

| | |
| -- | -- |
| **コンテンツ制作** | 記事を書く、動画を作る、ポッドキャストを録音して Zeabur で作ったものをシェア |
| **イベント主催** | 自分の街でミートアップ、ワークショップ、ハッカソンを開催 |
| **深い活用** | Zeabur を深く使い、問題を報告し、リアルな事例をシェア |
| **テンプレート貢献** | 高品質なテンプレートをマーケットプレイスとコミュニティに貢献 |

## あなたが得られるもの

* より進んだプロダクト体験
  * Pro プラン資格
  * 月額 30 USD の Zeabur クレジット（Pro プラン込み）
  * 未公開機能・ベータ機能への先行アクセス
* チームとの直接的なやりとり
  * アンバサダー専用スペース — Zeabur チームへの直通ライン
  * AMA — Zeabur チームとの直接対話
  * デベロッパーカウンシル — あなたのフィードバックが製品ロードマップに反映されます
* ブランドリソースのサポート
  * イベントスポンサーシップ
  * 大規模利用 / 高インパクトのケースは、Zeabur と 1 対 1 で拡張コラボレーションをお話しします

## 私たちが探している人

特定の属性は必要ありません — ただ、こんな開発者ならとても自然にフィットします：

* **ビルダー / クリエイター** — 自分の作品を Zeabur にデプロイし、プロセスを記録してシェアする方
* **コミュニティリーダー** — すでにある言語圏、Discord、Forum、オフラインイベントでハブになっている方
* **ヘビーユーザー** — Zeabur を本番環境のインフラとして深く使っている方

参加条件：直近 90 日以内にデプロイ実績がある、本人確認が可能、[コミュニティ行動規範](/ja-JP/community/code-of-conduct)に同意する。

## 私たちがお願いすること

クレジットがある以上、最低限のラインがあります — 形式は自由、KPI は設定しません。

6 か月に少なくとも 1 回、公に検証可能な貢献。以下のいずれかでかまいません：

* 1 本の記事、1 本の動画、1 エピソードのポッドキャスト
* 1 回のミートアップ、ワークショップ、講演
* 1 件公開されたテンプレート
* 1 件の公開デプロイ事例
* Forum / Discord での高品質な深掘り回答 5 件以上
* プロダクトチームに採用された問題報告や機能提案
* コミュニティイベントへの参加または共同設計と成果物

## よくある質問

**アンバサダープログラムと紹介プログラムの関係は？**

両者は独立しています — 紹介プログラムはすべての Zeabur ユーザーに平等に開かれており、アンバサダー特典ではありません。大量の紹介をもたらす高インパクトのアンバサダーには、拡張コラボレーションの枠組みで個別の条件を相談できます。

**他のプラットフォームのアンバサダーを兼任できますか？**

できます。私たちは独占を求めません — 開発者コミュニティでは複数のアイデンティティを持つことは自然なことで、衝突するものではありません。

**毎回の AMA に参加できなくても大丈夫ですか？**

問題ありません — AMA は録画されており、後から視聴できます。参加は歓迎しますが必須ではありません。

**6 か月貢献できなかったらどうなりますか？**

こちらから連絡して状況を伺います — 月次クレジットは一時停止、ステータスは保持、活動を再開すれば元に戻ります；恥をかかせることも強制退会もありません。

**友人を推薦できますか？**

できます。ホワイトリスト期間中は Zeabur から直接ご招待します；一般公開後は公募が始まります。

**Zeabur を公に批判してもいいですか？**

問題ありません。アンバサダーは広報担当ではありません — 開発者コミュニティで最も大切なのは率直さです。批判、競合との比較、誠意ある質問や不満、すべて歓迎します。

## 申請

アンバサダープログラムは現在ホワイトリスト制で、Zeabur から直接創設メンバーをご招待しています；一般公開後、ここに申請の入り口が表示されます。
