# 註冊網域

import { Callout, Steps } from 'nextra/components'

<Callout type="info">
  網域購買需要 **Developer** 或 **Team** 方案。Free 方案用戶會看到升級提示。
</Callout>

## 搜尋網域

在側邊欄開啟 **網域** 頁面，輸入想要的網域名稱。Zeabur 會同時查詢 9 個後綴（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）的可用性。

結果逐步載入——熱門後綴（`.com`、`.dev`、`.app`、`.io`）優先顯示，其餘隨後出現。每個結果旁即時顯示價格。

輸入完整網域如 `example.com` 時，頁面頂部會出現**最佳匹配**卡片。

## 購買網域

<Steps>

### 選擇網域

在搜尋結果中找到可用的網域，點擊**加入購物車**。

### 選擇註冊人資料

在購買彈窗中選擇已有的[註冊人資料](/domain/profiles)，或建立新的資料。如果設定了預設資料，會自動選中。

### 完成結帳

點擊**下一步**進入 Zeabur 結帳頁面。結帳流程與其他 Zeabur 產品相同，使用已綁定的付款方式。

付款後網域進入 **Pending** 狀態，等待註冊處理。大多數網域會在幾分鐘內變為 **Active**。

</Steps>

## 註冊人驗證

部分後綴在購買後要求註冊人驗證信箱地址：

1. 驗證信件發送到註冊人資料中的信箱地址。
2. 管理頁面的網域卡片上顯示 **驗證待處理** 標籤。
3. 點擊標籤或進入網域詳情頁，可重新發送驗證信件。

<Callout type="warning">
  長期未驗證的網域可能被註冊局暫停。購買後請及時完成驗證。
</Callout>
