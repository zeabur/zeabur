# 網域管理


網域管理頁面列出了你帳戶下註冊的所有網域。

## 網域狀態

每個網域具有以下狀態之一：

| 狀態 | 說明 |
|------|------|
| **已啟用** | 網域已註冊且正常運作。 |
| **待處理** | 註冊正在處理中。 |
| **已過期** | 網域已過期。 |
| **已暫停** | 網域被暫停（例如註冊人未完成驗證）。 |
| **失敗** | 註冊失敗。 |

網域較多時，可以使用狀態篩選下拉選單縮小列表範圍。

將在 30 天內到期的網域會在到期日期欄顯示警告。

## 自動續費

管理表格中每個網域都有自動續費開關。開啟後，Zeabur 會在網域到期前使用你綁定的付款方式自動續費。

你可以直接在表格列中切換自動續費，無需開啟詳情頁。

## 手動續費

手動續費網域的步驟：

<Steps>

### 開啟網域詳情

點擊網域開啟詳情頁。

### 點擊續費

點擊標題列中的 **續費** 按鈕，按鈕上會顯示續費價格。

### 完成結帳

完成結帳流程。

</Steps>

<Callout type="info">
  詳情頁同時顯示原始購買價格和續費價格，供你參考。
</Callout>

## Zeabur Email 整合

在網域詳情頁中，你可以一鍵將網域接入 **Zeabur Email**。選擇傳送區域後，所需的 DNS 記錄（DKIM、SPF、DMARC）會自動建立。

更多關於郵件網域驗證的資訊，請參閱 [Zeabur Email 網域管理](/email/domains)文件。
