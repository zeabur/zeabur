# 網域管理

網域管理頁面列出你帳戶下所有已註冊的網域。

## 網域狀態

每個網域具有以下狀態之一：

| 狀態 | 說明 |
|------|------|
| **Active** | 網域已註冊且正常運作。 |
| **Pending** | 註冊正在處理中。 |
| **Expired** | 網域已過期。 |
| **Suspended** | 網域被暫停（例如註冊人未驗證）。 |
| **Failed** | 註冊失敗。 |

網域較多時，使用狀態篩選下拉選單縮小列表範圍。

30 天內即將到期的網域會在到期日欄顯示警告。

## 自動續費

管理表格中每個網域都有自動續費開關。啟用後，Zeabur 會在網域到期前使用已綁定的付款方式自動續費。

直接在表格列中切換自動續費，無需開啟詳情頁。

## 手動續費

手動續費網域：

1. 點擊網域進入詳情頁。
2. 點擊頁首的 **續費** 按鈕，按鈕上顯示續費價格。
3. 完成結帳流程。

<Callout type="info">
  詳情頁同時顯示原始購買價格和續費價格，供參考。
</Callout>

## Zeabur Email 整合

在網域詳情頁，可以一鍵將網域接入 **Zeabur Email**。選擇發送區域後，所需的 DNS 記錄（DKIM、SPF、DMARC）會自動建立。

詳情參見 [Zeabur Email 網域管理](/email/domains) 文件。
