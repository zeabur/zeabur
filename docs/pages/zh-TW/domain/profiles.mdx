# 註冊人資料

網域註冊需要 WHOIS 聯絡資訊。註冊人資料讓你儲存一次資訊，在多次網域購買中重複使用。

## 資料欄位

| 欄位 | 必填 | 說明 |
|------|------|------|
| 名 | 是 | 註冊人名字 |
| 姓 | 是 | 註冊人姓氏 |
| 信箱 | 是 | 聯絡信箱（同時用於註冊人驗證） |
| 電話 | 是 | 聯絡電話 |
| 地址行 1 | 是 | 街道地址 |
| 地址行 2 | 否 | 公寓、套房等 |
| 城市 | 是 | 城市 |
| 州/省 | 是 | 州或省 |
| 國家 | 是 | 國家（從下拉選單選擇） |
| 郵遞區號 | 是 | 郵遞區號 |
| 組織 | 否 | 公司或組織名稱 |

## 建立資料

<Steps>

### 開啟資料分頁

在側邊欄進入 **網域** 頁面，選擇 **資料** 分頁。

### 填寫表單

點擊 **建立資料**，填寫必填欄位。從可搜尋的下拉選單中選擇國家。

### 儲存

點擊 **儲存**。新資料立即可用於網域註冊。

</Steps>

## 預設資料

點擊星標圖示將某個資料設為**預設**。購買網域時，預設資料會自動選中。你仍然可以在結帳時切換到其他資料。

## 編輯和刪除

- 點擊資料卡片的**編輯圖示**更新欄位。
- 點擊**刪除圖示**移除資料。

<Callout type="warning">
  刪除資料不會影響已用該資料註冊的網域。已註冊網域保留註冊時提交的聯絡資訊。
</Callout>

## 購買時建立資料

你也可以在網域購買彈窗中直接建立新資料，無需跳轉到資料分頁。新資料會儲存到你的帳戶，可用於後續註冊。
