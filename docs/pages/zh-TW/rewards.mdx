# 推薦與獎勵

Zeabur 提供多種獎勵計畫，讓你在使用和推廣的過程中獲得回饋。

- [推薦計畫](/rewards/referral) — 分享推薦碼，為朋友提供折扣、同時賺取佣金
- [合作夥伴計畫](/rewards/partner) — 以講師或顧問身份推廣 Zeabur，獲得合作獎勵
- [貢獻獎勵計畫](/rewards/contribution) — 貢獻模板或翻譯文件以獲得免費使用額度
- [開源贊助](/rewards/sponsor) — 為開源專案申請 Zeabur 贊助
- [兌換預付卡](/rewards/redeem-card) — 兌換預付信用額度至帳戶餘額
