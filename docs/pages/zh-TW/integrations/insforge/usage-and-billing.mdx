# 使用量與帳單

InsForge 提供詳細的使用量追蹤，讓您監控資源消耗和費用。

## 查看使用量

1. 在 Zeabur 面板中導航到您專案的 InsForge 整合頁面
2. 向下捲動到 **使用量概覽** 區塊

### 概覽分頁

概覽分頁顯示所選時間範圍的使用量摘要：

- **最大資料庫大小** — 資料庫儲存的峰值使用量
- **最大檔案儲存** — 檔案儲存的峰值使用量
- **總流量** — 消耗的總出站頻寬
- **EC2 計算** — 使用的總計算時數
- **AI Credits** — 消耗的 LLM AI 點數
- **AI Tokens** — 處理的 AI token 總數
- **MCP 呼叫** — MCP 協定呼叫總數
- **電子郵件請求** — 電子郵件 API 請求總數
- **函式呼叫** — 無伺服器函式執行總數

### 每日使用量分頁

每日使用量分頁提供按日期分類的資源使用量明細，以分頁表格顯示每天的所有指標。

### 時間範圍篩選

您可以依以下範圍篩選使用量資料：
- **1 天** — 過去 24 小時
- **7 天** — 過去一週
- **30 天** — 過去一個月
- **全部** — 所有可用資料

### 定價

| 項目 | 費率 |
| --- | --- |
| Compute Hour (Nano EC2 + EBS + Public IP) | $0.006 / 小時 |
| Database Size | $0.125 / GB / 月 |
| Storage | $0.021 / GB / 月 |
| Egress | $0.1 / GB |
| LLM AI Credits | $0.1 / credit |

### 帳單重要資訊

- **沒有人工限制** — 不存在 API rate limits、Row limits 或 Max User 上限。你只需為實際使用的計算和儲存付費。
- **統一帳單** — 所有 InsForge 成本會直接彙總到你既有的 Zeabur 專案帳單中。一個平台，一張帳單。
- **按用量計費** — 你只需要為後端實際消耗的資源付費。

### 帳單問題

如果您的 InsForge 實例因帳單問題而暫停：

1. 在 Zeabur 面板中前往 **帳單**
2. 綁定付款方式或儲值餘額
3. 返回 InsForge 整合頁面，點擊 **復原** 以重新扣款並恢復服務

<Callout type="info">
  如果您的實例因帳單問題而暫停，資料將被保留。解決帳單問題後復原實例即可恢復服務。
</Callout>
