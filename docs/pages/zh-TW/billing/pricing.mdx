# 定價

Zeabur 為你的專案提供按量計費的定價模式，您無需為未使用的資源買單，只需為您服務在這一分鐘實際使用的資源付費。

## 服務費率

Zeabur 有兩種伺服器類型：**共享叢集** 和 [**專用伺服器**](/dedicated-server)。這個費率表只適用於 **共享叢集**，我們不會收取專用伺服器的使用費用。

在 **共享叢集** 中使用的費率如下：

- vCPU - 不計費 ($0.0 / vCPU-分鐘)

- 記憶體 - **US$0.00025** / GB-分鐘 (約 US$10.8 / GB-月)
  - 我們只收取您服務實際使用的記憶體，服務 crash 或暫停了就不會繼續收取費用。

- 網路輸出費用 - **US$0.1** / GB (US$10 / 100GB)
  - 1GB 大約等於 1,000 名獨立訪客，在沒有 CDN 和壓縮的情況下，瀏覽您 1MiB 的網頁時會產生的流量。
  - [私有網路](/networking/private) 不算作輸出流量。

- 持久儲存空間 - **US$0.2** / GB-月

## 退款

如有任何退款相關問題，請至 [我們的 Discord 伺服器](https://zeabur.com/dc) 聯絡我們。

### 伺服器與訂閱方案

針對透過 **信用卡或其他非餘額付款方式** 購買的 **專屬伺服器** 和 **訂閱方案**（開發者方案、團隊方案），可以申請退款。您可以選擇：

- 將退款轉為 **Zeabur 餘額** — 核准後立即完成。
- 退回 **原付款方式** — 可能需要最多 7 天處理時間，且您需要支付交易手續費。

### 使用餘額購買的項目

對於使用 **Zeabur 餘額** 購買的伺服器或訂閱方案，退款只能退回至您的 Zeabur 餘額。

### 已購買的餘額

**已購買的餘額無法退款。**
