# 教育夥伴（Education Partner）

教育夥伴計畫面向**講師、社群領袖、Hackathon 主辦方與教育機構**。當你舉辦技術活動、開設教學課程或帶領工作坊時，可以透過活動碼（Event Code）讓參加者免費體驗 Zeabur，並依據活動後留存的付費使用者數量獲得獎勵。

## 合作模式

<Steps>

### 申請合作

前往 [partner.zeabur.com](https://partner.zeabur.com) 申請，說明你的課程或活動計畫。

### 取得活動碼

Zeabur 為你的活動建立專屬活動碼，供你發放給參加者。

### 進行活動

在課程、工作坊或 Hackathon 中帶領參加者使用 Zeabur 部署專案。

### 統計留存

活動碼發放期結束後一個月，統計透過你活動碼註冊且持續付費的使用者。

### 領取獎勵

依留存付費使用者人數核發獎勵。

</Steps>

## 活動碼機制

活動碼是一組專屬代碼，兌換後會把活動額度加入參加者的帳戶餘額，可用於 Zeabur 上的任何付費服務（部署、伺服器、AI Hub 等）：

- 每個活動碼與特定活動綁定，並有明確的有效期間（通常於活動當日結束）。
- 參加者可透過活動註冊連結（`https://zeabur.com/events?code=...`）完成註冊後自動兌換，或於登入後在帳單頁面手動兌換。
- 參加者兌換的完整步驟請參考 [兌換活動碼](/partner/event-partner/redeem-event-code)。

<Callout type="info">
活動碼兌換不會觸發推薦計畫的佣金機制——活動額度屬於贊助而非真實金流，推薦佣金僅在使用者實際付款成功時才會計算。
</Callout>

## 面額規格

活動碼面額為 **US$24** 或 **US$29** 兩種，結構為 Pro Plan 首月費用（US$19）加上 usage buffer：

- **US$24**（$19 + $5 buffer）：適合以部署專案為主的工作坊。
- **US$29**（$19 + $10 buffer）：適合 Hackathon、課程或 AI Hub 使用量較大的活動。

兩種面額都讓參加者可**首月免綁卡直接升級 Pro Plan** 體驗完整功能；差別在於 Pro 月費之外可用於其他服務的餘額多寡。

## 獎勵計算

| 項目 | 說明 |
| --- | --- |
| 統計時間 | 活動碼發放期結束後一個月 |
| 留存標準 | 參加者透過活動碼註冊，且在統計期間內升級為付費使用者並持續使用 |
| 基本獎勵 | 每位留存付費使用者 US$5 |
| 發放方式 | Zeabur Credits 或其他約定方式 |

### 範例

你在一場 50 人的工作坊中發放活動碼，活動後一個月有 8 位參加者持續使用 Zeabur 並升級為付費方案。你的獎勵為 8 × US$5 = **US$40**。

### 大型活動另案計算

針對大型 Hackathon、長期課程或影響力較高的系列活動，Zeabur 可視活動規模、主辦方背景與預估轉換，將單位獎勵調整至 **US$10 per 留存付費使用者**。此類合作需事先與 Zeabur 團隊討論，並於申請時註明。

## 適用場景

- **程式設計課程** — 帶學員將課程專案部署到雲端。
- **技術工作坊** — 現場實作活動，參加者即時體驗部署流程。
- **Hackathon** — 參賽團隊使用 Zeabur 快速部署作品。
- **線上課程** — 提供活動碼讓遠端學員實作練習。
- **社群活動** — 技術社群聚會、分享會搭配 demo 環境。

## 如何申請

前往 [partner.zeabur.com](https://partner.zeabur.com) 建立帳號並提交活動合作申請，填寫活動名稱、預計人數、時間與形式。Zeabur 團隊審核後會協助建立專屬活動碼。

如有其他疑問，歡迎在 [Zeabur 社群論壇](https://zeabur.com/forum) 提問。
