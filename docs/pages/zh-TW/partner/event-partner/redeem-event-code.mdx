# 兌換活動碼

活動碼（Event Code）由 Zeabur 教育夥伴（Education Partner）於講座、課程、工作坊或 Hackathon 發送給參加者，可在 Zeabur 兌換為體驗額度，用於升級訂閱方案、部署專案與其他付費功能。

本頁介紹活動碼的兩種兌換方法。活動主辦方可將本頁連結直接分享給參加者，避免重複說明。

<Callout type="info">
活動碼都有使用期限，太早或太晚皆無法兌換成功。請留意主辦方告知的有效期間。
</Callout>

## 方法一：活動註冊連結（最簡單，適合尚未註冊的人）

如果你還沒有 Zeabur 帳號，可以直接使用活動註冊連結。將你的活動碼加到以下網址後方：

```
https://zeabur.com/events?code=你的活動碼
```

例如活動碼為 `i_love_zeabur`，就打開：

```
https://zeabur.com/events?code=i_love_zeabur
```

打開後依照頁面引導完成註冊，活動額度會自動兌換到你的新帳戶。

## 方法二：在帳單頁面兌換（適合已有 Zeabur 帳號的人）

<Steps>

### 登入 Zeabur 帳號

前往 [zeabur.com/login](https://zeabur.com/login) 登入。

### 進入帳單管理頁

點擊右上角頭像選單中的「帳單」，或直接前往 [zeabur.com/account/billing](https://zeabur.com/account/billing)。

### 點擊「兌換積分」

在「帳戶餘額」區塊右上角找到「兌換積分」按鈕並點擊。

### 輸入活動碼

在彈出的對話框中輸入你的活動碼，點擊「兌換」。兌換成功後會在右下角顯示通知，帳戶餘額立即增加。

</Steps>

## 兌換成功後

活動額度會直接進入你的**帳戶餘額**，可用於 Zeabur 上的任何服務——部署專案、購買伺服器、使用 AI Hub 等，使用方式與自行加值的儲值金相同。

## 升級至 Pro Plan

前往 [Pricing 頁面](https://zeabur.com/pricing#upgrade) 手動點擊升級到 **Pro Plan** 解鎖進階功能（見 [Pro Plan](/pricing/pro-plan)）。升級時活動額度會優先用於支付 Pro 月費——額度足以覆蓋首月（US$19）時，升級過程**不需要綁定信用卡**，第二個月起才會進入一般計費週期。

## 常見問題

### 活動額度可以用在哪？

任何 Zeabur 上的付費項目——部署、資料庫、伺服器、AI Hub 等。它就是一般的帳戶餘額，使用方式與自行加值的儲值金相同。

### 我需要綁信用卡嗎？

兌換活動碼**不需要**綁卡。升級 Pro 方案時，若活動額度足以覆蓋首月費用就不需綁卡；不足時需綁卡補齊差額。後續月份想繼續使用 Pro 也需綁卡以便扣款。

### 活動結束後還能繼續用 Zeabur 嗎？

可以。想繼續用 Pro 的話，綁定信用卡後就會自動續訂；不想繼續付費就維持[免費方案](/pricing)即可，隨時也能回到 [Pricing 頁面](https://zeabur.com/pricing) 重新升級。

### 活動額度用完後會怎樣？

- **已在 Pro 方案**：下次結算會從綁定的信用卡扣款。
- **仍在免費方案**：超過免費配額的資源無法使用，可以[加值](https://zeabur.com/account/billing)或升級方案繼續使用。

### 活動碼過期 / 無效 / 其他問題

- **過期** — 過期後無法補救，請確認主辦方告知的有效期間。
- **無效** — 確認輸入是否完整、大小寫是否正確。
- **其他** — 前往 [Zeabur 社群論壇](https://zeabur.com/forum) 聯繫客服團隊。
