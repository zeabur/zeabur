# 銷售夥伴（Sales Partner）

銷售夥伴計畫面向**企業顧問、系統整合商、技術顧問公司**。當你協助企業客戶、機構或組織導入 Zeabur 時，可依實際參與的合作階段獲得經常性分潤。

## 適用場景

銷售夥伴計畫適合下列合作類型：

- **企業雲端遷移** — 協助企業將既有服務遷移至 Zeabur。
- **機構統一採購** — 教育機構、政府單位或企業 IT 部門的大量採購、合約採購，含 Voucher、需開發票、需合約等情境。
- **技術顧問導入** — 身為技術顧問或系統整合商，將 Zeabur 納入你為客戶設計的部署方案。
- **長期服務合作** — 替客戶提供部署、維運或金流代收等服務，並以 Zeabur 作為底層平台。

若金額未達門檻且為個人自助付款，請改走 [推薦計畫](/rewards/referral) 的自動流程。具體適用門檻與條件於合約中個別談定。

<Callout type="info">
同一筆交易只能走推薦計畫或銷售夥伴其中一條路徑，**兩者不疊加**。
</Callout>

## 分潤架構

銷售夥伴的分潤由兩個層級組成，依你實際參與的範圍計算。

### 第一層：交易費率

每筆交易獨立計算，費率隨交易金額採累進制。費率依夥伴類型、市場定位與合作規模**個別談定**，不對外公開標準值。

### 第二層：貢獻階段權重

依你在該筆交易中實際負責的階段，四階段權重加總為 100%。Zeabur 自行負責的階段不外付。

| 階段 | 定義 | 認定基準 |
| --- | --- | --- |
| Lead（引薦與開發） | 找到客戶 | 你促成雙方首次建立聯繫（含成立專屬通訊群組）之時間 |
| Close（結單與簽約） | 談成簽約 | 以雙方簽署之報價單為準 |
| Deliver（服務與維護） | 服務交付 | 客戶端問題回覆與操作協助皆由你完成，無需 Zeabur 工程團隊介入 |
| Collect（金流與發票） | 催收金流 | 由你完成收款、開立發票，無需 Zeabur 提醒或追帳 |

### 常見參與組合

| 情境 | Lead | Close | Deliver | Collect |
| --- | :---: | :---: | :---: | :---: |
| 全包 | ✓ | ✓ | ✓ | ✓ |
| 純介紹 | ✓ |   |   |   |
| 介紹 + 服務 | ✓ |   | ✓ |   |
| 服務 + 收款 |   |   | ✓ | ✓ |
| 簽約 + 代操（無 Lead） |   | ✓ | ✓ | ✓ |
| 介紹 + 服務 + 收款 | ✓ |   | ✓ | ✓ |

### 多方拆帳

同一筆交易可由不同合作夥伴負責不同階段。交易層封閉計算、不跨池，各方獨立結算、互不影響。

## 配套規則

- **結算幣別**：以 USD 為基準結算，匯兌成本由銷售夥伴自行承擔。
- **結算週期**：分潤以月為單位計算，經雙方確認後入帳至 Partner Portal。
- **提領等待期**：分潤入帳後需經過 holding 期方可提領。等待期內客戶若 churn 會自動扣除，期滿後已提領的部分不予追回。具體天數依合約約定。
- **最低銷售承諾**：合作期間設有最低銷售承諾，連續未達時 Zeabur 有權調整費率或終止合約。具體金額依合約約定。
- **Lead 認定**：Lead 階段需經 Zeabur 確認後方能鎖定。
- **階段鎖定**：Deal 登記時鎖定各階段責任歸屬，變更需雙方書面同意。
- **分潤存續**：客戶持續付費期間持續計算，客戶 churn 即停。

<Callout type="warning">
銷售夥伴路徑與推薦計畫不可套用於同一筆交易——每筆交易擇一，以避免雙重利益衝突。
</Callout>

## 合作流程

<Steps>

### 聯繫 Zeabur

前往 [partner.zeabur.com](https://partner.zeabur.com) 申請，或透過 [Zeabur 社群論壇](https://zeabur.com/forum) 聯繫團隊，說明你的客戶背景與預估合作規模。

### 洽談合約

Zeabur 團隊評估後與你洽談具體費率、階段權重與配套條件。個別合約內容保密。

### 簽署合約

確認條款後正式簽約，開始追蹤 Deal。

### 客戶導入

協助客戶完成評估、測試與上線。

### 月結分潤

客戶付款成功後，依合約條件進行月結分潤。

</Steps>

## 相關附約

部分合作會搭配獨立合約或附約處理，例如：

- **金流代收代付服務費**：若你擔任代收代付角色，相關手續費走獨立合約。
- **教育補助**：若合作包含學校或教育機構的教育補助，以獨立附約/MOU 處理。

## 如何申請

- 前往 [partner.zeabur.com](https://partner.zeabur.com) 建立帳號並提交申請，填寫你的公司資訊、客戶背景與預估合作規模。
- 也可於 [Zeabur 社群論壇](https://zeabur.com/forum) 聯繫 Zeabur 團隊說明合作意願。
