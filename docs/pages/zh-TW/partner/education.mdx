# 教育推廣

教育推廣計畫面向**講師、培訓機構和活動主辦方**，透過教學和技術活動將 Zeabur 介紹給學員，並依據留存轉換為付費用戶的數量獲得獎勵。

## 合作模式

1. **申請合作** — 聯繫 Zeabur 團隊，說明你的課程或活動計畫。
2. **取得 Event Code** — Zeabur 為你的活動建立專屬 Event Code，學員使用此 Code 可免費體驗 Zeabur 服務。
3. **進行教學 / 活動** — 在課程或活動中帶領學員使用 Zeabur 部署專案。
4. **統計留存用戶** — 活動結束一個月後，Zeabur 統計透過你的 Event Code 註冊且持續付費的用戶數。
5. **領取獎勵** — 每位留存付費用戶獎勵 US$5。

## Event Code 機制

Event Code 是一組專屬代碼，提供學員在活動期間使用 Zeabur 的體驗額度：

- 每個 Event Code 與特定活動綁定。
- 學員在 Zeabur 註冊時輸入 Event Code 即可獲得體驗額度。
- Event Code 有效期依活動安排而定，通常涵蓋活動期間及之後一段緩衝時間。

<Callout type="info">
Event Code 的兌換不會觸發推薦計畫的佣金機制——因為 Event Code 提供的是體驗額度，而非真實金流。推薦佣金僅在用戶實際付款成功時才會計算。
</Callout>

## 獎勵計算

| 項目 | 說明 |
| --- | --- |
| 統計時間 | Event Code 發放後一個月 |
| 留存標準 | 學員透過 Event Code 註冊，且在統計期間內升級為付費用戶並持續使用 |
| 獎勵金額 | 每位留存付費用戶 US$5 |
| 發放方式 | Zeabur Credits 或其他約定方式 |

### 範例

你在一場 50 人的工作坊中發放 Event Code。一個月後，有 8 位學員持續使用 Zeabur 並升級為付費用戶。你的獎勵為 8 × US$5 = **US$40**。

## 適用場景

- **程式設計課程** — 帶學員部署課程專案到雲端。
- **技術工作坊** — 現場實作活動，學員即時體驗部署流程。
- **Hackathon** — 參賽團隊使用 Zeabur 快速部署作品。
- **線上課程** — 提供 Event Code 讓遠端學員實操練習。

## 申請方式

請透過以下方式聯繫 Zeabur 團隊申請教育推廣合作：

- **Discord**：前往 [Zeabur Discord 伺服器](https://zeabur.com/dc) 開啟 ticket，說明你的活動計畫。
- 提供活動名稱、預計人數、時間和形式，Zeabur 團隊將為你安排 Event Code。
