# 定價

Zeabur 提供四種訂閱方案，以及一個客製化的 Enterprise 選項：

- **[Free Plan](/pricing/free-plan)** — US$0/月
- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 天免費試用）
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 天免費試用，建議用於正式環境）
- **[Team Plan](/pricing/team-plan)** — US$79/月含 3 個席位（每增加一席 +US$24/月）
- **Enterprise** — 客製定價，請[聯絡我們](https://zeabur.com/forum)
