# 快速開始

本指南將幫助您在幾分鐘內使用 Z-Send 發送第一封郵件。

## 前提條件

在開始之前，您需要：

1. 一個 Zeabur 帳號
2. 在 Zeabur 控制台中啟用 Z-Send 服務
3. 至少一個已驗證的寄件網域
4. 一個 API 金鑰

## 步驟 1：建立 API 金鑰

import { Callout, Steps } from 'nextra/components'

<Steps>

### 登入 Zeabur 控制台

訪問 [Zeabur Dashboard](https://dash.zeabur.com) 並登入您的帳號。

### 進入 Z-Send 管理頁面

在控制台中找到 Z-Send 服務並進入管理頁面。

### 建立新的 API 金鑰

1. 點擊「建立 API 金鑰」按鈕
2. 選擇權限類型：
   - **唯讀**：僅能查詢郵件狀態
   - **僅發送**：可發送郵件和查詢狀態（建議用於生產環境）
   - **完整權限**：包含所有操作權限
3. 可選：限制網域權限（只能使用特定網域發送）
4. 點擊「建立」

<Callout type="warning">
  建立後請立即儲存 API 金鑰，金鑰僅顯示一次！
</Callout>

</Steps>

## 步驟 2：驗證網域

要發送郵件，您需要先驗證寄件網域：

<Steps>

### 新增網域

在 Z-Send 管理頁面的「網域管理」中，點擊「新增網域」。

### 設定 DNS 記錄

系統會產生 DKIM、SPF 和 DMARC 記錄，您需要在網域 DNS 提供商處新增這些記錄。

### 等待驗證

DNS 記錄新增後，點擊「驗證」按鈕。驗證通常需要幾分鐘到幾小時。

</Steps>

## 步驟 3：發送第一封郵件

網域驗證通過後，您就可以發送郵件了！

### 使用 cURL

```bash
curl -X POST https://gateway.zeabur.com/api/v1/zsend/emails \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": ["user@example.com"],
    "subject": "來自 Z-Send 的問候！",
    "html": "<h1>你好！</h1><p>這是我從 Z-Send 發送的第一封郵件。</p>",
    "text": "你好！這是我從 Z-Send 發送的第一封郵件。"
  }'
```

### 使用 JavaScript/Node.js

```javascript
const response = await fetch('https://gateway.zeabur.com/api/v1/zsend/emails', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    from: 'hello@yourdomain.com',
    to: ['user@example.com'],
    subject: '來自 Z-Send 的問候！',
    html: '<h1>你好！</h1><p>這是我從 Z-Send 發送的第一封郵件。</p>',
    text: '你好！這是我從 Z-Send 發送的第一封郵件。'
  })
});

const data = await response.json();
console.log('郵件已發送:', data);
```

### 使用 Python

```python
import requests

response = requests.post(
    'https://gateway.zeabur.com/api/v1/zsend/emails',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
    },
    json={
        'from': 'hello@yourdomain.com',
        'to': ['user@example.com'],
        'subject': '來自 Z-Send 的問候！',
        'html': '<h1>你好！</h1><p>這是我從 Z-Send 發送的第一封郵件。</p>',
        'text': '你好！這是我從 Z-Send 發送的第一封郵件。'
    }
)

print('郵件已發送:', response.json())
```

### 使用 Go

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "from":    "hello@yourdomain.com",
        "to":      []string{"user@example.com"},
        "subject": "來自 Z-Send 的問候！",
        "html":    "<h1>你好！</h1><p>這是我從 Z-Send 發送的第一封郵件。</p>",
        "text":    "你好！這是我從 Z-Send 發送的第一封郵件。",
    }

    jsonData, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://gateway.zeabur.com/api/v1/zsend/emails", bytes.NewBuffer(jsonData))
    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    fmt.Println("郵件已發送！狀態:", resp.Status)
}
```
