# Zeabur 大使計畫

跟我們一起，讓 Zeabur 社群活起來。

大使是讓 Zeabur 產品落地社群的夥伴 — 你的每一個部署、每一篇分享、每一場活動，都在將 Zeabur 產品帶向開發者。

## 你會做的事

形式不限，社群裡最常見的產出形式有：

* **內容創作** — 寫文章、做影片、錄 podcast 分享你用 Zeabur 做的事
* **活動主辦** — 在你的城市辦社群聚會、工作坊、黑客松
* **深度使用** — 把 Zeabur 用得深、回報問題、把真實案例分享出來
* **模板貢獻** — 貢獻高品質模板到模板市集以及社群內

## 大使權益

* 更進階的產品使用
  * Pro 方案資格
  * 每月 30 美元 Zeabur 額度（含 Pro 方案）
  * 搶先使用未發佈與測試版功能
* 更直接與團隊交流
  * 大使專屬交流空間 — 直通 Zeabur 團隊
  * AMA — 跟 Zeabur 團隊直接對話
  * 開發者諮詢會 — 你的回饋直接影響產品藍圖
* 更多品牌資源協助
  * 活動贊助
  * 大規模使用 / 高影響力的個案，跟 Zeabur 一對一洽談加碼合作空間

## 我們在找的人

不需要符合特定身份 — 但下列幾種開發者最自然契合：

* **建立者 / 創作者** — 用 Zeabur 部署自己的作品、把過程記錄下來分享
* **社群領袖** — 你已經在某個語區、Discord、Forum、或線下活動是節點
* **重度使用者** — 把 Zeabur 當正式環境的基礎設施、深度地用

我們的要求：過去 90 天有部署紀錄、身份可被驗證、同意[社群行為準則](/zh-TW/community/code-of-conduct)。

## 我們對你的期待

有額度，就有最輕量的門檻 — 形式不限，不訂績效指標。

半年內至少一次可公開驗證的貢獻，下列任一即可：

* 一篇文章、一支影片、一集 podcast
* 一場社群聚會、工作坊、講座
* 一個上架的模板
* 一個公開的部署案例
* Forum / Discord 高品質深度解答累積五篇以上

## 常見問題

**大使計畫跟推薦計畫是什麼關係？**

兩者解耦 — 推薦計畫對所有 Zeabur 用戶平等開放，不是大使福利。高影響力大使若帶來大量推薦，可在加碼合作的框架下另案洽談個案條件。

**沒辦法每場 AMA 都參加怎麼辦？**

沒關係 — AMA 全程錄影、可事後補看，鼓勵出席但不強制。

**半年沒做出貢獻會發生什麼？**

活躍度與參與度未達預期者，大使權益會暫停。

**我可以提名朋友嗎？**

可以。白名單制度期間由 Zeabur 主動邀請；正式上線後開放公開申請。

## 申請

大使計畫目前為白名單制度，由 Zeabur 主動邀請首批創始團隊；正式上線後開放公開申請 — 屆時此處更新申請入口。
