# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以透過以下管道獲得幫助。

## 社群論壇

[Zeabur 社群論壇](https://zeabur.com/forum) 是取得幫助的主要管道。你可以建立貼文描述你的問題，我們的團隊會根據你的方案支援優先順序進行回覆。詳細的貼文建立方式和注意事項，請參考[社群論壇](./forum)頁面。

![new post](/community/forum/new-post.png)

![server](/community/help/server.png)

## 升級方案

如果你需要更快的回應時間和專屬支援，可以考慮升級你的方案。較高等級的方案在[社群論壇](./forum)上享有更短的保證回應時間。詳情請參閱[方案](../billing/plans)頁面。

## 電子郵件

你也可以透過電子郵件聯繫我們以獲得支援或詢問：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為了獲得更快的回應時間，我們建議使用[社群論壇](./forum)。不過，如果你偏好使用電子郵件溝通，歡迎隨時發送訊息給我們，我們會盡快回覆。
