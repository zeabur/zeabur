# **尋求幫助**

使用 Zeabur 時偶爾會遇到問題。您可以在我們的 [Zeabur 論壇](https://zeabur.com/forum) 向 Zeabur 團隊尋求幫助。
# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以透過以下管道獲得幫助。

## 社群論壇

[Zeabur 社群論壇](https://zeabur.com/forum) 是取得幫助的主要管道。你可以建立貼文描述你的問題，我們的團隊會根據你的方案支援優先順序進行回覆。詳細的貼文建立方式和注意事項，請參考[社群論壇](./forum)頁面。

![new post](/community/forum/new-post.png)

## Discord

你也可以加入[我們的官方 Discord 伺服器](https://zeabur.com/dc)與其他使用者和 Zeabur 團隊進行討論。

<Callout>
  Discord 是一個社群討論平台。雖然 Zeabur 工作人員可能會參與並回答問題，但我們不負責回覆那裡的所有訊息。如需有保證回應時間的官方支援，請改用[社群論壇](./forum)。
</Callout>

![Zeabur Forum](/community/get-help/Zeabur-Forum.png)

## **如何取得幫助**

在我們的論壇中，有一個名為 `New Post` 的按鈕，供大家建立問題票證。

![Zeabur Forum Overview](/community/get-help/zeabur-forum-overview.gif)

### **發文格式**

遵循格式清晰描述問題，能幫助我們更快定位並解決。

1. 選擇 **Issue Type**（問題類型）
2. 在 **Title**（標題）中簡要描述問題。
3. 選擇 **Project**（專案）或 **Server**（伺服器） - 會列出您 Zeabur 帳號中的所有專案/伺服器
4. 在服務中選擇有問題的服務
5. 貼上 logs 中的錯誤訊息。
6. 最後，加入內容上下文，幫助支援團隊全面了解問題全貌。

![Ticket System](/community/get-help/Ticket-System.png)

## 其他

![Others](/community/get-help/other-issues.png)

選擇「Others」標籤，盡量詳細描述以獲得 Zeabur 團隊最高效的回應

### **優先支援**

- Team 方案：半個工作日內回覆
- Dev 方案：3 個工作日內回覆
- Free 方案：5 個工作日內回覆

**其他聯絡方式**

雖然 Zeabur 論壇是我們的官方支援平台，但您也可以透過電子郵件聯絡我們進行支援或詢問：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為獲得更快的回應時間與社群資源存取，我們推薦使用 Discord。但若您偏好電子郵件溝通，歡迎發送訊息，我們會盡快回覆。
