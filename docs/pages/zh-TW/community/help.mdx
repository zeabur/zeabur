# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以在 [我們的官方 Discord 伺服器](https://zeabur.com/dc) 中尋求幫助。

![server](/community/help/server.png)

## 如何得到幫助

在我們的 Discord 伺服器中，我們有一個叫做 `問題求助` 的頻道，你可以在這裡發布你的問題。

![help channel](/community/help/help-channel.png)

### 發文格式

遵循以下格式來描述你的問題可以幫助我們更快地定位和解決你的問題。

- 在帖子標題簡單描述你的問題
- 在內容區塊中描述你的問題細節
- 在帖子中提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在你的 Zeabur 控制台的 URL 中找到它們。
  ![url](/community/help/url.png)

### 優先支援

你可以通過驗證你的帳戶來獲得優先支援。請參考[這篇文件](./verify)來進行驗證。

## 其他聯絡方式

雖然 Discord 是我們的主要社群平台，你也可以透過電子郵件聯繫我們以獲得支援或詢問：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為了獲得更快的回應時間和存取我們的社群資源，我們建議使用 Discord。不過，如果你偏好使用電子郵件溝通，歡迎隨時發送訊息給我們，我們會盡快回覆。
