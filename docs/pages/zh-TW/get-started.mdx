# **快速開始**

## **第一步：建立帳號**

您需要建立一個帳號才能開始在 Zeabur 上部署服務。

若您是 Zeabur 新使用者，請使用電子郵件註冊（第一個選項）。在此輸入您的電子郵件地址，然後點擊「使用電子郵件繼續」。

![SignUp.png](/start/get-started/SignUp.png)

接著會收到「郵件發送成功」的提示。

![SignUp Part 2.png](/start/get-started/SignUp%20Part%202.png)

開啟您的郵件收件匣（例如 Gmail、Outlook、Apple Mail），您會看到來自 Zeabur 的郵件。點擊郵件中的「登入」按鈕。

![Sign Up Callback.png](/start/get-started/Sign%20Up%20Callback.png)

若登入過程中開啟的瀏覽器不是您偏好的預設瀏覽器（例如開啟 Safari 而非 Chrome），您可手動複製「登入」按鈕下方的連結，並貼到您想要的瀏覽器中。

出現此登入介面即表示您已成功登入！

![Dashboard.png](/start/get-started/Dashboard.png)

請記得，您可在左側面板切換介面語言！我們目前支援繁體中文、簡體中文、英語、日語、西班牙語和印尼語。

![Languages.png](/start/get-started/Languages.png)

## **第二步：建立專案**

在 Zeabur 中，專案是一個單一容器，用來存放構成真實世界應用程式的所有相關服務。例如，若您的應用程式包含前端網站、後端 API 和資料庫，它們都應放在同一個專案中。

![Project structure](/start/project-structure.webp)

要開始建立專案，請點擊中央的**建立專案**按鈕。

![Create New Project.png](/start/get-started/Create%20New%20Project.png)

若是第一次建立專案，您可能會看到「帳號驗證」提示。

![Toast if not verified.png](/start/get-started/Toast%20if%20not%20verified.png)

您可選擇：

1. 使用**手機號碼**驗證。大多數國家和地區都支援此方式。若您的國家/地區不支援，或需要綁定多個 Zeabur 帳號，除[聯絡我們](https://zeabur.com/forum)外，也可參考第二和第三個選項。
2. 預先儲值一些**使用額度**。只要您未訂閱 Dev 方案、未購買專用伺服器，且未涉及嚴重濫用（例如違反 ToS 發起高流量 DoS 攻擊），我們不會扣除餘額。
3. 我們需要**信用卡**來驗證您的帳號，以確保操作順暢。請注意，除非您主動發起訂閱相關交易，否則不會產生任何扣款。

選擇您偏好的方式，然後點擊對應部分的「驗證」按鈕。

驗證完成後，您可重新建立專案，並看到成功建立的專案。

![After Verification.png](/start/get-started/After%20Verification.png)

## **第三步：部署服務**

Zeabur 可部署多種服務，包括資料庫（PostgreSQL、MySQL、Redis）、開源專案（n8n、WordPress）或您自己的自訂應用（從 GitHub 匯入或從資料夾上傳）。

- 若要部署資料庫，點擊「資料庫」直接部署常見資料庫。
    
    ![Databases.png](/start/get-started/Databases.png)
    
- 若要部署開源專案，Zeabur 的「模板」提供許多預先配置的常見開源專案，選擇「模板」即可一鍵部署。
    
    ![templates.png](/start/get-started/templates.png)
    
- 若要部署自己的自訂應用，可點擊「GitHub」或「本地專案」上傳程式碼，或點擊「Cursor」直接部署目前專案。
- 若您想部署的開源專案不在「模板」中，也可使用「Docker 映像」自行設定（屬於進階操作）。

以 GitHub 部署為例。首先將 Zeabur 帳號綁定至 GitHub，等待約 1 分鐘，然後搜尋您想部署的儲存庫：

![Github.png](/start/get-started/Github.png)

點擊儲存庫後，Zeabur 代理會自動分析您的程式碼並為您選擇最適合的部署方式。

![Build Plan.png](/start/get-started/Build%20Plan.png)

點擊「部署」即可一鍵部署專案。如有需要，可前往「設定」調整環境變數。

在設定中，將根目錄設為您的應用資料夾（例如“/”或子資料夾），透過鍵/值或編輯原始變數新增所需環境變數，然後點擊下一步確認。

![Configure Build Plan  .png](/start/get-started/Configure%20Build%20Plan%20%20.png)

## **使用 Zeabur AI 代理部署任意服務**

Zeabur 的 AI 代理讓您使用自然語言快速部署所需服務。

![Postgresql Examples.png](/start/get-started/Postgresql%20Examples.png)

點擊右側面板中的「Zeabur AI」進入 AI 助手介面。若遇到任何部署問題，也可向其求助！

## **第四步：啟動您的網站**

要讓您的網站可供他人存取，可在「網域」中綁定自己的網域：

![Bind domain](/start/bind-domain.webp)

等待約 30 秒後即可點擊網域。Zeabur 會為您處理 HTTPS 等設定。全部確認後，即可分享您的網站。

若需綁定自己的網域，可查看[網域綁定](https://zeabur.com/docs/deploy/domain-binding)文件取得更多資訊。

## **下一步做什麼？**

若希望使用 Google 或 GitHub 帳號登入，可在「設定」>「整合」中綁定 GitHub 和 Google 帳號。

![Account integration](/start/account-integration.webp)

接下來，您可繼續探索 Zeabur 豐富的模板庫以及各種功能（例如檔案管理、記錄）。如遇技術問題，可在 [Zeabur 官方論壇](https://zeabur.com/forum) 提交開發者票證。如需社群討論與互助，請加入我們的開發者社群 [Zeabur Discord 伺服器](https://zeabur.com/dc)。
