# 購買專用叢集

Zeabur 提供一鍵建立 Linode 專用叢集的流程，適合希望快速取得可用 Kubernetes 叢集，而不想自己手動建控制平面、節點池與網路設定的團隊。

## 購買前準備

- 確認你的 Zeabur 帳戶中有可用的付款方式或餘額。
- 決定叢集要部署的區域與節點規格。

## 購買步驟

1. 登入 Zeabur，前往[「伺服器」選單](https://zeabur.com/servers)。
2. 選擇「叢集」後，點選右上角的「建立」。
   ![cluster page](/cluster/cluster-page.webp)
3. 在建立流程中選擇 **Buy a cluster from Zeabur**。
   ![create cluster](/cluster/create-cluster.webp)
4. 選擇叢集區域和節點方案，然後點選 **Purchase now**。
   ![select cluster plan](/cluster/cluster-select-plan.webp)
5. 確認費用與設定後送出訂單。
6. 等待 Zeabur 完成叢集建立與初始化，建立時間大約會耗時 3-5 分鐘。
7. 建立完成後，你會在伺服器列表中看到新的叢集，可直接在 Zeabur 上用它來建立專案與部署服務。
   ![cluster information](/cluster/cluster-information.webp)

建立完成後，Zeabur 會替你接手叢集接入所需的基礎設定，讓你可以直接從 Dashboard 管理部署，而不需要自己逐步設定 Kubernetes 基礎元件。
