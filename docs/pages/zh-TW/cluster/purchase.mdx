# 購買叢集

Zeabur 提供一鍵建立 Linode 叢集的流程，適合希望快速取得可用 Kubernetes 叢集，而不想自己手動建控制平面、節點池與網路設定的團隊。

## 購買前準備

- 確認你的 Zeabur 帳戶中有可用的付款方式或餘額。
- 決定叢集要部署的區域與節點規格。

## 購買步驟

<Steps>

### 登入 Zeabur

前往[「伺服器」選單](https://zeabur.com/servers)。

### 選擇叢集並建立

選擇「叢集」後，點選右上角的「建立」。
![cluster page](/cluster/cluster-page.webp)

### 選擇從 Zeabur 購買叢集

在建立流程中選擇「從 Zeabur 購買叢集」。
![create cluster](/cluster/create-cluster.webp)

### 選擇區域和節點方案

選擇叢集區域和節點方案，然後點選「立即購買」。
![select cluster plan](/cluster/cluster-select-plan.webp)

### 確認費用並送出訂單

確認費用與設定後送出訂單。

### 等待叢集建立完成

等待 Zeabur 完成叢集建立與初始化，建立時間大約會耗時 3-5 分鐘。

### 叢集出現在伺服器列表中

建立完成後，你會在伺服器列表中看到新的叢集，可直接在 Zeabur 上用它來建立專案與部署服務。
![cluster information](/cluster/cluster-information.webp)

</Steps>

建立完成後，Zeabur 會替你接手叢集接入所需的基礎設定，讓你可以直接從 Dashboard 管理部署，而不需要自己逐步設定 Kubernetes 基礎元件。

## 下一步

- 若你仍在評估單機與多節點方案差異，請回到[叢集總覽](/cluster)。
