# 合作夥伴計畫

Zeabur Partner Program 為講師、活動主辦方和導入顧問提供合作機會，透過推廣 Zeabur 獲得獎勵回饋。

## 兩種合作模式

### 教育推廣

適合**講師、培訓機構、活動主辦方**。透過教學課程或技術活動推廣 Zeabur，使用 Event Code 讓學員免費體驗，並依留存付費使用者數獲得獎勵。

- 提供專屬 Event Code 供學員使用
- 一個月後統計留存付費使用者，每人獎勵 US$5
- 適合工作坊、線上課程、Hackathon 等場景

[了解更多 →](/partner/education)

### 導入顧問（Sales Partner）

適合**企業顧問、系統整合商、技術諮詢公司**。協助企業客戶導入 Zeabur，客戶持續付費期間獲得 15% 的經常性佣金。

- 觸發條件：單筆金額 > US$1,000 或機構統一採購
- 佣金模式：客戶持續付費期間，每月獲得 15% recurring commission
- 適合企業導入、機構採購等大型合作

[了解更多 →](/partner/sales-partner)

## 與推薦計畫的區隔

Zeabur 同時提供[推薦計畫](/rewards/referral)，讓所有使用者透過推薦碼賺取回饋。兩者的主要差異：

| 項目 | 推薦計畫 | 合作夥伴計畫 |
| --- | --- | --- |
| 適用對象 | 所有使用者 | 講師 / 活動主辦 / 企業顧問 |
| 觸發方式 | 推薦碼自動流程 | 人工審核與合約 |
| 獎勵機制 | 推薦碼佣金（依等級 20%–100%） | Event Code 留存獎勵 / 15% recurring commission |
| 金額門檻 | 無 | Sales Partner：單筆 > US$1,000 |

<Callout type="info">
同一筆交易只能走一條路——推薦碼或合作夥伴計畫，兩者不疊加。
</Callout>
