# 服務使用量

Zeabur 採用按量計費模式，並提供詳細圖表展示你的服務使用情況。

點擊你的服務，選擇 **Metrics** 標籤頁，即可查看圖表。

### 記憶體使用量

顯示服務執行時消耗的記憶體。

![memory](/manage/metric/Memory_Usage.png)

### CPU 使用量

顯示服務執行時調用的 CPU。

![cpu](/manage/metric/CPU_Usage.png)

### 網路使用量

顯示 Zeabur 各區域的服務執行網路流量。

![network](/manage/metric/Network_Usage.png)

### 磁碟使用量

對於 Google Cloud Platform (GCP) 部署，磁碟使用量限制為 1 GB。對於其他所有平台，磁碟使用量採用按量計費模式，可根據你的儲存需求靈活使用。

![disk](/manage/metric/disk_usage.png)
