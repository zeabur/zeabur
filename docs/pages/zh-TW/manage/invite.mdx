# 邀請成員

一個專案可以有兩位或更多成員，專案擁有者可以邀請成員加入專案共同管理服務。

## 如何邀請成員

### 步驟 1：開啟專案設定

首先，開啟你想邀請成員的專案。在專案頁面左上角點擊設定按鈕，即可進入專案設定頁面。

![Project Config](/manage/invite/Invite_team_member_step_1.png)

### 步驟 2：邀請成員

在側邊欄選擇團隊面板，在團隊面板中會看到邀請成員的按鈕。

![Project Settings](/manage/invite/Invite_team_member_step_2.png)

輸入你想邀請的成員的電子郵件地址，然後點擊邀請按鈕。該成員將收到一封加入專案的邀請郵件。

你也可以複製邀請連結並分享給想邀請的成員，對方只需開啟連結即可加入專案。

![Invite Member](/manage/invite/project_team_pages.png)

## 常見問題 (FAQ)

- **Q：可以邀請尚未在 Zeabur 註冊的成員嗎？**  
  A：目前只能邀請已在 Zeabur 註冊的成員。

- **Q：成員加入專案後擁有什麼權限？**  
  A：成員將擁有與專案擁有者相同的權限，但無法刪除專案。

- **Q：可以從專案中移除成員嗎？**  
  A：可以，在專案設定頁面的團隊面板中移除成員。
