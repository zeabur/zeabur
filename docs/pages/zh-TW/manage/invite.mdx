# 邀請成員

一個專案中可能有兩個或多個成員，專案所有者可以邀請成員加入專案來管理服務。

## 如何邀請成員

### 步驟一：打開專案設定

首先，打開您想要邀請成員的專案。單擊專案頁面左上角的配置按鈕，您將被導航到專案設定頁面。

![Project Config](/manage/invite/project-config.png)

### 步驟二：邀請成員

選擇側邊欄中的團隊面板，您將在團隊面板中看到邀請成員按鈕。

![Project Settings](/manage/invite/project-settings.png)

填寫您想要邀請的成員的電子郵件地址，然後單擊邀請按鈕。成員將收到加入專案的電子郵件邀請。

你也可以複製邀請連結並與您想要邀請的成員分享，然後成員只需打開連結即可加入專案。

![Invite Member](/manage/invite/invite.png)

## 常見問題

- **Q: 我能邀請一個未在 zeabur 上註冊的成員嗎？**
    - A: 目前，您只能邀請已在 zeabur 上註冊的成員。

- **Q: 成員加入專案後有哪些權限？**
    - A: 成員將擁有除了刪除專案之外的所有權限。

- **Q: 我可以從專案中刪除成員嗎？**
    - A: 是的，您可以在專案設定頁面的團隊面板中刪除專案中的成員。
