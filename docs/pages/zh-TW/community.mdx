# 社群

Zeabur 是跟在上面部署、分享、實際在跑東西的人一起長出來的。這裡是你接上來的入口。

- [大使計畫](/zh-TW/community/ambassador-programme) — 加入創始團隊：Pro 方案、每月額度、搶先試用，以及一條直通團隊的線
- [社群行為準則](/zh-TW/community/code-of-conduct) — 我們在各個 Zeabur 空間共同遵守的規範
- [Zeabur 支援](https://zeabur.com/support) — 提交支援請求，並向 Zeabur 團隊提供問題細節
