# 社群

Zeabur 是跟在上面部署、分享、實際在跑東西的人一起長出來的。這裡是你接上來的入口。

- [大使計畫](/zh-TW/community/ambassador-programme) — 加入創始團隊：Pro 方案、每月額度、搶先試用，以及一條直通團隊的線
- [社群行為準則](/zh-TW/community/code-of-conduct) — 我們在 Forum、Discord 以及每個 Zeabur 空間共同守的規範
- [社群論壇](/zh-TW/community/forum) — 提問、分享你做的東西、幫其他開發者
