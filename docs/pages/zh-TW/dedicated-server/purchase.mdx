# 從 Zeabur 購買主機

如果你沒有自己的伺服器，可以直接從 Zeabur 購買專用主機。Zeabur 與主要雲端供應商（AWS、GCP、Hetzner、Linode、Digital Ocean、Aliyun、Tencent Cloud、Volcengine）合作，提供具競爭力的價格——通常比直接向供應商購買還便宜。

<Callout type="info">
  購買的主機由 Zeabur 全權代管，你不需要處理 SSH 存取、作業系統更新或基礎設施維護。
</Callout>

## 購買步驟

<Steps>

### 開啟伺服器列表

登入後，點擊左側欄的 **Servers**，或直接造訪 [https://zeabur.com/servers](https://zeabur.com/servers)。

![我的伺服器頁面](/dedicated-server/purchase-servers-page.png)

### 點擊「Create」並選擇「從 Zeabur 購買伺服器」

點擊右上角的 **Create** 按鈕。在彈出的對話框中，選擇**從 Zeabur 購買伺服器**。

![建立伺服器對話框](/dedicated-server/purchase-create-dialog.png)

你還會看到其他選項：

- **固定 IP 伺服器** — 透過 SSH 連接你自己具有固定公網 IP 的伺服器（參見[新增主機](./register)）。
- **[Wonder Mesh](/wonder-mesh)** — 適用於沒有固定公網 IP 的伺服器（目前為 Beta 版）。

### 選擇主機方案

你會被導向主機選擇頁面。使用篩選器縮小選項範圍：

- **Provider** — 從 AWS、GCP、Hetzner、Linode、Digital Ocean、Aliyun、Tencent、Volcengine 中選擇。
- **Region** — 按洲別篩選（亞洲、中國、歐洲、北美、南美、大洋洲）。
- **CPU** — 按 vCPU 數量篩選（1–32）。
- **Memory** — 按記憶體大小篩選（2 GB–64 GB）。
- **Sort** — 按推薦、價格、CPU、RAM 或流量排序。

![選擇主機方案](/dedicated-server/purchase-select-server.png)

每個主機方案會顯示供應商、城市、vCPU 數量、記憶體、儲存空間、出站流量及月費。部分方案會顯示折扣標籤，標示 Zeabur 相較供應商官方價格的常態性折扣。

### 確認並付款

點擊主機方案後進入結帳流程。付款完成後，主機將自動佈建。佈建完成後（通常幾分鐘內），主機將出現在你的**我的伺服器**列表中，並顯示 **Running** 狀態。

</Steps>

## 購買後

當主機上線後，你可以：

- **[建立專案](/zh-TW/deploy/creating-projects)** 在新主機上部署服務。
- **[管理伺服器](./operate)** — 監控資源、設定配置等。
- **[設定防火牆規則](./firewall)** — 控制進出站流量。

## 常見問題

### 購買後可以升級方案嗎？

可以，你可以在伺服器的設定頁面自助升級方案。請注意，升級為單向操作——升級後無法降級至較低方案。

### 如果伺服器不符合需求怎麼辦？

透過 **Tencent Cloud**、**Aliyun** 或 **Volcengine** 購買的伺服器，Zeabur 提供一次免費退款。第二次退款需透過[社群論壇](https://zeabur.com/forum)提交申請。其他供應商（AWS、GCP、Hetzner、Linode、Digital Ocean）的伺服器，可以在伺服器設定頁面點擊**申請退款**直接操作。

### 計費方式為何？

購買的主機按月固定費率計費。選擇頁面上顯示的價格即為月費。主機將自動續約，除非你手動取消。

### 有哪些供應商可選？

Zeabur 目前提供 AWS、GCP、Hetzner、Linode、Digital Ocean、Aliyun、Tencent Cloud 和 Volcengine 的主機，資料中心遍佈亞洲、歐洲、北美、南美和大洋洲。
