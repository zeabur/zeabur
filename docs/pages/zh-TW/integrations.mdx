# 整合服務

Zeabur 提供與第三方服務的內建整合，讓您可以直接在專案中啟用強大的後端功能 — 無需另外部署。

## 可用的整合服務

### [InsForge](/zh-TW/integrations/insforge/overview)

AI 原生的 Supabase 替代方案，讓 AI 代理能夠自主建構和管理全端應用程式。包含 PostgreSQL 資料庫、JWT 身份驗證、S3 相容檔案儲存、無伺服器邊緣函式，以及用於 AI IDE 整合的 MCP 協定支援。

- **類別**：後端 / BaaS
- **計費**：按用量計費，獨立計費
- **網站**：[insforge.dev](https://insforge.dev)
