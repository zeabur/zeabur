# 預備案子域名

根據中國大陸法律法規，所有位於中國大陸境內的伺服器部署的服務必須完成 ICP 備案才可以訪問，這對於某些還處於開發階段的專案來說可能會造成一些困擾。

在傳統的做法中，如果你購買了中國大陸境內的伺服器，在沒有備案的情況下你可以使用 IP + 端口號的方式來訪問服務，但這個做法導致你無法使用網域來訪問服務，也沒辦法簽發 SSL 憑證。

對於這類情況，Zeabur 提供了預備案子域名的功能，讓你可以在完成了實名認證的前提下，暫時使用 Zeabur 提供的已備案網域來訪問你部署的服務，這對於開發階段或是僅用於測試的個人臨時服務來說非常方便，且完全合法合規。

## 使用條件

- 你的伺服器必須是從 Zeabur 購買的。
- 你的伺服器必須在中國大陸境內。
- 你的 Zeabur 帳號必須已經完成了中國大陸身份實名認證。
- 你的伺服器供應商必須已經在 Zeabur 支援預備案子域名：

## 使用方法

1. 選擇你部署在中國大陸境內的 Zeabur 托管伺服器的專案，然後選擇想要綁定預備案子域名的服務。
2. 選擇「網路」標籤頁。
3. 點擊「生成網域」，輸入你想要的子域名。

![](https://cdn.zeabur.com/pre-icp-1.png)

4. 點擊「建立網域」，如果你還沒有完成實名認證，Zeabur 會提示你完成實名認證。
5. 在實名認證視窗輸入你的姓名及身份證號，點擊提交就會自動跳轉到阿里雲的實名認證服務。

![](https://cdn.zeabur.com/pre-icp-2.png)

6. 完成實名認證後，重新點擊「生成網域」，就可以自動生成並綁定已備案的網域。

## 注意事項

- 使用預備案子域名即表示你已經同意對應雲端服務商的使用條款。
- Zeabur 嚴格禁止使用預備案子域名進行任何違反當地法律法規的行為，一旦發現 Zeabur 有權單方面終止服務並將實名資訊移交用戶所在國家/地區的執法機構。
- 不建議使用預備案子域名作為商用服務的正式網域，若測試/開發完成，建議進行正式的備案流程申請您自己的網域，Zeabur 免費提供備案申請的服務及支援。
