# 免費額度說明

Zeabur 為所有使用者提供免費額度，讓你無需付費即可體驗平台功能。

## 額度計算方式

免費額度是一筆**每月重置的虛擬 Credits**，用於抵扣在共享叢集上的資源使用費。額度依方案不同：

| 方案 | 每月免費額度 |
| --- | --- |
| Serverless（Free Plan） | 有限額度（依帳戶狀態） |
| Dev Plan | US$5（包含在月費中） |
| Team Plan | US$5（包含在月費中） |

對於付費方案，「免費額度」即為月費所包含的 Credits。例如 Dev Plan 每月 US$5 的月費已包含 US$5 的 Credits，若當月資源使用費不超過 US$5 則不會額外扣款。

## 適用範圍

免費額度**僅適用於共享叢集的資源使用費**，包括：

- 記憶體使用費
- 網路輸出流量費
- 持久儲存費

### 不適用於免費額度

以下費用**不會**從免費額度中扣除，需額外付款：

- **專用伺服器費用** — 專用伺服器的月租費獨立計算，不受免費額度影響。
- **AI Hub 點數** — AI Hub 使用獨立的點數系統，需單獨儲值。

## 額度用盡後的處理

### Serverless（Free Plan）

當月免費額度用盡後：

1. 所有服務會被**自動暫停**。
2. 你會收到額度用盡的通知。
3. 下個月額度重置後，服務會自動恢復。
4. 如需立即恢復，可以升級到 [Dev Plan](/pricing/dev-plan) 或儲值 Credits。

### 付費方案

當月包含的 Credits 用盡後：

1. 超出部分會從你的**信用卡或儲值餘額**自動扣款。
2. 如果扣款失敗，服務不會立即暫停，但會收到付款提醒。
3. 持續欠費可能導致服務暫停。

<Callout type="info">
你可以在 [Dashboard 帳單頁面](https://dash.zeabur.com/account/billing) 查看當月的額度使用情況。
</Callout>

## 常見問題

### 免費額度可以累積嗎？

不可以。免費額度每月重置，未使用的額度不會累積到下個月。

### 如何查看剩餘額度？

前往 [Dashboard 帳單頁面](https://dash.zeabur.com/account/billing)，可以看到當月的使用量和剩餘額度。
