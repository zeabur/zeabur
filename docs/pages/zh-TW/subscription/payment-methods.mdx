# 付款方式

Zeabur 的付款系統由 [Stripe](https://stripe.com/) 提供支援，支援多種付款方式。

## 支援的付款方式

### 信用卡 / 簽帳卡

Zeabur 接受所有 Stripe 支援的信用卡與簽帳卡，以**美元（USD）**計價：

- Visa
- Mastercard
- American Express
- JCB
- UnionPay（銀聯）

### 支付寶（Alipay）

如果你沒有支援 Stripe 的信用卡，可以使用支付寶進行**儲值**，以人民幣支付。

儲值的金額會轉換為 Zeabur Credits，用於扣抵訂閱費與資源使用費。

## 付款流程

### 訂閱付費方案

1. 前往 [Zeabur 方案頁面](https://zeabur.com/pricing) 或 [Dashboard 帳單頁面](https://dash.zeabur.com/account/billing)。
2. 選擇你要訂閱的方案（[Dev Plan](/pricing/dev-plan) 或 [Team Plan](/pricing/team-plan)）。
3. 輸入信用卡資訊（或使用已綁定的卡片），完成支付。
4. 訂閱成功後，方案立即生效。

### 儲值 Credits

1. 前往 [Dashboard 帳單頁面](https://dash.zeabur.com/account/billing)。
2. 在「儲值」區域點選儲值按鈕。
3. 選擇儲值金額，使用信用卡或支付寶完成付款。
4. Credits 會即時加入你的帳戶餘額。

![儲值 Credits](/billing/subscription/get-more-credits.png)

## 管理付款方式

### 綁定信用卡

在 [帳單頁面](https://dash.zeabur.com/account/billing) 的「支付方式」區域，輸入信用卡資訊後點選「加入」。

### 更換信用卡

1. 在帳單頁面找到「**解除綁定**」按鈕，解除當前信用卡。
2. 重新綁定新的信用卡。

### 解除綁定

如果你已取消所有付費訂閱，可以解除綁定信用卡。

<Callout type="warning">
如果你有進行中的付費訂閱，解除綁定信用卡可能導致下次扣款失敗。請先確認已取消訂閱或改用其他付款方式。
</Callout>

## 常見問題

### 信用卡被拒絕怎麼辦？

- 確認卡片餘額充足。
- 確認卡片支援國際交易（USD）。
- 聯繫發卡行確認是否有交易被攔截。
- 嘗試使用其他信用卡或改用支付寶儲值。

### 可以用 PayPal 付款嗎？

目前不支援 PayPal。請使用信用卡或支付寶。

### 扣款幣別是什麼？

所有費用以美元（USD）計價。信用卡公司可能會收取外幣交易手續費，具體費率請洽發卡行。

如有其他付款問題，請到 [社群論壇](https://zeabur.com/forum) 聯絡我們。
