# 退款政策

## 訂閱方案退款

針對 **開發者方案（Dev Plan）**、**專業方案（Pro Plan）** 或 **團隊方案（Team Plan）**：

- 取消訂閱後，你仍可繼續使用目前方案的功能直到當期帳單周期結束。周期結束後，帳號將自動切換至免費方案。

### 退款方式

你可以選擇以下方式接收退款：

| 退款方式 | 處理時間 | 手續費 |
| --- | --- | --- |
| **Zeabur Credits** | 核准後立即到帳 | 無 |
| **原付款方式退回** | 最多 7 天 | 約退款金額的 30%（Stripe 交易手續費） |

<Callout type="info">
選擇退為 Zeabur Credits 可以避免 Stripe 手續費，且到帳更快。Credits 可用於支付未來的訂閱費或資源使用費。
</Callout>

### 如何取消訂閱

1. 前往 [Dashboard 帳單頁面](https://dash.zeabur.com/account/billing)。
2. 下滑找到「**取消訂閱**」按鈕。
3. 確認取消。

![取消訂閱](/billing/subscription/unsubscribe.png)

<Callout type="warning">
降級到免費方案後，你目前部署的容器化服務將會被系統自動暫停。請在取消前確認已備份重要資料。
</Callout>

## 伺服器退款

伺服器的退款規則依雲端廠商而異：

### 非中國廠商（GCP、Hetzner、Linode、DigitalOcean）

在 Dashboard 的 Server 頁面**自行取消**即可，系統會自動處理退款至 **Zeabur Credits**，無需聯絡客服。

### 中國廠商（火山引擎、阿里雲、騰訊雲）

- **首次退款**：可在 Server 頁面自助取消，無條件退款至 **Zeabur Credits**。
- **第二次及後續退款**：需到 [Zeabur 社群論壇](https://zeabur.com/forum) 提交退款申請，由客服審核處理。

<Callout type="warning">
退款前請務必確認伺服器內所有資料已備份或確認不需保留——**主機銷毀後資料無法復原**。
</Callout>

### 退款注意事項

- 如果選擇退款至原付款方式，處理時間約 **7 個工作天**，退款金額會**扣除約 30% 的 Stripe 手續費**。
- 為避免在下一個計費週期產生費用，可提前關閉自動續約設定。

![關閉自動續約](/billing/pricing/auto-renewal.png)

## 儲值 Credits 退款

已儲值的 Credits 目前**不支援退款**。請依實際需求儲值。
