# 使用插件部署服務

Zeabur 提供多種插件，讓你無需離開開發環境，即可一鍵部署本機程式碼。

本篇文章將介紹如何透過 Zeabur 的 VS Code、Cursor、Chrome 和 Raycast 插件部署服務。

## VS Code / Cursor 插件

你可以在 Visual Studio Code 和 Cursor 編輯器中使用 Zeabur 插件，直接將本地專案部署至 Zeabur。

### 安裝插件

1. 開啟 VS Code 或 Cursor 編輯器。
2. 前往 Extensions 面板，搜尋「Zeabur」。
3. 點擊「安裝」，或直接前往 [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) 安裝。

### 使用方式

1. 開啟要部署的專案資料夾。
2. 點擊側邊欄的 Zeabur 插件圖示。
3. 點選「Deploy」後會自動開啟瀏覽器頁面。
4. 選擇部署地區。
5. 幾分鐘後即可完成部署。

## Chrome 插件

Zeabur 的 Chrome 插件讓你能從支援的 AI 程式碼生成平台（如 Lovable、Bolt.new 或 v0.dev）直接部署程式碼。

### 安裝插件

前往 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem)，點擊「加到 Chrome」。

### 使用方式

在支援的 AI 平台中撰寫程式碼時，畫面上會出現「Deploy with Zeabur」按鈕。點擊即可開始部署流程。

## Raycast 插件

使用 Zeabur 的 Raycast 插件，你可以快速部署專案、搜尋文件或範本模板。

### 安裝插件

開啟 Raycast Store，搜尋 Zeabur 插件並安裝。你也可以直接前往 [Raycast Zeabur 插件頁面](https://www.raycast.com/ridemountainpig/zeabur)，點擊「Install Extension」安裝插件。

### 使用方式

1. 安裝完成後，打開 Raycast 並輸入 `Zeabur`。
2. 選擇「Deploy Project」指令。
3. 選取你想部署的專案資料夾，按下 `⌘ + Enter`。
4. 選擇部署地區，幾分鐘後即可完成部署。
