# 部署按鈕

使用我們的部署按鈕，可一鍵輕鬆將服務部署到 Zeabur。該按鈕適用於任何 Markdown 或 HTML 檔案。它能將任何 [Zeabur 模板](https://zeabur.com/templates) 轉換為可部署的專案。

![button-demo](/deploy/button/button-demo.png)

## 什麼是部署按鈕

部署按鈕是一個簡單的 HTML 程式碼片段。你可複製該片段並貼到任何 Markdown 或 HTML 檔案中。每個按鈕會將使用者重定向至指定的模板。模板可為單一服務或由模板作者決定的服務群組。

按鈕也可附加作者的推薦碼；每位使用推薦碼註冊並升級至付費計劃的新使用者，將為作者帶來 5 美元的信用額度。

## 複製按鈕

按鈕與模板關聯，因此僅模板作者可產生按鈕。

前往 [Zeabur 儀表板](https://zeabur.com/account) 的帳戶頁面，選擇 **Template** 標籤，即可看到你建立的模板清單。

![created-templates](/deploy/button/created-template.png)

選擇所需模板，點擊分享按鈕。將彈出模態框顯示按鈕程式碼。

![popup](/deploy/button/popup.png)

若啟用推薦碼，按鈕程式碼將附加推薦碼（即你在 Zeabur 的使用者名稱）。

## 使用情境

這是向全世界分享你在 Zeabur 上部署的精彩專案的最佳方式。任何人只需一鍵即可部署你的專案。

例如，你可將按鈕貼到以下位置：

- GitHub README 檔案
- 部落格
- 文件與教學
- 社群個人資料

## 要求

需先建立模板才能取得部署按鈕。按照此[說明](https://zeabur.com/docs/zh-TW/template/template-in-code)建立模板。

## 收益

如上所述，每位使用你的推薦碼註冊並升級至付費計劃的新使用者，將為你帶來 5 美元信用額度。這些信用可用於支付你自己的服務費用。

更多詳情見我們的[推薦文件](https://zeabur.com/docs/zh-TW/billing/referral)。
