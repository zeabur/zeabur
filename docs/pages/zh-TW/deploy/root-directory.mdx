# **自訂根目錄**

對於某些 Git 儲存庫（如同時包含前端與後端的專案），需要手動指定專案的根目錄。

### **步驟 1：部署服務**

按照[部署服務](https://zeabur.com/docs/zh-TW/get-started)的指南部署你的應用程式。

## **步驟 2：更改專案根目錄**

點擊已部署的服務，進入設定標籤，向下捲動至 **Root Directory** 欄位。

![Root Directory](/deploy/root-directory/root_directory.png)

在此處設定你的專案根目錄。
