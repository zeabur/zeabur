# 執行命令

你可以在 Zeabur 服務內執行命令，用於安裝套件、執行程式、查看資訊，甚至備份特定資料。

## 開啟命令執行介面

開啟服務的 **Overview** 標籤頁，點擊 **Command** 即可顯示「Execute Command」介面。

![開啟命令執行介面](/deploy/command-execution/Command_part1.png)

命令執行介面：

- 命令輸出區域：顯示命令執行結果。成功時顯示輸出內容，失敗時顯示錯誤碼。

![命令執行介面](/deploy/command-execution/terminal.png)

## 執行命令

以簡單 `echo` 為例，輸入 `echo Hello, World!`，應看到輸出 `Hello, World!`。

![執行 hello world 命令](/deploy/command-execution/terminal_2.png)

輸入不存在的命令會顯示錯誤碼。

![命令執行失敗](/deploy/command-execution/terminal_3.png)

你還可以在 MySQL 或 PostgreSQL 等資料庫容器中執行 SQL 語句、使用 cURL 上傳日誌或備份檔案，甚至執行遷移程式（如 `prisma migrate`、`node seeder.js`）。

對於 MySQL，可複製「Command to connect to your MySQL」命令，刪除 `host` 與 `port` 參數（容器內無需遠端連線），將 `mysqlsh` 改為 `mysql`（容器內無 `mysqlsh`），最後加上 `-e` 執行 SQL 語句。

```bash
mysql --user=root --password=<password> --database=zeabur -e "SHOW DATABASES;"

![執行 MySQL 命令](/deploy/command-execution/MySQL_terminal.png)
