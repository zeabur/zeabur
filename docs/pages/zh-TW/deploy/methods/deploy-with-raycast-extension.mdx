# Raycast 擴充功能

Zeabur 提供多種擴充功能，讓你無需離開開發環境，即可一鍵部署本機程式碼。

本篇文章將介紹如何透過 Zeabur 的 Raycast 擴充功能部署服務。

## Raycast 擴充功能

使用 Zeabur 的 Raycast 擴充功能，你可以快速部署專案、搜尋文件或範本模板。

### 安裝擴充功能

開啟 Raycast Store，搜尋 Zeabur 擴充功能並安裝。你也可以直接前往 [Raycast Zeabur 擴充功能頁面](https://www.raycast.com/ridemountainpig/zeabur)，點擊「Install Extension」安裝。

### 使用方式

1. 安裝完成後，打開 Raycast 並輸入 `Zeabur`。
2. 選擇「Deploy Project」指令。
3. 選取你想部署的專案資料夾，按下 `⌘ + Enter`。
4. 選擇部署地區，幾分鐘後即可完成部署。
