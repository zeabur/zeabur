# VS Code / Cursor 擴充功能

Zeabur 提供多種擴充功能，讓你無需離開開發環境，即可一鍵部署本機程式碼。

本篇文章將介紹如何透過 Zeabur 的 VS Code 擴充功能部署服務。

## VS Code / Cursor 擴充功能

你可以在 Visual Studio Code 和 Cursor 編輯器中使用 Zeabur 擴充功能，直接將本地專案部署至 Zeabur。

### 安裝擴充功能

1. 開啟 VS Code 或 Cursor 編輯器。
2. 前往 Extensions 面板，搜尋「Zeabur」。
3. 點擊「安裝」，或直接前往 [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Zeabur.zeabur-vscode) 安裝。

### 使用方式

1. 開啟要部署的專案資料夾。
2. 點擊側邊欄的 Zeabur 擴充功能圖示。
3. 點選「Deploy」後會自動開啟瀏覽器頁面。
4. 選擇部署地區。
5. 幾分鐘後即可完成部署。
