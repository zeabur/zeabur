# 使用 Chrome 插件部署服務

Zeabur 提供多種插件，讓你無需離開開發環境，即可一鍵部署本機程式碼。

本篇文章將介紹如何透過 Zeabur 的 Chrome 插件部署服務。

## Chrome 插件

Zeabur 的 Chrome 插件讓你能從支援的 AI 程式碼生成平台（如 Lovable、Bolt.new 或 v0.dev）直接部署程式碼。

### 安裝插件

前往 [Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/zeabur/mdcaillllhdkbocjldfclmkcmilkjpem)，點擊「加到 Chrome」。

### 使用方式

在支援的 AI 平台中撰寫程式碼時，畫面上會出現「Deploy with Zeabur」按鈕。點擊即可開始部署流程。
