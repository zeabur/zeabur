# **建立服務**

在[建立專案](https://zeabur.com/docs/zh-TW/deploy/create-project)後，你可以開始部署服務。點擊 **Deploy New Service** 按鈕（或按 `CMD + K` / `Ctrl + K` 並選擇 **Deploy New Service**），你將看到包含各種部署選項的選單：

![Create Service](/deploy/Create-Services/add-service.png)

## **模板**

Zeabur 提供豐富的預配置模板庫，涵蓋熱門開源專案如 WordPress、n8n、Ghost 等。這些模板可一鍵部署。

1. 點擊 **Template** 瀏覽可用模板。
2. 搜尋或瀏覽你需要的模板。
3. 點擊模板並按照提示進行部署。

更多詳情見[從模板部署](https://zeabur.com/docs/zh-TW/template/template-in-code)。

![choose template](/deploy/Create-Services/choose-template.gif)

## **GitHub**

直接從你的 GitHub 儲存庫部署。這是部署自有應用最常見的方式。

1. 點擊 **GitHub** 連結你的 GitHub 帳號（若尚未連結）。
2. 連結後，搜尋要部署的儲存庫。
3. 點擊儲存庫，Zeabur 會自動分析程式碼並選擇最適合的建置方式。
4. 點擊 **Deploy** 開始部署。

![Search repository](https://zeabur.com/docs/_next/image?url=%2Fdocs%2F_next%2Fstatic%2Fmedia%2Fsearch-repository.db02d750.webp&w=3840&q=75)

更多詳情見[與 GitHub 整合](https://zeabur.com/docs/zh-TW/deploy/github)。

## **資料庫**

無需設定即可快速部署常見資料庫。

1. 點擊 **Databases** 查看可用資料庫清單。
2. 選擇需要的資料庫（PostgreSQL、MySQL、MongoDB、Redis 等）。
3. 資料庫將立即部署並可使用。

![Database](/deploy/Create-Services/database.png)

## **函數**

使用 Zeabur Functions 即時撰寫、執行與迭代程式碼。適合快速腳本、API 或 serverless 函數。

1. 點擊 **Functions** 建立新函數。
2. 選擇函數語言為 **JavaScript** 或 **Python**。
3. 在編輯器中直接撰寫程式碼。
4. 函數將部署完成，可透過 dashboard 存取。

![Functions](/deploy/Create-Services/functions.png)

## **本地專案**

無需 Git 儲存庫，直接上傳本地原始碼到 Zeabur。

1. 點擊 **Local Project**。
2. 拖放專案資料夾，或點擊瀏覽選擇資料夾。
3. Zeabur 將分析並部署你的專案。

![Local Project](/deploy/Create-Services/local-project.png)

適用於快速部署或不想先推送程式碼到儲存庫的情況。

上傳時 `/dist/*` 目錄將自動被忽略。

## **Docker 映像**

從 Docker Hub 或其他容器登錄檔部署任何預建置的 Docker 容器映像。

1. 點擊 **Docker Image**。
2. 輸入映像名稱（例如 `nginx:latest` 或 `ghcr.io/owner/image:tag`）。
3. 配置所需環境變數或連接埠。
4. 點擊 **Deploy**。

更多詳情見[使用 Docker 映像部署](https://zeabur.com/docs/zh-TW/deploy/customize-prebuilt)。

## **Cursor**

使用 Zeabur 擴充功能從 Cursor IDE 直接部署。

1. 點擊 **Cursor** 以在 Cursor IDE 中開啟你的專案。
2. 若未安裝 Zeabur 擴充功能，將提示安裝。
3. 使用擴充功能直接從 Cursor 部署目前專案。

為主要使用 Cursor 作為 IDE 的開發者提供無縫體驗。
