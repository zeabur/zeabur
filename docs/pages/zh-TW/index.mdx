**開始使用**

**Zeabur** 是一個能幫助你一鍵部署服務的平台，無論你使用什麼程式語言或開發框架，都能輕鬆完成部署。

你可以加入我們的 [Zeabur 論壇](https://zeabur.com/forum) 了解更多並獲得優先支援！

你也可以加入我們的 [官方 Discord 伺服器](https://zeabur.com/dc) 與社群交流想法並獲得協助。

### **Zeabur 的優勢**

### **方便快捷**

相較於傳統部署方式，Zeabur 讓你不必花時間學習複雜的部署知識，就能在幾分鐘內完成部署，把時間與精力專注於產品開發。

- AI Agent 自動識別專案類型
- 多家廠商、多地區的共享叢集 / 專用伺服器
- 友善直觀的 UI
- 開箱即用的 CI/CD
- 自動產生網域 / 支援自帶網域（BYOD）
- 多環境隔離
- Zeabur AI Hub
- 以及更多

### **一站式整合**

Zeabur 提供強大的一站式整合能力。

你可以在同一個專案中部署前端、後端、資料庫、訊息佇列、物件儲存等各類服務，無需分別部署到不同平台。

### 一鍵部署模板

Zeabur 在 [模板市集](https://zeabur.com/templates) 提供了大量模板。只要點擊 `Deploy` 按鈕，模板就會立即開始部署。幾分鐘內，無論是部署資料庫還是像 n8n 這樣的應用，都能準備就緒並開始使用。

更多資訊可參考 [Zeabur 模板頁面](https://zeabur.com/templates)。

### Zeabur AI Hub

Zeabur AI Hub 是一個統一平台，旨在讓開發者更容易使用強大的 AI 模型。你不需要為不同供應商分別管理帳號與 API Key，就能透過單一介面存取業界最受歡迎的模型。

- **統一存取**：在同一個平台中使用 OpenAI（GPT 系列）、Anthropic（Claude 系列）、Google（Gemini）、xAI（Grok）以及開源模型（Llama、DeepSeek、Qwen）等主流模型。
- **單一 API Key**：使用一個簡單的 API Key 即可呼叫任何支援的模型，省去管理多組憑證的複雜度。
- **相容 OpenAI**：API 與 OpenAI API 格式完全相容，可輕鬆整合至既有應用或工具（如 n8n、SillyTavern）中，無需更改程式碼。
- **全球低延遲**：在東京、舊金山等多個地區部署 API Endpoint，可選擇最近的伺服器以獲得最佳效能。

你可以在 [Zeabur AI Hub 頁面](https://zeabur.com/ai-hub) 了解更多並開始使用。

### **按量計費**

Zeabur 採用按量計費模式，代表你只需要為服務實際使用到的資源付費，而不是為整台伺服器付費。

你可以在[定價文件](https://zeabur.com/docs/zh-TW/billing/pricing)中了解更詳細的內容。
