# 基本介紹

**Zeabur** 是一個可以幫助你部署服務的平台，而且只需要透過幾個簡單的按鈕即可完成，無論你使用什麼程式語言或開發框架。

你也可以加入我們的 [社群論壇](https://zeabur.com/forum) 以了解更多以及獲得優先支援！

### Zeabur 的優勢

#### 方便快捷

對比傳統的部署方式，Zeabur 可以讓你在幾分鐘內完成部署，而不需要花費大量的時間去學習部署的相關知識，把時間精力專注於產品的開發。

- 開箱即用的 CI/CD
- 自動識別專案類型
- 快速生成網域
- 多環境隔離

#### 價格固定可預期，成本可控

Zeabur 提供固定價格方案，讓你的每月支出從一開始就清晰透明。無論是副業專案還是正式上線的服務，你都能精確掌握每月費用——沒有意外帳單、沒有隱藏費用，也沒有不可預期的用量波動。

這種可預期的計費方式讓你更容易編列基礎建設預算，把心力專注在開發上，而不是估算花費。你可以到[定價文件](/zh-TW/billing/pricing)中了解更詳細的內容。

#### 一站式整合

Zeabur 提供了強大的一站式整合功能。

你可以在一個專案中同時部署前端、後端、資料庫、消息佇列、物件儲存等各種服務，而不需要分別部署到不同的平台。
