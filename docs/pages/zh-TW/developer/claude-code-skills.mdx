# Claude Code Skills

Zeabur 官方 Claude Code 外掛將自然語言轉化為基礎設施操作。部署服務、租用伺服器、管理設定——全部不需要離開你的編輯器。

<Callout type="info">
  Claude Code Skills 需要 [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) 環境。請確認你已安裝並可正常使用 Claude Code。
</Callout>

## 安裝

<Steps>

### 新增 Zeabur 外掛到本機登錄檔

從 Claude Code 市集新增 Zeabur 外掛：

```bash copy
claude plugin marketplace add zeabur/agent-skills
```

### 安裝並啟用 Skills

在 Claude Code 中安裝並啟用所有 Zeabur skills：

```bash copy
claude plugin install zeabur@zeabur
```

</Steps>

安裝完成後，你就可以在 Claude Code 中透過自然語言操作 Zeabur 的所有功能。

## 支援的 Skills

### 租用 VPS

瀏覽並租用 AWS、Hetzner、Linode 等供應商的伺服器。

> 「幫我找一台 2 核 4GB 的便宜 VPS」

### 部署服務

將本機專案或模板部署到伺服器，自動偵測框架與建置設定。

> 「把這個專案部署到我的 Hetzner 伺服器上」

### 啟用 AI 模型

透過 Zeabur AI Hub 存取 GPT-4o、Claude、Gemini 等模型。

> 「我的 App 需要 AI 功能，幫我設定 API」

### 設定 Email

使用 Zeabur Email 為應用程式新增寄信功能，自動設定 SMTP。

> 「幫我的 App 加上寄信功能」

### 設定環境變數

建立、更新或連結跨服務的環境變數。

> 「幫我的後端服務設定 DATABASE_URL」

### 修改設定檔

編輯 nginx 設定、runtime 參數、容器組態等設定檔。

> 「更新 nginx 設定，啟用 gzip 壓縮」

### 設定端口

對應並開放服務端口，解決端口不匹配的問題。

> 「我的 App 監聽 8080 端口，幫我修正 proxy 設定」

### 綁定網域

綁定自訂網域或 `.zeabur.app` 免費子網域，自動設定 SSL 憑證。

> 「把 myapp.dev 綁定到這個服務」

### 設定健康檢查

配置健康檢查端點，當服務不健康時自動重啟。

> 「在 /api/health 設定健康檢查」

### 設定防火牆

控制伺服器的進站與出站流量規則。

> 「只允許我辦公室 IP 的流量」

### 設定容器鏡像

使用自訂 Docker Image 部署服務，或切換基礎鏡像。

> 「用我的私有 Docker Image 部署」

### 購買網域

搜尋、註冊並設定自訂網域。

> 「幫我買 myapp.dev 並指向我的 App」

### 管理資料庫

部署、查看、執行查詢、填入測試資料、處理資料庫遷移。

> 「資料庫裡面有什麼？幫我新增一個測試使用者」

### 除錯與排障

進入容器、讀取日誌、診斷並修復問題。

> 「我的 App 一直回傳 500 錯誤，幫我查」
