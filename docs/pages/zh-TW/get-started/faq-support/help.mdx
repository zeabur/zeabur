# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以聯繫 [Zeabur 支援](https://zeabur.com/support) 來取得團隊協助。

![Zeabur 支援頁面](/community/help/support-home.png)

## 如何得到幫助

開啟 [Zeabur 支援](https://zeabur.com/support) 並提交你的請求，請提供足夠的背景資訊，讓我們的團隊能夠調查問題。

![提交工單表單](/community/help/support-submit-ticket.png)

### 請求格式

遵循以下格式來描述你的問題可以幫助我們更快地定位和解決你的問題。

- 在標題簡單描述你的問題
- 描述問題細節、預期行為，以及你已經嘗試過的處理方式
- 如果問題與特定專案或服務相關，請提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在你的 Zeabur 控制台的 URL 中找到它們。

### 優先支援

優先支援會依照你的方案提供。你也可以通過驗證帳戶，幫助我們更快識別你的帳戶。請參考[這篇文件](./verify)來進行驗證。

## 其他聯絡方式

你也可以透過電子郵件聯繫我們以獲得支援或詢問：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為了獲得更快的回應時間，我們建議使用 [Zeabur 支援](https://zeabur.com/support)。不過，如果你偏好使用電子郵件溝通，歡迎隨時發送訊息給我們，我們會盡快回覆。
