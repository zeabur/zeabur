# 尋求幫助

有些時候你可能會在使用 Zeabur 的過程中遇到一些問題，你可以在 [Zeabur 社群論壇](https://zeabur.com/forum) 中尋求幫助。

## 如何得到幫助

在社群論壇中，你可以發布問題來獲得團隊和社群成員的協助。

### 發文格式

遵循以下格式來描述你的問題可以幫助我們更快地定位和解決你的問題。

- 在標題簡單描述你的問題
- 在內容區塊中描述你的問題細節
- 在文中提供你的 `projectID`、`serviceID` 或 `deploymentID`。你可以在你的 Zeabur 控制台的 URL 中找到它們。

### 優先支援

你可以通過驗證你的帳戶來獲得優先支援。請參考[這篇文件](./verify)來進行驗證。

## 其他聯絡方式

你也可以透過電子郵件聯繫我們以獲得支援或詢問：

- **電子郵件：** [contact@zeabur.com](mailto:contact@zeabur.com)

為了獲得更快的回應時間和存取我們的社群資源，我們建議使用社群論壇。不過，如果你偏好使用電子郵件溝通，歡迎隨時發送訊息給我們，我們會盡快回覆。
