# 平台概覽

**Zeabur** 是一個 AI 驅動的雲端部署平台，讓你在幾分鐘內就能把應用程式上線，不需要管理伺服器、設定 CI/CD 或處理基礎設施。

## 核心能力

### 自動化部署

- **自動偵測語言和框架** — 推送程式碼後，Zeabur 自動識別 Node.js、Python、Go、Java、.NET、Rust 等語言，並選擇最佳的建置方式。
- **開箱即用的 CI/CD** — 每次 `git push` 都會自動觸發建置和部署，零設定。
- **模板市集** — 上百個預設模板，一鍵部署 WordPress、n8n、PostgreSQL 等常用服務。

### AI 助理

Zeabur 內建 AI 助理（Agent），你可以用自然語言描述需求，它會自動幫你建立專案、部署服務、設定環境變數和綁定網域。

### AI Hub

透過 [AI Hub](/ai-hub)，你可以直接呼叫 GPT-4o、Claude、Gemini 等主流大型語言模型的 API，使用 OpenAI 相容格式，無需各自申請帳號。

### 一站式整合

在一個專案中同時部署前端、後端、資料庫、消息佇列、物件儲存等服務，不需要分散到不同平台。服務之間可以透過私有網路互聯，環境變數也能互相引用。

### 多環境隔離

每個專案可以建立多個環境（如 Production、Staging），各環境互相隔離，方便開發和測試流程。

### 彈性基礎設施

- **共用叢集** — 適合快速開始，無需管理伺服器。
- **專用伺服器** — 帶入你自己的機器（BYO Server），享有完整控制權。詳見[專用伺服器](/dedicated-server)。
