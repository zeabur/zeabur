# 定價

Zeabur 提供四個訂閱方案，以及可自訂的 Enterprise 選項：

- **[Free Plan](/pricing/free-plan)** — US$0/月
- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 天免費試用）
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 天免費試用）
- **[Team Plan](/pricing/team-plan)** — US$79/月（含 3 個席位，每增加一個席位 US$24）
- **Enterprise** — [聯絡我們](https://zeabur.com/forum)

如需完整功能比較和退款政策，請參閱[定價方案](/pricing/pricing-plans)。如需最新定價，請前往 [zeabur.com/pricing](https://zeabur.com/pricing)。
