# Pro Plan

Pro Plan 是 Zeabur 推薦用於正式環境的方案——每月 **US$19**，並提供 **14 天免費試用**。在 Dev Plan 的基礎上新增了**日誌轉發**、**更大的 CI 規格**、**更長的日誌保留時間**、**更高的 Email / 網域 / API 金鑰 / Webhook 配額**、**自訂服務圖示**以及**優先支援**。

完整功能清單與最新配額請參閱[方案比較表](/pricing)。

## 使用情境

- **小型正式環境部署** — 需要更長日誌保留時間、更大 CI 編譯規格及直接支援管道的服務。
- **高交易量電子郵件應用** — Pro Plan 的每月 Email 配額足以應對大多數早期 SaaS 和電商應用。
- **重視維運的工作負載** — 將日誌轉發至現有的可觀測性平台（如 Datadog、Grafana 等），讓 Pro 方案的服務與您的集中監控系統整合。
- **多網域前端應用** — Pro 的網域與 API 金鑰配額能涵蓋大多數多環境部署。

## 何時升級至 Team

若您需要**多位成員協同管理同一專案**、**角色與權限管理**、**發票與統一編號支援**，或**無限制的外部伺服器綁定**，下一步可考慮升級至 [Team Plan](/pricing/team-plan)（每月 US$79，含 3 個席位，每增加一個席位 US$24/月）。

<Callout type="info">
若需要大規模自訂方案、專屬支援及 SLA，請參閱 **Enterprise Plan** — [聯絡我們](https://zeabur.com/forum)了解詳情。
</Callout>
