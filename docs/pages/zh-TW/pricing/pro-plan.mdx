# Pro Plan（專業方案）

Pro Plan 是 Zeabur 為需要 **更高建置規格、更長日誌保留與強化可觀測性** 的開發者設計的方案，每月 **US$19**，在 Dev Plan 之上新增生產級能力。

<Callout type="info">
支援 **14 天免費試用** — 功能與付費版本完全一致，試用結束前不會收費。詳見 [14 天免費試用](/billing/subscription#14-天免費試用)。
</Callout>

## 方案內容

| 項目 | 說明 |
| --- | --- |
| 月費 | US$19 / 月 |
| 綁定外部伺服器 | 最多 10 台 |
| 建置 CI 規格 | 4C8G |
| 日誌保留 | 30 天 |
| 進階日誌搜尋 | 支援 |
| 日誌轉發（Log Drain） | 支援 |
| 自訂監控與告警 | 支援 |
| 檔案上傳限制 | 高 |
| 容器檔案管理 | 單檔上傳 |
| HA 服務部署 | 支援 |
| 網域與 IP 存取控管 | 支援 |
| 成員數 | 僅限 1 人（帳號擁有者） |

## 相較 Dev Plan 新增

包含 Dev Plan 所有功能，並額外提供：

- 最多 **10 台綁定外部伺服器**（Dev：3 台）
- **建置 CI 4C8G**（Dev：2C4G）— 更快、更強的建置能力
- 日誌保留 **30 天**（Dev：7 天）
- **進階日誌搜尋** — 篩選與查詢歷史日誌
- **日誌轉發（Log Drain）** — 將日誌導向第三方系統
- **高可用服務部署**
- **網域與 IP 存取控管** — 限制誰可以存取你的服務
- 更高的檔案上傳限制

## 適用場景

- **有流量的生產服務** — 更高 CI、HA 部署與更長日誌保留，保持生產環境穩定。
- **需要可觀測性的一人團隊** — 進階日誌搜尋、日誌轉發與自訂告警，讓除錯更輕鬆。
- **安全敏感服務** — 網域與 IP 存取控管，限制誰可以連到你的端點。
- **長時間建置** — 4C8G 建置 CI 更快，特別適合 monorepo。
