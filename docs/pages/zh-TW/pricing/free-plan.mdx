# Free Plan

Free Plan 是 Zeabur 的入門方案，適合首次嘗試 Zeabur 部署流程的使用者。無需綁定信用卡，註冊即可開始部署。

## 包含資源

| 資源 | 限制 |
| --- | --- |
| 月費 | $0 |
| 綁定外部伺服器 | 1 台 |
| 建置 CI 規格 | 2C4G |
| 日誌保留 | 48 小時 |
| 檔案上傳限制 | 低 |
| 容器檔案管理 | 單檔上傳 |
| 資料庫檢視 | 支援 |
| AI Agent（Zeabur AI） | 不包含 |
| 部署區域 | 共享叢集區域 |

## 資源費率

共享叢集按分鐘計費，與付費方案使用相同費率：

- **vCPU** — 不計費（$0.0 / vCPU-分鐘）
- **記憶體** — US$0.00025 / GB-分鐘（約 US$10.8 / GB-月）
- **網路輸出** — US$0.1 / GB
- **持久儲存** — US$0.2 / GB-月

完整說明請見 [Billing / Pricing](/billing/pricing)。

## 限制與注意事項

- **無 Zeabur AI / AI Agent** — AI 驅動的維運體驗僅在付費方案提供。
- **日誌保留較短** — Free 僅保留 48 小時。
- **綁定伺服器有限** — Free 最多只能綁定 1 台外部伺服器。
- **無 SLA 保障** — 免費方案不提供服務等級協議。
- **僅社群支援** — 優先技術支援需 Team 方案。

## 升級路徑

當你的需求超過 Free Plan 的範圍，可以升級到付費方案：

- **[Dev Plan](/pricing/dev-plan)** — US$5 / 月。需要 AI Agent、資料庫備份、網域管理與郵件系統的個人開發者。
- **[Pro Plan](/pricing/pro-plan)** — US$19 / 月。更高建置規格、30 天日誌保留、進階可觀測性、HA 部署與 IP 存取控管。
- **[Team Plan](/pricing/team-plan)** — US$79 / 月（含 3 席）。無限綁定伺服器、團隊協作、角色管理與優先支援。
- **Enterprise** — 客製化合約、專屬資源、合規支援，詳情請 [聯絡我們](https://zeabur.com/forum)。

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 比較各方案的完整功能。
