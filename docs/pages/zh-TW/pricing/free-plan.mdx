# Free Plan

Free Plan 是 Zeabur 的免費方案，專為個人探索與學習而設計。無需信用卡——只需註冊即可開始部署。

## 方案詳情

| 項目 | Free Plan |
| --- | --- |
| 月費 | US$0 |
| 可綁定外部伺服器數量 | 1 |
| 編譯 CI 規格 | 2C4G |
| 日誌保留時間 | 48 小時 |
| 容器檔案管理 | 單檔上傳 |
| 資料庫查看 | 支援 |
| 持久儲存空間 | 支援 |
| 自訂網域綁定 | 支援 |
| 網域購買（註冊） | 不支援 |
| 資料庫備份（匯出 / 匯入） | 不支援 |
| 日誌轉發 | 不支援 |
| Zeabur AI | 不支援 |
| Zeabur Email — 每日配額 | 0 |
| Zeabur Email — 每月配額 | 0 |
| 網域 | 0 |
| API 金鑰 | 0 |
| Webhooks | 0 |
| 論壇發文 | 限制（基本） |
| 優先支援 | 不包含 |
| SLA | 不包含 |

## 限制

- **自動休眠** — Free Plan 上的服務在一段時間無活動後會自動進入休眠。下次有請求進來時會自動喚醒，但可能會有幾秒鐘的冷啟動延遲。
- **無 SLA** — Free Plan 不包含服務等級協議。
- **無 Zeabur Email** — Free Plan 不支援透過 [Zeabur Email](/email) 寄送郵件。
- **無自動資料庫備份 / 匯入及日誌轉發** — 這些進階維運功能從 Dev Plan 起才提供。

<Callout type="info">
Free Plan 上的服務在閒置時會自動休眠以節省資源。若需要服務持續運行，請考慮升級至 [Dev Plan](/pricing/dev-plan)。
</Callout>

## 升級路徑

當您的需求超出 Free Plan 時，可升級至付費方案：

- **[Dev Plan](/pricing/dev-plan)** — US$5/月（14 天免費試用），適合有個人正式專案的獨立開發者。新增 Zeabur AI、資料庫備份、網域購買及 Zeabur Email。
- **[Pro Plan](/pricing/pro-plan)** — US$19/月（14 天免費試用），推薦用於正式環境。新增日誌轉發、進階配額、自訂服務圖示及優先支援。
- **[Team Plan](/pricing/team-plan)** — US$79/月（含 3 個席位，每增加一個席位 US$24），專為需要角色與權限管理的團隊協作而設計。
- **Enterprise** — 自訂方案，提供專屬支援及 SLA——[聯絡我們](https://zeabur.com/forum)了解詳情。

前往 [zeabur.com/pricing](https://zeabur.com/pricing) 比較所有方案。
