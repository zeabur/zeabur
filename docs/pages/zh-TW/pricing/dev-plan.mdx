# Dev Plan (Developer Plan)

Dev Plan 是 Zeabur 的入門付費方案，專為獨立開發者設計。每月 **US$5**，並提供 **14 天免費試用**，解鎖 Zeabur AI、自動資料庫備份、網域購買及 Zeabur Email 服務——非常適合個人正式專案和 side project。

## 適用情境

- **個人 side project** — 部署個人網站、部落格、API 服務等。
- **開發與測試** — 需要持久儲存空間（如資料庫）和自動備份的環境。
- **小型正式服務** — 流量不大但需要 24/7 不間斷運行的服務。
- **模板開發** — 建立和維護 Zeabur 模板，搭配[一鍵部署按鈕](/deploy/deploy-button)推廣您的專案。

## 何時升級至 Pro Plan

若您需要**日誌轉發**、**更高的 Email / 網域 / API 金鑰 / Webhook 配額**、**自訂服務圖示**或**優先支援**，下一步可考慮升級至 [Pro Plan](/pricing/pro-plan)（US$19/月，14 天免費試用）。詳見 [Pro Plan](/pricing/pro-plan) 頁面。
