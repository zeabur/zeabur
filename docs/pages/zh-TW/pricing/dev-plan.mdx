# Dev Plan（開發者方案）

開發者方案（Developer Plan）是 Zeabur 針對個人開發者設計的付費方案，每月 **US$5**，解鎖完整的部署功能，適合個人正式專案與 side project。

## 方案內容

| 項目 | 說明 |
| --- | --- |
| 月費 | US$5 / 月 |
| 包含額度 | US$5 / 月（可抵扣資源費用） |
| 專案數量 | 無限制 |
| 服務數量 | 無限制 |
| 自訂網域 | 支援 |
| 持久儲存 | 支援 |
| 部署區域 | 所有可用區域 |
| 成員數 | 僅限 1 人（帳號擁有者） |

### 費率

Dev Plan 的資源費率與 [方案總覽](/pricing/pricing-plans) 中列出的費率一致：

- **vCPU** — 不計費
- **記憶體** — US$0.00025 / GB-分鐘（約 US$10.8 / GB-月）
- **網路輸出** — US$0.1 / GB
- **持久儲存** — US$0.2 / GB-月

每月包含的 US$5 額度會自動抵扣資源使用費。超出額度的部分會從你的信用卡或儲值餘額扣款。

## 與 Free Plan 的差異

| 功能 | Free Plan | Dev Plan |
| --- | --- | --- |
| 月費 | 免費 | US$5 |
| 專案數量 | 有限制 | 無限制 |
| 服務數量 | 有限制 | 無限制 |
| 自訂網域 | 不支援 | 支援 |
| 持久儲存 | 不支援 | 支援 |
| 服務休眠 | 無流量自動暫停 | 持續運行 |
| SLA | 無 | — |

## 適用場景

- **個人 Side Project** — 部署個人網站、Blog、API 服務等。
- **開發與測試** — 需要持久儲存（如資料庫）的開發環境。
- **小型生產服務** — 流量不大但需要 24/7 運行的服務。
- **模板開發** — 建立和維護 Zeabur 模板，搭配[一鍵部署按鈕](/deploy/deploy-button)推廣專案。
