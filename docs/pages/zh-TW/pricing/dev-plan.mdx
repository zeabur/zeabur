# Dev Plan（開發者方案）

Dev Plan 是 Zeabur 為需要 AI Agent、資料庫備份與網域管理的個人開發者設計的付費方案，每月 **US$5**，解鎖個人開發者最需要的生產力功能。

<Callout type="info">
支援 **14 天免費試用** — 功能與付費版本完全一致，試用結束前不會收費。詳見 [14 天免費試用](/billing/subscription#14-天免費試用)。
</Callout>

## 方案內容

| 項目 | 說明 |
| --- | --- |
| 月費 | US$5 / 月 |
| 綁定外部伺服器 | 最多 3 台 |
| 建置 CI 規格 | 2C4G |
| 日誌保留 | 7 天 |
| 檔案上傳限制 | 中 |
| 容器檔案管理 | 單檔上傳 |
| Zeabur AI（AI Agent） | 包含 |
| 網域購買 | 支援 |
| 郵件系統 | 包含 |
| 資料庫備份匯出 | 支援 |
| 成員數 | 僅限 1 人（帳號擁有者） |
| 部署區域 | 所有可用區域 |

## 相較 Free Plan 新增

包含 Free Plan 所有功能，並額外提供：

- 最多 **3 台綁定外部伺服器**（Free：1 台）
- 日誌保留 **7 天**（Free：48 小時）
- **Zeabur AI** — 管理服務的 AI Agent
- **透過 Zeabur 購買網域**
- **郵件系統** — 發送交易郵件
- 更高的檔案上傳限制

## 資源費率

共享叢集按分鐘計費。完整費率請見 [Billing / Pricing](/billing/pricing)。

- **vCPU** — 不計費
- **記憶體** — US$0.00025 / GB-分鐘（約 US$10.8 / GB-月）
- **網路輸出** — US$0.1 / GB
- **持久儲存** — US$0.2 / GB-月

| 功能 | Free Plan | Dev Plan |
| --- | --- | --- |
| 月費 | $0 | US$5 |
| 綁定外部伺服器 | 1 台 | 3 台 |
| 日誌保留 | 48 小時 | 7 天 |
| Zeabur AI | — | 包含 |
| 網域購買 | — | 支援 |
| 郵件系統 | — | 包含 |
| 檔案上傳限制 | 低 | 中 |

## 適用場景

- **個人 Side Project** — 部署個人網站、Blog、API 服務等。
- **AI 輔助工作流** — 讓 Zeabur AI 協助管理與排錯。
- **小型生產服務** — 流量不大但需要 24/7 運行的服務。
- **模板開發** — 建立和維護 Zeabur 模板，搭配 [一鍵部署按鈕](/deploy/deploy-button) 推廣專案。
