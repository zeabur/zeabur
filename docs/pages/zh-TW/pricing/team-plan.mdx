# Team Plan（團隊方案）

團隊方案（Team Plan）是 Zeabur 為團隊協作設計的方案，支援多成員管理和統一帳單，適合需要多人共同管理專案的團隊。

## 方案內容

| 項目 | 說明 |
| --- | --- |
| 計費方式 | 依用量計費 |
| 專案數量 | 無限制 |
| 服務數量 | 無限制 |
| 自訂網域 | 支援 |
| 持久儲存 | 支援 |
| 部署區域 | 所有可用區域 |
| 成員數 | 多人（依 seat 數） |
| 發票 / 統編 | 支援 |

### Seat 機制

Team Plan 支援多成員協作。團隊成員數依 seat 計算，具體費率請參考 [zeabur.com/pricing](https://zeabur.com/pricing)。

### 費率

Team Plan 的資源費率與 Dev Plan 相同，請參考[方案總覽](/pricing/pricing-plans)。

## 發票與統編

Team Plan 支援開立發票及設定統一編號，適合需要報帳的企業和組織。詳細說明請參考[發票與統編](/subscription/invoicing)。

## 與 Dev Plan 的差異

| 功能 | Dev Plan | Team Plan |
| --- | --- | --- |
| 計費方式 | US$5/月 + 用量 | 依用量計費 |
| 成員數 | 1 人 | 多人（依 seat） |
| 成員權限管理 | 不適用 | 支援 |
| 發票 / 統編 | 不支援 | 支援 |
| 適用對象 | 個人開發者 | 團隊 / 企業 |

<Callout type="info">
如需更大規模的客製化方案、專屬支援與 SLA，請參考 **Enterprise 方案**，詳情請[聯絡我們](https://zeabur.com/dc)。
</Callout>

## 適用場景

- **新創團隊** — 多位開發者共同管理部署環境。
- **企業內部專案** — 需要統一帳單和發票的團隊。
- **接案公司** — 為不同客戶建立獨立專案，團隊統一管理。
- **開源社群** — 多位維護者協同管理服務部署。
