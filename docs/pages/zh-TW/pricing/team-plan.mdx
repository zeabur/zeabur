# Team Plan（團隊方案）

Team Plan 是 Zeabur 為需要 **協作、無限綁定伺服器與優先支援** 的小團隊設計的方案。起價 **US$79 / 月**，內含 3 個席位，每額外席位 **US$24 / 月**。

<Callout type="warning">
Team 方案**不**包含免費試用。若想先評估平台，建議先試用 Dev 或 Pro 方案。
</Callout>

### 席位機制

Team Plan 支援多成員協作，每個席位皆擁有 Pro 等級功能。基礎訂閱包含 **3 席**，額外席位為每席每月 **US$24**。

範例：5 人團隊支付 `$79 + 2 × $24 = $127 / 月`。

## 角色與權限

Team Plan 的席位分為 **Viewer**、**Editor**、**Admin** 三種角色。依據每位成員實際需要的權限分配角色 — 從純粹的唯讀檢視、日常開發操作，到完整的管理控制。

| 具體功能 | Viewer | Editor（即將推出） | Admin |
| --- | :---: | :---: | :---: |
| 查看 Team 成員 | ✅ | ✅ | ✅ |
| 邀請新成員 | ❌ | ❌ | ✅ |
| 修改成員角色 | ❌ | ❌ | ✅ |
| 移除成員 | ❌ | ❌ | ✅ |

| 查看 Project / Service | ✅ | ✅ | ✅ |
| 建立 Project / Service | ❌ | ✅ | ✅ |
| 修改 Service 設定 | ❌ | ✅ | ✅ |
| 刪除 Project / Service | ❌ | ❌ | ✅ |
| 查看部署紀錄 | ✅ | ✅ | ✅ |
| 手動觸發 Deploy | ❌ | ✅ | ✅ |
| Rollback 部署 | ❌ | ✅ | ✅ |
| 查看 Logs | ✅ | ✅ | ✅ |
| 搜尋 Logs | ❌ | ✅ | ✅ |
| 下載 Logs | ❌ | ✅ | ✅ |
| 查看 Database / Server | ✅ | ✅ | ✅ |

| 修改付款方式 | ❌ | ❌ | ✅ |
| 修改訂閱方案 | ❌ | ❌ | ✅ |

- **Viewer** — 對團隊資源的唯讀權限。適合利害關係人、稽核人員或新進同事的上手期。
- **Editor** _（即將推出）_ — 可建置與部署：建立／修改 Project、觸發部署、Rollback、搜尋與下載 Logs。但無法刪除資源，也不能管理成員或帳務。
- **Admin** — 完整的管理權限，包含成員管理、破壞性操作與帳務。任何影響 Team 本身或訂閱的操作都需要 Admin 角色。

## 相較 Pro Plan 新增

包含 Pro Plan 所有功能，並額外提供：

- **無限** 綁定外部伺服器（Pro：10 台）
- 日誌保留 **90 天**（Pro：30 天）
- **內含 3 個席位**（每席 Pro 等級）
- **團隊專案協作**
- **角色與權限管理**
- **優先客服與技術支援**
- 容器**批次檔案操作**
- **企業採購與發票開立**

## 發票與統編

Team Plan 支援開立正式發票及設定統一編號，適合需要報帳的企業和組織。詳細說明請參考 [發票與統編](/subscription/invoicing)。

## 適用場景

- **新創團隊** — 多位開發者共同管理部署環境。
- **企業內部專案** — 需要統一帳單和正式發票的團隊。
- **接案公司** — 為不同客戶建立獨立專案，團隊統一管理。
- **開源社群** — 多位維護者協同管理服務部署。
