# 方案

Zeabur 提供四個訂閱方案，以及可自訂的 Enterprise 選項：

- **[Free Plan](/pricing/free-plan)** — US$0/月，適合個人探索。
- **[Dev Plan](/pricing/dev-plan)** — US$5/月，14 天免費試用。適合獨立開發者。
- **[Pro Plan](/pricing/pro-plan)** — US$19/月，14 天免費試用。推薦用於正式環境。
- **[Team Plan](/pricing/team-plan)** — US$79/月（含 3 個席位，每增加一個席位 US$24）。適合團隊使用。
- **Enterprise Plan** — [聯絡我們](mailto:contact@zeabur.com)取得專屬於您使用情境的方案。

如需完整功能比較和退款政策，請參閱[定價方案](/pricing/pricing-plans)。如需最新定價，請前往 [zeabur.com/pricing](https://zeabur.com/pricing)。
