# 網域註冊

import { Callout } from 'nextra/components'

網域註冊服務讓你在 Zeabur 控制台中直接註冊、管理和設定網域，無需第三方註冊商。

## 功能

- **網域搜尋** — 同時查詢 9 個後綴（`.com`、`.net`、`.org`、`.io`、`.dev`、`.app`、`.co`、`.me`、`.xyz`）的可用性和即時價格
- **DNS 管理** — 完整的 DNS 編輯器，支援 A、AAAA、CNAME、MX、TXT、SRV、CAA、NS 記錄及 Cloudflare 代理
- **自動和手動續費** — 為每個網域開關自動續費，或隨時手動續費
- **註冊人資料** — 儲存 WHOIS 聯絡資訊，在多次註冊中重複使用
- **註冊人驗證** — 處理部分後綴購買後要求的信件驗證
- **Zeabur Email 整合** — 一鍵將網域接入 Zeabur Email，DNS 記錄自動建立

<Callout type="info">
  網域購買需要 **Developer** 或 **Team** 方案。
</Callout>

## 開始使用

1. 在側邊欄開啟 **網域** 頁面
2. [註冊網域](/domain/register) — 搜尋、比較價格並購買
3. [管理網域](/domain/manage) — 查看狀態、切換續費、接入 Zeabur Email
4. [設定 DNS 記錄](/domain/dns) — 在詳情頁新增和編輯 DNS 記錄
5. [設定註冊人資料](/domain/profiles) — 儲存聯絡資訊以加快註冊
