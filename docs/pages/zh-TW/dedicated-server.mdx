# 伺服器

除了共享叢集，你也可以將自己的伺服器**註冊到 Zeabur**，讓 Zeabur 替你管理主機上的服務部署、防火牆規則和資源分配——無需 SSH 進去手動操作。

## 為什麼使用伺服器

- **完整資源獨享**：服務不與其他使用者共享硬體，資源穩定可預測。
- **自攜主機（BYOH）**：已在 AWS、GCP、Hetzner 或自建機房購買伺服器？一鍵掛載到 Zeabur，立即享用 Zeabur 的部署管理能力。
- **地理位置靈活**：在共享叢集尚未覆蓋的區域，用自己的伺服器補足覆蓋範圍。
- **高穩定性生產環境**：對 SLA 與安全性有更高要求的大型系統。

## Zeabur 幫你管理什麼

一旦伺服器註冊，你就能透過 Zeabur Dashboard 完成以下工作，無需直接操作主機：

- 部署和更新服務
- 查看即時日誌與效能指標
- 設定環境變數
- 管理持久儲存空間與網路設定

<Callout type="info">
  伺服器按月計費（固定費用），而非按資源使用量計費。適合硬體需求穩定的生產工作負載。
</Callout>

## 開始使用

1. **[購買主機](/zh-TW/dedicated-server/purchase)** — 從 Zeabur 購買專用主機，享有優惠價格
2. **[新增主機](/zh-TW/dedicated-server/register)** — 將既有主機掛載到 Zeabur
3. **[管理伺服器](/zh-TW/dedicated-server/operate)** — 設定防火牆、監控資源、調整設定

## 常見問題

### 使用伺服器需要訂閱方案嗎？

不需要，你不需要訂閱方案即可使用伺服器。不過，我們建議訂閱開發者方案以發揮 Zeabur 的完整潛力——備份等功能僅提供給開發者方案的訂閱者。開發者方案用戶還可獲得來自工作人員的優先支援，而免費用戶可以透過 [Zeabur 論壇](https://zeabur.com/forum)獲得協助，但沒有保證的回覆時間。詳情請參閱[訂閱方案](/zh-TW/billing/plans)。

### 我只想架設小型網站，需要同時購買方案和伺服器嗎？

你可以在沒有訂閱方案的情況下，在伺服器上部署服務。不過，備份等特定功能僅限開發者方案的用戶使用。如果你想完整使用 Zeabur 的平台功能和優先支援，我們建議在伺服器之外訂閱開發者方案。更多資訊請參閱[社群與支援](/zh-TW/community/forum)。
