# 暫停服務

Zeabur 允許你直接從儀表板暫停（停止）個別服務。被暫停的服務會停止運行、停止消耗資源、停止產生費用——同時保留所有設定，方便你隨時恢復。

## 暫停服務

暫停服務的步驟：

1. 在 Zeabur 儀表板中開啟你的服務。
2. 前往 **Settings**。
3. 點擊 **Suspend Service**。

服務會立即停止。暫停期間：

- 服務容器會被關閉，不再運行。
- 該服務**不會產生運算費用**。
- 所有設定都會保留，包括環境變數、網域繫結和儲存空間掛載。
- [儲存空間](/operations/volumes)中的資料會保留。

## 恢復服務

恢復已暫停服務的步驟：

1. 在儀表板中開啟已暫停的服務。
2. 前往 **Overview**。
3. 點擊 **Restart Service**。

Zeabur 會使用最新的設定啟動一個新的部署。服務會經過正常的[健康檢查](/operations/health-checks)流程，並在就緒後開始接收流量。

## 使用情境

### 暫時性維護

當你需要對相依基礎設施進行維護（例如資料庫遷移），並希望確保在維護期間沒有流量到達應用程式時，可以暫停服務。

### 節省費用

如果你的服務只在特定時段或特定任務中需要使用（例如預備環境或批次處理服務），在不使用時暫停它們可以降低費用。

### 偵錯

暫停出現異常的服務，使其停止處理請求，同時你進行調查。找到並修復問題後，重新部署並恢復服務。

## 暫停與刪除的比較

| | 暫停 | 刪除 |
| --- | --- | --- |
| 服務停止運行 | 是 | 是 |
| 設定保留 | 是 | 否（寬限期過後） |
| 儲存空間保留 | 是 | 否（寬限期過後） |
| 可以復原 | 是，立即可用 | 有限時間內可聯繫客服 |
| 計費 | 停止 | 停止 |

<Callout type="warning">
  刪除服務是一個破壞性操作。刪除後有短暫的寬限期，在此期間你可以聯繫客服嘗試復原服務。寬限期過後，所有資料——包括儲存空間——都會被永久移除。如果你只需要暫時停止服務，請使用**暫停**。
</Callout>

## 已暫停的服務與網域

當服務被暫停時，繫結到該服務的任何網域都會顯示錯誤頁面。如果你需要顯示自訂的維護頁面，請考慮使用另一個輕量級服務（例如靜態網站）來提供維護通知，並在維護期間將網域的 DNS 指向該服務。
