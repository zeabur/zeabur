## 如何使用 Zeabur 將你的 v0 專案部署到網路

本指南是將 **v0** 產生的程式碼轉換為真實的線上網站，並與朋友或客戶分享的最簡單方法。

### 概述

**v0** 非常適合建立應用程式的設計和功能，但要將它上線，你需要「託管」服務。我們強烈推薦 **Zeabur**，因為它替你處理繁重的技術工作。只要幾次點選，就能取得你的程式碼並將它放在一個即時可用的 URL 上。

**工作流程：**

- **建立（Create）：** 在 **v0** 中產生你的應用程式。
- **完善（Refine）：** 將程式碼儲存到你的電腦，並使用 **Cursor** 進行檢查。
- **部署（Deploy）：** 在 **Zeabur** 上即時託管。

> 📝 重要提示：本指南重點介紹將你的網站上線（前端）。如果你的應用程式需要儲存使用者資料或登入資訊，則需要資料庫。我們將在後續教學中介紹如何新增資料庫！

### 先決條件

- GitHub 帳戶。
- 電腦上已安裝 [Cursor](https://cursor.sh/)。
- [Zeabur](https://zeabur.com/) 帳戶。
- 本機已安裝 Node.js（建議版本 18+）。

## 第 1 步：將 v0 同步到 GitHub

v0 有直接整合功能，可以為你建立 GitHub 儲存庫，無需手動複製程式碼。

1. **找到 GitHub 整合：**
   - 在 v0 編輯器中開啟你的專案。
   - 查看右上角的標題列。
   ![前往 GitHub](https://cdn.zeabur.com/v0%20migration/v0%20github%20part%201.png)
   - 點選 **"Add to Codebase"** 或 **GitHub 圖示**按鈕。

2. **連接到 GitHub：**
   - 將出現一個選單。選擇 **"Connect to GitHub"**。
   - 如果你之前沒有連接過，請按照授權提示進行操作。

3. **建立儲存庫：**
   - 選擇 **"Create New Repository"**。
   - 輸入你的專案名稱（例如 `my-v0-app`）。
   - 點選 **"Create"**。
   - 完成後，你將看到一條成功訊息，其中包含指向新 GitHub 儲存庫的連結。**複製此 URL。**
    ![連接到 GitHub](https://cdn.zeabur.com/v0%20migration/v0%20github%20part%202.png)

4. **推送到 GitHub：**
   - 確保你要使用的分支（通常是 `main`）被選為作用中的分支。
    ![設定作用中的分支](https://cdn.zeabur.com/v0%20migration/v0%20github%20part%203%20.png)
   - 點選 **"Push to GitHub"** 將你的程式碼推送到 GitHub。

## 第 2 步：設定本機環境（Cursor）

現在，你將從 GitHub 把程式碼帶到本機電腦，以驗證相依套件並進行正式環境調整。

1. **複製儲存庫：**
   - 開啟 **Cursor**。
   - 按 `Cmd + Shift + P`（Mac）或 `Ctrl + Shift + P`（Windows）開啟命令面板。
   - 輸入 `Git: Clone` 並選擇它。
   - 貼上你剛才複製的 GitHub URL，並選擇電腦上的一個資料夾來儲存它。

2. **安裝相依套件：**
   - 在 Cursor 中開啟終端機（`Ctrl + ~`）。
   - 執行安裝命令（v0 專案通常使用 `npm`）：

   ```bash
   npm install
   ```

3. **環境設定：**
   - 找到 `.env.example` 檔案（如果存在）並將其複製以建立一個 `.env` 檔案。
   - **重要：** 如果你的 v0 應用程式使用了 API 金鑰（OpenAI、Supabase 等），你必須手動將這些金鑰新增到你的本機 `.env` 檔案中。出於安全考量，v0 不會將你的密鑰匯出到 GitHub。

4. **本機測試：**
   - 執行開發伺服器以確保應用程式在本機正常運作。

   ```bash
   npm run dev
   ```

## 第 3 步：在本機編輯器中開啟並安裝 Zeabur 擴充功能

為了使部署變得極其簡單，你將使用 Zeabur 擴充功能。

1. 前往編輯器中的擴充功能市集。
2. 搜尋「Zeabur」並點選**安裝**。

![下載 Zeabur 擴充功能](https://cdn.zeabur.com/replit%20migration%20/Download%20Zeabur%20Extensions.png)

## 第 4 步：一鍵部署

現在你的專案資料夾已在 Cursor 中開啟：

1. 點選左側活動列中新的 **Zeabur 圖示**（通常是一個「Z」標誌）。
2. 點選 **"Deploy"** 按鈕。
3. 你的編輯器將請求在瀏覽器中開啟連結的權限。點選 **"Open"**。

![點選部署並開啟](https://cdn.zeabur.com/replit%20migration%20/Click%20Deploy%20and%20Open%20.png)

## 第 5 步：讓 Zeabur AI 代理處理剩下的事情

一旦你被重新導向到 Zeabur，平台的 AI 代理就會接管。它會分析你的程式碼以了解專案的結構、相依套件和建置需求。

Zeabur 代理將：

- 識別出你有一個全端應用程式（例如 Next.js）。
- 自動設定建置和啟動命令。
- 預先佈建所有必要的服務。

![Zeabur 代理處理其餘部分](https://cdn.zeabur.com/new%20agent.png)

系統將提示你選擇專案和部署區域。之後，Zeabur 會處理其他所有事情。

你可以即時觀看服務的建置過程並使其完全運作。部署成功後，Zeabur 甚至會為你提供一個即時網域。

## 第 6 步：網域和網路

1. 在 Zeabur 中點選你的服務。
2. 前往 **Networking**（網路）分頁。
3. 點選 **Generate Domain**（產生網域）以取得 `.zeabur.app` 網域，或點選 **Custom Domain**（自訂網域）以連接你自己的網域。
4. 造訪該 URL。你的 v0 應用程式現在是一個已完全部署的正式應用程式。

## 第 7 步：連接資料庫（首選 Supabase）

如果你的 v0 應用程式允許使用者登入、建立帳戶或儲存資料，則它需要資料庫。
目前，你已成功部署**前端**（視覺介面）。為了啟用資料和身份驗證功能，你將連接 Supabase 資料庫，並將連線詳細資訊新增為 Zeabur **變數**。

我們已準備了一份專門的指南，帶你逐步取得 Supabase 金鑰並將其新增到 Zeabur：

**👉 按照[此指南](https://zeabur.com/templates/SUPABS)在 Zeabur 上建立並連接 Supabase。**
