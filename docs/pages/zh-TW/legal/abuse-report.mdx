# 濫用回報指引

Zeabur 與各家網路安全機構合作，共同打擊犯罪及惡意使用行為。我們鼓勵您向我們回報任何違反 [公平使用指南](/legal/fair-use-guideline) 的行為，因此若您發現任何違反 [公平使用指南](/legal/fair-use-guideline) 的網站或行為，請立即向我們回報。

若您是政府部門或者是網路安全機構，也可以與我們聯絡來接入自動化回報系統。

## 聯絡途徑

我們提供以下兩種聯絡途徑：

- 電子信箱：[contact@zeabur.com](mailto:contact@zeabur.com)
- Discord 社群：[Zeabur Discord](https://zeabur.com/dc)

您寄送的回報均會在 2 個工作日內進行回覆並說明受理情況。若 2 個工作日未收到回覆，請再次聯絡我們。
