# 部署 Nuxt 專案

在 Zeabur 上部署 Nuxt 專案。

### 第一步: 建立 Nuxt 專案（可選）

打開終端機，輸入以下指令，並透過 `nuxi` 來建立 Nuxt 專案：

<Tabs items={['pnpm', 'npx']}>
  <Tabs.Tab>
    ```bash copy
    pnpm dlx nuxi init <project-name>
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    npx nuxi init <project-name>
    ```
  </Tabs.Tab>
</Tabs>

再來你需要安裝套件，並在終端機輸入以下指令：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash copy
    pnpm install --shamefully-hoist
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    npm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    yarn
    ```
  </Tabs.Tab>
</Tabs>

<Callout type="info" emoji="ℹ️">
  這裡需要注意由於 pnpm 的 `node_modules` 結構與其他套件管理器不同，因此需要使用 `--shamefully-hoist` 來安裝套件。
  並確保在部屬前你有新增 `.npmrc` 檔案在專案的根目錄，並在檔案中加入 `shamefully-hoist=true`。
</Callout>

````.npmrc filename=".npmrc" copy
shamefully-hoist=true
````

### 第二步: 部署 Nuxt 專案

在你的專案中，點選 **Deploy service** 或者 **Add new service** 按鈕，並選擇 **Deploy your source code**.

![deploy](/deploy/common/select-service.en-US.png)

搜尋想要部署的 Nuxt 程式碼儲存庫，並點選匯入，你的 Nuxt 應用程式就將自動開始部署。
