# 在 Zeabur 上部署 Flutter

[Flutter](https://flutter.dev) 是一個由 Google 開發的開源 UI 軟體開發工具包，用於構建跨平台的移動應用程式。

本文將幫助你在 Zeabur 上部署 Flutter 網站。

## 步驟一：建立一個 Flutter 專案

### 方法一：使用 `flutter create` 指令建立專案

請先根據 [Flutter 官方文件](https://docs.flutter.dev/get-started/install) 安裝 Flutter。

你可以輸入下面的指令來檢查 Flutter 是否已經安裝好。

```bash
flutter doctor
```

如果你看到所有的檢查項目都是綠色的，那麼恭喜你，你已經安裝好了 Flutter。

然後，使用 `flutter create` 指令建立一個新的 Flutter 專案。

```bash
flutter create myapp
cd myapp
flutter run -d chrome
```

### 方法二：使用模板建立專案

我們為你準備了一個模板來幫助你快速開始 Flutter 專案： [zeabur/flutter-template](https://github.com/zeabur/flutter-template)。

你可以在 GitHub 頁面上點選 `Use this template` 按鈕來使用模板。

## 步驟二：部署到 Zeabur

### 方法一：使用 Zeabur CLI

首先，確保你已經按照 [Zeabur CLI](/deploy/deploy-in-cli) 的指南安裝好了 Zeabur CLI。

然後，在你剛初始化好的 Flutter 專案中，將尚未提交到版本控制的檔案加入到 Git 中。

```bash
git init
git add -A
git commit -m "Initial commit"
```

接著，使用下面的指令部署你的服務。

```bash
zeabur
```

### 方法二：使用 Zeabur 控制台

前往 [Zeabur 控制台](https://dash.zeabur.com) 並建立一個新專案。

然後，點選 `Deploy New Service` 按鈕，選擇從 GitHub 部署。

<Callout>
    如果你是第一次在 Zeabur 上從 GitHub 部署專案，你需要先配置並安裝 GitHub App。
    你可以在 [這裡](/deploy/github) 查看更詳細的指南。
</Callout>

當你選擇好儲存庫和分支後，Zeabur 將會自動開始建置你的服務。

Zeabur 會自動檢測到你的服務是由 Flutter 建置的，所以你不需要做任何額外的配置。你的部署將會在幾分鐘內完成。

## 下一步

部署完成後，你需要為你的 Flutter 網站配置網域。

打開服務頁面的 "Domain" 分頁，然後點選 "Generate Domain" 或 "Custom Domain"。

關於如何為你的服務綁定網域的更多資訊，請參考 [綁定網域](/deploy/domain-binding)。
